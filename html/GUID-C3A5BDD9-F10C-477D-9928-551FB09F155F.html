涂抹(Smear)

将 RGB 颜色转化为 UV 坐标偏移。

通过使用纹理定义如何重新映射 UV 坐标，该工具可用于将涂抹效果添加到 UV 坐标。会将“输入 RGB”(In Rgb)颜色转化为颜色控制盘上的 HSV 颜色。会基于 HSV 颜色位置确定极面坐标（颜色控制盘上的角度和距颜色控制盘中心的距离）。会将这些极面坐标转化为矩形坐标并输出为 UV 偏移。

输入 RGB (In Rgb)

确定偏移 UV 的距离和方向的 RGB 颜色值。

在“创建”(Create)选项卡中找到该工具（请参见“创建”(Create)选项卡）。

若要使用该工具，请参见使用涂抹工具。
