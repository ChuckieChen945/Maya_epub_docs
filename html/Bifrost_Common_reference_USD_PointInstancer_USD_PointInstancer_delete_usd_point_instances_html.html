delete_usd_point_instances

从 USD PointInstancer 中删除点实例。

输入

enable

禁用此选项可跳过删除点实例，并改为输出未修改的后台文件。

stage

保留 PointInstancer 基本体的 USD 后台文件。

point_instancer_path

要从中删除点实例的基本体（PointInstancer 类型）的路径。

invert

删除所有与选定 mode 的规则不匹配的点实例。

mode

控制如何删除点实例。

  • Random：根据概率权重阈值删除点实例。每个实例都会生成一个随机数，如果该数小于指定的 probability_weight，则将其删除。
  • Inside Geometry：删除包含在其他几何体中的点实例。例如，这可用于删除边界网格内的点实例。
  • By Indices：连接包含点实例索引的 array ，以获取要删除点的直接控件。通过将 get_usd_point_instancer_attributes 节点的 positions 输出端口连接到 get_array_indices 节点，可以推断点实例索引。

probability_weight

选择 Random 模式后，如果为点实例生成的随机数低于此阈值，则将删除该点实例。

seed

用于生成随机数的种子。不同的种子将产生一组不同的删除点实例。

geometries

选择 Inside Geometry 模式时要使用的一个或多个几何体。

indices

选择 By Indices 模式时要使用的一个或多个 long 或 array。

输出

out_stage

修改后的 USD 后台文件。

out_prim_path

修改后的 PointInstancer 的基本体路径。

new_point_count

修改后的 PointInstancer 的点实例计数
