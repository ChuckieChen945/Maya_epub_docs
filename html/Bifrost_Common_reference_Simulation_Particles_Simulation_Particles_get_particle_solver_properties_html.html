get_particle_solver_properties

此项在编辑 simulate_particles 复合时使用。可用于轻松访问粒子解算器对象上的所有特性。

如果将 simulate_particles 节点设为可编辑，则有一个为自定义工作创建的复合，当前此复合所处位置有些深，可以导航到以下位置以找到它：simulate_particles: solve_particles: advance_one_frame: advance_one_timestep: custom_effects。可以在此处构建用于直接修改粒子系统的复合。

输入

solver_object

此项应连接到 simulate_particles 复合中的内部 solver_object。

输出

particles

此项为包含所有粒子的粒子系统对象。它是 simulate_particles 复合最终输出的内容。

solver_settings

此项为传递到 simulate_particles 的粒子解算器设置。可以通过将此项连接到 get_particle_solver_settings 复合来获取各个设置。为了轻松地将自定义特性向下传递到解算器，可以在 particle_solver_settings 和 simulate_particles 之间添加 set_property 调用。然后，可以对解算器设置执行 get_property 来获取自定义特性。这可避免一直不断地创建端口。

sources

此项为传递到 simulate_particles 的一组源。

colliders

此项为传递到 simulate_particles 的一组碰撞对象。

fields

此项为传递到 simulate_particles 的一组影响场。

current_time

此项为当前子步内的当前解算时间（以秒为单位）。

step_length

一个子步的长度（以秒为单位）。

frame_length

一帧的长度（以秒为单位）。

frame_end_time

正在处理的当前帧的结束时间。

after_start_frame

如果模拟在开始帧时间之后进行，则为 true。
