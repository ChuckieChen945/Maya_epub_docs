This Version is needed to run on Maya 2017 Update 1. Previous versions will fail to load due to changes in Maya's API## BUGFIXES

MtoA is a bugfix release. It contains the following changes :

  • Support for Maya 2017 Update 1
  • Textures Gamma could change suddenly during an IPR session
  • Fixed texture lock happening on windows when "Auto-Convert Textures to TX" was enabled
  • Fixed hidden standins display in Viewport 2
  • Support overrides on connected attributes
  • Fixed Image planes with Motion blur
  • maketx could not be executed on linux as syncolor files weren't found
  • Release Shave Haircut extension for Maya 2016 Ext2 and 2017
