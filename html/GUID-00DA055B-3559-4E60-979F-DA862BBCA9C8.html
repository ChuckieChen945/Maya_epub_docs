创建文件引用

  1. 通过执行下列操作之一来访问“引用选项”(Reference Options)窗口：
    • 在“大纲视图”(Outliner)中的空白区域上单击鼠标右键，然后从显示的菜单中选择“引用 > 创建引用”(Reference > Create Reference) > 。
    • 选择“文件 > 创建引用”(File > Create Reference) > 。
  2. 在引用选项(Reference Options)中，设定应用于新文件引用的选项。
    注： 如果从包括 XGen 描述的场景创建文件引用，则必须在“引用选项”(Reference Options)中启用“使用名称空间”(Use namespaces)。请参见引用选项(Reference Options)。无法从包含 assemblyDefinition 或 assemblyReference 节点的 Maya 场景创建文件引用。请参见场景集合节点(Scene Assembly nodes)。
  3. 单击“引用”(Reference)。
  4. 在文件浏览器中，导航到要引用的文件，单击其名称，再单击“引用”(Reference)。

    引用文件及其内容将加载在场景中，其引用节点将显示在“大纲视图”(Outliner)中，由 图标表示。属于引用文件的对象由 图标表示。

文件引用创建选项

有关创建文件引用时常用的选项，请参见下表。

目标 操作

使用定位器创建引用。

  1. 访问“引用选项”(Reference Options)窗口。
  2. 确保“分组”(Group)和“定位器”(Locator)选项已启用。
  3. 单击“引用”(Reference)。

延缓加载引用。

  1. 访问“引用选项”(Reference Options)窗口。
  2. 在“常规选项”(General Options)中，启用“延缓”(Deferred)。

在创建引用后，它将以空载状态添加到场景中，且仅在手动加载时才会显示。

使用名称空间创建引用。

请参见名称空间。

重要： 使用名称空间创建引用之后，切勿删除此名称空间，否则使用此名称空间的所有被引用对象都将受到影响，不会再按预期工作。如果不再需要将引用对象与特定名称空间相关联，请使用“名称空间编辑器”(Namespace Editor)将其移除。请参见移除名称空间。
  1. 访问“引用选项”(Reference Options)窗口。
  2. 在“名称空间选项”(Namespace Options)部分中，确保启用了“使用名称空间”(Use Namespaces)。

    请参见在引用或导入时创建名称空间。

使用共享显示层创建引用。

  1. 访问“引用选项”(Reference Options)窗口。
  2. 在“共享节点选项”(Shared Node Options)中，启用“显示层”(Display Layers)。

在创建引用时，与子场景关联的层将集成到父场景中。

使用共享着色网络创建引用。

  1. 访问“引用选项”(Reference Options)窗口。
  2. 在“共享节点选项”(Shared Node Options)中，启用“着色网络”(Shading Networks)。

在创建引用时，与子场景关联的着色网络将集成到父场景中。
