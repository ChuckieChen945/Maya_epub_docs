终止模拟

要在 Bifrost 中停止正在运行的模拟，可以使用“播放”(Play)按钮在主场景中停止播放，也可以按 Esc 键。这些方法具有不同的效果。

在主场景中停止播放

在主场景中使用“播放”(Play)按钮停止播放时，不会有新帧排队等待计算。但是，可能需要一些时间来完成当前帧的计算，之后您才能重新获得控制权，以及继续编辑场景。

使用此方法，只需再次单击“播放”(Play)即可从上一帧继续播放。在此期间，您可以导航视图、调整着色和照明等。您甚至可以在返回到上一个模拟帧之前跳转到其他帧以继续播放。

请注意，如果修改图表或其输入，则需要从模拟的第一帧继续播放。

使用 Esc 键终止

当您按 Esc 键终止模拟时，Bifrost 会尝试关闭已在进行中的计算。这适用于中止较长的模拟，这样，您几乎可以立即继续工作。

并非每个计算都会立即停止。即使您可以与图表和场景进行交互，某些进程也将继续在后台运行并占用资源，直到完成为止。在终止序列开始和结束时，会向 Maya 脚本编辑器记录以下消息 - 它们之间的停顿表示终止图表用了一些时间。

// Warning: Bifrost job terminating... (bifrostGraphShape1) // 
// Warning: Bifrost job terminated. (bifrostGraphShape1) //

在使用 Esc 键终止后继续播放

要在终止图表后继续计算，必须在重新开始播放之前先后退一帧以回到上一个完成的帧。有时也可以从上一帧继续播放，但这不能保证可行。

仅当启用“选项 > 在使用 Esc 键终止后可继续”(Options > Resumable after Esc Termination)时，按 Esc 键后才能以此方式继续播放。默认情况下，对于新场景，此选项处于禁用状态以节省内存。

与前面一样，如果修改图表或其输入，则需要从模拟的第一帧继续播放。
