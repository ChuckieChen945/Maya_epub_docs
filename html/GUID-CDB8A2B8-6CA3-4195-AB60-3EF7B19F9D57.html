选择用于“Paint Effects”的 UV 集（仅多边形）

默认情况下“Paint Effects”会使用 UV 集 map1，但可以将“Paint Effects”笔划链接到多边形曲面上不同的 UV 集。例如，如果为不同的投影定义了不同的 UV 集，可以将“Paint Effects”笔划链接到提供最佳结果的 UV 集。

将“Paint Effects”笔划链接到某个 UV 集时，会链接附加到多边形的所有笔划。不能将各个笔划链接到不同的 UV 集。

将附加的“Paint Effects”笔划链接到一个不同的 UV 集

  1. 选择包含要链接的“Paint Effects”笔划的对象。
  2. 选择“窗口 > 关系编辑器 > UV 链接 > Paint Effects/UV”(Windows > Relationship Editors > UV Linking > Paint Effects/UV)以打开“关系编辑器”(Relationship Editor)。

    左侧面板中会列出附加到选定多边形的所有“Paint Effects”笔划。

    右侧面板中会列出包含选定多边形的 UV 集的网格节点。如果选择了多个多边形，则仅会列出最后一个选定的多边形。

  3. 在左侧面板中，单击“Paint Effects”笔划。在右侧面板中，“Paint Effects”笔划所链接到的 UV 集会变为亮显。
  4. 在右侧面板中，单击希望将“Paint Effects”笔划链接到的 UV 集。
