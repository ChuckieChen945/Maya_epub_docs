is_face_edge_open

确定指定的面边是否是开放边。开放面边是指不与其他面边相邻的边。也称为边界边。

例如：

 ┌───────────────┬────────────────┐
 │      s2       │       s2       │
 │             ▲ │                │
 │             │ │                │
 │s3    f0   s1│ │ s3    f1     s1│
 │             │ │                │
 │             │ │                │
 │      s0       │       s0       │
 └───────────────┴────────────────┘

指代面 f0、侧边 s1 的面边不是开放边，但侧边 s2、s3 和 s0 是开放边。

输入

face_edge

要查询的面边。

face_offset

相应网格的面偏移数组。

face_vertex_adjacent_edge

相应网格的面顶点相邻数组。（请参见 update_mesh_adjacency）

输出

is_open

如果 face_edge 是开放边，则为 true；否则为 false。
