set_geo_property_data

设置现有几何特性的数据值。例如，如果输入对象上存在 point_size，则可以使用此节点逐点更改大小。

如果不存在该特性，则应先使用 set_geo_property 创建它。

对于某些常用特性，还可以使用特定复合，例如 set_point_position。

输入

geometry

几何对象。

property

特性的名称。如果不存在该特性，则不会修改对象。

data

基于每个组件的一组值（组件是特性针对的组件）。

  • 如果数组小于组件数，则将使用特性的默认值扩展它。
  • 如果数组大于组件数，则会被截断。
  • 如果数组类型错误，则不会修改对象。

输出

out_geometry

修改后的对象。
