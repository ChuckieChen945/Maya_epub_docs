平面映射选项(Planar Mapping Options)

通过从假想平面沿一个方向投影 UV 纹理坐标，将其映射到选定曲面网格上。

使用这些选项可设置在选择“UV > 平面”(UV > Planar)时发生的情况。

投影操纵器(Projection Manipulator)

适配投影到(Fit projection to)

默认情况下，投影操纵器将根据以下两个设置之一自动定位：

最佳平面(Best plane)

如果要为对象的一部分面映射 UV，可以选择“最佳平面”(Best plane)，投影操纵器将捕捉到直接对准选定面的一个角度和旋转。

边界框(Bounding Box)

将 UV 映射到对象的所有面或大多数面时，此选项非常适用。它将捕捉投影操纵器以适配对象的边界框。此选项处于启用状态时，必须从方向中选择一个“投影”(Project)，以确定投影操纵器的方向。

投影源(Project from)

选择轴（X、Y、Z），以便投影操纵器指向对象的大多数面。例如，趴在栅格上海龟模型的大多数面均指向沿 Y 轴的某个位置，而站在栅格上的马模型的大多数面则指向沿 X 轴或 Z 轴的某个位置。

如果模型的大多数面不是直接指向沿 X、Y 或 Z 轴的某个位置，则选择“摄影机”(Camera)选项。此选项根据当前的活动视图为投影操纵器定位。

保持图像宽度/高度比率(Keep image width/height ratio)

启用此选项可保持图像的宽度与高度之比，避免图像扭曲。禁用此选项后，映射的 UV 将在“UV 编辑器”(UV Editor)中填充 0 到 1 的坐标。

在变形器之前插入投影(Insert projection before deformers)

当多边形对象应用了变形时，需要使用“在变形器之前插入投影”(Insert projection before deformers)选项。如果该选项处于禁用状态，并且为变形设置了动画，则纹理放置会受顶点位置更改的影响。这将导致“游移”纹理。

启用该选项时，可在多边形对象应用变形之前将纹理放置应用到该对象。即：系统在变形器依存关系图节点之前插入纹理放置依存关系图的节点，纹理即使在变形之后亦能“粘住”几何体。

UV 集(UV Set)

创建新 UV 集(Create new UV set)

启用此选项可创建新 UV 集并将通过投影创建的 UV 放置在该集中。在“UV 集名称”(UV Set name)框中键入 UV 集名称。

手动调整投影操纵器(Manually adjusting the projection manipulator)

可以使用操纵器控制柄以交互方式调整投影的 UV 贴图，或使用“通道盒”(Channel Box)或“属性编辑器”(Attribute Editor)来输入值以更改“投影中心”(Projection Center)、“旋转”(Rotation)和“缩放”(Scale)。这些值对应于将纹理映射到多边形上时显示的操纵器控制柄。

投影中心(Projection Center)

投影中心将定义可投影纹理贴图的 X、Y 或 Z 轴中的原点。默认情况下，这是在 X、Y 或 Z 轴中选定面的中心。

旋转(Rotate)

键入一个值，以便在 3D 视图中围绕 X、Y 或 Z 轴旋转投影，并随后旋转纹理。

投影宽度(Projection Width)

设置相对于 3D 投影轴的投影宽度 (U)。

投影高度(Projection Height)

设置相对于 3D 投影轴的投影高度 (V)。

图像中心(Image Center)

此值表示投影 UV 的中心。更改此值时，将相应平移中心。

旋转角度(Rotation Angle)

此值更改 UV 在 2D 窗口中旋转的角度。拖动滑块或输入值可旋转图像。

图像比例(Image Scale)

此值表示 2D 贴图相对于 2D 中心点的宽度 (U) 或高度 (V)。
