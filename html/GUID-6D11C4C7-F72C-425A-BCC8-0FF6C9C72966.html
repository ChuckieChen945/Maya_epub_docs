将纹理指定给 UV 集

使用关系编辑器将纹理指定给 UV 集。

选择“窗口 > 关系编辑器 > UV 链接 > 以 UV 为中心”(Windows > Relationship Editors > UV Linking > UV-Centric)以打开关系编辑器的 UV 集编辑视图。

目标 操作

将纹理指定给 UV 集。

单击左列中的“UV 集”(UV Set)，然后单击右列中的纹理，以使其亮显。

从 UV 集移除纹理。

单击左列中的“UV 集”(UV Set)，然后单击右列中的纹理，以使其不会亮显。

暂时使纹理不显示在软件渲染中。

从纹理列表上方的“编辑”(Edit)菜单中，选择以下选项之一：

  • “忽略纹理”(Ignore Texture)：隐藏选定纹理。
  • “渲染时使用纹理”(Use Texture When Rendering)：使用选定纹理。
  • “隔离纹理”(Isolate Texture)：隐藏除选定纹理以外的所有其他纹理。
  • “渲染时使用所有纹理”(Use All Textures When Rendering).7

利用“关系编辑器”(Relationship Editor)，还可以轻松访问其“编辑”(Edit)菜单中的多个 UV 集编辑操作。
