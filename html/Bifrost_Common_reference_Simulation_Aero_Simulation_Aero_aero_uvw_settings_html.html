aero_uvw_settings

创建一些通过 Aero 模拟平流传递的 UVW 坐标集。该复合将插入 aero_solver_settings 复合上的 addtional_settings 端口。生成的特性表示整个 Aero 体的 UVW 坐标。

输入

常规

enable

启用 UVW 特性创建。

name

输入特性的名称。请注意，特性名称将带有前缀 voxel_。仅平流传递具有此前缀的特性。

start_frame

创建 UVW 坐标的帧。

use_end_frame

可以启用该项以移除特定帧之后的 UVW 坐标。

end_frame

将移除其后的 UVW 坐标的帧。

UVW 重置

enable_resetting

如果启用，将定期重置 UVW 坐标。

number_of_sets

启用重置时要使用的 UVW 坐标集数。当_禁用_重置时，将仅创建一个 UVW 集。

period

UVW 坐标重置之间的秒数。

offset_ratio

使用多个 UVW 坐标集时，第二个 UVW 坐标集将从第一个 UVW 坐标集偏移 offset_ratio x period 秒。第三个 UVW 坐标集将偏移 offset_ratio x period 秒的两倍，依此类推。

blending_weight_profile

在 UVW 坐标集之间淡入和淡出时使用的配置。请注意，给定的配置将用于淡入，而 1 减去此配置将用于淡出。这仅在使用多个 UVW 坐标集时适用。

输出

uvw_settings

具有所有 UVW 设置的 Bifrost 对象。
