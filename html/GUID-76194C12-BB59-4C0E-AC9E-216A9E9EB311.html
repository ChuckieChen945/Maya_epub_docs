XGen 编辑器工具栏

在 XGen 编辑器工具栏上可以使用常用功能。也可从 XGen 工具架使用这些选项。

预览(Preview)

单击此选项可以显示由描述的当前属性设置所生成的基本体。典型的 XGen 工作流包括在场景视图中预览对基本体属性所做的修改，然后再进行渲染。可以从下列“预览”(Preview) 选项中选择：

自动更新预览(Update Preview Automatically)

修改属性值后会自动显示预览。除其他“预览”(Preview) 选项之外，还可以开启此选项。

关闭该选项后，预览图标将更改为此图标

仅预览当前描述(Preview Current Description Only)

显示当前在 XGen 编辑器中选择的描述的预览。

预览集合中的所有描述(Preview All Descriptions in the Collection)

显示当前在 XGen 编辑器中选择的集合中包含的所有描述的预览。

预览所有集合(Preview All Collections)

显示场景中的所有集合包含的所有描述的预览。

仅预览选定对象(Preview Selected Objects Only)

仅选定的基本体显示在预览中。

清除预览(Clear Preview)

清除基本体生成的显示。可以从下列“清除预览”(Clear Preview) 选项中选择：

仅清除当前描述(Clear Current Description Only)

清除当前在 XGen 编辑器中选择的描述的预览显示。

清除集合中的所有描述(Clear All Descriptions in the Collection)

清除当前在 XGen 编辑器中选择的集合中包含的所有描述的预览显示。

清除所有集合(Clear All Collections)

清除场景中的所有集合中每个描述的预览显示。

仅清除选定对象(Clear Selected Objects Only)

清除选定对象的预览显示。

创建描述(Create Description)

打开“创建 XGen 描述”(Create XGen Description)窗口，可以在其中创建新集合和描述。

绑定面片(Bind Patches)

可以更改将哪些面绑定到选定描述的面片。请参见绑定面片。

添加导向(Add Guides)

切换“添加导向”(Add Guide)工具。使用此工具可以将导向添加到几何体的曲面并在曲面上移动导向。

使用鼠标左键添加插值导向。添加到曲面时，将从现有的相邻导向生成这些导向的形状和缩放。例如，可以形成现有导向并添加新导向，然后添加带有现有导向插值形状的新导向。这是最常用的基于导向的描述类型。

使用鼠标中键添加默认长度，即与曲面法线对齐的笔直导向。默认导向忽略现有邻近导向的形状。如果需要从曲面法线投影的未成形导向，请使用默认导向。

您可以在场景视图中的导向上单击鼠标右键以访问移动、复制和粘贴导向等的选项。有关可用于操纵导向的更多工具的信息，请参见 XGen 工具选项。

导向可见性(Guide Visibility)

用于开启和关闭导向可见性。

导向选择(Guide Selection)

用于启用导向选择。

镜像导向(Mirror Guides)

用于将选定导向的镜像版本添加到几何体的反面（沿 X 轴）。使用此选项可以通过选定导向的镜像放置和形状来快速填充几何体。

面片/几何体可见性(Patch / Geometry Visibility)

（仅限 XGen 工具架）切换面片和几何体显示。

XGen 面片/几何体(XGen Patch /Geometry)

从切换几何体和面片显示转到仅面片。

基本体选择(Primitive Selection)

用于启用基本体选择。使用此选项可以选择场景中的各个基本体。

消隐选定基本体(Cull Selected Primitives)

用于消隐选定基本体。请参见消隐。
