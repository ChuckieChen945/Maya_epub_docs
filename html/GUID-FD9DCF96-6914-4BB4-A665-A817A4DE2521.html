使用 OpenSubdiv 平滑网格

使用“网格 > 平滑”(Mesh > Smooth)平滑网格时或创建细分曲面代理时，OpenSubdiv 是默认的细分方法。

使用 OpenSubdiv 平滑网格

  1. 选择要平滑的网格或面。
  2. 选择“网格 > 平滑”(Mesh > Smooth) > 。
  3. 在显示的“平滑选项”(Smooth Options)窗口中，设置 OpenSubdiv Catmull-Clark 选项。
  4. 单击“平滑”(Smooth)。

    Maya 通过添加新的多边形，然后平滑顶点及其连接的边来平滑网格。

  5. 在“属性编辑器”(Attribute Editor)中，选择“polySmoothFace”选项卡 >“OpenSubdiv 控件”(OpenSubdiv Controls)。

    使用 OpenSubdiv 控件(OpenSubdiv Controls)调整平滑网格。

使用 OpenSubdiv 创建细分曲面代理网格

  1. 选择要平滑的网格。
  2. 选择“网格 > 平滑代理 > 细分曲面代理”(Mesh > Smooth Proxy > Subdiv Prox) > 。
  3. 在显示的“细分曲面代理选项”(Subdiv Proxy Options)窗口中，设置 OpenSubdiv Catmull-Clark 选项。
  4. 单击“平滑”(Smooth)。

    Maya 会平滑选定的网格，并将其放置在原始的未平滑网格（代理）中。Maya 还会创建包含两种网格的新组，并为每个网格创建新层。

  5. 在“属性编辑器”(Attribute Editor)中，选择“polySmoothProxy”选项卡 >“OpenSubdiv 控件”(OpenSubdiv Controls)。

    使用 OpenSubdiv 控件(OpenSubdiv Controls)调整平滑网格。
