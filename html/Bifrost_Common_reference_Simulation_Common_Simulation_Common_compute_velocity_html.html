compute_velocity

此项计算给定几何体的两个帧之间的速度，并将其存储在几何体上。适用于为模拟中使用的移动网格提供速度。通常，模拟会自动计算输入源或碰撞对象网格的速度，但此项可用于自定义解算器。另一个用途是在 convert_to_volume 之前使用，以使体积具有速度。不能在循环（例如 do_while 或 for_each）内调用此复合，因为它使用反馈端口保留几何体的前一状态。但是，用户可以传入一组对象，然后它将输出一个数组，因此不需要循环。

输入

geometry

连接单个几何体或一组几何体。当前，这些几何体必须是点对象或网格。如果点数或拓扑数自最后一帧发生变化，则计算的速度将为零。

start_frame

这是从其开始计算速度的帧。在开始帧处或开始帧之前计算的速度将为零，然后后续帧将使用与最后一帧的位置差距来计算速度。

输出

out_geometry

这是具有其他计算几何特性 point_velocity 和 point_velocity_prev（上一帧的速度）的输入几何体（单个对象或对象数组，具体取决于输入）。
