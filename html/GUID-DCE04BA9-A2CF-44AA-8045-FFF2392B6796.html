转化当前选择选项(Convert Selection Options)

选择“选择 > 转化当前选择”(Select > Convert Selection)时，子菜单中提供以下选项，可用于将现有的多边形组件当前选择转化为顶点、边、面等。

到顶点(To Vertices)
将现有多边形组件选择更改为与最初选定的组件相关联的顶点。也可以通过按 Ctrl+F9 选择此选项。

例如，如果有一个由四边多边形（四边形）构成的多边形网格，且选定其四面，然后选择“选择 > 转化当前选择 > 到顶点”(Select > Convert Selection > To Vertices)，选择会转化为与之前选定的面相关联的顶点。

到顶点面(To Vertex Faces)
将现有多边形组件的当前选择更改为与原始选择的多边形组件相关联的顶点面。

在将逐顶点颜色信息指定给多边形面时，将使用顶点面。可以将颜色指定给与面相关联的每个顶点。即使面之间共享顶点，每个共享这个顶点的面仍有一种颜色。面顶点处的颜色将跨面融合在一起。

到顶点周长(To Vertex Perimeter)
将当前组件选择（顶点、UV、边或面）转化为沿着顶点周长的边界。
注： 多边形标记菜单中也提供此命令。（在选定多边形组件后按 Ctrl + 单击鼠标右键。）
到边(To Edges)
将当前多边形选择更改为与选定多边形组件关联的任何多边形边。也可以通过按 Ctrl+F10 选择此选项。
到循环边(To Edge Loop)
选择与原始选择相关联的多边形边，然后沿多边形网格的任意一个方向将边的选择扩展为循环边，即可更改当前组件选择。

循环边是由其共享顶点按顺序连接的多边形边的路径。

到环形边(To Edge Ring)
更改当前组件选择，方法是选择与原始选择关联的多边形边，然后沿多边形网格在任意方向上将选择的边扩展为环形边。

环形边是通过其共享曲面按顺序连接的多边形边的路径。

到包含的边(To Contained Edges)
将当前选择更改为任何多边形的边，可根据最初选择的组件来选择这些边。
到边周长(To Edge Perimeter)
将当前选择的组件（顶点、UV、边或面）沿着其周长转化为边的边界。
注： 多边形标记菜单中也提供此命令。（在选定多边形组件后按 Ctrl + 单击鼠标右键。）
到面(To Faces)
将当前选择更改为与最初选定的组件关联的任何多边形面。
到面路径(To Face Path)
通过选择与原始选择关联的多边形边，更改当前组件选择，然后沿着多边形网格的任一方向以多边形面的形式展开所选的边。
到连接面(To Connected Faces)
将当前组件选择（顶点、UV、边或面）转换为所有相邻面。
要包含的面(To Contained Faces)
将当前多边形组件选择更改为可选择的任何多边形面，具体取决于原始选择是否包含区域。
到面周长(To Face Perimeter)
将当前组件选择（顶点、UV、边或面）转化为沿着面周长的边界。
到 UV(To UVs)
将现有多边形组件选择更改为与最初选定多边形组件关联的 UV 纹理坐标。也可以通过按 Ctrl + F12 选择此选项。

例如，如果您拥有一个已选定几个面的多边形网格，然后选择“选择 > 转化当前选择 > 到 UV”(Select > Convert Selection > To UVs)，该选择将被转化为与之前选定面关联的 UV 纹理坐标。

尽管 UV 选择将显示在场景视图中的网格上，但是要编辑或修改选定的 UV 纹理坐标，必须使用 UV 编辑器。(UV > UV 编辑器(UV > UV Editor))

到 UV 壳(To UV Shell)
将现有的多边形组件选择更改为与原始选择的多边形组件关联的 UV 壳。

例如，如果多边形网格已有多个面被选中，并且随后选择了“选择 > 转化当前选择 > 到 UV 壳”(Select > Convert Selection > To UV Shell)，则该选择会转化为与之前选定的面关联的 UV 壳。

若要编辑或修改选定的 UV 壳，必须使用 UV 编辑器。

到 UV 壳边界(To UV Shell Border)
将现有多边形组件选择更改为与最初选定的多边形组件相关联的 UV 壳上的边界 UV。

若要编辑或修改选定的边界 UV，必须使用 UV 编辑器。

到 UV 周长(To UV Perimeter)
将当前选择的组件（顶点、UV、边或面）沿着其周长转化为 UV 的边界。
注： 多边形标记菜单中也提供此命令。（在选定多边形组件后按 Ctrl + 单击鼠标右键。）
到 UV 循环边(To UV Edge Loop)
将当前选定的顶点、UV 或边更改为循环边。与原来选择的对象关联的任何多边形边，以及沿同一循环边的任一方向并且在任意 UV 边界处终止的任何边都将被转化。
到壳(To Shell)
将当前选定的 UV 扩展到关联壳上的所有 UV。
到壳边界(To Shell Border)
将当前选定的 UV 转化为沿关联壳的边界的 UV。
提示： 只要在选定多边形组件的情况下按住 Ctrl 键，同时单击鼠标右键，即可在上下文相关多边形标记菜单中使用许多转化当前选择功能。
