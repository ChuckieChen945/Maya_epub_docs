log_file_io

记录文件 I/O 操作的结果。完整的复合名称为：File::Common::Internal::log_file_io。

输入

in_passthrough

端口类型是：auto。它应为 File::Common::FileOperationSingleResult 或 File::Common::FileOperationMultiResults。

filepath

端口类型是：string。

kind

端口类型是：File::Common::FileOperationKind。已执行的 I/O 操作的类型。

输出

out_passthrough

推断端口类型将与 in_passthrough 相同。

父页面: Internal
