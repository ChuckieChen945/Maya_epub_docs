安装

要在 Maya 中使用 Arnold，您需要下载并安装 MtoA 插件，然后配置 Maya 以使用它。

请注意，针对每个受支持的操作系统，为每个受支持的 Maya 版本都提供了一个插件：

  • Windows 64 位
  • Linux 64 位
  • Mac OS X（64 位 10.7 Lion 或更高版本）

此外，还提供了一个疑难解答页面。
