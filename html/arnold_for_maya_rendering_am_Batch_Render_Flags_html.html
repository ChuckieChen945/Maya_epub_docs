批渲染标志

下面是 Arnold 批渲染的最新渲染标志列表。渲染器“arnold”的特定选项：Arnold 渲染器。

通用标志
-rd path 存储输出图像文件的目录。
-im filename 图像文件输出名称。
-rt int 渲染类型（0 = 渲染，1 = 导出 ass，2 = 导出并运行 kick）。
-lic boolean 打开或关闭许可。

帧编号选项
-s float 动画序列的开始帧。
-e float 动画序列的结束帧。
-b float 动画序列的帧数或步长。
-pad int 输出图像帧文件的扩展名的数字位数。

渲染层和渲染通道
-rl boolean name(s)
-rp boolean name(s)
-sel boolean name(s)
-l boolean name(s)

摄影机选项
-cam name 指定将渲染哪个摄影机。
-rgb boolean 启用或禁用 RGB 输出。
-alpha boolean 启用或禁用 Alpha 输出。
-depth boolean 启用或禁用深度输出。
-iip 忽略图像平面。渲染之前关闭所有图像平面。

分辨率选项
-x int 设置最终图像的 X 分辨率。
-y int 设置最终图像的 Y 分辨率。
-percentRes float 使用分辨率百分比渲染图像。
-ard float 渲染图像的设备宽高比。

采样选项
-ai:as int 摄影机采样数（抗锯齿）
-ai:hs int 间接漫反射采样数。
-ai:gs int 间接镜面反射采样数。
-ai:rs int 折射采样数。
- ai:bssrdfs int SSS 采样数。

采样区间限定
-ai:cmpsv boolean 启用采样区间限定。
-ai:aovsc boolean 采样区间限定会影响 AOV。
-ai:aasc float 采样区间限定值。

光线深度选项
-ai:td int 总深度。
-ai:dif int 间接漫反射深度。
-ai:glo int 间接镜面反射深度。
-ai:rfl int 反射深度。
-ai:rfr int 折射深度。
-ai:atd int 自动透明度深度。
-ai:att float 自动透明度阈值。

运动模糊
-ai:mben boolean 启用运动模糊。
-ai:mbdf boolean 启用对象变形运动模糊。
-ai:mbrt int 快门位置
-ai:mbms int 运动步数。
-ai:mbfr int 快门长度。
-ai:mbst int 运动开始。
-ai:mben int 运动结束。

灯光
-ai:llth float 低灯光阈值。
-ai:ll int 灯光链接模式。（0 - 无，1 - Maya 灯光链接）
-ai:sl int 阴影链接模式。（0 - 无，1 - 沿用灯光链接，2 - Maya 阴影链接）

细分
-ai:mxsb int 最大细分级别。

渲染设置
-ai:threads int 设置线程数。
-ai:bscn int 渲染块扫描。（0 - 顶部，1 - 底部，2 - 左，3 - 右，4 - 随机，5 - 交织，6 - 螺旋，7 - Hilbert）
-ai:bsz int 渲染块大小。
-ai:bass boolean 导出二进制 ASS 文件。
-ai:exbb boolean 导出 ASS 文件中的边界框。
-ai:aerr boolean 出错时中止。
-ai:alf boolean 许可失败时中止。
-ai:slc boolean 跳过许可检查。
-ai:uopt string 用户选项字符串。

纹理
-ai:txamm boolean 启用自动 MipMap。
-ai:txaum boolean 接受非 MipMap 纹理。
-ai:txaun boolean 接受非平铺纹理。
-ai:txett boolean 使用现有的平铺纹理。
-ai:txat int 自动分片大小。
-ai:txmm float 最大纹理缓存内存 (MB)。
-ai:txmof int 打开的纹理的最大数量。
-ai:txpfs boolean 每个文件纹理的统计信息。
-ai:txdb float 漫反射光线的纹理模糊。
-ai:txgb float 光泽光线的纹理模糊。

功能覆盖
-ai:fotx boolean 忽略纹理贴图。
-ai:fosh boolean 忽略着色器。
-ai:foat boolean 忽略大气着色器。
-ai:folt boolean 忽略灯光。
-ai:fosw boolean 忽略阴影。
-ai:fosd boolean 忽略细分。
-ai:fodp boolean 忽略置换贴图。
-ai:fobp boolean 忽略凹凸贴图。
-ai:fosm boolean 忽略顶点法线平滑处理。
-ai:fomb boolean 忽略运动模糊。
-ai:fosss boolean 忽略 SSS。
-ai:fodof boolean 忽略景深。

搜索路径
-ai:sppg string 插件搜索路径。
-ai:sppr string 程序搜索路径。
-ai:spsh string 着色器搜索路径。
-ai:sptx string 纹理搜索路径。

日志记录
-ai:lfn string 日志文件名。
-ai:lmw int 警告最大数量。
-ai:ltc int 日志输出到控制台。
-ai:ltf int 日志输出到文件。
- ai:lve int 详细级别。（0 - 错误，1 - 警告，2 - 信息，3 - 调试）

统计信息/分析器
-ai:ste 支持布尔值的统计信息。
-ai:stf 字符串统计信息文件名。
-ai:stm 整数统计信息模式。
-ai:pfe 支持布尔值的数据图。
-ai:pff 字符串数据图文件名。

Mel 回调
-preRender string 添加渲染前执行的 Mel 代码。
-postRender string 添加渲染后执行的 Mel 代码。
-preLayer string 添加在每个渲染层之前执行的 Mel 代码。
-postLayer string 添加在每个渲染层之后执行的 Mel 代码。
-preFrame string 添加在每一帧之前执行的 Mel 代码。
-postFrame string 添加在每一帧之后执行的 Mel 代码。
-insertPreRender string 插入在渲染前执行的 Mel 代码。
-insertPostRender string 插入在渲染后执行的 Mel 代码。
-insertPreLayer string 插入在每个渲染层之前执行的 Mel 代码。
-insertPostLayer string 插入在每个渲染层之后执行的 Mel 代码。
-insertPreFrame string 插入在每一帧之前执行的 Mel 代码。
-insertPostFrame string 插入在每一帧之后执行的 Mel 代码。

区域选项
reg 设置自定义矩形区域进行渲染 (xmin xmax ymin ymax)。XY 从左上角 0,0 处开始。

渲染设备
- ai:device 渲染设备（0：CPU，1：GPU）。
- ai:gpu int 用于渲染的 GPU 的索引（与 manGpuSel 结合使用，可以设置单个 GPU 进行渲染）。
ai:manGpuSel 启用/禁用手动 GPU 选择（布尔）。
