为多边形顶点颜色设置动画

可以为应用于多边形顶点的逐顶点颜色数据设置动画。例如，可以创建水晶从暗淡无光变为发出绿光的动画。

为顶点颜色设置动画

  1. 开始之前，请确保对象具有一些构建历史。如果没有构建历史，逐顶点颜色功能将无法设置关键帧。

    可以通过选择“编辑网格 > 变换”(Edit Mesh > Transform)在模型上快速创建历史，而无需移动任何内容。

  2. 在给定的帧上，为选定顶点指定颜色。颜色指定操作项（例如“绘制顶点颜色工具”(Paint Vertex Color Tool)）位于网格显示(Mesh Display)菜单的“顶点颜色”(Vertex Color)部分下。
  3. 选择“网格显示 > 为顶点颜色设置关键帧”(Mesh Display > Set Keyframe for Vertex Color)。

    不能使用其他任何关键帧设置方法。

删除顶点颜色动画

  1. 执行下列操作之一：
  • 删除 polyColorPerVertex 节点。
  • 如果从网格中删除构建历史（编辑 > 按类型删除 > 历史(Edit > Delete by Type > History)），Maya 也将删除顶点颜色动画。

逐顶点颜色动画疑难解答

为什么在为顶点颜色设置动画后播放速度变慢？

Maya 将为每个面的每个顶点创建动画曲线。出于此原因，如果为过多的顶点设置动画，播放性能将大大降低。

请尽量限制为其设置动画的顶点的数量。另一种方法是删除静态动画曲线。

为什么会收到警告“活动对象不具有可设置关键帧的属性”(Active objects have no keyable attributes)？

必须首先应用颜色。如果没有指定颜色，将无法设置关键帧。

为什么会收到警告“当前操纵器和活动对象不具有可设置关键帧的属性”(Current manipulator and active objects have no keyable attributes)？

必须向网格中添加构建历史。如果网格没有构建历史，则顶点颜色更改不会设置关键帧。

可以通过在将转换设置为 0 的情况下使用“编辑网格 > 变换”(Edit Mesh > Transform)在模型上快速创建历史。
