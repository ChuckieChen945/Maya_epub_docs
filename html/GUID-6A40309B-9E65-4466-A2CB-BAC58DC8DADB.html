通过移动对象创建运动路径

以下过程使用“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)（在“动画”(Animation)菜单集中），通过将对象从场景中的一个位置移动到另一个位置来创建运动路径动画。或者，也可以使用“约束 > 运动路径 > 连接到运动路径”(Constrain > Motion Paths > Attach to Motion Path)将对象连接到现有路径。有关详细信息，请参见将对象附加到运动路径。

创建路径动画

  1. 选择要使用运动路径设置动画的对象，并将它移动到开始位置。
  2. 将当前时间设定为路径动画的开始时间。
  3. 选择“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)。

    将在指定的开始时间创建一条具有位置标记的 CV 曲线。

  4. 增加当前时间并将对象移动到新位置。
  5. 再次选择“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)。

    将在开始时间标记到对象的当前位置（在该位置放置一个新标记）之间绘制一条跨度曲线。

  6. 只要您希望对对象位置设置动画，便可通过选择“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)继续更改时间和位置。

    设定最后一个运动路径关键帧的时间是运动路径动画的结束时间。

请注意，使用“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)创建路径动画后，可以使用前面几节介绍的所有其他路径动画工具细化基本路径动画。

同样，您可以选择预先存在的路径曲线上已设置动画的对象，并使用“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)扩展动画的长度（从而延伸路径曲线）。

提示： 如果您不喜欢对象在特定标记时间的位置，则将当前时间更改为该标记的时间并将对象移动到新位置。选择“约束 > 运动路径 > 设置运动路径关键帧”(Constrain > Motion Paths > Set Motion Path Key)，曲线的形状将更改，以便标记现在位于对象的新位置。
