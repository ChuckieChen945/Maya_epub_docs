使用顶点贴图控制 XGen 属性

您可以使用“绘制顶点颜色工具”(Paint Vertex Color Tool)将 XGen 属性值绘制到绑定网格的顶点上。例如，绘制遮罩或颜色时会直接在网格上逐顶点设置这些值。请注意，贴图的分辨率取决于绑定网格的分辨率。如果需要更高的分辨率，则可绘制使用 Texel 分辨率的 Ptex 贴图（请参见使用 Ptex 贴图控制基本体。

  1. 单击要使用顶点贴图控制的属性旁边的表达式图标 。
  2. 在“XGen 表达式编辑器”(XGen Expression Editor)中，单击“采样数”(Samples)选项卡并浏览到“全局 > 颜色 > UI > map_vertexcolormap”(Global > Color > UI > map_vertexcolormap)。

    “XGen 表达式编辑器”(XGen Expression Editor)显示以下表达式： 

    $a=vmap('${DESC}/colorSet1/');#vpaint
$a
  3. 单击“接受”(Accept)。
  4. 在“XGen 编辑器”(XGen Editor)，单击使用颜色顶点贴图设置的遮罩或属性旁边的图标 。
  5. 使用“绘制顶点颜色工具”(aint Vertex Color Tool)绘制网格。
  6. 单击图标 以保存顶点贴图。
