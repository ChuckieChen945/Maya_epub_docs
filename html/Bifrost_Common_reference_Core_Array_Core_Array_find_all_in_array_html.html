find_all_in_array

返回 array 中值等于 value 的元素的所有 indices。此节点适用于可与 equal 进行比较的所有类型的数组。

它可用于查找数组中与某个值完全匹配的所有值，例如：

Array: [0, 1, 2, 0, 1, 2]
Value: 1
Invert: False
Indices: [1, 4]
Array: [0, 1, 2, 0, 1, 2]
Value: 1
Invert: True
Indices: [0, 2, 3, 5]

还可以用于在布尔数组中查找 True 值。这使您能够间接地查找某些查询返回 True 的所有值，例如“less than three”。 

Original array: [0, 1, 2, 3, 4, 5]
Apply less than 3 to the array: [True, True, True, False, False, False]
Value: True
Invert: False
Indices: [0, 1, 2]

输入

array

要与之比较的值的数组。

value

要在数组中查找的值。这是一个 equal 比较，因此只包括完全相等的值。

invert

如果为 True，则查找数组中不等于 value 的所有值。

输出

indices

与查询匹配的索引数组。

父页面: Array
