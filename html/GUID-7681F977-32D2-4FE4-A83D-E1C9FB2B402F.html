导出权重

当平滑蒙皮曲面时，Maya 将在每个关节创建一个权重贴图。使用“导出权重贴图”(Export Weight Maps)菜单项，可以同时导出所有权重贴图。

基于选定对象的当前 UV 导出蒙皮权重贴图。蒙皮权重贴图相对于蒙皮几何体的 UV 参数化。

重要： 不能将蒙皮权重贴图导出或导入到晶格。

如果要导出蒙皮权重，最好将 UV 指定给蒙皮几何体。如果蒙皮几何体不具有 UV，导出结果可能（取决于蒙皮几何体的复杂度）是不需要的。如果蒙皮几何体已指定 UV 并且在 UV 中存在重叠，导出结果可能是重叠区域中不需要的。

有关为多边形或细分曲面创建 UV 的信息，请参见映射 UV。

注： 如果希望基于空间位置而非 UV 传递蒙皮权重，请参见复制平滑蒙皮权重。使用“复制蒙皮权重”(Copy Skin Weights)，您的源和目标蒙皮几何体不必具有相同 UV。

导出权重贴图

  1. 选择蒙皮对象。如果为对象创建父对象或分组，单独选择各个对象。
  2. 选择“蒙皮 > 导出权重贴图”(Skin > Export Weight Maps) > 以打开导出权重贴图选项。
  3. 设定这些选项并单击“导出”(Export)。

    对于大多数应用程序，仅“贴图大小 X”(Map Size X)和“贴图大小 Y”(Map Size Y)选项可用。如果您计划使用专门的贴图编辑技术，其他选项很有用。大多数情况下，默认选项设置适用。

  4. 在显示的“写入映射”(Write Map)文件浏览器中，为将创建以保存贴图文件的文件夹（或目录）指定路径和名称，然后单击“写入”(Write)。

    默认情况下，Maya 将指定的文件夹名称置于当前项目的 sourceimages 文件夹下。

  5. 当 Maya 列出将写入到磁盘的贴图数目并提示是否要继续时，单击“是”(Yes)。

    Maya 将图像文件（每个关节一个）置于指定的文件夹中。例如： 

    jackie_back_root.iff
jackie_jaw.iff
jackie_left_ankle.iff

    Maya 也创建名为 folder.weightMap 的权重贴图文件，创建位置与您为图像指定的文件夹（默认为 sourceimages）位置相同。

    权重贴图文件的内容定义曲面几何体和图像文件之间的关系。您可以使用文本编辑器查看和编辑该文件的内容。Maya 使用权重贴图文件导入图像。导入权重贴图时，权重贴图文件必须同包含图像文件的文件夹位于同一位置。请参见导入权重。
