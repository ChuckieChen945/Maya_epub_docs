Display Layer options menu

Display toggle options appear as you click Options under the Display panel in the Channel Box/ Layer Editor.

  • Windows > General Editors > Channel Box/ Layer Editor > Display > Options

Make New Layers Current

If enabled, each new display layer created will be the current display layer highlighted/selected. This can enable you to have the latest created display layer active for prim assigments immediately.

Add New Objects to Current Layer

If enabled, any USD prim or object in your Outliner hierarchy that is created will automatically get added to the current layer (your selected layer is highlighted in blue).

Show Namespace

If enabled, displays your namespace hierarchy.
