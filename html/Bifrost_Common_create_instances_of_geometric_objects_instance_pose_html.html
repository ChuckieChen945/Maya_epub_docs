调整实例变换

点对象的 point_position 特性用于确定实例放置位置。此外，还有其他几个特性可能会影响实例化几何体的变换。

可以使用 set_geo_property 创建这些特性（如果它们尚不存在）。如果存在，则可以使用 set_geo_property_data 更改其值。这些特性应针对点对象的 point_component。

  • point_size：用于在所有轴上按统一比例缩放形状的 float。
  • point_scale：用于按非统一比例缩放的 float3。
  • point_orientation：以四元数表示方向的 float4。

此外，还可以在 geo_instance 或 render_archive_instance 等节点上设置 transform 参数。如果主几何体偏移于世界中心，并且您希望补偿该几何体的所有实例，同时仍能够单独控制特定实例的大小和方向，这将很有用。
