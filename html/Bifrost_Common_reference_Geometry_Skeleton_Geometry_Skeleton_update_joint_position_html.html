update_joint_position

根据关节的变换矩阵更新关节位置。

虽然变换数据使用双精度，但返回的位置是 array<float3>，这是为了便于进行传统的几何操作（例如空间查询）。

输入

skeleton

要更新的骨架。

use_rest_state

如果启用此项，则使用 joint_pivot_matrix（表示变换的静止状态）而不是 joint_world_matrix 来获取其位置。

输出

out_skeleton

更新的骨架，包含 joint_position 几何特性。

joint_position

从更新的 joint_position 几何特性生成的 array<float3> 数据。
