BUGFIXES

MtoA 1.4.1.1 is a bugfix release. It contains the following fixes :

  • Custom AOVs were randomly missing
  • Renaming a snapshot in the Arnold RenderView sometimes crashed Maya
  • Exporting the scene when active camera in the viewport is non-renderable could skip the outputs definition
  • Rollback to arnold default message callback during batch sessions
  • Export to .ass creates the arnold options node if it's missing
