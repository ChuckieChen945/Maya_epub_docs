set_geo_property

创建和初始化针对组件（如几何对象上的点）的特性。

如果对象上已经存在该特性，则可以改用 set_geo_property_data。对于某些常用特性，还可以使用特定复合，例如 set_point_position。

输入

geometry

几何对象。

property

特性的名称。

default

特性的默认值。

data

特性的数据，通常每个分量对应一组值。数组将调整大小以匹配 target 分量中的元素数。注意：对于 data 类型为 array<uint> 的偏移特性（例如 face_offset 或 strand_offset），数组的大小将调整为元素数加一，以符合偏移数组模式。有关偏移数组的详细信息，请参见 access_offset_array。

target

特性针对的几何组件，例如 point_component、face_component、face_vertex_component、voxel_component。

interpolation_mode

指定在修改几何体时应如何对数据进行插值。

  • Nearest：最近相邻插值。
  • Linear：通过加权和进行线性插值。
  • DefaultValue：无插值，改为使用特性的 default 值。

输出

out_geometry

修改后的几何体。
