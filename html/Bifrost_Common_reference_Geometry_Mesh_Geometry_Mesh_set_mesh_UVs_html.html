set_mesh_UVs

在网格中设置 UV 坐标数据。

输入

mesh

要设置 UV 数据的网格。

uv_number

UV 坐标数据集的编号。例如，使用值 5 会将名为 face_vertex_uv5 的 UV 集添加到网格中。如果存在具有此名称的现有 UV 坐标集，则将被替换。

coordinates

要在网格中设置的 UV 坐标数据。数据可以是 2D (float2) 或 3D (float3) 坐标。

indices

将面顶点映射到给定坐标数组中元素的逐面顶点索引。如果为空，则坐标将被视为逐顶点值，网格的 face_vertex 数组将用于为坐标数组编制索引。

输出

out_mesh

添加了 UV 坐标的网格。
