使用代理引用

下表概述了使用“引用编辑器”(Reference Editor)处理代理引用时可执行的常见任务的步骤。由于代理引用与文件引用相关联，您可能还需要参考以下相关主题。

提示：

在 Maya 场景中工作时，通过在被引用对象上单击鼠标右键，然后从菜单中进行相应选择，也可以使用“引用编辑器”(Reference Editor)的某些菜单操作。

目标 操作

将代理引用添加到场景中的现有文件引用。

  1. 选择“文件 > 引用编辑器”(File > Reference Editor)。
  2. 在引用编辑器中，选择要添加代理引用的现有文件引用，并选择“代理 > 添加代理”(Proxy > Add Proxy) > 。
  3. 在“代理选项”(Proxy Options)窗口中，通过以下任一方法设定“代理标记选项”(Proxy Tag Options)：键入希望用作代理引用的标签/标记的文本字符串；或者从文本字符串下拉菜单中选择现有标记，然后单击“代理”(Proxy)按钮。
  4. 在文件浏览器中，导航到希望用作代理引用的文件，单击其名称，然后单击“引用”(Reference)。

代理文件将添加到代理集内可用代理的列表中。在“引用编辑器”(Reference Editor)中，包含新代理的文件引用会同时显示一个图标，以指示该文件引用存在一个或多个代理引用。

加载一个或多个现有代理引用或在它们之间切换。

  1. 选择“文件 > 引用编辑器”(File > Reference Editor)。
  2. 在“引用编辑器”(Reference Editor)中，选择与要加载到该场景中的代理引用相对应的文件引用。
  3. 在引用编辑器菜单中，选择“代理 > 将代理重新加载为”(Proxy > Reload Proxy As)，然后从下拉列表中显示的代理标记列表中选择所需的代理文件。

通过加载与选定的代理标记相关联的代理引用文件更新选定的文件引用。

从场景中移除代理引用。

  1. 选择“文件 > 引用编辑器”(File > Reference Editor)。
  2. 在“引用编辑器”(Reference Editor)中，选择要从场景中移除的文件引用。
  3. 在引用编辑器菜单中，选择“代理 > 移除代理”(Proxy > Remove Proxy)，然后从下拉列表中显示的代理标记列表中选择所需的代理文件。

选择性预加载代理引用。

  1. 选择“文件 > 打开场景”(File > Open Scene) > 。
  2. 在“引用”(Referencing)区域中，启用“选择性预加载”(Selective Preload)，然后单击“打开”(Open)。
  3. 在文件浏览器中，选择包含要加载的文件和代理引用的文件，然后单击“打开”(Open)。
  4. 在“预加载引用编辑器”(Preload Reference Editor)中，执行下列操作之一：
    • 单击某个引用旁边的复选框，以指示要加载该引用。
    • 在某项目上单击鼠标右键，然后从显示的菜单中选择“调入已加载的引用”(Bring in Reference Loaded)或“调入未加载的引用”(Bring in Reference Unloaded)。
    • 在某项目上单击鼠标右键，然后从显示的菜单中选择“将活动代理设置为”(Set Active Proxy To)并选择要从显示的子菜单中加载的代理。

将引用编辑保存到引用文件。

  1. 选择“文件 > 引用编辑器”(File > Reference Editor)。
  2. 在“引用编辑器”(Reference Editor)中，单击被引用文件，然后选择“文件 > 保存引用编辑”(File > Save Reference Edits)。

使用共享编辑创建代理集。

  1. 从“引用编辑器”(Reference Editor)中选择一个现有文件引用。
  2. 从引用编辑器中，选择“代理 > 添加代理”(Proxy > Add Proxy) > 。
  3. 在“代理选项”(Proxy Options)窗口中，启用“共享编辑”(Shared Edits)。
  4. 单击“代理”(Proxy)按钮。

    此时将显示文件浏览器。

  5. 选择文件并单击“代理”(Proxy)将文件作为代理引用添加到现有文件引用。

重命名代理标记。

  1. 在“引用编辑器”(Reference Editor)中，单击要重命名其代理标记的代理引用以使其亮显。
  2. 在引用编辑器中，选择“代理 > 切换代理标记”(Proxy > Switch Tag for Proxy)。
  3. 选择要重命名其代理标记的代理或引用并选择“定义新标记”(Define New Tag)。
  4. 此时将显示“定义新代理标记”(Define New Proxy Tag)窗口。
  5. 在字段中键入新代理标记的名称并单击“确定”(OK)。

重用代理标记。

  1. 在“引用编辑器”(Reference Editor)中，单击以亮显要重命名其代理标记的代理引用。
  2. 在引用编辑器中，选择“代理 > 切换代理标记”(Proxy > Switch Tag for Proxy)。
  3. 选择要重命名其代理标记的代理或引用并选择要重用的代理标记。
  4. 此时将显示“定义新代理标记”(Define New Proxy Tag)窗口，后跟可用标记的列表（如果存在）。已在代理集中使用的标记将呈灰色并显示在该列表末尾。选择可用代理标记。
