dynamic_wave_settings

此项包含 simulate_dynamic_waves node 使用的所有全局设置。二维模拟的范围由边界特性定义。如果启用了 periodic，则在这些边界之外重复操作。

输入

Global Properties

periodic

在空间中重复显示波浪图案。这样可以使用较小的面片对较大的网格平面进行变形，但平铺的重复可能会很明显。

resolution_x

模拟的 X 分辨率。值越大，细节越精细，但计算速度越慢，并且使用的内存也越多。它应为 2 的幂次方，例如 128、256、512、1024。

resolution_z

模拟的 Z 分辨率。值越大，细节越精细，但计算速度越慢，并且使用的内存也越多。它应为 2 的幂次方，例如 128、256、512、1024。

space_scale

X 和 Z 各方向上面片的线性比例因子。如果值介于 0 和 1.0 之间，则收缩结果；如果值大于 1.0，则展开结果。如果在缩放后波移动过快或过慢，您可以调整 gravity。

speed_scale

缩放模拟的总体速度。值为 1.0 表示适合面片大小的速度，以米为单位（忽略 Maya 单位设置），并且假定重力为 9.8。

Simulation

substeps

这是每帧的模拟步数。通常，默认值 1 就足够了，但对于快速移动的波浪，尤其是当 capillary_size 相对较高时，可能需要更高的步数才能稳定或获得良好的结果。

gravity

由重力产生的加速度。默认值 9.8 表示地球表面的重力强度（以 m/s² 为单位）。值越小，产生的波浪越高且移动速度越慢，值越大，产生的波浪越低且移动速度越快。可以调整此值以更改比例。

friction

此项主要是随着时间的推移对波施加阻尼，使其消亡。通常，值为零或非常小的值可以获得良好的结果。

start_frame

要从其开始模拟的帧。

Bounds

边界确定要计算的水面在 X 和 Z 方向上的尺寸。模拟仅限于此区域，但如果启用了“periodic”，则输出波将在此区域之外无缝环绕。

bounds_method

此项确定如何为模拟定义边界。

  • Min_Max：边界由特性 min_bound 和 max_bounds 确定

  • Size_Center：边界由特性 size 和 center 确定

  • From_Geometry：边界由连接到 bounds_geometry 的几何体确定

min_bound

模拟在 X 和 Z 方向上的最小范围。（当 bounds_method 为 Min_Max 时）

max_bound

模拟在 X 和 Z 方向上的最大范围。（当 bounds_method 为 Min_Max 时）

size

模拟在 X 和 Z 方向上的总体大小（当 bounds_method 为 Size_Center 时）

center

模拟边界在 X 和 Z 方向上的中心位置（当 bounds_method 为 Size_Center 时）

bounds_geometry

当 bounds_method 设置为 From_Geometry 时，边界由连接到 bounds_geometry 的对象的 xz 边界确定。连接几何体（基于点的对象）。这可以是任何点对象或网格。

Wave

wave_height

波高度的人为倍增。如果值介于 0.0 和 1.0 之间，则降低波高度，如果值大于 1，则增加波高度。

use_horizontal_displacement

使网格的顶点在水平方向和垂直方向发生位移。这会导致波的形状更尖锐、更不圆滑。它启用 wave_peaking。

wave_peaking

启用了 use_horizontal_displacement 时，此项控制相对于 wave_height 的水平位移量，这将产生尖波。可调整此值以避免输出网格中出现自相交。值大于 1.0 时，往往会产生自相交。普通水波是摆线形状，波浪越高，也就越尖。

decay_width

这是与边界的距离（相对于模拟范围），在此距离内按 decay_alpha 对模拟施加阻尼。此项用于在模拟边界处停止波反射和创建。它通常应该是非常小的值，否则对模拟施加阻尼的程度可能会变得非常大。

decay_alpha

这是模拟边界附近的阻尼量，用于避免边界处发生波反射。它应该非常小，但适当高就可以消除不需要的边界波。通常，如果 decay_width 较大，则此值应较小，以避免对模拟过度施加阻尼。

ocean_depth

用于计算波浪运动的水深。在浅水中，波浪往往较长、较高及较慢。

capillary_size

毛细波（曲面张力传播的较小、较快的涟漪，有时可在重力传播的较大波浪顶部看到）的最大波长（厘米）。毛细波通常仅在比例较小且分辨率较高的情况下可见，因此在许多情况下，可以让此值保留为 0.0 以避免执行不必要的计算。

Sources

use_advection

如果启用此项，则波将在 XZ 方向上按漂移速度平流。这可用于模拟流动的河流或水流中的湍流涡流等对象。漂移可能只是某个恒定速度（在这种情况下，它还可用于模拟摄影机在水面上飞行），也可以按每个像素映射它。例如，可以将 fractal_turbulence_field 连接到 drift，使水旋转起来。

drift

启用了 use_advection 时，此速度用于推动液体。可以使用常量值 (float2) 或连接向量场。这可用于流动的液体。

shallow_bottom

这可用于模拟连续变化的深度。要实现此目的，必须将几何体或 wave_map 连接到 depth 特性。当与底部的距离小于波长时，波速会减慢。因此，波浪在靠近海岸时往往会越来越靠近并堆起，并且变得与海岸线平行。此项模拟输入深度中最小深度与最大深度之间的几个不同深度，然后根据局部深度值在它们之间插值。num_depth_levels 确定要模拟的离散深度数。

depth

启用了 shallow_bottom 时，此项确定整个模拟中水的局部深度，这会影响波速度。它可以直接连接到定义水底的网格。如果网格未涵盖模拟的整个范围，则网格上的最小边界将表示对象外部的深度。可以使用表示深度的波贴图来代替网格。

num_depth_levels

启用了 shallow_bottom 时，此项确定要计算的离散深度模拟数。计算的深度分布在输入深度的最小边界与最大边界之间。输出模拟将根据某点处的局部深度对这些 FFT 模拟进行插值。

Output Options

output_slope

如果启用此项，输出波将具有坡度几何特性，即波的 XZ 坡度（无波峰）。

output_velocity

如果启用此项，输出波将具有速度几何特性，即曲面上的 XYZ 速度。

输出

solver_settings

这是具有波模拟的所有特性的对象。它应连接到 simulate_dynamic_wave 节点的设置。
