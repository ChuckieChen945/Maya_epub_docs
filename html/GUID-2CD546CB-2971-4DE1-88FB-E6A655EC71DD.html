“UV 编辑器”(UV Editor)的“工具”(Tool)菜单

从工具栏中也可访问该菜单中的大多数项目。

隐藏 UV 工具包(Hide UV Toolkit)/显示 UV 工具包(Show UV Toolkit)

切换 UV 工具包的显示。

选择(Select)、移动(Move)、旋转(Rotate)、缩放(Scale)
请分别参见选择工具、移动工具、旋转工具和缩放工具。
晶格(Lattice)

通过允许出于变形目的围绕 UV 创建晶格，将 UV 的布局作为组进行操纵。有关详细信息，请参见晶格工具。

移动 UV 壳(Move UV Shell)

可用于通过在壳上选择单个 UV 在 UV 编辑器的 2D 视图中选择并重新定位 UV 壳。可以自动防止已重新定位的 UV 壳与其他 UV 壳重叠。有关详细信息，请参见移动 UV 壳工具。

平滑(Smooth)

使用该选项可以按交互方式展开或松弛 UV。选择一组 UV，然后将“展开”(Unfold)或“松弛”(Relax)控件拖动到右侧。有关详细信息，请参见平滑 UV 工具。

涂抹(Smudge)

将选定 UV 及其相邻 UV 的位置移动到用户定义的一个缩小的范围。有关详细信息，请参见涂抹 UV 工具。

调整(Tweak)

可用于在 UV 编辑器中重新定位 UV，无需使用操纵器。有关详细信息，请参见调整 UV 工具。

笔刷

优化(Optimize)

均匀隔开 UV。详细信息请参见展开 UV 和优化 UV 工具。

展开(Unfold)

围绕接缝展开 UV。特别适用于对称的有机对象。详细信息请参见展开 UV 和优化 UV 工具。

切割(Cut)

沿边分离 UV。详细信息请参见切割 UV 工具。

抓取(Grab)

选择 UV 并在基于笔刷的区域中沿拖动方向移动 UV。详细信息请参见抓取 UV 工具。

固定(Pin)

锁定 UV，使其他基于笔刷的操作无法移动它们。详细信息请参见固定 UV 工具。

收缩(Pinch)

向工具光标的中心拉近顶点。详细信息请参见收缩 UV 工具。

缝合(Sew)

沿边焊接 UV。详细信息请参见缝合 UV工具。

涂抹(Smear)

按与曲面上拖动方向的原始位置相切的方向移动 UV。详细信息请参见涂抹 UV 工具。

对称(Symmetrize)

有关详细信息，请参见对称 UV 工具。
