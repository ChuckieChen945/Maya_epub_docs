在“属性编辑器”(Attribute Editor)或“通道盒”(Channel Box)中更改属性值

“通道盒”(Channel Box)或“属性编辑器”(Attribute Editor)中背景为紫色的属性值受与其他属性之间的连接控制。无法手动编辑这些属性值。

目标 操作

为属性输入一个值

在文本框中单击，然后键入新值并按 Enter 键。

对于布尔 (on/off) 值，键入 1 或 on，或者键入 0 或 off。

请注意，在“通道盒”(Channel Box)中不能设置旋转和平移限制。必须改为使用“属性编辑器”(Attribute Editor)进行设置。若要同时为许多对象设置限制，请使用“属性总表”(Attribute Spread Sheet)（“窗口 > 常规编辑器 > 属性总表”(Windows > General Editors > Attribute Spread Sheet)）中的“全部”(All)选项卡。

使用文件名填充文件路径

将文件从文件浏览器拖放到“属性编辑器”(Attribute Editor)的“图像名称”(Image Name)或“注释”(Notes)字段中。此外，还可以将其拖放到“窗口文本”(Window Text)和“滚动”(Scroll)字段中。

更改值并将键盘焦点返回到视图窗口，以便使用热键。

键入值，然后按 Enter 键。

更改值并将键盘焦点保留在编辑框中。

键入值，然后按 Enter 键。

注： 按 Enter 键后，值将亮显，方便输入新值和查看结果。

同时为多个属性输入相同的值。

按住 Shift 键并单击、按住 Ctrl 键并单击或拖过多个文本字段即可选择文本字段。然后键入新值并按 Enter 键。
注： 仅选择字段，而不选择属性名称。

更改“通道盒”(Channel Box)中滑块的敏感度。

  • 单击滑块速度图标可以在慢/中/快滑块之间切换。
  • 单击双曲线图标可以切换到双曲线缩放。这样就能够非常快速地更改值。将其运用到需要大幅进行调整的值上。

使用鼠标调整数值属性的值。

  • 单击属性，然后使用鼠标中键向左或向右拖动。
  • 使用“通道盒”(Channel Box)工具栏中的图标来控制鼠标移动距离与数值更改之间的映射。
    • 注： 这些设置也会影响视图中编辑器内不可见滑块的速度。
复制/粘贴“通道盒”(Channel Box)属性 在“通道盒”(Channel Box)中的属性上单击鼠标右键并选择“复制选定属性值”(Copy Selected Attribute Values)，然后在场景中选择其他对象，单击鼠标右键并选择“粘贴选定属性值”(Paste Selected Attribute Values)。
注： 如果在场景中选择多个对象，则仅复制选择的第一个项目的属性。

输入相对于当前值的一个值。

  • 键入 +=n，当前值加 n。
  • 类型 -=n，当前值减 n。
  • 键入 *=n，当前值乘 n。
  • 键入 /=n，当前值除 n。
  • 后缀 % 表示基于百分比进行操作（例如，+=10% 表示每个选定值都加上当前值的 10%）。

将属性快速链接到其他值。（仅适用于“属性编辑器”(Attribute Editor)）

在文本框中键入 =time 或 =frame，以便使用保留关键字 time 或 frame 并将属性值链接到当前时间或当前帧。也可以包括简单的运算符，例如 =frame/2、=time*3.5。

创建复杂表达式

在文本框上单击鼠标右键，然后选择“创建新表达式”(Create New Expression)（“属性编辑器”(Attribute Editor)）或“表达式”(Expressions)（“通道盒”(Channel Box)）。

提示：
在“通道盒”(Channel Box)中键入值时：
  • 按 Enter 键可设置值，并将焦点保持在“通道盒”(Channel Box)中。
  • 按 Enter 键可将焦点返回视图窗口（这样即可使用热键）。
