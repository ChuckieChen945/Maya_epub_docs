coarsen_voxel_property

根据指定特性的值，减少体积的细节。这有助于减少从 VBD 文件导入的体积的内存要求。

输入

volume

输入体积（对象）。

property

粗化所基于的特性的名称。

voxel_size

输出中允许的最小细节大小（如果大于零）。当此值为零时，将保留原始细节大小。

max_relative_error

输出中允许的相对于细节大小和边界框（对于带符号距离体积）或相对于绝对值（对于其他体积，包括雾）的最大差异（与输入体积相比）。首先尝试默认值，然后根据结果上下调整。如果误差阈值太低，放大时可能会出现瑕疵，如果误差阈值设置得太高，将会减少内存并节省计算时间。

add_sampling_error_property

将名为 voxel_sampling_error 的特性添加到每个体素上都有采样误差的输出体积。使用 volume_scope 节点检查此特性，以防 max_relative_error 未按预期工作。

max_magnitude

如果大于零，则将删减其绝对值大于此值的所有体素。这可用于将带符号距离场限制为 0 级左右的小带宽。

输出

out_volume

粗化体积（对象）。
