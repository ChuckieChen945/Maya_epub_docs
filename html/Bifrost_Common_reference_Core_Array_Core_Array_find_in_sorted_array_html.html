find_in_sorted_array

查找已排序数组中某项的第一个匹配项的索引，从数组的 start 到终点。sort_array 节点可用于对数组进行排序。在已排序数组中查找多个元素比使用 find_in_array 在未排序数组中查找要快。

输入

sorted_array

要搜索的已排序数组。

start

搜索开始处的索引。

value

要查找的值。

输出

found_index

第一个匹配项的索引，或者第一个大于 value 的元素的索引（如果未找到第一个匹配项）。如果该值大于数组中的最后一个元素，则返回数组的大小。

found

如果找到该值，则为 True；否则为 False。

父页面: Array
