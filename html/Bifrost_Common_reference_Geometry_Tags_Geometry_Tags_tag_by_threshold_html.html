tag_by_threshold

标记其特性值在阈值范围内的几何体组件。例如，这可用于标记其浮点特性小于最大值、大于最小值或在某个区间内的点。

输入

geometry

包含要标记的组件的几何体。

geometry_component

要标记的组件类型。默认情况下，此节点将标记点。

elements_to_tag

（可选）连接其他标记节点的输出或指定输入 geometry 上现有标记特性的名称。此输入的标记数据将与使用指定的 merge_mode 生成的标记数据合并。例如，通过将两个标记节点链接在一起并将第二个节点的 merge_mode 设置为 Union，可以合并这两个标记节点的结果。

这是解释的自动端口，可以接受 array<bool>、array<long>、array<uint> 或输入 geometry 上标记的字符串名称。

merge_mode

控制如何将 elements_to_tag 中的标记数据与此节点生成的标记数据合并：

  • Replace：忽略 elements_to_tag。
  • Union：对 elements_to_tag 和生成的标记数据执行布尔并集。
  • Intersection：对 elements_to_tag 和生成的标记数据执行布尔交集。
  • Subtraction：从 elements_to_tag 中减去生成的标记数据。

threshold_property

用于将阈值与之进行比较的几何特性。这可以是任何以指定的 geometry_component 为目标的 float 或 array<float> 几何特性。

标记

tag

要创建的标记特性的名称。

invert_tag

对标记条件求反。例如，反转 < & > 将标记超出 min 和 max 的区间（即，小于 min 或大于 max）的组件。

阈值

threshold_type

确定要应用的阈值类型：- <：小于 max。- <=：小于或等于 max。- >：大于 min。- >=：大于或等于 min。- < & >：小于 max 并且大于 min。- <= & >：小于或等于 max 并且大于 min。- < & =>：小于 max 并且大于或等于 min。- <= & =>：小于或等于 max 并且大于或等于 min。

min

阈值的下限。

max

阈值的上限。

诊断

point_size

点显示的世界空间大小。

offset_amount

面显示的世界空间偏移。增加此值可防止发生 Z 轴冲突。

strand_width

股显示的世界空间宽度。

triangle_size

用于控制面顶点角显示大小的归一化值。值的范围为 [0, 0.5]。值为 0.5 表示覆盖范围为面顶点相邻边一半边长的三角形。

输出

geometry_with_tag

以几何特性形式添加了标记的 geometry。

tag_data

包含所计算的组件标记的布尔数组。此数组的大小等于 geometry_component 中的元素数。如果 vector_property 中对应于此数组中某个元素的元素在阈值范围内，则此数组中的该元素为 true。
