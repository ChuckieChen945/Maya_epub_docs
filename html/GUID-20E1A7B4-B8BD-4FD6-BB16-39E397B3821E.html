geoConnector 节点

几何体连接器属性(Geo Connector Attributes)

镶嵌细分因子(Tessellation Factor)

该属性表示在镶嵌细分曲面中多边形的近似数量。

摩擦力(Friction)

设定碰撞粒子在弹离碰撞曲面时其平行于曲面的速度减小或增大的量。

值为 0 意味着粒子不受摩擦力的影响。值为 1 表示粒子完全沿曲面法线反射。如果“弹性”(Resilience)为 0，而“摩擦力”(Friction)为 1，则粒子不反弹。

仅 0 和 1 之间的值对应于自然摩擦力。值超出此范围表示夸大了响应。

弹性(Resilience)

设定反弹程度。值为 0 表示粒子碰撞没有反弹。值为 1 表示粒子完全反弹。值介于 0 和 -1 之间表示粒子穿过曲面同时折射出背面。值大于 1 或小于 -1 表示提高粒子的速度。
