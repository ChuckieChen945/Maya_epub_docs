创建和编辑集

您可以创建两种不同类型的集：集和快速选择集。二者均可包含任何类型的对象、组件或组，但快速选择集不能添加到划分（请参见集和划分）。您还可以创建嵌套集，使一个集包含其他集。

从 Maya“创建”(Create)菜单或“大纲视图”(Outliner)上下文菜单创建集，然后在“大纲视图”(Outliner)或“关系编辑器”(Relationship Editor)中查看和编辑集。

创建集

创建集

  1. 选择多个对象、对象组件或组。
  2. 从 Maya 主菜单中，选择“创建 > 集 > 集”(Create > Sets > Set) > 。

    如果所有集成员都是对象，也可以在“大纲视图”(Outliner)中单击鼠标右键以显示其上下文菜单，然后选择“集 > 创建集”(Sets > Create Set) > 。

  3. （可选）输入集的名称并将集添加到划分中。

创建快速选择集

  1. 选择多个对象、对象组件或组。

  2. 从 Maya 主菜单中，选择“创建 > 集 > 快速选择集”(Create > Sets > Quick Select Set)。

    如果所有集成员都是对象，也可以在“大纲视图”(Outliner)中单击鼠标右键以显示其上下文菜单，然后选择“集 > 创建快速选择集”(Sets > Create Quick Select Set)。

  3. 输入集的名称。

    也可以单击“添加到工具架”(Add to Shelf)按钮，将其作为图标添加到当前活动工具架，以便于访问。

选择集成员

选择集成员后，成员将在视口和“大纲视图”(Outliner)中亮显。集成员既可以是属于集一部分的对象、组件或组，也可以是嵌套在另一个集内的集。

若要选择集成员，请执行以下操作之一：

  • 在大纲视图中的集节点内单击集成员的名称。

    如果选择的对象是一个集成员（在视口或“大纲视图”(Outliner)中），则该集的名称会在“大纲视图”(Outliner)中以深灰色亮显来指示成员身份。

  • 在“大纲视图”(Outliner)中的集节点或集成员的名称上单击鼠标右键，然后选择“选择集成员”(Select Set Members)以选择该集的所有成员。

    这些对象在“大纲视图”(Outliner)和视口中亮显。

  • 在“关系编辑器”(Relationship Editor)（“窗口 > 关系编辑器 > 集”(Windows > Relationship Editor > Sets)）中，在“集”(Sets)面板中选择集，并在“对象”(Objects)面板中选择其成员。
  • 对于快速选择集，请从 Maya 主菜单中选择“选择 > 快速选择集”(Select > Quick Select Sets)，然后选择集，或者在工具架中单击集的图标（如果已创建）。

若要在选择集时自动选择集成员，请启用“首选项 > 选择 > 修改器 > 在大纲视图中自动选择集成员”(Preferences > Selection > Modifiers > Auto-select set members in the Outliner)。

在大纲视图中显示或隐藏集成员

若要在“大纲视图”(Outliner)中显示或隐藏集成员，请在“大纲视图”(Outliner)中的集上单击鼠标右键，然后切换“显示集成员”(Show Set Members)选项，或从“大纲视图”(Outliner)菜单中选择“显示 > 集成员”(Display > Set Members)。

这将切换所有集中的集成员在大纲视图中的显示。

您还可以通过在“属性编辑器”(Attribute Editor)中切换“在大纲视图中隐藏”(Hidden in Outliner)属性来显示或隐藏特定集。

将成员添加到集

可以将其他对象、组件或组添加到现有集。也可以通过将一个集添加到另一个集来创建嵌套集。

若要将成员添加到集，请执行以下操作之一：

  • 在“大纲视图”(Outliner)中，使用鼠标中键将任何节点或集拖放到集节点上，或者将其拖动到集中的集成员内。

    新成员将转到列表的底部。

    注： 如果使用鼠标中键将集成员拖动到另一个集，则会将其复制到该集，而不会从其原始集中移除。这是因为同一对象可以是多个集的成员。
  • （仅适用于对象）在“大纲视图”(Outliner)中，右键单击对象，然后选择“集 > 将当前选择添加到集”(Sets > Add Selection to Set)。

  • 在“关系编辑器”(Relationship Editor)中，在“集”(Sets)面板中选择集，然后在“对象”(Objects)面板中选择要添加到选定集的项目。

从组中移除成员

可以通过执行以下操作之一从集中移除成员或者从父集中移除嵌套集：

  • 使用鼠标中键，将一个或多个集成员拖动到“大纲视图”(Outliner)中的当前集之外的任意位置。

  • 在“大纲视图”(Outliner)中的一个或多个集成员或其集节点上单击鼠标右键，然后选择“从当前集中移除”(Remove from Current Set)。

  • （仅适用于对象）在“大纲视图”(Outliner)中，右键单击对象，然后选择“集 > 从集中移除当前选择”(Sets > Remove Selection from Set)。
  • 在“关系编辑器”(Relationship Editor)中，在“集”(Sets)面板中选择一个集成员，然后选择“编辑 > 从集中移除亮显的项”(Edit > Remove Highlighted from Set)。

删除集

删除集时将仅删除集节点，而不会删除其成员。若要删除集，请执行以下操作之一：

  • 在“大纲视图”(Outliner)中选择集，然后按 Delete 键。
  • 在“关系编辑器”(Relationship Editor)中，在“集”(Sets)面板中选择一个集，然后选择“编辑 > 删除亮显对象”(Edit > Delete Highlighted)。
