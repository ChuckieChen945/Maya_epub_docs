在现有图像和纹理上绘制

您可以将现有图像（Maya 支持的任何文件格式）打开到画布，然后在上面进行绘制。有关 Maya 支持的文件格式的信息，请参见支持的文件格式。

您也可以使用“Paint Effects”修改现有纹理。

在现有图像上绘制

  1. 选择“画布 > 打开图像”(Canvas > Open Image)。将显示“打开”(Open)（Windows 和 Mac OS X）或“打开图像”(Open Image) (Linux) 窗口。
  2. 若要显示 Maya 支持的文件格式的图像文件，请执行下列操作：
    • (Linux) 选择“读取为”(Read as)旁边的图像（UNIX 格式）。
    • (Windows) 选择“文件类型”(Files of type){旁边的图像（所有格式）。
    • (Mac OS X) 选择“显示”(Show)旁边的图像（所有格式）。
  3. 选择要导入到画布中的图像。

  4. 单击“打开”(Open)（Windows 和 Mac OS X）或“打开图像”(Open Image) (Linux)。

  5. 如果画布上存在未保存更改的图像，则系统会提示您保存。

    单击“是”(Yes)保存现有图像。

    “保存图像”(Save Image) (Linux) 或“保存”(Save)（Windows 和 Mac OS X）窗口将打开。键入包括文件扩展名的图像名称（例如，forest.iff），然后单击“保存图像”(Save Image) (Linux) 或“保存”(Save)（Windows 和 Mac OS X）。

    单击“否”(No)清除该画布而不进行保存，然后打开导入的图像。

  6. 在图像上绘制（请参见在画布上绘制）。
  7. 保存图像（“画布 > 保存”(Canvas > Save)）。“Paint Effects”将保存对所导入图像的更改。

在现有纹理上绘制

  1. 使用鼠标中键，将纹理从“Hypershade”拖到画布上。
  2. 如果画布上存在未保存更改的图像，则系统会提示您保存。

    单击“是”(Yes)保存现有图像。“保存图像”(Save Image) (Linux) 或“保存”(Save)（Windows 和 Mac OS X）窗口将打开。键入包括文件扩展名的图像名称（例如，forest.iff），然后单击“保存图像”(Save Image) (Linux) 或“保存”(Save)（Windows 和 Mac OS X）。

    单击“否”(No)清除该画布而不进行保存，然后打开导入的图像。

  3. 在纹理上绘制（请参见在画布上绘制）。
  4. 保存纹理（“画布 > 保存”(Canvas > Save)）。“Paint Effects”将保存对纹理引用的图像文件的更改。
