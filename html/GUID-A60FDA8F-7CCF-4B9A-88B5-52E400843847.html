XGen 交互式修饰导向(Guide)和线性线(Linear Wire)修改器属性

使用“导向”(Guide)和“线性线”(Linear Wire)修改器可确定头发形状并使其变形，以及驱动头发动画或模拟。“导向”(Guide)和“线性线”(Linear Wire)修改器可用于相同的用途。“导向”(Guide)修改器使用插值方法和区域贴图对头发形状进行其他控制。

有关详细信息和这些修改器之间的异同，请参见 XGen 交互式修饰修改器。

融合(Blend)
特定于“导向”(Guide)修改器。可用于控制每个导向对周围头发的影响范围。增加该值会放大每个导向的影响范围，使其对头发的影响成为相邻导向的平均值。

使用遮罩和幅值渐变时，调整“融合”(Blend)有助于控制导向形状、动画和模拟如何影响头发。

平滑度(Smoothness)
特定于“线性线”(Linear Wire)浏览器。设置每根头发上每条线具有的控制量。设置为 0 时，头发由最近的线控制。将“平滑度”(Smoothness)设置为较高的值时，会融合其他线之间的控制影响。“破损”(Breakage)属性控制从根部到尖端插值的衰减。
破损(Breakage)
特定于“线性线”(Linear Wire)浏览器。控制插值的衰减（通过设置从发根到发梢沿着头发的衰减半径）。

设置为 0 时，将逐渐从根部到尖端应用插值。较高的“破损”(Breakage)值增加在发梢的影响量，这样头发将在控制线分离时断开。

引用状态(Reference State)
特定于“线性线”(Linear Wire)浏览器。设置用于为头发设置动画的曲线的静止位置。

输入导向(Input Guide)/输入线(Input Wire)

特定于“导向”(Guide)和“线性线”(Linear Wire)修改器。

创建(Create)
单击“创建”(Create)将在网格曲面上生成导向/线。默认情况下，导向的密度是头发密度的 10%。这还会为导向和线创建一个由下列各项组成的新节点网络：
  • inGuide，源于主描述节点的子描述节点。
  • 雕刻修改器节点，允许使用交互式修饰工具雕刻导向。
  • 比例修改器节点，用于缩放导向的长度。
  • inGuide_base 节点，用于控制导向的生成。
使用选定曲线作为导向(Use Selected Curve as Guide)/使用选定曲线作为线(Use Selected Curve as Wire)
用于将场景、被引用文件或 Alembic 缓存文件中的选定 Maya 曲线转化为导向/线。
使导向动力学化(Make Guides Dynamic)/使线动力学化(Make Wires Dynamic)
打开使导向动力学化选项(Make Guides Dynamic Options)窗口，从中可以启用通过 nHair 系统中的动态曲线为导向/线设置动画的选项。

区域控制(Region Control)

特定于“导向”(Guide)修改器。在网格对象的曲面上创建导向后，可以使用“区域控制”(Region Control)属性。

使用区域贴图(Use Region Map)
启用和禁用区域贴图。
区域遮罩(Region Mask)
控制“区域贴图”(Region Map)的效果。您可以将绘制的纹理或现有的纹理文件指定给此属性，从而控制区域贴图在何处具有较强和较弱的效果。

值为 0.0 表示该区域应忽略区域贴图，而值 1.0 意味着应该完全支持贴图。介于 0.0 和 1.0 之间的值将使非区域导向按该值衰减。“区域遮罩”(Region Mask)可用于淡出该区域控制。该属性用作“区域贴图”(Region Map)效果上的倍增。

有关如何为该属性创建遮罩的信息，请参见使用遮罩。

区域贴图(Region Map)
区域贴图提供了覆盖导向插入放置方法和头发行为的方式。通常，区域贴图根据不同的颜色将网格曲面分为若干不同的区域。每个区域中的导向仅控制同一区域中的头发插值。通过在绑定网格曲面上绘制纹理文件或将现有的纹理文件指定给网格，可以创建区域贴图。

当您为定义网格上的区域而绘制单个颜色时，区域贴图效果最佳。使用硬边笔刷可以保证各个区域的边界清晰。例如，要想在一个头皮网格上创建头发部分，那就把网格的一侧绘制成蓝色而另一侧绘制成红色。贴图会强制源自红色区域的基本体忽略蓝色区域的导向插值，反之源自蓝色区域的基本体也会忽略红色区域的导向插值。

有关区域贴图和导向插值的信息，请参见区域贴图和遮罩。
