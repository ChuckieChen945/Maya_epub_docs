project_vector

将一个向量投影到另一个向量，然后返回平行和正交组件。

输入

vector

要投影的向量。

onto_vector

要投影到的向量。

输出

parallel_vector

first 在 second 上的投影。这是与 second 平行的 first 的组件。

orthogonal_vector

从 first 减去 parallel_vector 后的剩余部分。这有时称为 first 在 second 上的阻碍，它是与 second 正交的 first 的组件。对于三维向量，这也相当于 first 在垂直于 second 的平面上的投影。
