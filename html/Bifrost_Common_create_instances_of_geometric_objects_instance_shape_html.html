使用 set_instance_shape 创建实例

set_instance_shape 是一个基本节点，用于为点对象的所有点指定实例形状。与 set_instance_geometry 不同，您不能直接连接几何体。而是必须连接 geo_instance 的输出以实例化单个几何体，或连接 selector_instance 的输出以在多个几何体之间进行选择。

  1. 将 set_instance_shape 节点添加到图表中。

  2. 将点对象（例如粒子系统的输出）连接到 points 输入。

  3. 请执行下列操作之一：

    • 要实例化单个形状，请添加 geo_instance 节点。将要实例化的几何体连接到其 geometry 输入。（可选）将要用作替代对象以供在视口中预览的简单网格连接到其 preview_geometry 输入。将其输出连接到 set_instance_shape 节点的 instance_shape 输入。
    • 要实例化磁盘中的渲染归档文件或序列，请添加一个 render_archive_instance 节点，并将其输出连接到 set_instance_shape 节点的 instance_shape 输入。
    • 要在多个实例形状之间进行选择，请添加一个 selector_instance 节点，并将其输出连接到 set_instance_shape 节点的 instance_shape 输入。

  4. 将生成的 instances 输出到场景。
