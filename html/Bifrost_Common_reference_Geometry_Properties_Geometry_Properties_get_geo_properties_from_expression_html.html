get_geo_properties_from_expression

从表达式获取几何特性及其对应的名称。

输入

geometry

将搜索其特性的几何体。

property_expression

要匹配特性的表达式。

component

要匹配的特性的目标组件。

输出

properties

与表达式匹配的几何特性的数组。

property_names

与 properties 输出对应的几何特性名称的数组。

is_indexed

布尔数组，用于指示是否为找到的每个特性建立索引。建立索引的特性会将组件映射到数组中其相应值的索引，如 face_vertex_uv 或 face_vertex_normal，而不是直接映射到 point_position 等值。

index_property_name

与 is_indexed 输出对应的几何特性名称的数组。
