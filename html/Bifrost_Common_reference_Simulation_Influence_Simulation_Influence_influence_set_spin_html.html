influence_set_spin

可以在模拟上设置 point_spin 特性，从而控制旋转速度。自旋在内部保持为旋转向量，但可以选择使用 Euler 角度进行设置。可以使用向量场进行设置，以便进行更多控制。可以使用代理字段访问当前 point_spin，该字段将返回旋转向量值。请注意，旋转向量类似于轴角度，但旋转向量是轴与角度的乘积，因此是 float3 而不是 float4。

要使用此节点，请将其输出插入模拟节点（如 simulate_aero、simulate_particles 或 simulate_mpm）的 influences 输入。

输入

influence

可以选择将影响场 out_influence 连接到此端口。例如，可以连接 mask_influence 节点以遮罩此影响的效果。可以将任意数量的影响场呈菊花链状连接在一起，以构建复杂的场。

spin

这是用于替换 point_spin 的值。可以从 vectorField 进行连接，以随空间变化值。

rotation_type

这用于确定传入的旋转类型。point_spin 特性在内部使用旋转向量，因此，如果设置为 Euler，它将在内部转化为旋转向量。

  • Euler：绕每个轴的旋转角度（以度为单位）。当前始终使用 XYZ 旋转顺序。
  • RotationVector：采用旋转向量格式，其中向量方向表示旋转轴，向量长度表示旋转量。

输出

out_influence

可以连接到模拟节点或其他影响节点的输出影响对象。
