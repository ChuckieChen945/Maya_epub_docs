set_usd_point_instances_invisible

通过设置 USD PointInstancer 的“invisibleIds”属性来隐藏其特定实例。

输入

stage

保留 PointInstancer 基本体的 USD 后台文件。

prim_path

返回数据的基本体（PointInstancer 类型）的路径。

invisible_ids

用于隐藏点实例的 ID 数组。要了解点实例 ID 的值，可以将 get_usd_point_instancer_attributes 节点的 positions 输出端口连接到 get_array_indices 的 array 输入端口。get_array_indices 的 indices 输出端口将返回所有 ID。可以构建包含其中某些 ID 的较小数组以隐藏特定实例。

enable

禁用此选项可跳过将实例设置为不可见，并改为输出未修改的后台文件。

replace

启用时，ìnvisible_ids 将替换 PointInstancer 基本体上存在的任何现有“invisibleIds”属性的数据。禁用时，它会将数据添加到 PointInstancer 基本体上存在的任何现有“invisibleIds”属性。

输出

out_stage

修改后的 USD 后台文件。
