附加到 nCache 选项(Append to nCache Options)

附加时间范围(Append time range)

用于为想要创建并附加到当前对象已启用缓存的 nCache 定义“缓存时间范围”(Cache time range)。

当前时间(Current time)

仅指定已缓存并附加到已启用缓存的当前帧。

当前时间到时间滑块末端(Current time to Time slider End)

指定要创建并附加到已启用缓存的 nCache 的“缓存时间范围”(Cache time range)从当前时间指示器的当前时间位置开始，到“播放结束时间”(Playback end time)结束。

时间滑块(Time slider)

指定要创建并附加到已启用缓存的 nCache 的“缓存时间范围”(Cache time range)为整个播放范围。

缓存末尾到时间滑块末端(End of cache to Time slider end)

指定要创建并附加到已启用缓存的 nCache 的“缓存时间范围”(Cache time range)从已启用缓存的“源结束”(Source end)开始，到“播放结束时间”(Playback end time)结束。

缓存末尾到结束(End of cache to end)

指定要创建并附加到已启用缓存的 nCache 的“缓存时间范围”(Cache time range)从已启用缓存的“源结束”(Source end)开始，到“动画结束时间”(Animation end time)结束。

开始/结束(Start/End)

指定要创建并附加到已启用缓存的 nCache 的“缓存时间范围”(Cache time range)从“开始/结束”(Start/End)字段中指定的“开始”(Start)时间开始，到该字段中指定的“结束”(End)结束。

开始/结束(Start/End)

使用这些字段可以为“缓存时间范围”(Cache time range)指定“开始”(Start)时间和“结束”(End)时间。

“每 # 个帧求值一次”(Evaluate every # frame(s))和“每 # 次求值保存一次”(Save every # evaluation(s))

在创建 nCache 时，默认情况下，每个帧将使用一个对象模拟样本并保存到缓存。

“每 # 个帧求值一次”(Evaluate every # frame(s))指定了 nCache 创建过程中的采样频率。例如，值为 2 会在“缓存时间范围”(Cache time range)的每隔一帧处缓存当前对象的模拟。

“每 # 次求值保存一次”(Save every # evaluation(s))指定在创建 nCache 期间保存哪些采样。例如，值 2 指定仅将“每 # 个帧求值一次”(Evaluate every # frame(s))选项指定的每隔一个采样保存到您的 nCache。

有关详细信息，请参见 nCache 创建。
