创建 Alembic 缓存文件

您可以从场景中的多边形或 NURBS 对象创建 Alembic 缓存文件。将对象保存到 Alembic 缓存时，原始几何的对象层次将保留在文件中。

您可以使用“Alembic 导出”(Alembic Export)窗口中的许多高级选项(Advanced Options)来指定要保存到缓存文件中的内容。

创建 Alembic 缓存

  1. 执行下列操作之一：
    • 若要将场景中的所有对象保存到缓存文件中，请选择“缓存 > Alembic 缓存 > 将所有内容导出到 Alembic”(Cache > Alembic Cache > Export All to Alembic) > 。
    • 若要保存选定对象，请选择要缓存的对象，然后选择“缓存 > Alembic 缓存 > 将当前选择导出到 Alembic”(Cache > Alembic Cache > Export Selection to Alembic) > 。
  2. 在“Alembic 导出”(Alembic Export)窗口中，设定所需的常规选项(General Options)。
  3. （可选）若要将动态或自定义属性添加到缓存，请展开属性(Attribute)区域并执行以下操作：
    • 在“属性”(Attribute)字段中键入属性的名称，然后单击“添加”(Add)。
    • 在“通道盒”(Channel Box)中选择属性，然后单击“来自通道盒”(From Channel Box)。

    添加的属性会显示在“属性”(Attribute)下的表中。

  4. （可选）在高级选项(Advanced Options)区域中，设定所需的选项。
    注： 若要使 Alembic 缓存的几何体继承原始几何体的着色网络，请启用 UV 写入(UV Write)。请参见合并 Alembic 缓存与 Maya 对象。
  5. （可选）在回调(Callback)区域中，键入 MEL 或 Python 回调字符串。
  6. 单击“导出全部”(Export All)/“导出当前选择”(Export Selection)。
