添加或移除折痕集成员

向折痕集添加成员

  1. 在“折痕集编辑器”(Crease Set Editor)（“窗口 > 关系编辑器 > 折痕集”(Window > Relationship Editors > Crease Sets)）中，从列表中选择折痕集。
  2. 在场景中，选择要添加到折痕集的组件（多边形边、顶点）。
  3. 在“折痕集编辑器”(Crease Set Editor)中，请执行下列操作之一：
    • 单击“添加”(Add)。
    • 在“折痕集”(Crease Set)列表中的一个折痕集上单击鼠标右键并选择“添加选定组件”(Add Selected Components)。

从折痕集中移除成员

  1. 在“折痕集编辑器”(Crease Set Editor)中，选择要从列表中移除的折痕集。
  2. 在场景中，选择要从折痕集中移除的组件。
  3. 在“折痕集编辑器”(Crease Set Editor)中，请执行下列操作之一：
    • 单击“移除”(Remove)。
    • 在“折痕集”(Crease Set)列表中的一个折痕集上单击鼠标右键并选择“移除选定组件”(Remove Selected components)。
