“运动学”(Kinematics)首选项

可以在首选项(Preferences)窗口的“运动学”(Kinematics)类别中设置以下首选项。

若要返回到出厂默认设置，请在此窗口中选择“编辑 > 还原默认设置”(Edit > Restore Default Settings)。请注意，这将重置“首选项”(Preferences)窗口中每个类别的首选项。

反向运动学(Inverse Kinematics)

关节大小(Joint size)

更改骨架关节大小的显示大小。范围从 0.01 到 5.0。

IK/FK 关节大小(IK/FK joint size)

对于包含融合的反向运动学和正向运动学 (IK/FK) 动画的骨架，此选项会为 IK 和 FK 骨架中的关节和骨骼设置显示大小。该选项不影响 IK/FK 融合骨架中的关节或骨骼的显示大小。

IK 控制柄大小(IK handle size)

更改 IK 控制柄的显示大小。范围从 0.01 到 5.0。

IK/FK 融合显示(IK/FK blending display)

对于包含融合 IK/FK 动画的骨架，该选项指定哪些骨架会显示在场景视图中。

无(None)

仅 IK/FK 融合骨架会显示在场景视图中。

IK

仅 IK 骨架会显示在场景视图中。

FK

仅 FK 骨架会显示在场景视图中。

二者(Both)

FK、IK 和 IK/FK 融合骨架会显示在场景视图中。

注：

默认情况下，关节和骨骼会显示深海军蓝色。但是如果骨架同时包含 IK 和 FK，那么每个骨架会显示为一种不同的颜色（“FK”为黑色，“IK”为深红色/棕色，而“融合”会显示粉红色/洋红色）。
