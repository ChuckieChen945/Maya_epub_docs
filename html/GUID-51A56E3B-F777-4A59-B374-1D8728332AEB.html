环境球(Env Ball)

在环境（真实世界或计算机生成的环境）中，使用高反射的铬球图像可以重新创建该环境。由于铬球中的反射可提供（将近）360 度的环境视图，因此可以重新创建。

与程序纹理背景或使用曲面建模的背景相比，“环境球”(Env Ball)纹理背景的渲染速度更快捷。因此，可以将复杂的背景替换“环境球”(Env Ball)纹理替换（通过渲染该环境中铬球的图像即可），以缩短渲染时间。

在“创建”(Create)选项卡中找到此纹理。若要使用该纹理，请参见使用“环境球”(Env Ball)模拟反射。

图像(Image)

指“环境球”(Env Ball)纹理使用的纹理。若要在环境中使用高反射铬球的单个图像，请将“文件”纹理映射到“环境球”(Env Ball)纹理的“图像”(Image)属性中，然后为“文件”(File)纹理的“图像名称”(Image Name)属性指定图像文件。

倾斜度(Inclination)

指球图像围绕垂直轴的旋转运动（以弧度为单位）。有效范围为 0 到 3.142。滑块范围为 0 到 3.142。默认设置为 0。

将“倾斜度”(Inclination)值设定为用于生成图像文件（通过拍摄或渲染环境中的反射球即可生成）的摄影机的倾斜度（指摄影机的视图和环境的 YZ 平面之间的角度）。例如，如果该摄影机与 YZ 平面平行，请将“倾斜度”值设定为 0。创建球图像时，摄影机的倾斜度和零度仰角将为要使用的视图提供最佳分辨率。

仰角(Elevation)

指球图像围绕水平轴的旋转运动。有效范围和滑块范围均为 -1.571 到 1.571。默认设置为 0。

将“仰角”(Elevation)值设定为用于生成图像文件（通过拍摄或渲染环境中的反射球即可生成）的摄影机的仰角（指摄影机的视图和环境的栅格平面（即：Y 轴向上场景的 XZ 平面）之间的角度）。例如，如果摄影机的仰角为 90 度（即，它直接位于该球之上），请将“仰角”(Elevation)值设定为 1.571。如果摄影机与栅格平面平行，请将“仰角”(Elevation)值设定为 0。创建球图像时，摄影机的倾斜度和零度仰角将为要使用的视图提供最佳分辨率。

Eyespace

使“环境球”(Env Ball)纹理的“图像”(Image)文件的位置相对于窗口、而不是摄影机视图而定义。

如果“Eyespace”已启用，则即使摄影机视图已更改，“环境球”(Env Ball)纹理的“图像”(Image)文件也会自动匹配环境的“背景”(Backdrop)图像文件（如果它们最初都是从相同视点中拍摄/渲染的）。但是，反射以距离无限远的背景为基础。即：反射平面会被忽略（请参见投影几何体）。当“Eyespace”启用时，“渲染”(Rendering)速度会更快。默认设置为禁用。

反射(Reflect)

使“图像”(Image)文件成为映射到背景的反射。。

如果“反射”(Reflect)已禁用，则系统将使用实体纹理球形投影映射“图像”(Image)文件。这允许为背景图像的几何体建模，然后将背景图像映射到该几何体，例如，将阴影、照明效果或雾添加到场景中。默认设置为已启用。

投影几何体(Projection Geometry)

“投影几何体”(Projection Geometry)属性可控制“环境球”(Env Ball)纹理中的反射外观。它们可定义原始环境中天空和/或房间墙壁的位置，以便 Maya 可以计算曲面在重新创建的环境中的反射。

例如，如果原始环境由栅格平面和天空组成，则应将“天空半径”(Sky Radius)值设定为该天空的半径，将“底”(Bottom)值设定为栅格平面和反射球之间的距离。如果原始环境由房间组成，则应将“底”(Bottom)、“顶”(Top)、“左侧”(Left)、“右侧”(Right)、“前”(Front)以及“后”(Back)的值设定为每面墙与反射球之间的距离。

在某些情况下，可能需要使用与原始环境不同的值。例如，可以仅使用“天空半径”(Sky Radius)属性来定义房间的墙壁，而不使用“底”(Bottom)、“顶”(Top)、“左侧”(Left)、“右侧”(Right)、“前”(Front)和“后”(Back)的值。因为“天空半径”(Sky Radius)将定义球体，因此，模拟环境中的曲面反射将更平滑，且几乎不太可能变模糊。还可以将“天空半径”(Sky Radius)与“底”(Bottom)、“顶”(Top)、“左侧”(Left)、“右侧”(Right)、“前”(Front)和/或“后”(Back)组合使用。

天空半径(Sky Radius)

指原始环境的天空的半径。有效范围是 0 到无限。滑块范围为 0 到 20。默认值为 0。

“底”(Bottom)、“顶”(Top)、“左侧”(Left)、“右侧”(Right)、“前”(Front)和“后”(Back)

指反射球和原始环境的每面墙之间的距离。有效范围是 0 到无限。滑块范围为 0 到 20。默认值为 0。
