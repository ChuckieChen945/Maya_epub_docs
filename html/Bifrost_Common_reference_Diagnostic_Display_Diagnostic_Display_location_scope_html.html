location_scope

位置范围可视化 GeoLocation 以可视化空间查询结果，例如 get_closest_location、get_points_in_radius、get_raycast_location。

空间查询节点从某个位置执行空间搜索并查找几何体上的位置。这些查询的结果可以可视化为箭头 - 从开始搜索的位置到找到的位置。

用法示例

要将此项与 get_closest_locations 一起使用，请将它的输出位置插入到位置范围的位置。然后，在位置范围中，使用 get_closest_locations 正在使用的相同数据填充几何体和位置。

该过程与 get_raycast_locations 基本相同，但还会使用光线投射方向所用的相同数据填充位置范围的 raycast_directions。

输入

locations

要可视化的几何体位置。箭头将指向这些位置的位置。这些位置可以是一维或二维位置数组。

geometry

按位置采样的几何体。如果用于可视化 get_closest_locations，则此几何体应该是插入到其 geometry 端口的同一几何体。

positions

从其绘制箭头的位置。如果用于可视化 get_closest_locations，这些位置应该是插入到该节点的位置。

raycast_directions

此项与光线投射一起使用时，可用于绘制未与几何体接触的“缺失”光线。这些将是不同的颜色并且更小。如果用于可视化 get_raycast_locations，则应将方向插入到此处。

诊断模式

arrows

允许或禁止绘制箭头。

circles_at_origin

允许或禁止在原点（位置）绘制圆。

circles_at_sample

允许或禁止在采样处绘制圆。即使未提供用于绘制箭头的位置，也会绘制这些圆。

size

每个箭头的大小。

miss_size_factor

与 raycast_directions 一起使用时，表示“缺失光线”的箭头的相对大小。
