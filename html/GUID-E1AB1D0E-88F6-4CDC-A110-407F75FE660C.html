“帮助”(Help)菜单

帮助(Help)

搜索(Search)
打开搜索(Search)字段，以帮助您查找菜单、命令、工具或对象。
Maya 帮助(Maya Help)
在默认 Web 浏览器中加载 Maya 帮助。还可以在界面中的任意位置按 F1 键来加载帮助。

了解(Learn)

快速入门(Quick Tour)

启动界面导览，其中指出了重要元素和功能。非常适合首次使用 Maya UI 的用户。

交互式教程

访问基础知识简介、建模简介、动画简介以及照明和着色简介交互式教程。非常适合想要以更有趣、更身临其境的方式学习 Maya 和三维的用户。选择“查看所有交互式教程...”(View All Interactive Tutorials...)以在单独的浏览器中打开社区 AREA 网站并查看所有可用教程。

YouTube 学习频道(YouTube Learning Channel)
启动 Youtube 上的 Maya 教学频道，其中提供了 Autodesk 授权的受信任内容。该频道提供了不同工作流、功能和互操作性主题的各种视频。

提高(Advance)

脚本参考(Scripting Reference)
MEL 命令参考(MEL Command Reference)

在 Web 浏览器中加载 Maya 嵌入式语言 (MEL) 命令和脚本的索引。

节点和属性参考(Node and Attribute Reference)

在 Web 浏览器中加载 Maya 节点和属性的索引。

支持(Support)

服务和支持(Services and Support)
Autodesk Account

启动 Autodesk Account 网站，您可以在该网站中跟踪和管理您的 Autodesk 产品、许可、服务和权益。

发行说明更新(Release Notes Updates)

启动 Autodesk Knowledge Network 上的 Maya 发行说明页面，您可以在其中找到此版本 Maya 的最新发行说明。

支持中心(Support Center)

启动 Autodesk Knowledge Network 上的 Autodesk Maya 支持与学习概述页面，您可以在其中访问不同级别支持的链接。

反馈(Feedback)
报告问题(Report a Problem)

启动报告错误网站，您可以在其中报告 Maya 的问题。

Maya Ideas 社区(Maya Ideas Community)

启动 Maya Ideas 论坛，您可以在该论坛中提出建议和投票以改进 Maya。

加入产品研究(Join Product Research)
桌面分析计划(Desktop Analytics Program)

使用 Maya 时，会自动允许 Maya 收集桌面分析数据。首次启动应用程序时，将显示一个对话框，通知您此情况。若要禁用数据收集，请选择“桌面分析”(Desktop Analytics)并禁用对话框中的复选框。

Autodesk App Store
启动 Autodesk App Store 网站，您可以在其中购买和下载适用于 Maya 的应用程序。

关于(About)

关于 Maya(About Maya)
显示有关 Maya 副本的版权和许可信息。
