输入颜色空间列表

使用默认的 OCIO v2 颜色管理系统时，可用的输入空间取决于场景使用的配置文件。只能使用在配置中定义的空间。

配置的作者还可以使用颜色空间的 categories 属性来限制可供文件纹理、图像平面等选择的颜色空间。如果使用这些颜色空间，Maya 将包括任何具有 file-io 类别的颜色空间。

随 Maya 一起安装的配置文件名为 config.ocio，这些文件位于安装位置中的 resources/OCIO-config/ 子目录中。但是，您的项目或工作室可能使用具有不同可用空间的不同配置。

注： 旧版 SynColor 颜色管理系统提供与 OCIO v2 旧版配置相同的输入空间。

Maya 默认输入空间

Maya 默认配置用于新场景。它包括以下输入空间：

输入空间 描述
ACES2065-1 使用 ACES2065-1 颜色空间的场景线性值。
ACEScct 使用 ACEScct 颜色空间的准对数值。
ACEScg 使用 ACEScg 颜色空间的场景线性值。
AdobeRGB 在 Adobe RGB(1998) 颜色空间中编码的图像。
ARRI LogC (v3-EI800)/AlexaWideGamut 使用 Alexa Wide Gamumit 主颜色以及 Log C v3 的 EI 800 曲线编码的图像。
DCI-P3 D65 为 DCI-P3 显示编码且具有 D65 艺术白点的图像。
Gamma 2.2/Rec.709 使用 Rec.709/sRGB 主颜色和 Gamma 2.2 编码的图像。
Log film scan (ADX10) 使用 Academy Density Exchange 编码的扫描胶片板。
Raw 非颜色数据，如法线贴图、深度贴图等。
Rec.1886/Rec.709 video 使用 Rec.709 编码的视频。
RED Log3G10/REDWideGamutRGB 使用 RED Log3G10 曲线在 REDWideGamutRGB 颜色空间中编码的图像。
scene-linear DCI-P3 D65 使用 DCI-P3 主颜色和 D65 艺术白点的场景线性值。
scene-linear Rec.2020 使用适用于 UHDTV 的 Rec.2020 主颜色的场景线性值。
scene-linear Rec.709-sRGB 使用 Rec.709/sRGB 主颜色的场景线性值。
Sony SLog3/SGamut3 Sony SLog3 格式的数字摄影机片段。
sRGB 使用 sRGB 主颜色和 Gamma 编码的图像。

Maya 旧版输入空间

Maya 旧版配置用于在 Maya 2020 和先前版本中保存的场景。它包括以下输入空间：

输入空间 说明 主颜色
ACES2065-1 使用 ACES2065-1 颜色空间的场景线性值。 已转化为渲染主颜色。
ACEScg 使用 ACEScg 颜色空间的场景线性值。 已转化为渲染主颜色。
Arri LogC Arri Log C v3 格式的数字摄影机片段。 应用学院输入设备变换 (IDT) 后，转化为渲染主颜色。
camera Rec 709 使用由适用于 HD 视频摄影机的 Rec.709 指定的主颜色和 Gamma 编码的图像。请注意，几乎没有摄影机实际使用此编码。但是，这与 1.8 到 2.2 之间的 Gamma 非常类似，因此如果具有用 2.0 左右的 Gamma 编码的图像，或者由于输入和视图变换之间的差异而获取有用的类似视频的对比度增强，则它可能很有用。 在所有三个 RGB 通道上应用单个曲线后，从 Rec.709/sRGB 主颜色转化为渲染主颜色。
Gamma 1.8 Rec 709 使用 Rec.709/sRGB 主颜色和 Gamma 1.8 编码的图像。 在所有三个 RGB 通道上应用单个曲线后，从 Rec.709/sRGB 主颜色转化为渲染主颜色。
Gamma 2.2/Rec.709 使用 Rec.709/sRGB 主颜色和 Gamma 2.2 编码的图像。 在所有三个 RGB 通道上应用单个曲线后，从 Rec.709/sRGB 主颜色转化为渲染主颜色。
gamma 2.4 Rec 709 (video) 使用 Rec. 709/sRGB 主颜色709/sRGB 主颜色和 Gamma 2.4 编码的图像。这将应用由适用于 HD 视频显示器的 ITU-R BT.1886 指定的 Gamma 的反转。 在所有三个 RGB 通道上应用单个曲线后，从 Rec.709/sRGB 主颜色转化为渲染主颜色。
Log film scan (ADX) 使用 Academy Density Exchange 编码的扫描胶片板。 应用学院输入设备变换 (IDT) 后，转化为渲染主颜色。
Log-to-Lin (cineon) 使用 Cineon 样式对数编码的扫描胶片板。 不进行主颜色转化。在所有三个 RGB 通道上应用单个曲线。
Log-to-Lin (jzp) 使用 Josh Pines 或“枢轴”记录编码的扫描胶片板。 不进行主颜色转化。在所有三个 RGB 通道上应用单个曲线。
Raw 非颜色数据，如法线贴图、深度贴图等。 无转化。
scene-linear DCI-P3 使用 DCI-P3 主颜色和 DCI 白点的场景线性值。 已转化为渲染主颜色。
scene-linear Rec 2020 使用适用于 UHDTV 的 Rec.2020 主颜色的场景线性值。 已转化为渲染主颜色。
scene-linear Rec.709-sRGB 使用 Rec.709/sRGB 主颜色的场景线性值。 已转化为渲染主颜色。
Sony SLog2 Sony SLog2 格式的数字摄影机片段。 应用学院输入设备变换 (IDT) 后，转化为渲染主颜色。
sRGB 使用 sRGB 主颜色和 Gamma 编码的图像。 在所有三个 RGB 通道上应用单个曲线后，从 Rec.709/sRGB 主颜色转化为渲染主颜色。
