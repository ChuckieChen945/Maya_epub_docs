特殊环境设置 - Windows

bifcmd 使用与 Bifrost 一起打包的第三方库来运行。

将 C:\Program Files\Autodesk\Bifrost\<maya_version>\<bifrost_version>\bifrost\thirdparty\bin 添加到 PATH 环境变量。

如果复合包含 USD 节点，请将 USD 安装目录也添加到 PATH 环境变量 - 例如，如下所示：

set PATH=%PATH%;C:\Program Files\Autodesk\Bifrost\<maya_version>\<bifrost_version>\bifrost\thirdparty\bin;C:\Program Files\Autodesk\Bifrost\<maya_version>\<bifrost_version>\bifrost\packs\usd_pack\<usd_version>\thirdparty\<usd-usd_version>\bin
重要信息：

如果您安装了多个版本的 Maya 或 Bifrost，请在要运行 bifcmd 的命令窗口中设置此路径。在全局环境变量中进行设置时，会导致与其他版本的 Maya 和 Bifrost 发生冲突。
