设定输出网格颜色集

为流体输出网格生成逐顶点着色数据，并将流体转化为多边形对象后，可以将其中一个逐顶点颜色不透明度或白炽度颜色集指定为多边形的当前颜色集（如果希望能够以交互方式查看它们）。

设定输出网格颜色集

  1. 选择流体对象的输出网格。

    如果尚未设定流体以生成逐顶点着色数据，或尚未将流体转化为输出网格，请现在执行该操作。请参见设定流体输出网格特性和将流体对象转化为多边形。

  2. 从“建模”(Modeling)菜单集中，选择“网格显示 > 颜色集编辑器”(Mesh Display > Color Set Editor)。

    此时将会显示“颜色集编辑器”(Color Set Editor)，其中列出了从流体对象生成的逐顶点颜色集。

  3. 在“颜色集编辑器”(Color Set Editor)中，选择要用作多边形对象当前颜色集的逐顶点颜色集。

    例如，如果希望使用来自于流体对象的“白炽度”(Incandescence)渐变的逐顶点数据给流体输出网格上色，请选择“incandescencePV(RGBA)”。

    如果在“颜色集编辑器”(Color Set Editor)中未显示流体对象的数据，请单击“更新”(Update)。

  4. 单击“更新”(Update)。
  5. 在“属性编辑器”(Attribute Editor)中，单击“polySurfaceShape”选项卡。
  6. 在“网格组件显示”(Mesh Component Display)区域中，启用“显示颜色”(Display Colors)。

    流体输出网格现在应显示由选定颜色集所定义的颜色。在“网格控制”(Mesh Control)区域中，“当前颜色集”(Current Color Set)属性应设定为选定逐顶点数据集。
