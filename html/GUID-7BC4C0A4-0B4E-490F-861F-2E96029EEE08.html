选择角色集

选择角色集

  1. 选择“角色 > 选择角色集节点”(Character > Select Character Set Node) > charactern，其中 charactern 是角色集的默认名称。

此时会选择角色集。（请注意，角色集中的对象不处于选中状态。）

选择角色集中的对象

  1. 选择“角色 > 选择角色集成员”(Character > Select Character Set Members) > charactern，其中 charactern 是角色集的默认名称。

此时会选择构成角色集的对象，但不会选择角色集本身。

从“关系编辑器”(Relationship Editor)中选择角色集

  1. 如果在“角色编辑”(Character Editing)模式下使用“关系编辑器”(Relationship Editor)，则可以选择某个角色集，使其在工作区中当前处于选中状态。若要执行该操作，请选择角色集，然后选择“编辑 > 选择角色集”(Edit > Select Character Set)。
