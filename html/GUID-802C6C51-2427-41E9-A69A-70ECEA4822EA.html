注：
如果您使用的是 macOS，除非另有指定，否则请使用以下特殊键，而不是其 Windows 等效键：
  • Alt = Option
  • Ctrl = Control
  • Windows 键 = Command
此外，如果您在 macOS 上使用单按钮或双按钮鼠标，请确保根据需要在界面首选项(Interface Preferences)中设置“鼠标跟踪”(Mouse Tracking)选项。

在模态窗口中打开“颜色选择器”(Color Chooser)。使用该热键（而非从样例）打开时，您可以将颜色保存到“颜色历史”(Color History)或使用“滴管”(Eyedropper)选择或检查屏幕上的颜色。此外，还可以加载、保存和编辑选项板。但是，您无法在场景对象上设置颜色属性。
