get_usd_attribute_value

返回特定帧处 USD 属性的值。

输入

stage

连接 USD 后台文件。

prim_path

具有属性的基本体的路径。

attribute_name

要获取的属性的名称。

type

与 USD 属性类型对应的 Bifrost 类型，例如，float3 对应 Normal3f 或 Color3f，string 对应 Token 或 Asset，依此类推。可以单击鼠标右键设置端口类型，或简单地将值与所需类型连接起来。

frame

要获取属性值的帧。

输出

value

属性的值（如果成功）。success 为 false 时，此为该类型的默认值，例如：0 或空字符串。

success

如果已成功返回属性的值，则返回 true。
