detach_faces

将指定的 faces 与网格的其余部分断开。这些面与网格中其他面之间的边界上的点将重复，并且 faces 将被修改以引用新点。如果选中 duplicate_all_border_points，则所有边界将被复制，并且某些点可能会保持孤立状态。

例如：

       mesh
  p0─────p1─────p2
  │      │      │ 
  │  f0  │  f1  │ 
  │      │      │ 
  p3─────p4─────p5
                          `duplicate_all_border_points`
         │                             │
         ▼                             ▼

      out_mesh                       out_mesh
 p0─────p1 p6─────p2         p0─────p1 p6─────p8 p2
 │      │  │      │          │      │  │      │
 │  f0  │  │  f1  │          │  f0  │  │  f1  │
 │      │  │      │          │      │  │      │
 p3─────p4 p7─────p5         p3─────p4 p7─────p9 p5

如果 faces 设置为 f1，则：

  • 点 p1 和 p4 重复，重复项将被 f1 参照。
  • detached_points 是 p2、p5、p6 和 p7。
  • detached_border_points 是 p2、p5、p6 和 p7。
  • 如果有些面完全位于所选内容中，则 detached_border_points 将不同于 detached_points。

输入

mesh

要修改的网格。

faces

包含要分离的面索引的数组。

properties

要传递到新几何体的几何特性的名称，例如 point_color。确保包含所需的所有几何特性，包括那些以面以外的组件（例如点和面顶点）为目标的几何特性。不是几何特性的特性将始终进行传递。

  • 使用空格分隔多个几何特性名称。
  • 您可以在表达式中使用通配符 *，例如使用 point_user* 来传递所有以字符串“point_user”开头的几何特性。
  • 默认值 * 将传递所有几何特性。
  • 使用空字符串可阻止传递任何几何特性，网格模式需要的几何特性除外（point_position、face_offset 和 face_vertex）。
  • 使用 * 后跟 ! 可排除特性。例如，* !point_user* 将包括以字符串“point_user”开头的特性以外的所有特性

duplicate_all_border_points

将复制边界上的所有点。有关此用法的示例，请参见 extrude_faces。

输出

out_mesh

分离了指定面的新网格对象。

detached_points

分离的面所引用的点。

detached_border_points

沿分离边界的点。

border_face_edges

父页面: Mesh
