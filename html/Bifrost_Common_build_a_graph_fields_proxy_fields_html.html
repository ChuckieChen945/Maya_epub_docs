代理场

通过代理场，可以使用基于点的几何体和体积的特性创建场。这样，您可以执行将场值与存储在几何体上的特性（例如 point_age 或自定义权重）结合起来的计算。例如，可以随着粒子变老而增加噪波场的振幅。

有两种类型的代理场：

  • 使用 property_proxy_field 节点从基于点的几何体创建代理场。指定要使用的特性的名称（例如 point_age），然后使用 field_type 输入以指定它是标量场还是向量场。
  • 使用 voxel_proxy_field 节点从体积创建代理场。除了特性和场类型外，还必须指定如何在空间中对其值进行插值。

可以将代理场与其他场组合在一起。结果也是一个代理场。

与普通场的主要区别在于，为了对代理场进行采样，必须使用 sample_field_with_proxies 并将几何体与要使用的特性连接起来。
