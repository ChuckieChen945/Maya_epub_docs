使用 FCheck

重要： 自 Maya 2020 起，FCheck 在 macOS 上不再受支持。可以在应用程序首选项（“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)）中选择其他默认图像查看器。

“FCheck”是图像和序列查看器，可以缓存 RAM 中的图像，因此可以实时回放并拖动图像。可以在“FCheck”中使用工具执行下列操作来完成颜色校正、图像编辑和错误检查：

  • 显示凸雕、底片和图像边界
  • 调整亮度、Gamma、饱和度和偏移设置
  • 更改大小、分辨率和放大率
  • 在 RGB 通道之间切换

FCheck 随 Maya 一起安装，适用于 Windows 和 Linux。Maya 将“FCheck”用作图像预览程序来显示渲染的中间结果。可以从 Maya 或 Standalone 直接调用 FCheck。

您可以使用“FCheck”来改变预渲染图像或动画序列的视图和外观。例如，可以调整亮度、Gamma 和饱和度。可以通过以下方法访问“FCheck”的功能：

  • FCheck 命令行选项
  • FCheck 热键
  • “FCheck 鼠标”(FCheck Mouse)控件
  • FCheck 控制栏（工具栏）。
  • Windows 中的“文件”(File)菜单，用于打开和保存图像或序列。
