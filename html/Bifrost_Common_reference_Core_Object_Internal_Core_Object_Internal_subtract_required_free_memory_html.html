subtract_required_free_memory

减去不使用内存池的操作所需的可用内存。

输入

enable

如果为 true，则允许设置可用内存，否则将禁止。

id

用于在记录期间标识节点的字符串 ID。

bytes

从所需的总可用内存中减去的字节数。

bytes_per_element

从所需的总可用内存中减去的每元素（例如体素）字节数。

percentage_of_physical_memory

从所需的总可用内存中减去的总物理内存的百分比。

percentage_of_object_memory

从所需的总可用内存中减去的对象占用内存的百分比。

geometry

输入几何体
