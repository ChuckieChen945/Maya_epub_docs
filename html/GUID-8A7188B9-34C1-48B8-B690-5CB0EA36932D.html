雕刻变形器

雕刻变形器用于创建任意类型的圆化变形效果。例如，在为面部动画设置角色时，可以使用雕刻变形器控制角色的下巴、眉毛或脸颊动作。

通过雕刻变形器，可以使用一个称为雕刻球体的球形影响对象或所创建的任何 NURBS 对象来变形对象。

雕刻工具

可将下列对象用作雕刻工具：隐式球体或 NURBS 对象。

隐式球体是您操纵以创建变形效果的球形线框影响对象。隐式球体的变形效果取决于雕刻变形器的模式。

除了雕刻球体，还可以将次 NURBS 对象用作雕刻工具。将 NURBS 对象用作雕刻工具也称为使用自定义雕刻工具。请参见将 NURBS 或多边形对象用作雕刻变形器的雕刻工具。

注：

删除雕刻节点或变形对象不会删除自定义雕刻工具。

翻转模式

翻转模式下的雕刻变形器在雕刻工具的中心有一个隐式定位器。当雕刻工具靠近几何体时，将发生变形。此模式称为翻转模式，这是因为当雕刻工具的中心通过曲面时，变形曲面将翻转到雕刻工具的另一侧。

投影模式

在投影模式下，雕刻变形器会将几何体投影到雕刻工具的曲面上。投影的范围取决于雕刻变形器的“衰减距离”(Dropoff Distance)属性。

“衰减距离”(Dropoff Distance)指定直接投影到雕刻工具上的投影范围，而“最大置换”(Max Displacement)属性指定是直接投影到雕刻工具上、投影在雕刻工具内还是投影在雕刻工具外。

“最大置换”(Max Displacement)为 1.0 时，投影发生在雕刻球体的曲面上。这是使用投影模式通常要实现的效果。但是，通过更改“最大置换”(Max Displacement)，可以从雕刻球体的曲面置换投影。“最大置换”(Max Displacement)介于 0 和 1.0 之间时，投影发生在雕刻工具内。“最大置换”(Max Displacement)大于 1.0 时，投影发生在雕刻工具的曲面外部。“最大置换”(Max Displacement)为 0 时，几何体将投影到雕刻工具的中心。

“最大置换”(Max Displacement)小于 0 时，投影几何体在其通过雕刻球体的中心进行投影时会翻转。

拉伸模式

在拉伸模式下，当移动雕刻球体使其远离几何体时，几何体的受影响曲面会拉伸或凸起，以与雕刻球体保持在一起。拉伸方向从拉伸原点定位器标记的点延伸到雕刻球体的曲面。

在拉伸模式下创建雕刻变形器时，可以像处理任意对象一样选择和移动拉伸原点定位器，或使其成为雕刻工具的子对象并一起移动它们。根据要创建的效果，还可以使定位器成为动画中其他某个对象的子对象。
