纹理映射

若要将纹理应用到对象，您可以将纹理映射到对象材质上的属性。纹理所连接到的属性确定纹理是如何使用的，以及它是如何影响最终结果的。

Maya 具有许多可以映射到对象上的纹理。有关这些纹理的详细信息，请参见 2D 和 3D 纹理。

例如，如果将 Maya 的 2D 黑白“棋盘格”(Checker)纹理连接到对象的材质的颜色属性，则已应用了一个颜色贴图；棋盘格图案决定对象的哪些部分显示为黑色以及哪些部分显示为白色（或者其他颜色，如果调整了纹理的颜色属性）。

若要映射纹理，请参见映射 2D 或 3D 纹理。

如果将黑白“棋盘格”(Checker)纹理连接到透明度属性，则已应用了一个透明度贴图；棋盘格图案决定对象的哪些部分不透明以及哪些部分透明。

有关如何将纹理粘着到对象的材质的描述，请参见映射方法。

常用的纹理映射

若要映射纹理，请参见映射 2D 或 3D 纹理。

颜色贴图

通过将纹理映射到对象的材质的“颜色”(Color)属性，可创建描述对象的颜色的颜色贴图。

若要了解有关如何使用颜色的详细信息，请参见公用表面材质属性。

透明度贴图

通过将纹理映射到对象的材质的“透明度”(Transparency)属性，可创建透明度贴图，使用该贴图可以使对象的各个部分不透明、半透明或完全透明。

若要了解有关如何使用透明度的详细信息，请参见公用表面材质属性。

镜面反射贴图

通过将纹理映射到对象的材质的“镜面反射”(Specular)属性，可创建镜面反射贴图，使用该贴图可描述光亮如何显示在对象上（通过控制高光）。

若要了解有关如何使用高光的详细信息，请参见常见表面材质镜面反射着色属性。

反射贴图

通过将纹理映射到对象的材质的“反射的颜色”(Reflected Color)属性，可创建反射贴图，使用该贴图可描述对象如何反射其周围对象。

若要了解有关如何使用反射的详细信息，请参见真实反射。

凹凸贴图

通过将纹理映射到对象的材质的“凹凸”(Bump)属性，可创建凹凸贴图，使用该贴图可将曲面凹凸细节的效果添加到曲面。

若要了解有关凹凸贴图的详细信息，请参见凹凸贴图。

置换贴图

使用置换贴图可以在渲染时向曲面添加真实维度，这是一个可能会减少或消除创建复杂模型的需求的过程。

若要了解有关如何使用置换贴图的详细信息，请参见置换贴图。
