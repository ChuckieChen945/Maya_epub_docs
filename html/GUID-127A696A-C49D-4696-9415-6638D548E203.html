使用顶点法线硬化或软化多边形着色

可以通过手动编辑与多边形网格关联的顶点法线，修改单个多边形的着色。

目标 操作

显示顶点法线。

选择网格，然后选择“显示 > 多边形 > 顶点法线”(Display > Polygons > Vertex Normals)。

使选定边在着色模式中显得尖锐。

选择“网格显示 > 硬化边”(Mesh Display > Harden Edge)。

使选定边在着色模式中显得软。

选择“网格显示 > 软化边”(Mesh Display > Soften Edge)。

按角度或纹理边界硬化选定边

选择“网格显示 > 软化/硬化边”(Mesh Display > Soften/Harden Edges) > ，然后选择适当的选项。

将顶点法线锁定或解除锁定到其当前方向。

选择顶点，然后选择“网格显示 > 锁定法线”(Mesh Display > Lock Normals)或“网格显示 > 解除锁定法线”(Mesh Display > Unlock Normals)。

强制法线指向特定向量（通过输入数字值）。

选择已锁定顶点，然后选择“网格显示 > 设置顶点法线”(Mesh Display > Set Vertex Normal) > 。

禁用“锁定法线”(Lock Normals)和“解除锁定法线”(Unlock Normals)。

使用文本框输入向量的 X、Y 和 Z 旋转值，然后单击“设置法线”(Set Normal)。

强制法线指向特定向量（通过调整操纵器）。

选择顶点，然后选择“网格显示 > 顶点法线编辑工具”(Mesh Display > Vertex Normal Edit Tool)。

使用操纵器来调整法线方向。

注： “法线”(Normals)功能可创建实际更改法线的历史节点。撤消这些操作之一的正确方式是删除节点。

不要尝试通过使用相反设置再次应用这些操作之一来撤消该操作的效果。这样做可能不会获得所需的效果，只会将另一个节点添加到依存关系图。

将顶点法线设定为与面法线相同的方向

  1. 选择一个或多个顶点或顶点面组件。
  2. 选择“网格显示 > 设置为面”(Mesh Display > Set to Face) > 。
  3. 设定选项，然后单击“设置为面”(Set to Face)。

    这将具有与硬化相邻边相同的效果。

    提示：

    使用“顶点/面”(Vertex/Face)选择允许您在逐顶点逐面的基础上设定法线。

平均顶点法线

  1. 选择一个或多个顶点或顶点/面组件。
  2. 选择“网格显示 > 平均化法线”(Mesh Display > Average Normals) > 。
  3. 设定选项，然后单击“平均化法线”(Average Normals)。
    提示：

    使用“顶点/面”(Vertex/Face)选择允许您在逐顶点逐面的基础上设定法线。

可执行以下任意操作：

  • 可以平均单个顶点的顶点面法线。
  • 可以增加容差并平均同一区域带几个顶点的法线，以使该区域有展平外观。
  • 可以拾取接缝反面的顶点，并平均它们以平滑接缝。
  • 使用较小容差，可以选择沿接缝的所有顶点，同时将单独平均每个相邻顶点组。

注释

  • 使用“软化边”(Soften Edge)或“硬化边”(Harden Edge)等这些操作可能会达到它们的效果，然而，它们通常会强制显式法线方向。
  • “平均化法线”(Average Normals)在对象空间中有用，因此，如果您希望平均跨两个不同形状的法线，确保每个形状的场景层次都有相同的变换。
  • 当希望操纵顶点法线的方向，并锁定其方向时，请使用“顶点法线编辑工具”(Vertex Normal Edit Tool)（“网格显示 > 顶点法线编辑工具”(Mesh Display > Vertex Normal Edit Tool)）或“设置顶点法线”(Set Vertex Normal)（“网格显示 > 设置顶点法线”(Mesh Display > Set Vertex Normal)）。也就是说，使用这些功能设置顶点法线表示您希望覆盖 Maya 自动计算的法线，然后将其锁定在您设置的位置中。
  • 当解除锁定之前锁定的顶点法线时，Maya 将基于其默认法线计算，自动计算面的法线。
