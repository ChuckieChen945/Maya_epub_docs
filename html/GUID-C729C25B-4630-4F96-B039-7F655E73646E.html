“IK 解算器”(IK solver)节点

这些属性是所有 IK 解算器所公用的。

有关详细信息，请选择“帮助 > 节点和属性参考”(Help > Node and Attribute Reference)。

最大迭代次数(Max Iterations)

指定在计算末端效应器如何到达 IK 控制柄的过程中 IK 解算器将采用的最大迭代次数。如果在“最大迭代次数”(Max Iterations)之后，无法满足“容差”(Tolerance)，IK 解算器将停止。默认值为 2147483647。较大的值（如默认值）意味着满足“容差”(Tolerance)时 IK 解算器通常将停止。

容差(Tolerance)

指定在计算末端效应器到达 IK 控制柄的过程中 IK 解算器所找到的精度。IK 解算器在满足“容差”(Tolerance)之后，将停止计算。默认值为 0.000。

ikSystem

仅当选中某个 IK 解算器节点时，才会显示该节点。

IK 系统节点组织 Maya 中的所有默认 IK 解算器。IK 系统控制在系统中使用 IK 解算器的所有 IK 控制柄是否将捕捉到其末端效应器，IK 解算器是否处于活动状态，以及 Maya 对解算器求值的顺序。

ikSystem

“ikSystem”窗口会列出 IK 系统中的 IK 解算器。默认情况下，可用的 IK 解算器有“旋转平面解算器”(Rotate-Plane Solver)（也称为 ikRPsolver，用于 IK 旋转平面控制柄）、“单链解算器”(Single-Chain Solver)（也称为 ikSCsolver，用于 IK 单链控制柄）和“ikSplineSolver”（用于 IK 样条线控制柄）。列出 IK 解算器的顺序就是 Maya 对解算器求值的顺序。

全局捕捉(Global Snap)

指定使用 IK 系统的任何解算器的所有 IK 控制柄是否都将捕捉回到其末关节的位置。启用“全局捕捉”(Global Snap)会禁用每个 IK 控制柄的“捕捉启用”(Snap Enable)属性。默认设置为启用。

全局解算(Global Solve)

指定使用 IK 系统的任何解算器的所有 IK 控制柄是否都处于活动状态。如果禁用该选项，则仅可以使用正向运动学 (FK) 造型形成由 IK 控制柄控制的关节链。默认设置为启用。
