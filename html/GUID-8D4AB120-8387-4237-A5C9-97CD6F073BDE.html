显示或隐藏工具架

默认情况下，在经典 Maya 工作区中加载 Maya 工具架，但您可以将其关闭并重新定位。

显示 Maya 工具架

  1. 选择“窗口 > UI 元素”(Windows > UI Elements)，然后激活“工具架”(Shelf)。
  2. 如果工具架显示在单独的窗口中，并且您希望将其还原到视口顶部的默认位置，请转到工作区(Workspace)字段，然后从列表中选择重置当前工作区(Reset Current Workspace)。

隐藏 Maya 工具架

  1. 选择“窗口 > UI 元素”(Windows > UI Elements)，然后禁用“工具架”(Shelf)。
  2. 您还可以撕下工具架，具体方法是：拖动工具架左侧边的双线，使其成为一个单独的窗口，然后关闭该窗口。
