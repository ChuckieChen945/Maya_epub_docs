Windows 菜单

主页

返回应用程序主页中心，快速访问最近打开的文件，以及“入门”、“学习”、“新特性”和“社区”等内容。也可以通过 Maya UI 左上角的主页图标或 Alt + Home 热键执行此操作。

工作区

从列表中选择预定义的“工厂”工作区或自定义工作区。工作区是各种窗口和面板以及其他界面选项的有序排列。请参见工作区。

预定义的工作区包括默认的“Maya 经典”(Maya Classic)工作区，以及另外几个专用于执行各组任务的工作区。

此外，还有以下选项可用：
  • 将“<工作区>”重置为出厂默认值”(Reset "<Workspace>" to Factory Default)/将“<工作区>”重置为之前保存的版本”(Reset "<Workspace>" to Previously Saved)：将丢弃您对当前工作区所做的任何更改。预定义工作区将还原到其默认配置，而自定义工作区将还原到上次保存的配置。
  • 将当前工作区另存为(Save Current Workspace As)：将当前配置另存为新的自定义工作区。如果指定现有自定义工作区的名称，则将覆盖旧配置。
  • 导入工作区文件(Import Workspace Files)：导入一个或多个自定义工作区定义文件。
  • 删除当前工作区(Delete Current Workspace)：将从首选项中移除当前自定义工作区。不能删除预定义工作区。
  • 禁用停靠/取消停靠(Disable Docking/Undocking)：防止停靠和取消停靠窗口和面板。它不会阻止对工作区进行其他更改，例如，打开、关闭或移动窗口以及收拢面板等。

UI 元素(UI Elements)

此子菜单中的菜单项可以控制各种界面用户面板和元素的可见性，包括“状态行”、“工具架”、“时间滑块”等等。

    该子菜单中的选项包括：

  • 隐藏所有 UI 元素(Hide All UI Elements) - 隐藏所有 UI 部分，只显示面板和菜单。
  • 显示所有 UI 元素(Show All UI Elements) - 显示所有 UI 部分。
  • 还原 UI 元素(Restore UI Elements) - 将 UI 部分的可见性还原为选择“隐藏所有 UI 元素”(Hide All UI Elements)之前的状态。如果某些部分在选择“隐藏所有 UI 元素”(Hide All UI Elements)之前已经隐藏，则它们仍将处于隐藏状态。

Bifrost 浏览器

打开 Bifrost 浏览器以导入 Bifrost 图表或打开 Bifrost 示例场景。请参见适用于 Maya 的 Bifrost Extension。

输出窗口(Output Window)
提供各种诊断和状态更新。请参见输出窗口(Output Window)。
最小化应用程序(Minimize Application)
在操作系统中将 Maya 最小化。
提升主窗口(Raise Main Window)

提升应用程序窗口(Raise Application Windows)
将所有编辑器窗口都置于主窗口之前。
