Cryptomatte
请参见《Arnold 用户手册》中的 Cryptomatte

Arnold 支持 Cryptomatte，这是由 Jonah Friedman 开发的一款 ID 蒙版创建工具。凭借对运动模糊、透明度和景深的支持，它可以自动创建 ID 蒙版。可以使用名称、对象名称空间和材质名称组织场景中的 ID 蒙版。

在“AOV”(AOVs)下可获得以下 Cryptomatte AOV。

  • crypto_asset：为所有资源创建相同的蒙版。
  • crypto_material：基于指定的着色器创建蒙版。
  • crypto_object：基于对象名称创建蒙版。

自动为您设置 Cryptomatte 着色器。

  • 在渲染设置(Render Settings)窗口的 AOV(AOVs) 选项卡中的“AOV 浏览器”(AOV Browser)下，创建您所需的 Cryptomatte AOV。选择一个 AOV，并在属性编辑器中查看它。它应该显示在自定义 AOV 下的“着色器”(Shader)中。

  • 渲染场景。Arnold 渲染视图(Arnold RenderView)窗口中现在应该会提供 Cryptomatte AOV。
