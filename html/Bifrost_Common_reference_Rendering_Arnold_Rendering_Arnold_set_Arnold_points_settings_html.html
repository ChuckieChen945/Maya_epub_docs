set_Arnold_points_settings

设置 Arnold 特定的 points（在 Arnold 中：particles）设置，以便在 Arnold 中作为 Arnold 原生设置应用。这些设置不影响视口，仅影响 Arnold 渲染。

这些设置与 Arnold 中的对应选项匹配。有关详细信息，请参见 Arnold 用户手册中的 particles 相关信息。

用法

它也可以与 set_Arnold_geo_settings 结合使用，后者用于设置对 Arnold 几何体通用的设置。

Arnold 点

points_mode_override

设置 Arnold 的粒子 mode，其控制点渲染为圆盘、球体还是四边形。这与 Bifrost points 的工作方式不同。

Default 使用 Bifrost 设置的默认解释。对于 Arnold 粒子不支持的几何体，这些点会变为几何体的 Arnold ginstance。

min_pixel_width

设置屏幕上点的最小大小。有关 Arnold 中 min_pixel_width 的行为，请参见 Arnold points 节点参考。
