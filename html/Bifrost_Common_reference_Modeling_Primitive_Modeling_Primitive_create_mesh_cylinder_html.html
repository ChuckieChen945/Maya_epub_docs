create_mesh_cylinder

使用法线和 UV 创建圆柱体网格。

输入

radius

圆柱体的半径。

height

圆柱体的高度。

axis_segments

围绕圆柱体圆周的分段数。

height_segments

沿圆柱体长度的分段数。

top_cap

是否为圆柱体创建顶部封口。

bottom_cap

是否为圆柱体创建底部封口。

create_UVs

是否为网格生成 UV 坐标。

position

创建网格的位置。

up_axis

网格将沿其定向的向量。

base_to_pivot

如果启用，则会在 position 之上而不是以它为中心创建网格。

标记(Tags)

create_tags

选中后，标记将添加到输出网格中。

top_tag、bottom_tag、sides_tag

圆柱体各个部分的标记名称。如果保留为空，则不会输出该标记。
