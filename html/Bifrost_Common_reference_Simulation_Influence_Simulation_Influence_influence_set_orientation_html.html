influence_set_orientation

可以使用 float3 或 VectorField 在模拟上设置 point_orientation 特性，以定义点旋转。point_orientation 在内部保持为四元数，但是此节点仅允许使用 Euler 角度或旋转向量进行设置，然后在内部将其转化为四元数。使用 influence_set_property 或 modifier_influence，可以以 float4 四元数形式设置 point_orientation。VectorField 是 float3，不直接支持四元数，因此它们必须采用旋转向量或 Euler 格式。可以使用 VectorField 代理获取当前 point_orientation，值将自动转化为旋转向量。请注意，旋转向量类似于轴角度，但旋转向量是轴与角度的乘积，因此是 float3 而不是 float4。

要使用此节点，请将其输出插入模拟节点（如 simulate_aero、simulate_particles 或 simulate_mpm）的 influences 输入。

输入

influence

可以选择将影响场 out_influence 连接到此端口。例如，可以连接 mask_influence 节点以遮罩此影响的效果。可以将任意数量的影响场呈菊花链状连接在一起，以构建复杂的场。

orientation

这是用于替换 point_orientation 的值。可以从 VectorField 进行连接，以随空间变化值。

rotation_type

这用于确定传入的旋转类型。point_orientation 特性在内部定义为四元数。将从 Euler 角度或旋转向量转化为四元数。

  • Euler：绕每个轴的旋转角度（以度为单位）。当前始终使用 XYZ 旋转顺序。
  • RotationVector：采用旋转向量格式，其中向量方向表示旋转轴，向量长度表示旋转量。

输出

out_influence

可以连接到模拟节点或其他影响节点的输出影响对象。
