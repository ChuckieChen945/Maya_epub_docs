define_usd_variant_set

定义 USD 变量集。将输出连接到 define_usd_prim 或类似节点的 variant_set_definitions 端口。对于每个变量，可以创建另一个具有相应 variant_set_name 和特定 variant name 的基本体定义，然后将其连接到同一 define_usd_prim 或类似节点的 children 端口。

输入

variant_set_name

变量集的名称。

selection

将作为默认选择的变量的名称。

输出

variant_set_definition

变量集定义。

父页面: VariantSet
