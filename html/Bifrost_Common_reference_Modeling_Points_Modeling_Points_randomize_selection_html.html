randomize_selection

随机化放置在实例对象的点上的形状。

输入

points

要为其点指定随机形状 ID 的实例对象。

weights

将为点指定随机形状的概率。

  • 小于等于 0 的值表示永不随机化点形状。
  • 0 到 1 范围内的值表示随机化点形状的几率。例如，0.5 表示随机化点形状的概率为 50%。
  • 大于等于 1.0 的值表示始终随机化点形状。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

随机化

min、max

要生成的最小和最大随机形状 ID。随机值在 [min, max) 范围内，表示大于或等于 min 但小于 max。

seed

随机数生成器的种子。

distribution

此 FCurve 可控制随机 ID 的分布。默认形状没有偏移，但您可以编辑曲线以使用较高或较低的 ID。如果 ID 的顺序有意义（例如，如果它们按大小顺序排列），这非常有用。要进一步控制分布，请考虑使用 randomize_selection_by_probabilities，而不是此复合。

特性

property

可确定要实例化的形状的特性的名称。如果图表未使用默认 point_instance_id，则可以更改此项。特性的类型必须为 array<long>。

component

property 端口的目标组件。此项通常可以保持不变。

输出

out_points

具有随机化实例选择的实例对象。

instance_ids

在几何体上设置的 point_instance_id 值。
