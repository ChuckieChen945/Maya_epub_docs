将引用编辑导出到脱机文件

可以将场景中的引用编辑（例如对几何体或新节点连接的更改）导出到 .editMB 或 .editMA 引用编辑文件。然后，可将该编辑文件应用于其他对象以对其调用相同的更改。

例如，如果从基础循环行走动画开始，可以将几个不同的调整（如阔步走、拖脚走和迈进）导出到不同的文件中。然后可以将这些编辑应用于其他角色。

可以导出场景中某个资产、某个被引用文件或一组选定节点上的编辑内容。导出引用编辑之前，可以查看它们以确保文件中包含正确的编辑。请参见列出引用编辑。

注：

悬挂连接（到未导出节点的连接）和引用编辑将保存到具有占位符名称空间的编辑文件。应用这些编辑时，Maya 会将占位符替换为应用该文件的引用的名称空间，或者使用“:”代表导入到主场景。

不会导出默认节点。例如，如果更改了默认着色器 (standardSurface1)，然后导出了编辑，则更改的 standardSurface1 不会导出。

从选定节点导出引用编辑

从一组选定节点导出引用编辑

  1. 按住 Shift 键的同时在场景中选择要从中导出编辑内容的节点。
  2. 选择“文件 > 导出到脱机文件”(File > Export to Offline File)。

    将显示文件浏览器。

  3. 导航到要存储编辑内容的文件夹并输入文件的名称。默认情况下，Maya 将这些文件放置在当前项目中名为 scenes\edits 的子文件夹中。
  4. 单击“导出到脱机文件”(Export to Offline File)。

    Maya 会创建包含引用编辑的指定文件，扩展名为 .editMA 或 .editMB。不能在 Maya 中打开这些文件，但可以将其指定给其他文件。有关详细信息，请参见指定脱机文件。

还可以导出特定被引用文件的编辑内容。

导出被引用文件上的编辑内容

导出被引用文件上的编辑内容

  1. 通过执行下列操作之一来访问导出到脱机文件选项(Export to Offline File Options)：
    • 在“大纲视图”(Outliner)中，在要从中导出编辑内容的被引用文件上单击鼠标右键，然后从显示的菜单中选择“导出到脱机文件”(Export to Offline File) > 。
    • 在“引用编辑器”(Reference Editor)（“文件 > 引用编辑器”(File > Reference Editor)）中，选择要从中导出编辑内容的被引用文件，然后选择“文件 > 导出到脱机文件”(File > Export to Offline File) > 。
  2. 在“导出到脱机文件选项”(Export to Offline File Options)中，设置导出选项，然后单击“导出到脱机文件”(Export to Offline File)。
  3. 导航到要存储编辑内容的文件夹并输入文件的名称。默认情况下，Maya 将这些文件放置在当前项目中名为 scenes\edits 的子文件夹中。
  4. 单击“导出到脱机文件”(Export to Offline File)。
