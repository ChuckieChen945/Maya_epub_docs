wave_set_foam

这将在输入几何体位置的位置对输入泡沫模拟进行采样，并在几何体上设置泡沫特性。不同的输入泡沫将汇总在一起。

输入

geometry

这是要对泡沫进行采样的网格或点对象。

foam

这是一组泡沫对象，通常是 simulate_foam 节点的输出。不同的输入泡沫将汇总在一起。

Sample Options

sampler

这是在几何体位置对泡沫图像进行采样时在泡沫图像数组上使用的插值方法。

  • Nearest：最靠近点的像素：无插值

  • Linear：在最近点之间线性插值

  • Quadratic：更平滑地插值

  • Bspline：最平滑的插值

magnitude

泡沫量的比例。这也可能是标量场，例如 fractal_noise_field。

offset

添加到泡沫。这也可能是标量场，例如 fractal_noise_field。

输出

out_geometry

这是在其上设置了泡沫几何特性的输入几何体。
