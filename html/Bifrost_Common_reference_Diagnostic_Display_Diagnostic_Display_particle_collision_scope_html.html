particle_collision_scope

可视化粒子对象的碰撞特性，以帮助诊断问题。要使此节点正常工作，必须在 particle_solver_settings 节点中至少启用 store_surface_collisions 或 store_self_collisions 之一。

单击节点下方的 D 标志以切换视口中的诊断显示。请注意，为显示相关信息，也必须在所有父复合中启用此标志。

输入

particles

要在其上可视化碰撞的输入粒子对象。

显示选项

碰撞位置基于粒子在碰撞时的确切位置，如果粒子在碰撞后的剩余帧内移动，则可能与粒子的当前位置不同。

display_collide_position

在曲面碰撞的位置绘制彩色点。请注意，必须在 particle_solver_settings 节点上启用 store_surface_collisions。

display_self_collide_position

在粒子与其他粒子碰撞的位置绘制彩色点。请注意，必须在 particle_solver_settings 节点上启用 store_self_collisions。

display_surface_normal

绘制箭头，显示碰撞点处的法线。这始终是曲面法线，而不是双面碰撞使用的潜在翻转法线。请注意，必须在 particle_solver_settings 节点上启用 store_surface_collisions。

display_point_id

显示碰撞和自碰撞粒子的 ID。请注意，必须在 particle_solver_settings 节点中启用 store_surface_collisions 和/或 store_self_collisions。

范围设置

这些输入都是解释的 auto 端口，允许您连接数组或场以及单个值。

point_size

诊断点的显示大小。例如，要匹配粒子大小，可以为 point_size 创建标量 property_proxy_field 场，并将其连接到此端口。

arrow_size

诊断箭头的显示长度。例如，要匹配粒子速度，可以为 point_velocity 创建向量 property_proxy_field，将其连接到 length，然后将结果连接到此端口。

arrow_color

用于箭头的颜色。

point_color

用于点的颜色。默认情况下，曲面碰撞和自碰撞的颜色相同。要使用不同的颜色，请为 point_self_collision 创建标量 property_proxy_field，将其连接到 switch_fields 节点的 condition 端口，将结果连接到此端口，然后使用 switch_fields 节点在两个 vector_field 节点（为 true 和 false 情况定义不同的颜色）之间切换。

输出

diagnostic_geometry

由诊断显示着色和使用的完整几何体数组。此对象数组可以包含用于碰撞点、自碰撞点、法线箭头和点 ID 的单独对象。

collide_index

与曲面碰撞的粒子的索引数组。您可以将此选项与 get_from_array 一起使用，以访问各种粒子特性数组的碰撞粒子值。

self_collide_index

与其他粒子碰撞的粒子的索引数组。您可以将此选项与 get_from_array 一起使用，以访问各种粒子特性数组的自碰撞粒子值。
