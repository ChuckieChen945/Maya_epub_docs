get_face_edge_face_vertices

获取对应于指定 face_edge 的起点和终点的面顶点索引。请注意，面边是有方向的。起点是面顶点索引较小的面顶点，而终点是面顶点索引较大的面顶点。请注意，如果起始面顶点是面中的最后一个面顶点，则结束面顶点将是面中的第一个面顶点。（即，索引将折返。）

例如：

 ┌───────────────┐
 │fv3   s2    fv2│
 │             ▲ │
 │             │ │
 │s3    f0   s1│ │
 │             │ │
 │             │ │
 │fv0   s0    fv1│
 └───────────────┘

对于指代面 f、侧边 s1 的面边，first_face_vertex 为 fv1，second_face_vertex 为 fv2。

输入

face_edge

要查询的面边。

face_offset

相应网格的面偏移。

输出

first_face_vertex

面边的第一个面顶点。

second_face_vertex

面边的第二个面顶点。
