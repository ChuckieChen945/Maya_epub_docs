find_mesh_islands

查找网格的孤岛。孤岛是一组面，该组中的每个面都与同一组中的其他面相邻。一个面不能是多个孤岛的一部分。

例如，以下网格具有两个面和一个孤岛：

  ┌────┬────┐
  │    │    │
  │    │    │
  └────┴────┘

再举一个示例，以下网格具有三个面和两个孤岛：

  ┌────┬────┐ ┌────┐
  │    │    │ │    │
  │    │    │ │    │
  └────┴────┘ └────┘

输入

mesh

输入网格。

输出

face_islands

面孤岛的二维数组。外部维度是面孤岛的数组。内部维度是特定孤岛的面索引。

face_island_count

在输入 mesh 中找到的面孤岛数。
