FCheck 中的图像调整键

重要： 自 Maya 2020 起，FCheck 在 macOS 上不再受支持。可以在应用程序首选项（“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)）中选择其他默认图像查看器。

使用以下键可以更改图像的亮度、Gamma、饱和度或偏移，也可以更改通道和操纵颜色。

请注意，只有将图像写入磁盘后才能保留在“FCheck”中执行的图像修改和调整。有关详细信息，请参见保存图像或序列。

Windows/Linux

操作
  重置所有“FCheck 信息”(FCheck Info)窗口选项。
d 将图像重置为更改亮度、Gamma 和饱和度之前图像所处的状态。
I 显示反转凹凸；类似于按 N 键，但会显示负增益。
L 根据加权 RGB，控制图像亮度。若要在线绘图和点之间切换，请在按住 Shift 键的同时按 L 键。
N 显示法线凹凸。通过按 和 键增加深度。
U 更新或重新加载图像。
W 保存当前图像。
Shift+w 保存整个动画。
= 将亮度增加 1%。
Shift+= 将亮度增加 10%。
Alt+= 将亮度增加 50%。
- (键盘，非数字键盘) 将亮度降低 1%。
Shift+-（键盘，非数字键盘） 将亮度降低 10%。
Alt+-（键盘，非数字键盘） 将亮度降低 50%。
Ctrl+=/- 按 Ctrl - 或 Ctrl + 会将图像还原到原始亮度值。
]（右方括号） 将 Gamma 增加 1%。
Shift+] 将 Gamma 增加 10%。
Alt+] 将 Gamma 增加 50%。
[（左方括号） 将 Gamma 降低 1%。
Shift+[ 将 Gamma 降低 10%。
Alt+[ 将 Gamma 降低 50%。
Ctrl+] 或 [ 按 Ctrl ] 或 Ctrl [ 会将图像还原到原始 Gamma 值。
/ 将偏移增加 1%。
Shift + / 将偏移增加 10%。
Alt+/ 将偏移增加 50%。
\ 将偏移降低 1%。
Shift+\ 将偏移降低 10%。
Alt+\ 将偏移降低 50%。
Ctrl+\ 或 / 将图像还原到原始偏移值。
‘ 将饱和度增加 1%。
Shift+‘ 将饱和度增加 10%。
Alt+‘ 将饱和度增加 50%。
; 将饱和度降低 1%。
Shift+; 将饱和度降低 10%。
Alt+; 将饱和度降低 50%。
Ctrl+‘/; 按 Ctrl ‘ 或 Ctrl ; 会将图像还原到原始饱和度值。
F1 切换为红色位平面通道控制。
F2 切换为绿色位平面通道控制。
F3 切换为蓝色位平面通道控制。
F4 切换为所有颜色 (RGB) 位平面通道控制。这是“FCheck”中的默认通道控制。
Shift+F1 切换为青色位平面通道控制。
Shift+F2 切换为洋红位平面通道控制。
Shift+F3 切换为黄色位平面通道控制。
F11 允许覆盖图像。
  使用数字键盘键进行以下操作
/ 降低添加到凹凸图像的颜色百分比。
* 增加添加到凹凸图像的颜色百分比。
Ctrl + / 通过将值重置为零，撤消所做的任何颜色更改。
Ctrl+* 通过将值重置为 100%，撤消所做的任何颜色更改。
