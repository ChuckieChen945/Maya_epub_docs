编辑粒子碰撞事件

可以使用“粒子碰撞事件编辑器”(Particle Collision Event Editor)来更改粒子碰撞事件。

编辑粒子碰撞事件

  1. 从 FX 菜单集中，选择“nParticle > 粒子碰撞事件编辑器”(nParticles > Particle Collision Event Editor)。

    此时将出现“粒子碰撞事件编辑器”(Particle Collision Event Editor)。

  2. 从“粒子碰撞事件编辑器”(Particle Collision Event Editor)的“对象”(Objects)面板中，选择与要编辑的事件关联的粒子对象。

    与选定粒子对象关联的事件将显示在“事件”(Events)面板中

  3. 在“事件”(Events)面板中，选择要编辑的事件。
  4. 对碰撞事件进行相应的更改。

    有关碰撞事件选项的信息，请参见粒子碰撞事件编辑器。
