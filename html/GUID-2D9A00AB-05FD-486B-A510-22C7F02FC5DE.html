非法节点名称疑难解答

合法节点名称以 a-z 或 A-Z 以及下划线中的任意字符开头，并且后跟 a-z、A-Z、下划线或数字组成的字符序列。

以前，Maya 将在创建和重命名节点时接受非法名称；例如，ns:0abc。从 Maya 2015 开始，非法节点名称将不再有效。每次使用非法节点名称打开、引用或导入场景时，将显示一个对话框，通知您更新场景并更正非法节点名称。

若要对当前 Maya 会话禁用该对话框，请启用该复选框。若要永久禁用该对话框，请将 MAYA_ENABLE_LEGACY_INCORRECT_NODE_NAMES 环境变量设定为 1。

若要列出所有包含非法名称的节点，请在 MEL 或 Python 中的“脚本编辑器”(Script Editor)中执行命令 listNodesWithIncorrectNames。有关详细信息，请参见“技术文档 > MEL 命令和 Python 命令”。

若要重命名节点，请使用大纲视图或属性编辑器。
