缩放关键帧选项(Scale Keys Options)

注： 本节介绍了“曲线图编辑器”(Graph Editor)和“摄影表”(Dope Sheet)的“编辑”(Edit)菜单中的“缩放关键帧选项”(Scale Keys Options)。有关“关键帧”(Key)菜单中的“缩放关键帧选项”(Scale Keys Options)的信息，请参见缩放关键帧选项(Scale Keys Options)。
注： 可以使用“动画”(Animation)首选项（在“设置”(Settings)下）的“曲线图编辑器”(Graph Editor)区域中的“关键帧大小”(Key Size)选项，更改“曲线图编辑器”(Graph Editor)图表视图中显示的关键帧的大小。
时间范围(Time Range)

指定关键帧集的动画曲线的时间范围。从下列选项中进行选择：

全部(All)

将时间范围设置为无限制。

开始/结束(Start/End)

设置从“开始时间”(Start Time)到“结束时间”(End Time)的时间范围。

时间滑块(Time Slider)

设置由“时间滑块”(Time Slider)的“播放开始时间”(Playback Start Time)和“播放结束时间”(Playback End Time)定义的时间范围。

开始时间(Start Time)

设置时间范围的开始时间。“开始时间”(Start Time)仅在启用“开始/结束”(Start/End)时可用。

结束时间(End Time)

设置时间范围的结束时间。“结束时间”(End Time)仅在启用“开始/结束”(Start/End)时可用。

方法(Method)

选择下列选项之一：

缩放/枢轴(Scale/Pivot)

对选定范围内的时间和值应用倍增。这还将设置枢轴点，各轴上从这里开始进行缩放。

启用该选项后，将缩放动画，而不影响位于指定范围前后的动画的曲线。与缩放区域相邻的关键帧和动画曲线分段会同时偏移已压缩或展开的缩放范围量，并保留其形状。

开始/结束(Start/End)

压缩或展开选定范围以适配“新开始/结束时间”(New Start/End Times)字段指定的范围。

仅缩放指定关键帧(Only scale specified keys)

启用该选项后，“缩放关键帧”(Scale Keys)仅影响选定关键帧。禁用该选项后，“缩放关键帧”(Scale Keys)影响所有关键帧。

时间缩放/枢轴(Time scale/pivot)

设置时间倍增。第一个字段设置关键帧集的所有时间的倍增值。第二个字段设置从其开始执行缩放的枢轴点。

提示： 您可以使用“时间缩放/枢轴”(Time Scale/Pivot)选项将一定范围的动画的持续时间减半。
值缩放/枢轴(Value scale/pivot)

设置倍增值。第一个字段设置关键帧集的所有值的倍增值。第二个字段设置从其开始执行缩放的枢轴点。

新开始/结束时间(New start/end times)

设置压缩或展开关键帧集的时间范围。“新开始/结束时间”(New Start/End Times)仅在启用“开始/结束”(Start/End)时可用。
