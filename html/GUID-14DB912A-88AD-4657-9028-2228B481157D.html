保存和加载 XGen 文件

您可以保存 XGen 集合和描述文件，然后使用“导出”(Export)和“导入”(Import)命令将其加载到其他场景或集合。可以从“建模”(Modeling)菜单集中的“生成”(Generate)菜单或 XGen 编辑器中的“文件”(File)菜单访问这些命令。默认情况下，Maya 会将导出的集合和描述保存到用户目录中的 xgen 文件夹中。例如：

C:/Users/user/Documents/xgen/collection.xgen

可以保存和加载以下集合和描述文件：

  • 集合文件 (.xgen) 包含关联的描述和面片绑定。

    请参见 XGen 集合。

  • 描述文件 (.xdsc) 包含指向在源描述中用到的 Ptex、点 (.xuv) 和 XGen 修改器文件的位置的指针。
    注： 在加载导出的描述后，您可能需要浏览到源描述中所用到的任何 Ptex、点 (.xuv) 和修改器文件的源位置。

    您可以选择要将描述添加到其中的集合。如果没有关联的集合，Maya 会创建一个。请参见 XGen 描述？。

    重要： 当您将描述导入到当前场景时，目标曲面几何体与原始曲面几何体必须具有相同的拓扑。否则，任何与描述关联的 Ptex 贴图将不起作用。
  • 必须通过从 XGen 编辑器中选择“文件 > 导出修饰”(File > Export Grooming)和“文件 > 导入修饰”(File > Import Grooming)分别导出和导入与可修饰样条线关联的文件。请在使用“导出修饰”(Export Grooming)或“导入修饰”(Import Grooming)前导出和导入描述或集合。请参见 XGen 编辑器的“文件”菜单。

您可以使用 Python 的导出和导入命令来保存和加载 XGen 修改器 (.xgfx) 文件。请参见使用 Python 命令导出和导入 XGen 文件。当您将修改器加载到描述中时，它必须生成与源描述相同的基本体类型。您可以在 Maya 或 Python 会话内执行该命令。
