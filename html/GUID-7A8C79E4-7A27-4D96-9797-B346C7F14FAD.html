将对象发送到 Flame

您可以使用“文件 > 发送到 Flame”(File > Send to Flame)从 Maya 导出作品。

“发送到”(Send to)选项用于支持常见的迭代往返传输工作流，从而实现制作过程从内容创作应用程序中的场景开始，将内容导出到 Maya 以添加细节，然后合并回原始场景中。

导出到 Flame

  1. 选择“文件 > 发送到 Flame”(File > Send To Flame)，然后选择所需的导出子选项：
    • 发送选定对象作为新场景(Send Selected as New Scene)

      仅将选定对象发送到 Flame。（如果未选择任何对象，则 Maya 将发送整个场景。） 选定对象将导入到活动 Batch 或 Batch FX 中的新 Action 节点中。

    • 更新当前场景(Update Current Scene)

      在 Flame 中更新当前 Action 示意图中的匹配几何体。

    • 将选定对象添加到当前场景(Add Selected to Current Scene)

      将选定对象添加到当前 Action 示意图中。

    Maya 将启动 Flame，并且场景将导入到上次使用的 Batch 示意图中的新 Action 节点。
    • 如果 Flame 已在运行，则场景将导入到活动 Action 示意图中。
    • 如果未显示 Action 示意图，则场景将导入到最近打开的 Batch 示意图中的新 Action 节点。
