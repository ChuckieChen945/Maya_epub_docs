align_matrix_chain

变换选定范围的变换矩阵，使其朝指定的目标向量对齐。

输入

transforms

要解算的变换的 array<double4x4>，解释为从头至尾的连续链。

enable

如果启用此项，将处理此节点。否则，将跳过任何计算并按原样输出输入 transforms。

start_index

定义变换链处理操作的起始数组位置。

end_index

定义变换链处理操作的结束数组位置。支持负值。例如，-1 表示处理到链中的最后一个变换，-2 表示处理到倒数第二个变换，依此类推。

from_vector

起始向量，表示要旋转的向量。这通常是从开始到结束变换位置的预解算方向。

to_vector

要使选定范围的变换矩阵朝其对齐的目标向量。

rotate_target_vector

如果启用此项，则旋转输入 to_vector 目标。

rotation_axis

要用于旋转输入 to_vector 的旋转轴。

rotation_radians

围绕输入 rotation_axis 的旋转量。

输出

out_transforms

解算的 transforms。
