nCloth 菜单

创建(Create)

创建被动碰撞对象
使选定对象成为被动碰撞对象，并创建相应的 nRigid 节点。被动对象是参与 Maya Nucleus 系统碰撞的非模拟 nCloth。

选择“nCloth > 创建被动碰撞对象”(nCloth > Create Passive Collider) > 以设置使碰撞选项。

创建 nCloth(Create nCloth)
使选定多边形网格成为 nCloth 对象。

选择“nCloth > 创建 nCloth”(nCloth > Create nCloth) > 以设置创建 nCloth 选项。

获取 nCloth 示例(Get nCloth Example)

将内容浏览器打开到“nCloth 示例”(nCloth Examples)文件夹。“nCloth 示例”(nCloth examples)是 Maya 的 .ma 文件，可用于快速启动您自己的 nCloth 项目。使用鼠标中键将文件从“内容浏览器”(Content Browser)拖动到场景中进行加载。

编辑(Edit)

删除历史(Delete History)
从当前 nCloth 对象中删除指定类型的构建历史。

选择“nCloth > 删除历史”(nCloth > Delete History) > 以设置删除历史选项。

显示当前网格(Display Current Mesh)
在场景视图中，显示选定 nCloth 对象的当前网格或输出网格。这是显示模拟效果的网格。
显示输入网格(Display Input Mesh)
在场景视图中，显示选定 nCloth 对象的原始多边形网格或输入多边形网格。
移除 nCloth
将选定的 nCloth 或被动对象转化为规则多边形网格对象。如果 nCloth 对象的 Nucleus 解算器与其他 Nucleus 对象（如 nParticle 对象）无关联，则还将移除 nCloth 的 Nucleus 节点。

特性(Properties)

转化 nCloth 输出空间(Convert nCloth Output Space)
可用于更改

选择“局部空间 -> 世界空间”(Local Space -> World Space)以将 nCloth 输出网格设置为世界空间输出。转化为世界空间时，只有 nCloth 的输出网格受 Maya Nucleus 解算器的影响。每个输入和输出网格都将拥有自己的变换节点。输入网格使用现有变换节点，而输出网格设置为新变换节点的子对象。

选择“世界空间 -> 局部空间”(World Space -> Local Space)以将 nCloth 输入和输出网格设置为局部空间输出。转化为局部空间时，nCloth 的输入和输出网格受 Maya Nucleus 解算器的影响，且它们会共享输入网格的变换节点。

静止形状(Rest Shape)
指定 nCloth 对象的静止形状的设定方式。nCloth 的静止形状决定模拟的静止长度和弯曲角度。它不能决定 nCloth 对象的位置或平移，但是可以决定 nCloth 对象通过拉伸、压缩和弯曲阻力想要达到的形状。

有关此子菜单的选项的详细信息，请参见“静止形状”(Rest Shape)子菜单。

贴图(Maps)

绘制纹理特性
使用“绘制 nCloth 属性工具”(Paint nCloth Attributes Tool)作为当前工具。使用该工具，可以在 nCloth 对象上绘制任何可用的属性以创建纹理贴图。绘制的 nCloth 属性值将存储在磁盘上的纹理文件中。

如果选择属性的 ，则会显示“绘制 nCloth 属性工具”(Paint nCloth Attributes Tool)设置。有关此工具的设置的详细信息，请参见 Artisan 工具设置。

有关可绘制的 nCloth 属性的信息，请参见 nCloth 碰撞属性和 nCloth 动力学特性。

绘制顶点特性
使“绘制 nCloth 属性工具”(Paint nCloth Attributes Tool)成为当前的工具。使用此工具，可在 nCloth 对象上绘制任何可用属性。使用此工具绘制的 nCloth 属性值将逐顶点存储在 nCloth 对象上。

如果选择属性的 ，则会显示“绘制 nCloth 属性工具”(Paint nCloth Attributes Tool)设置。有关此工具的设置的详细信息，请参见 Artisan 工具设置。

有关可绘制的 nCloth 属性的信息，请参见 nCloth 碰撞属性和 nCloth 动力学特性。

将纹理贴图转化为顶点贴图
将当前 nCloth 对象上指定的特性纹理贴图转化为在 nCloth 的顶点上存储的值。
注：

仍必须设定贴图类型属性才能使用已转化的顶点贴图。

有关支持贴图属性的信息，请参见 nCloth 碰撞属性和 nCloth 动力学特性中的“动力学特性贴图”。

将顶点贴图转化为纹理贴图
将当前 nCloth 对象上的指定顶点特性值转化为纹理贴图。
注：

仍必须将贴图类型属性设定为使用已转化纹理贴图。

有关支持贴图属性的信息，请参见 nCloth 碰撞属性和 nCloth 动力学特性中的“动力学特性贴图”。
