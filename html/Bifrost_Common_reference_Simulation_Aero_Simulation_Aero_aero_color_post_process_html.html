aero_color_post_process

可以在 Aero 模拟后使用此节点，以防止发射的颜色随时间变暗。如果使用 aero_source_color 特性创建颜色特性，则会创建 voxel_emission_density 特性，此节点可以使用此特性取消预乘颜色。此外，还有一个步骤是向外扩散颜色，以避免在最终渲染中出现深色镶边。

输入

aero_volume

此项应连接到 simulate_aero node 的输出。

enable

关闭此项可禁用效果。如果禁用，将使其成为穿过节点。

color

要取消预乘并扩散到零密度区域的体素特性的名称。

alpha

要用于 Alpha 取消预乘的体素特性的名称。aero_source_color_property 节点将自动创建特性 voxel_emission_density。

voxels_to_dilate

此项确定将颜色值扩散到 voxel_fog_density 为零的区域中的距离。约为 2 个体素的值通常足以避免出现深色镶边。

clamp_colors

如果启用此项，则在取消预乘后，颜色将钳制为 0.0 到 1.0。

输出

aero_processed_color

更正了指定颜色特性的 aero_volume。
