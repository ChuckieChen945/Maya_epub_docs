“推拉工具”(Dolly Tool)选项

向前和向后平移摄影机。也可以按住 Alt 键并单击鼠标右键（或是按住 Alt 键并单击鼠标左键和中键）。按住 Ctrl + Alt 并单击鼠标左键在要推拉到的区域周围进行拖动框选。

    打开“推拉工具”(Dolly Tool)选项

  • 在“视图”(View)面板的面板菜单中，选择“视图 > 摄影机工具 > 推拉工具”(View > Camera Tools > Dolly Tool) >
比例(Scale)

调整摄影机移动的速度。滑块范围为 0.01 到 10。默认值为 1。

局部(Local)

如果启用，则在摄影机视图中进行拖动，可以让摄影机朝向或远离其兴趣中心移动。如果禁用，则在摄影机视图中进行拖动，可以让摄影机及其兴趣中心一同沿摄影机的视线移动。“本地”默认情况下处于启用状态。

兴趣中心(Center of Interest)

如果启用“兴趣中心”(Center of Interest)，在摄影机的视图中按住鼠标中键并拖动可向前移动摄影机的兴趣中心或将摄影机的兴趣中心移离摄影机。如果禁用，则在摄影机视图中进行拖动，可以让摄影机朝向或远离其兴趣中心移动。可以使用 Ctrl + Alt + 鼠标左键用蚂蚁线选择区域，将兴趣中心捕捉到该区域的中心。

“兴趣中心”(Center of Interest)默认情况下处于禁用状态。

如果启用“兴趣中心”(Center of Interest)（和/或“局部”(Local)）和“边界框”(Bounding box)，且当前处于四面板视图中，则在任何视图中使用鼠标中键进行拖动，系统都会显示末端带有小 x 的红线，表示“兴趣中心”(Center of Interest)。

朝向中心(Towards center)

禁用此选项，可以在开始推拉时朝向“推拉工具”(Dolly Tool)图标的当前位置进行推拉。

捕捉长方体推拉到(Snap box dolly to)

当使用 Ctrl + Alt + 拖动（或在 Mac OS X 上使用 Control + Option + 拖动）推拉摄影机时，长方体推拉会将兴趣中心移动到框选区域中。

表面(Surface)

如果启用，则在对象上执行长方体推拉（Ctrl + 拖动）时，兴趣中心将移动到对象的曲面上。如果禁用平滑着色模式（且特别是在场景中存在许多可见对象时），计算曲面点速度会较慢。

边界框(Bounding box)

如果启用，则在对象上执行长方体推拉（Ctrl + 拖动）时，兴趣中心会移动到对象的边界框的中心。“边界框”(Bounding Box)默认情况下处于启用状态。

正交模式
与正交摄影机一起使用时，推拉工具可以在“推拉”(Dolly)模式中运行，以便摄影机前后平移；或者，推拉工具可以在“缩放”(Zoom)模式中运行，使“正交宽度”(Orthographic Width)发生更改且您可以控制摄影机的可见场景范围。
