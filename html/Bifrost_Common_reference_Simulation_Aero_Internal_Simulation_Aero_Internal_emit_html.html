emit

此节点向 Aero 体发射，并将结果放置在 outAeroBody 中。

输入

source

source 是包含体素化源的对象或对象数组。如果 source 包含不同时间的对象，我们假定该数组按字母（时间、对象）顺序排序，这样，我们可以按该数组中的索引和获知的给定对象的时间戳将不同时间的同一对象关联起来。source 可以包含以下命名特性，这些特性将覆盖该节点的输入参数：

  • “amount”(float)：时间步期间发射的烟量（发射速率）。

  • “voxel_temperature”(float)：温度发射速率。

  • “continuous”(float)：如果大于零，将连续发射，否则，将仅在开始帧处发射。

  • “emitVelocity”(float3)：将添加到源化体素的速度的均匀加速度。

  • “density”(float)：源发射的物质的物理密度（质量/体积）。对于烟灰，密度应为 1800kg/m^3。

  • “inheritGeoVelocity”(float)：用于确定源化体素将如何继承源本身速度的因子。值为零表示不继承。

  • “fluidDetailSize”(float)：确定源发射时的分辨率级别。

sourcingAcceleration

源 BIF 上将添加到源化体素的速度的体素化加速通道的名称。如果为空，则源内体素的加速度将设置为零。

time

当前时间。

timeStep

此时间步的大小（增量时间）。

velocityTimeStep

用于速度的此时间步的大小（增量时间）。

frame

当前帧。

frameStartTime

帧开始的时间（可用于子帧插值）

frameEndTime

帧结束的时间（可用于子帧插值）

coarsestDOFDepth

平铺树中应发生发射的最粗糙深度。

number_of_upres_levels

平铺树中将保留 upres 通道（雾密度和温度）的最精细级别的数量

simulationDomain

将在以世界空间坐标定义的此边界框中发射。

aeroBody

Aero 体（对象）。
