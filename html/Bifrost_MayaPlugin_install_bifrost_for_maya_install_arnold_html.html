安装 Arnold for Maya 插件进行渲染

具有 Bifrost 的 Maya 安装包含兼容版本的 Arnold for Maya (MtoA)。但是，如果您已安装较新版本的 Bifrost，则可能还需要更高版本的 MtoA。有关要求的最低 MtoA 版本，请参见您的 Bifrost 版本对应的发行说明，或者，只需使用可用的最新版本。

要下载最新版本，请访问 Autodesk Account 门户。

有关在 Maya 中安装 Arnold 的帮助，请参见安装 - Arnold for Maya。

对于不支持直接渲染 Bifrost 内容的渲染器，可以导出为 Alembic 和 OpenVDB 等标准格式。
