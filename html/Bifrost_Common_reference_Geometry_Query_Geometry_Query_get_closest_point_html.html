get_closest_point

返回输入几何体上最接近每个输入位置的点（顶点）的位置和索引。然后，可以使用 sample_property 获取这些位置上的几何体的特性的插值。

输入

geometry

要在其上查找最近点的几何体。仅支持基于点的几何体。

positions

要从中查找最近点的位置。

use_cutoff_distance

仅在指定的 cutoff_distance 中搜索。

cutoff_distance

要在其中搜索的距离。如果最近位置太远，则 found 数组的对应元素为 false。

输出

locations

包含位置的数组。您可以将其插入到 sample_property 节点的 locations 端口。

point_indices

包含几何体的点索引的数组。可以使用这些索引从包含几何体特性数据的数组获取对应值。

found

布尔值的匹配数组。如果元素为 false，则相应的位置无效，如果使用它对几何体进行采样，将直接返回默认值。这可能是因为中止距离内没有点，也可能是因为几何体不受支持。
