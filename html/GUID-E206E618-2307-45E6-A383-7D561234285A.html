for

一个 for 循环具有以下格式：

for (initialization; condition; change of condition) {
	statement;
	statement;
	 ...
}

for 语句后面的括号必须包含三个以分号隔开的部分。很重要的一点是了解这些部分之间的关系：

  • 初始化设置循环变量的初始值，例如，$i = 0 或 $i = $v+1。在循环开始之前仅运行该表达式一次。
  • 在循环的每个迭代开始时检查该条件。如果为真，则执行块。如果为假，则循环结束，同时在循环后继续执行。例如，$i < 5 或 $i < size ($words)。
  • 在循环的每个迭代结束后运行条件更改。该表达式应进行某些更改，以使每个迭代更接近于循环的结束目标。例如，$i++ 或 $i += 5。

在执行每个语句之前，一个 for 循环会对终止条件求值。条件将比较变量、属性或常量值。

int $i;
for ($i = 10; $i > 0; $i--) {
	print($i+"...\n");
}
print("Blastoff!!!");
