This bugfix release fixes the following issues:

  • A recent regression could prevent batch renders from working properly when executed from Maya interface.
  • When standins are exported without cameras, MtoA shouldn't consider the list of renderable cameras in the scene
  • Photometric lights filenames containing Maya environment variables weren't expanded
#3186 MAYA_RENDER_DESC_PATH set incorrectly in mtoa.mod
#3179 Standin exports without cameras shouldn't loop over cameras
#3182 Photometric lights path not expanded
