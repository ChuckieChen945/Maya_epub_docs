points_array_to_strand_trails

从一系列基于点的几何体创建股轨迹。轨迹会将每个几何体的第一个点彼此连接。

如果点几何体具有 ID（point_id 几何特性），则 point_id 将用于标识哪些点应连接在一起。如果没有可供使用的 ID，将通过假定点计数在不同对象之间不发生改变来生成股。在点计数不相等并且缺少 point_id 的情况下，将以不正确的顺序生成股。

另请参见：create_particle_trails。

输入

points_array

一组基于点的几何体，它们将连接在一起形成股。

properties

要传递到输出股的特性。这是要传递的基于表达式的特性列表，对于特定特性，请使用以空格分隔的列表：
point_size point_color。

若要传递每个传入特性，请使用 *。

若要排除特定特性，请使用 !：
* !point_color。

默认为 *，表示将传递所有受支持的传入特性。
