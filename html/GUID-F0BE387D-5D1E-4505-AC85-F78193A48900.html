“UV 编辑器”(UV Editor)的“UV 集”(UV Sets)菜单

通过从显示的下拉列表中选择 UV 集的名称，选择要在 UV 编辑器中编辑的 UV 集。请参见 UV 集。

UV 集编辑器(UV Set Editor)
打开 “UV 集编辑器”(UV Set Editor)。
将 UV 复制到 UV 集(Copy UVs to UV Set)
将当前 UV 布局复制到现有或新的 UV 集。
创建空 UV 集(Create Empty UV Set)
在当前对象上创建一个新的空 UV 集。然后可以使用其中一种映射/投影方法在集中创建 UV。有关详细信息，请参见创建 UV 集选项(Create UV Set options)。
删除当前 UV 集(Delete Current UV Set)
删除当前活动 UV 集。
重命名当前 UV 集(Rename Current UV Set)
用于为当前活动 UV 集指定新名称。
逐实例共享(Per Instance Sharing)
提供用于导航当前 UV 集实例的选项。
  • 选择共享实例(Select Shared Instances)：选择共享当前 UV 集的所有实例。
  • 共享实例(Share Instances)：共享当前 UV 集的所有选定实例。
  • 将选定实例作为当前实例(Make Selected Instance current)：将选定实例设置为当前处于活动状态。
自动加载纹理(Auto-load textures)
确定是否自动加载选定 UV 集的关联纹理。
当前 UV 集
可用于切换显示所有选定对象的当前显示的 UV 集。
