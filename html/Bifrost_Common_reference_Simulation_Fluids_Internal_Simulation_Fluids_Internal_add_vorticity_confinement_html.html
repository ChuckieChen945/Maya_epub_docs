add_vorticity_confinement

此节点向输入体的速度场添加漩涡约束，并将结果放置在 outBody 中。

输入

timeStep

此时间步的大小（增量时间）。

vorticityConfinementStrength

漩涡约束的强度。[0;1] 之间的较小值可获得更好的结果。

vorticityConfinementScale

漩涡约束力的空间比例。介于 0 和 1 之间的数字，0 表示最精细的比例，1 表示最粗糙的比例。

coarsestDOFDepth

平铺树中应发生传输的最粗糙深度。

mask

乘以漩涡约束的可选遮罩。

body

Aero 体（对象）。
