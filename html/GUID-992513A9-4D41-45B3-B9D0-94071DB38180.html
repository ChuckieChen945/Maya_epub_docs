使用 Bifrost“禁用体积”(Kill Volume)和“剪裁”(Clipping)输入网格

可以将多边形对象用作输入网格，以设置 Bifrost 模拟区域周围的边界。使用输入网格可以：

  • 将 Bifrost 发射限制到多边形对象的边界框区域。请参见 Bifrost 禁用体积属性。
  • 预览 bifrostShape 或 Bifrost 网格的切片以进行精密分析。请参见 bifrostShape 属性。
  • 为“平铺视图”(Tile View)隔离关注区域。请参见使用平铺视图可视化 Bifrost 体素分辨率。

添加“禁用体积”(Kill Volume)或“剪裁”(Clipping)网格

  1. 在属性编辑器中显示适当的选项卡：
    • 使用“禁用体积”(Kill Volume)限制发射的主要特性。
    • 用于剪裁显示、网格或“平铺视图”(Tile View)的形状。
  2. 在属性编辑器菜单中，选择“列表”(List)，然后禁用“自动加载选定属性”(Auto Load Selected Attributes)。
    这将禁用 Maya 加载选定对象的属性这一默认行为。选择和定位多边形对象时，当前属性现在将在属性编辑器中保持加载状态。
    注： 或者，可以选择属性编辑器底部的“复制选项卡”(Copy Tab)，以在浮动窗口中创建 Bifrost 对象属性的副本。
  3. 选择和定位多边形对象。
  4. 在“禁用体积”(Kill Volume)或“剪裁”(Clipping)属性组中，单击“使用选定对象”(Use Selected)。
    提示： 添加输入网格后，可以禁用网格的着色，以免它阻止模拟的视图。为此，请在多边形对象的形状节点中，展开“显示”(Display)属性组，启用“启用绘制覆盖”(Enable Drawing Overrides)，然后禁用“着色”(Shading)。
  5. （可选）启用“自动加载选定属性”(Auto Load Selected Attributes)以使 Maya 恢复为默认行为。

移除“禁用体积”(Kill Volume)或“剪裁”(Clipping)网格

  1. 在属性编辑器中显示适当的选项卡。
  2. 在属性编辑器菜单中，选择“列表”(List)，然后禁用“自动加载选定属性”(Auto Load Selected Attributes)。
  3. 按 Alt + D 以取消选择所有对象，然后在 Bifrost 对象的属性编辑器的“禁用体积”(Kill Volume)或“剪裁”(Clipping)属性组中，单击“使用选定对象”(Use Selected)。
  4. （可选）启用“自动加载选定属性”(Auto Load Selected Attributes)以使 Maya 恢复为默认行为。
