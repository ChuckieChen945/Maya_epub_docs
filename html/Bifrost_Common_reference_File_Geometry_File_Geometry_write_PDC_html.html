write_PDC

此节点将 Bifrost 点云写入 Maya PDC 文件。

输入

points

要写入的 Bifrost 点。

enable

如果为 true，则启用写入文件。否则，节点将以穿过方式工作。

filename

文件名。使用 # 包含输入 frame 编号。# 的出现次数确定填充量。例如，file####.pdc 解析为 file0001.pdc、file0002.pdc 等。

directory

目录。

frame

帧编号。您可以将 time 节点的 frame 输出连接到 to_long 节点，然后在此处连接结果。

overwrite

如果为 true，将覆盖现有文件。

create_directories

如果为 true，则创建输出目录层级（如果不存在）。

输出

out_points

使输入点穿过。

success

如果已成功写入文件，则为 True。
