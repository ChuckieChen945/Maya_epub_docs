filter_offset_array

过滤输入 offset_array，保留 indices_to_keep 输入指定的元素。要过滤由偏移数组表示并具有关联数据数组的组件（如股或网格面）时，可以使用此节点。

例如，有一个包含三个股的股几何体，其中第一个股引用三个点 [0, 1, 2]，第二个股引用三个点 [3, 4, 5]，最后一个股引用四个点 [6, 7, 8, 9]，此股几何体将包含一个名为 strand_offset 的偏移数组，其值将为 [0, 3, 6, 10]。

要过滤此几何体以仅保留第一个股和最后一个股，则将 [0, 2] 传入到 indices_to_keep 中。输出 filtered_offset_array 将为 [0, 3, 8]，target_compaction_indices 将为 [0, 1, 2, 6, 7, 8, 9]。

然后，target_compaction_indices 可以连接到一个 get_from_array 节点，该节点将仅过滤对应于保留的各个股（第一个股和最后一个股）的点的元素。例如，point_position、point_size 和 point_tangent 都可以使用相同的 target_compaction_indices 数组进行过滤。

输入

offset_array

要过滤的偏移数组。

indices_to_keep

要在 offset_array 中保留的元素的索引。

输出

filtered_offset_array

仅包含 indices_to_keep 数组指定的元素的已过滤偏移数组。

target_compaction_indices

索引数组，用于精简与偏移数组关联的数据数组。此输出可以连接到 get_from_array 以过滤这些关联的数据数组。
