交替

注：

交替在 Mac OS X 和 Linux 中可用。在 Windows 中，请使用 FieldAssembler。有关详细信息，请参见关于 FieldAssembler。

交替 工具用于交替场渲染图像。场渲染图像是专为在 NTSC 设备上进行视频播放而设计的图像序列中的图像。每个场或图像仅包含半个全帧，并且每个场都存储在单独的文件中。设计将在 NTSC 设备上播放的动画时，以这种方式渲染图像非常有用。但是，若要在计算机屏幕上查看这些图像或将其传递到某些其他计算机程序，需要重新将其集合为完整图像。发生这种情况时，最终将有一半的文件，并且每个文件包含的数据翻倍。

使用交替

  1. 在 Linux shell 中，键入： 

    interlace [-a # # #] [-d] [-e #] [-r] [-t str] base_image_name
  2. 选项 描述

    -a [# # #]

    指定要交替的动画，三个编号是开始帧、结束帧和帧数。当前，开始帧、结束帧和帧数都必须是整数。

    -d

    指定偶场为第一个场；否则，第一个场为奇场。该选项应与渲染设置窗口中的场选项一致。

    -e

    指定扩展名的填充长度。

    -r

    删除原始场图像。

    -t

    指定文件扩展名尾部（例如，sgi）。

    -u

    将帧解开放到场中。

    -help

    显示帮助文件。
