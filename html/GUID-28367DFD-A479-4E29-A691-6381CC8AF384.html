XGen 缓存(Caching)子菜单

使用以下“缓存”(Caching)选项，可以保存和加载交互式修饰样条线的描述。旧版 XGen 描述不支持这些“缓存”(Caching)选项。

创建新缓存(Create New Cache)

将选定描述中的头发保存到 Alembic 缓存中，然后将缓存文件加载到场景。

选择“生成 > 创建新缓存”(Generate > Create New Cache) > 以设置缓存选项。

注： “文件格式”(File Format)是交互式修饰样条线唯一支持的 Alembic 导出高级选项。

“创建新缓存”(Create New Cache)选项类似于 Alembic 导出选项。

导入缓存(Import Cache)

用于从交互式修饰样条线描述中加载包含头发的 Alembic 缓存文件。

请参见 Alembic 导入选项。
注： “适配时间范围”(Fit Time Range)和“将当前时间设置为开始帧”(Set Current Time to Start Frame)是交互式修饰样条线唯一支持的导入选项。
导出缓存(Export Cache)

将选定描述中的头发保存到 Alembic 缓存文件，以便稍后能够加载。

选择“生成 > 导出缓存”(Generate > Export Cache) > 以设置缓存选项。

（可选）在“导出缓存”(Export Cache)选项窗口的“高级选项”(Advanced Options)区域中，设置以下选项来创建第三方 Alembic 导入器支持的标准 Alembic 文件：

  • “多个变换”(Multiple Transform)，用于为缓存文件中的每根头发创建变换和曲线节点。

    否则，这些头发将作为曲线组节点缓存到描述变换节点下。

  • “写入最终宽度”(Write Final Width)，用于将每根头发上每个 CV 的最终宽度保存到 Alembic 缓存。每个 CV 的最终宽度是使用每个 CV 的宽度（基于“宽度”(Width)笔刷）、“宽度比例”(Width Scale)以及其他基于宽度的描述节点属性（例如，“锥化”(Taper)）计算得出的。
注： “文件格式”(File Format)是交互式修饰样条线唯一支持的另外一个 Alembic 导出“高级选项”(Advanced Option)。

“导出缓存”(Export Cache)选项类似于 Alembic 导出选项。

禁用选定对象的所有缓存(Disable All Caches On Selected)/启用选定对象的所有缓存(Enable All Caches On Selected)

暂时禁用属于选定描述的缓存节点和文件。选择“启用选定对象的所有缓存”(Enable All Caches On Selected)可重新启用这些缓存节点和文件，以便播放。

替换缓存(Replace Cache)

将属于选定描述的 Alembic 缓存替换为新生成的缓存。

选择“生成 > 替换缓存”(Generate > Replace Cache) > 以设置缓存选项。

删除缓存(Delete Cache)

删除属于选定描述的缓存节点和文件。

选择“生成 > 删除缓存”(Generate > Delete Cache) > 以设置缓存选项。

删除缓存之前的节点(Delete Nodes Ahead of Cache)

删除连接到选定描述的所有上游节点，包括修改器和“雕刻”(Sculpt)节点，同时保留样条线缓存节点。

如果头发或毛发已保存到缓存文件且不再需要修改修饰，请使用此选项。此工作流假定当前缓存包含对角色修饰进行最终渲染所需的全部数据。
