members_if

返回结构（向量或矩阵），在此结构中，基于布尔结构从其他两个结构中选择成员。

示例

对于某一节点，bool3 (True, False, False) 连接到 condition 端口，float3 (3.0, 4.0, 5.0) 连接到 true_case 端口，float3 (9.0, 8.0, 7.0) 连接到 false_case 端口，则此节点返回值为 (3.0, 8.0, 7.0) 的 float3。

输入

condition

布尔结构值，用于从 true_case 或 false_case 端口选择相应的成员值。

true_case

此结构包含要在条件为 true 时采用的值。

false_case

此结构包含要在条件为 false 时采用的值。

输出

output

此结构包含最终选择。
