render_archive_instance

创建引用磁盘上的 Arnold 渲染归档（.ass 文件）的实例形状。实例形状可以连接到 create_instances、set_instance_shape、set_instance_geometry 或 selector_instance。

输入

archive_file

渲染归档的路径和文件名。使用 #### 表示序列的帧编号。

frame_property

指定要实例化序列的哪一帧的点对象特性的名称。应确保图表在每一帧逐点更新此特性。

实例

preview_geometry

要用于预览的可选几何对象。如果连接了对象，它将仅显示在视口中，而不用于渲染。

name

可选标识符。

transform

要应用于实例化几何体的可选变换矩阵。

渲染

operators

要应用的任何 Arnold 运算符（可选）。可以将运算符添加到图表中，方法是从大纲视图拖动运算符（当“显示”>“仅 DAG 对象”(Display > DAG Objects Only)处于禁用状态时），然后将其连接到相应端口。

operators

Arnold 运算符可以插入此处，以应用于生成的几何体。

输出

render_archive_instance

可以连接到 create_instances 或类似节点的实例形状。
