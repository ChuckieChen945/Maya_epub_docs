“GPU 缓存”(GPU Cache)子菜单

输入
将基于 Alembic 的 GPU 缓存文件加载到场景中。请参见创建或导入 GPU 缓存。

在“GPU 缓存导入选项”(GPU Cache Import Options)中，在“导入”(Import)中启用“适配时间范围”(Fit Time Range)以将当前场景的时间滑块范围更新为与已导入 GPU 缓存文件的时间滑块范围匹配。启用“将当前时间设置为开始帧”(Set Current Time to Start Frame)，以将场景的当前时间设置为已导入 GPU 缓存文件的开始帧。

“导出全部”(Export All)和“导出当前选择”(Export Selection)

将所有对象或选定的多边形和 NURBS 对象导出到基于 Alembic 的 GPU 缓存文件中。如果在场景中选择多个对象，并选择“导出当前选择”(Export Selection)，则可以将每个对象保存到同一缓存文件或各个文件中。默认情况下，GPU 缓存文件保存在当前 Maya 项目的 cache\alembic 文件夹中。

请参见创建或导入 GPU 缓存。
选择“缓存”(Caching) > “GPU 缓存”(GPU Cache) > “导出全部/导出当前选择”(Export All/Selection) > 以查看 GPU 缓存导出选项。
刷新所有(Refresh All)
在场景中重新加载所有的 GPU 缓存节点。
