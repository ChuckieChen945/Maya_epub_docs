scale_points

将基于点的几何体的当前 point_scale 值与指定的缩放因子相乘。这将影响实例的大小以及点形状。但是，它不会影响整个几何对象的大小。

输入

points

具有要缩放的点的几何体。

weights

要应用的 scale 因子的比例。

  • 权重为 0 时，保持当前缩放比例不变。
  • 权重在 0 到 1 范围内时，按给定的 scale 因子的某个比例进行缩放。
  • 权重为 1 时，按精确的 scale 因子进行缩放。
  • 权重大于 1 时，缩放比例大于给定的 scale 因子。
  • 权重小于 0 时，进行负缩放。这将反转任何实例化形状。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

scale

缩放时使用的因子。例如，0.25 表示小 4 倍，4 表示大 4 倍。

这是一个解释的自动端口，其接受以下类型：

  • 连接单个 float 以按相同的因子统一缩放所有点。
  • 连接 array<float> 以按不同的量统一缩放每个点。
  • 连接单个 float3 以按相同的 XYZ 因子缩放所有点。
  • 连接 array<float3> 以按不同的 XYZ 因子缩放每个点。
  • 连接 string 以按具有相应名称的特性的值进行缩放（每个点的 float 或 float3）。
  • 连接 ScalarField 或 VectorField 以按每个点所在位置处的场的值进行缩放。

输出

out_points

具有已缩放点的几何体。

point_scale

在几何体上设置的 point_scale 值。
