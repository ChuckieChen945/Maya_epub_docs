LookdevX for Maya

LookdevX is a new material authoring plug-in for Maya. It lets you graph shading networks and create materials for USD objects. With LookdevX, you can efficiently build procedural materials that work across multiple renderers.

To get started, visit the Getting started with LookdevX page for resources and an overview of working with LookdevX.
