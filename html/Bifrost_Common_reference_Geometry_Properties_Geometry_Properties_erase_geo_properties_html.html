erase_geo_properties

从输入几何体删除几何特性。

输入

geometry

要从中删除几何特性的几何体。

components

被要删除的特性作为目标的组件。连接一个或多个包含组件名称的字符串或字符串数组，例如 point_component、face_component 等。不会删除将此处未包含的组件作为目标的特性。

properties

要删除的几何特性的名称，例如 point_color。确保包含所需的所有几何特性。不会删除不是几何特性的特性。

  • 使用空格分隔多个几何特性名称。
  • 您可以在表达式中使用通配符 *，例如使用 point_user* 来删除所有以字符串“point_user”开头的几何特性。
  • 默认值 * 会删除所有几何特性。
  • 使用空字符串可防止删除任何几何特性。
  • 若要包含除指定特性之外的所有特性，请使用 *（以包含所有特性），然后使用 ! 分别为特定特性或模式添加前缀以将其排除。例如，* !point_foo !point_bar* 将包括除“point_foo”以及以字符串“point_bar”开头的那些特性以外的所有特性。

输出

out_geometry

不包含指定特性的输入 geometry。
