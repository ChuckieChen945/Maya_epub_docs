write_Field3d

此节点将 Bifrost 体积写入 Field3D 文件。

输入

volume

要写入的 Bifrost 体积。

properties

要写入的特性的列表（以空格分隔）。使用 *（星号）指定所有特性。

enable

如果为 true，则启用写入。否则，节点将用作穿过。

filename

文件名。使用 # 包含输入 frame 编号。# 的出现次数确定填充量。例如，file####.3ed 解析为 file0001.f3d、file0002.f3d 等。

directory

目录。

frame

帧编号。您可以将 time 节点的 frame 输出连接到 to_long 节点，然后在此处连接结果。

overwrite

如果为 true，将覆盖现有文件。

create_directories

如果为 true，则创建输出目录层级（如果不存在）。

filter_property

用于确定要存储的体素的体素特性。

threshold

将存储幅值大于 threshold 的体素。

file_format

用于 Field3D 的文件格式。如果不确定，请使用 Ogawa。

field_name

如果指定此项，则覆盖 Field3D 文件中存储的内部划分名称。

preview_name

如果指定此项，则使用给定名称存储预览划分中所有特性的降低采样版本。

preview_level

预览字段的分辨率的推导方式为原始分辨率除以 2 再乘以 preview_level 次幂。

输出

out_volume

使输入体积穿过。

success

如果已成功写入文件，则为 true。

file_path

已写入文件的完整路径。
