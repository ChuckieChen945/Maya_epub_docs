randomize_point_scale

随机化 point_scale 特性。这将影响实例的大小以及点形状。但是，它不会影响整个几何对象的大小。

输入

points

具有要缩放的点的几何体。

weights

控制应用的随机化量。

  • 权重为 0 时，不应用随机化。
  • 权重在 0 到 1 范围内时，按 min 和 max 之间的值的一部分进行随机化。
  • 权重为 1 时，完全按 min 和 max 之间的值进行随机化。
  • 权重大于 1 时，点移动的距离大于给定值。
  • 权重在 0 到 1 范围以外时，按 [min, max) 范围以外的量进行随机化。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

随机化

min、max

最小和最大缩放因子。例如，0.25 表示小 4 倍，4 表示大 4 倍。仅支持 float 值，但您可以使用 axis_to_scale 控制因子如何应用于轴。

seed

随机数生成器的种子。

axis_to_scale

控制如何应用缩放因子：

  • XYZUniform：每个点将一个随机缩放因子应用于所有 3 个 XYZ 轴。
  • XYZSeparately：每个点将三个单独的随机缩放因子应用于 XYZ 轴。
  • XOnly：每个点将一个随机缩放因子应用于 X 轴。
  • YOnly：每个点将一个随机缩放因子应用于 Y 轴。
  • ZOnly：每个点将一个随机缩放因子应用于 Z 轴。

scale_existing_values

当此选项为 true 时，随机缩放因子将与任何现有 point_scale 值相乘。当此选项为 false 时，随机值将覆盖任何现有的缩放因子。

distribution

用于控制随机分布的 FCurve。使用默认形状时，在范围最小值附近生成较多点，而在范围最大值附近生成较少点。

输出

out_points

具有已缩放点的几何体。

point_scale

在几何体上设置的 point_scale 值。
