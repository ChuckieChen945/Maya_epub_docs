雕刻目标工具

除了常规雕刻工具以外，Maya 还提供了一部分专用于处理融合变形的工具。这些工具适用于将一个或多个融合变形特定的变形作为目标，而不会对其他融合变形产生负面影响。有关融合变形工作流的详细信息，请参见形变创作。

在“雕刻”(Sculpting)工具架上和“网格工具 > 雕刻工具”(Mesh Tools > Sculpting Tools)菜单中，将显示以下工具。有关与这些工具关联的公用设置列表，请参见雕刻工具设置。

工具名称 图标 用途

平滑目标工具(Smooth Target Tool)

通过平均化顶点的位置，将相对彼此的顶点位置拉平。可以使用此工具仅平滑当前处于“编辑”(Edit)模式下的融合变形目标。

克隆目标工具(Clone Target Tool)

将雕刻编辑从指定的融合变形目标复制到当前处于“编辑”(Edit)模式的融合变形目标。融合变形和目标源可以通过“工具设置”(Tool Settings)编辑器中的“融合变形源”(Blend Shape Source)和“目标源”(Target Source)属性设置。

在该工具处于活动状态时按住 Ctrl 键将暂时切换到“擦除目标工具”(Erase Target Tool)，而按住 Shift 键将暂时切换到“平滑目标工具”(Smooth Target Tool)。

遮罩目标工具(Mask Target Tool)

隐藏绘制区域的当前目标的雕刻编辑。

按住 Ctrl 键的同时进行绘制可移除遮罩并取消隐藏编辑，或者按住 Shift 键可平滑遮罩/未遮罩区域。

擦除目标工具(Erase Target Tool)

为当前处于“编辑”(Edit)模式下的融合变形目标移除雕刻操作。
注： 将此工具与“遮罩目标工具”(Mask Target Tool)结合使用时请务必小心，因为您仍可擦除当前处于隐藏状态的编辑。
