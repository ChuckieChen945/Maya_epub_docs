update_transform_chain

使用各种更新方法更新解释为从父对象到子对象的连续链的现有 array<Transform>。

输入

transforms

要更新的 array<Transform>。数组元素应按从父对象到子对象的顺序排序。

parent_transform

数组中第一个 Transform 的父对象变换。如果未连接任何对象，则此项将默认为单位变换。

update_matrices

用于更新 transforms 链的变换矩阵数组。这些矩阵应反映选定的 update_method，以便正确解释。

update_method

定义应如何解释 update_matrices 输入以及如何更新变换。

  • From Pivot：使用新的枢轴矩阵重新计算所有变换矩阵分量。

  • From Operator：使用新的运算符矩阵重新计算非枢轴变换矩阵分量。

  • From Local：计算运算符矩阵，并直接使用局部空间动画结果重新计算移动矩阵。

  • From World：计算运算符矩阵，并直接使用世界空间动画结果重新计算移动矩阵。在使用反向运动学时，通常使用此方法。

overwrite

如果启用此项，则使用输入 update_matrices 覆盖选定的变换分量。否则，将 update_matrices 乘以从 update_method 选择的现有变换分量。

输出

out_transforms

更新的 transforms。
