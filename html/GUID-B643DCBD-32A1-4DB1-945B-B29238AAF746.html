搜索首选项

可以设置以下首选项以更改搜索(Search)功能的工作方式。

使用上次运行的命令自动填充搜索字段(Autofill search field with last-run command)
确定搜索字段是否在每次打开时预填充最近运行的命令。默认值为禁用。

工具和命令搜索模式

搜索字段为空时显示历史记录(Show history when search field is empty)
确定最近搜索的词是否显示在空白搜索字段的下方。默认值为启用。
在生成搜索结果时包含运行时命令名称(Include runtime command names while generating search results)
确定运行时命令是否显示在搜索结果列表中（例如 polyExtrude）。默认值为启用。
在生成搜索结果时包含关键字(同义词)(Include keywords (synonyms) while generating search results)
确定关键字是否显示在搜索结果列表中。如果您通常确切知道要搜索的内容，则可能需要禁用此选项以避免混乱。默认值为启用。
