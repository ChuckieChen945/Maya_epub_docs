安装适用于 Autodesk Maya 的 Unreal Live Link

注意：

在安装之前，请确保您运行的是 Maya 2020 或更高版本以及 Unreal 5.0 或更高版本。Live Link 要求在本地安装 Unreal Engine 才能正常运行。

您可以在 Windows 或 Linux 上安装适用于 Autodesk Maya 的 Unreal Live Link。执行下列操作之一：

在 Windows 上：

  1. 执行 .msi 安装程序文件，然后按照说明进行操作。

在 Linux 上：

  1. 运行脚本“./UnrealLiveLinkForMaya#.sh”
    • 如果作为“su”安装，则插件会安装到 /usr/autodesk/ApplicationPlugins/UnrealLiveLinkForMaya
    • 否则，该插件会安装到 $HOME/Autodesk/ApplicationPlugins/UnrealLiveLinkForMaya

安装程序会安装插件的 Maya 端，然后自动触发第二次安装来安装 Unreal 端。您只能在安装的第一 (Maya) 部分中取消安装。

注意：

如果选择将 Live Link 安装到自定义目录路径，请确保将该路径添加到 Maya.env 中的 MAYA_PLUG_IN_PATH 和 MAYA_SCRIPT_PATH 环境变量。

卸载适用于 Autodesk Maya 的 Unreal Live Link

在 Windows 上：

  1. 打开“设置”(Settings)。

  2. 选择“应用程序”(Apps)以显示已安装应用程序的列表。

  3. 单击“适用于 Autodesk Maya 的 Unreal Live Link”(Unreal Live Link for Autodesk Maya)。

  4. 单击“卸载”(Uninstall)。

在 Linux 上：

我们建议使用 OpenSSL 1.1.x。如果您的 Linux 发行版未安装 OpenSSL 1.1，则可能会看到 OpenSSL 错误。

  1. 退出 Maya（如果正在运行）。（在活动 Maya 会话期间卸载插件会使 Maya 无响应。）

  2. 删除安装文件夹（如安装部分中所述）。
