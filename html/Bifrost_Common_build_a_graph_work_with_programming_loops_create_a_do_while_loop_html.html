创建 do_while 循环

do_while 循环会重复计算，直到满足条件。例如，您能够以迭代方式近似计算某个值，直到误差低于给定的容差。

  1. 将 do_while 节点添加到图表中。

  2. （可选）可以通过绘制与“新端口”(New Port)图标的连接来创建所需的任何端口。右键单击端口以对其进行重命名或其他更改。

  3. 进入复合并为要在循环中执行的计算创建子图。

    • 默认情况下，output 节点上的 looping_condition 端口为 false。连接某个计算的输出，该计算确定是否要执行另一次循环迭代。请注意，do_while 循环始终至少解算一次，然后使用 looping_condition 确定是否再次解算。
    • 所有 do_while 循环都需要至少一对“状态”端口。状态端口包含匹配的输入和输出端口，以便将输出值沿用为下一次循环迭代的输入。要创建一对状态端口，请右键单击复合的 input 或 output 节点中的某个端口，展开“设置端口状态”(Set Port State)菜单，然后选择匹配的端口。（可选）您可以通过在循环复合外部连接或设置一个值，来初始化第一次迭代的输入值。
    • 请不要删除 max_iterations 输入端口，即使您在计算中不使用它。它可以防止当 looping_condition 的输入始终为 true 时出现无限循环。您可以在复合外部为 max_iterations 设置或连接一个值。

不是状态端口的输出均为“辅助”。其输出值只是最后一次循环迭代计算得出的结果。
