read_bif_particles

此节点从 Maya Bifrost 流体 (.bif) 文件读取 Bifrost 点云。

输入

filename

文件名。使用 # 包含输入 frame 编号。# 的出现次数确定填充量。例如，file####.bif 解析为 file0001.bif、file0002.bif 等。

directory

目录。

frame

帧编号。您可以将 time 节点的 frame 输出连接到 to_long 节点，然后在此处连接结果。

enable_bounds

仅加载流体中位于定义的最小/最大边界内的平铺。当从自适应液体模拟中读取粒子时，预计粒子半径可能会超出为流体表面或网格创建级别时指定的边界。这是由于 point_size 值越大，在自适应模拟中需要的细节就越少。

min_bound

最小边界框范围。

max_bound

最大边界框范围。

输出

points

输出 Bifrost 点云。

success

指示操作是否成功的布尔值。
