add_attribute_connection

将来自 source 属性的连接添加到指定 position 中的连接列表。

输入

stage

要在其中添加属性连接的 USD 后台文件。

prim_path

包含属性的基本体的路径。

attribute_name

属性的名称。

source

要连接到的源属性的完整路径。

position

有关 USD 列表操作的更多详细信息，请参见有关 UsdListPosition 的 USD SDK 文档。

输出

out_stage

修改后的后台文件。

success

指示操作是否成功的布尔值。
