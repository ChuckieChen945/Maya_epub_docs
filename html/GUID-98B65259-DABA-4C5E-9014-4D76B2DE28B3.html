选择多边形组件

若要对多边形网格进行多种类型的操作，必须首先选择要修改的相关多边形组件。

下表中描述的方法用于选择各种类型的多边形组件。

目标 操作
选择多边形顶点

将选择类型设定为顶点：

  • 在多边形网格上单击鼠标右键，然后从弹出菜单中选择“顶点”(Vertex)。

或

  • 如果已选定多边形网格或其他组件类型，则可通过按 F9 键将选择类型更改为顶点。

然后，选择多边形网格与线相交处出现的点。

选择多边形边

将选择类型设定为边：

  • 在多边形网格上单击鼠标右键，然后从弹出菜单中选择“边”(Edge)。

或

  • 如果已选定多边形网格或其他组件类型，则可通过按 F10 键将选择类型更改为顶点。

然后，选择多边形网格上的线。

选择多边形面

将选择类型设定为面：

  • 在多边形网格区域上单击鼠标右键，然后从弹出菜单中选择“面”(Face)。

或

  • 如果已选定多边形网格或其他组件类型，则可通过按 F11 键将选择类型更改为顶点。

然后，选择面中心处的点。

选择多边形顶点、边和面

通过执行下列操作之一将选择类型设置为多重选择：

  • 在多边形网格区域上单击鼠标右键，然后从弹出菜单中选择“多重”(Multi)。

然后，选择多边形网格上的组件。

在多组件选择模式下，将选择离鼠标光标最近的顶点、边或面。

选择多边形网格的 UV 纹理坐标

将选择类型设定为 UV：

  • 在多边形网格区域上单击鼠标右键，然后从弹出菜单中选择“UV”。

或

  • 如果已选定多边形网格或其他组件类型，则可通过按 F12 键将选择类型更改为顶点。

然后，选择多边形网格与线相交处出现的点。

虽然可以在场景视图中选择 UV，但若要查看多边形网格的 UV 布局并进行其他 UV 编辑操作，则必须使用 UV 编辑器（“UV > UV 编辑器”(UV > UV Editor)）。

可以将多边形组件的现有选择更改为其他组件类型。例如，如果当前已选定多边形面，则可改为选择与这些面关联的顶点。有关详细信息，请参见更改选择类型。

如果选择一个多边形动作，Maya 会自动将选择转化为多数情况下多边形动作所期望的组件类型。如果 Maya 将顶点转化为边或面，或将边转化为面，则转化后的选择中包含触动初始选择的所有组件。例如，如果您选择一条边并尝试删除顶点，Maya 会自动选择这条边两端的顶点并删除这些顶点。但是，如果您选择两端顶点并尝试删除边，Maya 将选择所有触动这两个顶点中任意一个顶点的边。
