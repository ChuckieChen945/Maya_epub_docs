read_Alembic

从 Alembic 文件读取 Bifrost 点、股和网格，并输出结果对象。

输入

filename

文件名。

directory

目录。

frame

帧编号。

frame_rate

帧速率。

interpolation

用于在子帧位置处插入数据的插值方法。

properties

要读取的特性的列表（以空格分隔）。

输出

points

输出 Bifrost 点。

strands

输出 Bifrost 股。

meshes

输出 Bifrost 网格。

局限

  • 加载时不保留对象层次。
  • 不支持加载 NURBS 和细分曲面。
  • 不明确加载变换，而是在加载时变换数据。
  • 不会加载边界框数据。
  • Bifrost 目前不支持类型为半精度浮点型的数据，这种数据会加载到类型为浮点型的数据中。
  • Bifrost 目前不支持类型为 wchar 的数据，因此将不加载 wchar 字符串。
  • Bifrost 未明确支持四元数，这种数据会加载到 float4 数据中。
  • Bifrost 未明确支持颜色类型，这种类型的数据会加载到对应的浮点和 uchar 类型数据中。
  • Bifrost 未明确支持法线类型，这种类型的数据会加载到对应的浮点和双精度类型数据中。
