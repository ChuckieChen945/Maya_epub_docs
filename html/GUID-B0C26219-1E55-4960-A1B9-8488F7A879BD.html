显示 UV

在 UV 编辑器（“UV > UV 编辑器”(UV > UV Editor)）的 2D 视图中查看和编辑 UV。只要在场景视图中选定多边形或细分曲面网格，UV 就会显示出来。取消选择 3D 网格时，UV 将不会继续显示在 UV 编辑器中。

默认情况下，选定对象的所有 UV 都显示出来。可以使用“隔离选择”(Isolate Select)特征显示 UV 的子集。有关详细信息，请参见显示 UV 的子集。

注：

UV 的 2D 表示不能显示在 Maya 场景视图中。

在 UV 编辑器中显示 UV

  1. 选择“UV > UV 编辑器”(UV > UV Editor)以显示 UV 编辑器。
  2. 执行下列操作之一：
  • 在场景视图中，使用标记菜单将选择类型设定为对象，然后选择多边形或细分曲面网格。

    选定网格的 UV 壳将在“UV 编辑器”(UV Editor)中显示。

  • 在场景视图中，使用标记菜单将选择类型设定为 UV，然后选择多边形或细分曲面网格上的任何顶点。

    与网格上选定的顶点相关联的 UV 将显示在 UV 编辑器中。

    • 提示： 由于纹理分辨率与您的显卡不兼容，具有许多纹理的大型场景有时看起来模糊或不显示 UV。 尝试将纹理分辨率钳制到与您的显卡兼容的值。有关在此场景中执行哪些操作的帮助，请参见纹理模糊或“超出 GPU 纹理 RAM，纹理加载失败”错误消息疑难解答。
