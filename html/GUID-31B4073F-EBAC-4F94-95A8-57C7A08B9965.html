运算符元素

运算符元素指定要应用于颜色数据的运算。您可以在 ProcessList 元素中添加任意多的运算，并按应用顺序排列好。

注： 每个运算符的 inBitDepth 属性必须与链中上一个运算符的 outBitDepth 属性匹配。请参见通用运算符属性。
本节内容
  • 通用运算符属性
    存在所有颜色变换运算符元素通用的一些属性。
  • LUT1D
    LUT1D 元素指定要应用的 1D LUT。
  • LUT3D
    LUT3D 元素指定要应用的 3D LUT。
  • Matrix
    Matrix 元素为颜色值相乘指定一个矩阵，也可以选择应用偏移。
  • Range
    Range 元素通过缩放值和偏移值将输入域映射到输出范围。
  • Gamma
    Gamma 元素对颜色值应用 Gamma 校正。
  • ExposureContrast
    ExposureContrast 元素可模拟调整图像的曝光，并修改浅色和深色之间的对比度。
  • Log
    Log 运算符应用对数或反对数函数，在线性和 Cineon 样式编码之间转化。
  • ASC_CDL
    ASC_CDL 运算符表示美国电影摄影师协会的颜色确定表格式（*.ccc 或 *.cdl）的颜色变换。
  • Reference
    Reference 元素按路径或别名引用其他 CTF 文件。将外部文件导入到 ProcessList 的相应位置。
  • DynamicParameter
    DynamicParameter 元素向兼容应用程序发送信号指示某个参数是动态的。如果变换用于显示，则 CTF 文件中定义的参数值将被忽略，而改用应用程序中设置的值。
