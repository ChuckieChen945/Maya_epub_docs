nComponent 节点

这是 nConstraint 组件属性节点。它可存储所有特定于对象的 nCloth 和 nParticle 约束属性。nComponent 节点中的属性仅应用于约束原点的特定对象。

当您调整 nComponent 节点的属性时，该值将添加到在 dynamicConstraint 节点上设置的值，从而增加这些特性对于选定组件的效果。

Nucleus 组件属性(Nucleus Component Attributes)

元素(Elements)

确定哪些元素会用于此 nComponent 节点约束的约束链接。

来自索引列表(From Indice List)

使用已连接到此节点的选择集中明确定义的点、边或面。

边界(Borders)

仅使用边界（仅被一个面共享的边“边界”(Borders)。）“组件类型”(Component Type)属性决定是否使用顶点、边或面的边界。

全部(All)

使用已连接到此节点的选择集中定义的一组顶点、边、面或所有这些的组合。

组件类型(Component Type)

确定要将此节点的属性应用于哪些类型的几何体。

无(None)

关闭此 nComponent。

点(Point)

将设置应用于点（顶点）。

面(Face)

将设置应用于面。

边(Edge)

将设置应用于边。

对象(Object)

将设置应用于对象。

强度(Strength)

确定此动力学约束中，节点的组件与其他组件之间约束链接的强度。动力学约束链接的强度基于链接到的两个组件的强度相乘。例如，如果每个约束的组件的“强度”(Strength)值为 0.5，则动力学约束链接的“强度”(Strength)值为 0.25 (0.5× 0.5)。

值为 0 意味着约束链接不起作用。默认值为 1。

强度贴图类型(Strength Map Type) / 强度贴图(Strength Map)

“强度贴图类型”(Strength Map Type)确定此 nCloth 组件的强度贴图类型。选择“无”(None)（无贴图），“逐顶点”(Per-vertex)（逐顶点应用贴图）或“纹理”(Texture)（使用“强度贴图”(Strength Map)属性指定的纹理贴图）。

“强度贴图”(Strength Map)指定要用作强度贴图的纹理贴图。此属性仅当“强度贴图类型”(Strength Map Type)设置为“纹理”(Texture)时可用。键入纹理文件的路径，或单击贴图按钮，以使用 Maya“纹理”(Texture)节点。

粘合强度(Glue Strength)

确定要打断该 nComponent 节点的约束链接的阻力。此值与整个场景或对象的比例相关。

“粘合强度”(Glue Strength)为 1 表示该约束不会断开。“粘合强度”(Glue Strength)为 0 表示该约束会立即断开，就好像根本不存在约束一样。默认值为 1。

由于链接在开始帧形成，一旦链接被打断，它们将无法重组，除非“粘合强度”(Glue Strength)属性设置回 1。

粘合强度贴图类型(Glue Strength Map Type) / 粘合强度贴图(Glue Strength Map)

“粘合强度贴图类型”(Glue Strength Map Type)确定此 nCloth 组件粘合强度贴图的类型。选择“无”(None)（无贴图），“逐顶点”(Per-vertex)（逐顶点应用贴图）或“纹理”(Texture)（使用“粘合强度贴图”(Glue Strength Map)属性所指定纹理贴图）。

“粘合强度贴图”(Glue Strength Map)指定要作用粘合强度贴图的纹理贴图。此属性仅当“粘合强度贴图类型”(Glue Strength Map Type)设置为“纹理”(Texture)时可用。键入纹理文件的路径，或单击贴图按钮，以使用 Maya“纹理”(Texture)节点。

权重(Weight)

确定此节点的组件对受其约束的其他组件影响如何。值为 0 时表示此组件可以被其他组件操纵（推、拉等），但不会对其他组件施加任何影响。

例如，如果希望节点的组件可以更多的控制动力学约束的行为，可以增加此节点的“权重”(Weight)。

权重贴图类型(Weight Map Type) / 权重贴图(Weight Map)

“权重贴图类型”(Weight Map Type)确定此 nCloth 组件的权重贴图类型。选择“无”(None)（无贴图），“逐顶点”(Per-vertex)（逐顶点应用贴图）或“纹理”(Texture)（使用“权重贴图”(Weight Map)属性指定的纹理贴图）。

“权重贴图”(Weight Map)指定作用权重贴图的纹理贴图。此属性仅当“权重贴图类型”(Weight Map Type)设置为“纹理”(Texture)时可用。键入纹理文件的路径，或单击贴图按钮，以使用 Maya“纹理”(Texture)节点。

切线强度(Tangent Strength)

确定当“组件类型”(Component Type)设置为“面”(Faces)时，运动在局部切线方向上的阻力。值在 0 到 1 范围内是有效的。默认值为 0.5。（此设置不会影响组件类型，与“面”(Faces)不同）

组件索引(Component Indices)

此截面以顶点、边或面填充，索引在创建此约束时对应的组件选择。当元素属性设置为“来自索引列表”(From Indice List)时，可以修改列出的索引。

例如，如果创建了一个“组件到组件”(Component to Component)约束，通过选择两个 nCloth 对象上的顶点，该约束点的顶点索引将显示在“组件索引”(Component Indices)区域。

注：

当“使用集”(Use Sets)选项处于启用状态时创建动力学约束（在“创建动力学约束选项”窗口中），“组件索引”(Component Indices)区域将不包含任何数据。
