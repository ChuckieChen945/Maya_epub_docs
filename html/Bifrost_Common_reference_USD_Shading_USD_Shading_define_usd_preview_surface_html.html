define_usd_preview_surface

此复合演示了在 Bifrost-USD 中创建材质的一种可能方法。其创建一个基本体，该基本体使用 UsdPreviewSurface 着色器定义基本材质。

输入

path

新基本体的路径，例如：/materials/my_material。如果此节点的输出连接到基本体定义下游的 children 端口，则该路径将被附加到该处指定路径的后面。

基础

diffuse_color

漫反射颜色或反照率。

texture_file_path

要使用的纹理文件的路径，而不是 diffuse_color。

emissive_color

发射的灯光颜色。

镜面反射

clearcoat

透明涂层的权重。

index_of_refraction

半透明对象和镜面反射对象的折射率。

metallic

金属度。金属对象用 1，电介质对象用 0。

opacity

材质的不透明度。

roughness

镜面反射的粗糙度。0 表示完全镜面反射，1 表示最大粗糙度。

其他输入

attribute_definitions

为材质连接一个或多个属性定义（可选）。可以在端口上单击鼠标右键，然后选择 “连接节点”(Connect Node) 以快速添加和连接 define_usd_attribute。

输出

material_definition

材质基本体定义。可以将其连接到 add_to_stage 节点的 prim_definitions 端口，或连接到其他 define_usd_prim 或类似节点的 children 端口。也可将其连接到 define_usd_prim 或类似节点的 material 端口，以将材质绑定到基本体。
