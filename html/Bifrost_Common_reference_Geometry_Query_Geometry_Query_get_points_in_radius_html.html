get_points_in_radius

返回输入几何体上位于输入位置指定半径内的点的位置。然后，可以使用 sample_property_2D 获取这些位置上的几何体的特性的插值。

输入

geometry

要在其上查找最近点的位置的几何体。仅支持基于点的几何体。

positions

要从中查找最近点的位置数组。

radius

从每个输入位置开始搜索的半径。

enable_max_number

限制每个位置返回的点数。

max_number

每个位置要返回的最大点数。

输出

locations

一组位置数组。每个数组包含距对应输入位置最近的点的位置，并按距离从最近到最远进行排序。可以将此输出插入到 sample_property_2D 节点的 locations 端口。

point_indices

一组包含对应点的索引的数组。可以使用这些索引从包含几何体特性数据的数组获取对应值。
