创建 iterate 循环

iterate 循环按顺序处理多个元素。例如，您可以使用 iterate 循环计算连接到迭代目标输入端口的一组值的乘积。

  1. 将 iterate 节点添加到图表中。

  2. （可选）可以通过绘制与“新端口”(New Port)图标的连接来创建所需的任何端口。右键单击端口以对其进行重命名或其他更改。

  3. 进入复合并为要在循环中执行的计算创建子图。

    • 如果您希望复合可以迭代数组中的项目，请在输入数组端口上单击鼠标右键，然后选择“设置端口迭代次数目标”(Set Port Iteration Target) >“true”。在循环的第 N 次迭代时，任何输入迭代目标的值均为连接数组中的第 N 个项目。
    • 如果已设置一个或多个输入迭代目标，则可以删除 max_iterations 端口。从而，您无需连接数组的大小，即可确保所有项目均已得到处理。
    • 还可以将输出端口设置为迭代目标。该值是一个数组，其中第 N 个项目为第 N 次迭代的结果。
    • 您不需要将输入数组端口设置为迭代目标，您还可以使用 max_iterations 端口直接控制迭代次数。
    • 要将中间值（例如部分总和）从一次迭代传递到下一次迭代，请使用一对“状态”端口。状态端口包含匹配的输入和输出端口，以便将输出值沿用为下一次循环迭代的输入。要创建一对状态端口，请右键单击复合的 input 或 output 节点中的某个端口，展开“设置端口状态”(Set Port State)菜单，然后选择匹配的端口。（可选）您可以通过在循环复合外部连接或设置一个值，来初始化第一次迭代的输入值。
    • iterate 循环中的所有输出都必须为迭代目标或状态端口。
