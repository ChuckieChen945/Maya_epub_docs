set_Arnold_strands_settings

设置 Arnold 特定的 strands（在 Arnold 中：curves）设置，以便在 Arnold 中作为 Arnold 原生设置应用。这些设置不影响视口，仅影响 Arnold 渲染。

这些设置与 Arnold 中的对应选项匹配。有关详细信息，请参见 Arnold 用户手册中的 curves 相关信息。

用法

它也可以与 set_Arnold_geo_settings 结合使用，后者用于设置对 Arnold 几何体通用的设置。

转化

match_ends 设置 Arnold 曲线的端点是否与 strands 匹配。为了使端点与曲线模式匹配，必须延伸端点。如果禁用此选项，则会按原样使用顶点。

Arnold 曲线

mode_override

Arnold 中的 mode 是指股渲染为摄影机面对的带状、已定向的带状还是有厚度。

Bifrost 具有类似的设置，默认值表示使用 Bifrost 设置的默认 Arnold 解释。其余设置会直接设置 Arnold 设置。

basis_override

Arnold 中的 basis 描述从控制点形成曲线的方式。

Bifrost（仅线性）strands 的默认转化是转化为 Catmull-Rom，具有匹配的端点 (match_ends)。可以使用此默认解释，也可以直接设置 basis。

override_min_pixel_width

Bifrost 股目前没有 min_pixel_width，但这可能会在未来发生变化，因此它作为一个替代公开。

有关 Arnold 中 min_pixel_width 的行为，请参见 Arnold curves 节点参考。
