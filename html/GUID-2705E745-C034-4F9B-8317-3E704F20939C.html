替换 nCache 帧选项(Replace nCache Frames Options)

不能为具有多个缓存节点的任何 Nucleus 对象替换 nCache 帧。

当前时间(Current time)

启用该选项后，将替换当前对象的缓存帧，并生成原始未更改缓存和描述文件的备份（默认情况下，命名为 backup#_originalCacheName）。

例如，如果要替换 nCache 中的帧 12，则通过该选项可以仅为帧 12 创建缓存，然后用新创建的缓存帧替换选定对象的当前缓存中的帧 12。“当前时间”(Current time)在默认情况下处于启用状态。

开始/结束(Start/End)

启用该选项后，将为当前对象的缓存替换从“开始替换帧”(Start replace fram)到“结束替换帧”(End replace frame)的帧范围，并会 成原始的未更改缓存和描述文件的备份（默认情况下，命名为 backup#_originalCacheName）。

例如，如果要替换 nCache 中的帧 10 到 20，则通过该选项可以为帧 10 到 20 创建缓存，然后用新创建的缓存替换选定对象的当前缓存中的帧范围。默认情况下“开始/结束”(Start/End)处于禁用状态。

“每 # 个帧求值一次”(Evaluate every # frame(s))和“每 # 次求值保存一次”(Save every # evaluation(s))

在创建 nCache 时，默认情况下会对每个帧提取对象模拟的一个采样并将其保存到 nCache 中。

“每 # 个帧求值一次”(Evaluate every # frame(s))指定在创建 nCache 期间提取采样的频率。例如，值为 2 会在“缓存时间范围”(Cache time range)的每隔一帧处缓存当前对象的模拟。

“每 # 次求值保存一次”(Save every # evaluation(s))指定在创建 nCache 期间保存哪些采样。例如，值 2 指定仅将“每 # 个帧求值一次”(Evaluate every # frame(s))选项指定的每隔一个采样保存到您的 nCache。

有关详细信息，请参见 nCache 创建。

使用模拟数据替换(Replace with simulation data)

如果启用，要替换的缓存帧将替换为新模拟数据。

删除备份文件(Delete backup files)

启用该选项时，将从磁盘上的 nCache 文件中删除要替换的缓存帧。该删除为永久性删除，并且无法通过编辑 > 撤消(Edit > Undo)或 Ctrl+z 进行撤消。

无插值(No interpolation)

启用该选项时，在对象的当前缓存帧与要替换的帧之间不会出现插值。

插值帧(Interpolate frames)

启用该选项时，在要替换的帧之前和之后的缓存帧将进行插值，以在对象的当前缓存帧与要替换的帧之间创建平滑过渡。

重命名插值帧(Rename interpolated frames)

启用该选项时，原始的 nCache 文件将重命名为在“重命名插值帧”(Rename Interpolated Frames)字段中指定的名称。

开始替换帧/结束替换帧(Start replace frame / End replace frame)

通过这些字段可以为要替换的缓存帧范围指定开始帧和结束帧。“开始替换帧”(Start replace frame)和“结束替换帧”(End replace frame)仅在启用“开始/结束”(Start/End)时可用。

开始插值帧范围/结束插值帧范围(Start interpolation frame range / End interpolation frame range)

通过这些字段可以为要插值的缓存帧范围指定开始帧和结束帧。通常，插值范围在要替换的帧之前开始，在要替换的帧之后结束。“开始插值帧范围”(Start interpolation frame range)和“结束插值帧范围”(End interpolation frame range)仅在启用“插值帧”(Interpolation frames)时可用。

入切线/出切线(In tangent/Out tangent)

通过这些下拉菜单可以为要替换的 nCache 帧之前（“入切线”(In tangent)）和之（“出切线”(Out tangent)）的缓存插值指定切线类型。当要在对象的替换缓存帧与其 nCache 的其余帧之间平滑插值时，该功能很有用。

步长(Step)

将从插值范围的第一帧向最后一帧对变形进行阶跃插值，以创建跳转到替换帧/从替换帧跳转的过渡。
