source_color_property

这样，用户可以轻松地将颜色特性添加到模拟源的其他特性，例如 source_particles 或 source_air 节点。

输入

property

这是要为模拟生成的颜色特性的名称。请注意，根据模拟的性质，创建的特性名称也可能具有 point_ 或 voxel_ 前缀。

base_color

这是发射期间要使用的颜色特性的值。它可以是 float3、array<float3> 或 vectorField。如果是 array<float3>，则数组的大小应与源的输入几何体匹配。如果源几何体具有 Maya 的 RGB 颜色集，则可以从源几何体执行 get_geo_property（类型为 array<float3>），并将其插入此端口。

输出

out_object

包含特性设置的对象。应将其插入到模拟源节点的 additional_properties 端口。
