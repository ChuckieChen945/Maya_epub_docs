find_all_true_in_array

计算其元素是输入布尔数组中值为 true 的元素的索引的数组。在从一个数组或一组数组中删除元素时，可以在计算映射时使用此项。indices 数组中的每个元素都对应于原始数组中要保留的一个元素的索引。

例如，假设您有多个数组：

[ 0, 10, 20, 30, 40, 50]
[ 0, 11, 21, 31, 41, 51]
[ 0, 12, 22, 32, 42, 52]

并且您希望基于以下数组的元素是否为负值来从其中移除元素：

[ 0, 1, -2, -3, -4, 5]

将该数组连接到 greater_or_equal 的 first 端口，并将 second 值设置为 0，将产生以下结果：

[true, true, false, false, false, true]

将该结果连接到 find_all_true_in_array 可提供要保留的索引：

[0, 1, 5]

最后，将每个原始数组与这些索引一起连接到 get_from_array 节点，可以产生所需的结果：

[ 0, 10, 50]
[ 0, 11, 51]
[ 0, 12, 52]

输入

boolean_array

布尔数组。每个 true 元素的索引将添加到输出 indices 数组。

输出

indices

boolean_array 中其元素值为 true 的元素的索引数组。请注意，此数组中的元素将按升序排序。

父页面: Array
