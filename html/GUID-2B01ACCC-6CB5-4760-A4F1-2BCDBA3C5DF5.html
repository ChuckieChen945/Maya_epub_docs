编辑最大影响

可以更改可影响蒙皮对象点数的关节或影响对象的数量。一般而言，绑定蒙皮时，可以在“平滑绑定选项”(Smooth Bind Options)中设定初始“最大影响”(Max influences)值。

编辑最大影响

  1. 选择要编辑其最大影响的蒙皮对象。
  2. 选择“蒙皮 > 编辑影响 > 设置最大影响”(Skin > Edit Influences > Set Max Influences)。
  3. 在“设置最大影响”(Set Max Influences)窗口中设定“最大影响”(Max Influences)。
  4. 单击“应用”(Apply)以设定新值。
