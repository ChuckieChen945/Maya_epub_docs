set_Arnold_mesh_settings

设置 Arnold 特定的 mesh（在 Arnold 中：polygons）设置，以便在 Arnold 中作为 Arnold 原生设置应用。这些设置不影响视口，仅影响 Arnold 渲染。

这些设置与 Arnold 中的对应选项匹配。有关详细信息，请参见 Arnold 用户手册中的 polygons 相关信息。

用法

它也可以与 set_Arnold_geo_settings 结合使用，后者用于设置对 Arnold 几何体通用的设置。

细分

设置与 Arnold 细分相关的各种设置。对于 3D 应用程序中常用的基本细分，将 subdiv_type 设置为 Catmull-Clark，并使用 subdiv_iterations 作为级别。

它还会设置 subdiv_adaptive_error、subdiv_adaptive_metric、subdiv_adaptive_space、subdiv_frustum_ignore、subdiv_smooth_derivs、subdiv_uv_smoothing。有关 Arnold 细分的详细信息，请参见“Arnold 节点参考”文档中的 polymesh。

置换

设置与 Arnold 置换相关的各种设置，包括 disp_autobump、disp_height、disp_padding、disp_zero_value。请注意，这不会指定置换着色器，它仅设置置换设置，为此，请使用 assign_material。

autobump_visibility_by_ray_type 是位掩码，请参见 Arnold_ray_bitmask 文档了解相关信息。

有关 Arnold 网格置换的详细信息，请参见“Arnold 节点参考”文档中的 polymesh。

法线(Normals)

smoothing 可用于禁用法线的平滑插值。
