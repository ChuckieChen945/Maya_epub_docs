mesh_to_level_set

此节点将输入网格几何体体素化为等值面（带符号距离场）。它还用于对网格点、边和面上存储的各种特性进行体素化。

输入

mesh

要进行体素化的网格几何体。

detail_size

指定体积中体素的（世界空间）大小。

volume_mode

在“实体”(Solid)模式下，网格被体素化为具有实心内部。在“壳”(Shell)模式下，网格被体素化为具有空心内部的厚曲面。

properties

要进行体素化的特性的列表（以空格分隔）。

adaptivity

用于将网格转化为体积的自适应性类型。

adaptivity_property

自适应性参数设置为“VariedFromProperty”时，指定自适应性的特性名称。

max_relative_error

自适应性设置为“Automatic”时，允许的相对于输出细节大小和边界框的最大采样误差。

resolution_fields

指定边界框（用于定义空间中的区域）和我们应细化到的深度级别。零表示最精细级别，值增加时，细节大小按 2 的幂相应增加。在任何级别都可以有多个边界。

volume_subdivision_structure

输出 Bifrost 体积的体积细分结构。

min_hole_radius

如果为正值，则启用实体体素化模式，此模式非常适合处理自相交点和小孔。在此模式下，将桥接半径小于 min_hole_radius 的间隙和孔。请注意，在 Shell 模式下，此项不发挥作用。

thickening

以体素单位指定输出体积上的加厚量，以避免在给定分辨率下不显示薄特征。

输出

level_set

对应于输入网格的等值面。
