自定义首选项文件夹的位置

默认情况下，Maya将首选项文件存储在以下路径中：

Windows

  • ..\Documents\maya\<版本>\prefs

Mac OS X

  • /Users/<用户名>/Library/Preferences/Autodesk/maya/<版本>/prefs
    • 注： 若要打开“首选项”(Preferences)目录，请执行以下操作：
    • 选择“查找器” > “转到” (Finder > Go)，按 Alt 键，Library 文件夹将显示在菜单中。
    • 从终端窗口导航到“首选项”(Preferences)目录，然后键入 open。

Linux

  • ~<用户名>/maya/<版本>/prefs

若要更改 Maya首选项文件夹的位置，请使用 MAYA_APP_DIR 环境变量。有关详细信息，请参见文件路径变量。必须使用操作系统（而不是 Maya.env）设置此环境变量。

注： 如果在保存首选项时磁盘空间不足，则首选项可能损坏或无法恢复。请确保硬盘驱动器上的可用空间足以保存首选项。如果磁盘空间不足，请释放一些空间，然后再退出应用程序。
