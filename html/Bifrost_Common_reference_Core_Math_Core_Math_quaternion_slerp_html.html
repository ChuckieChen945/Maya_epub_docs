quaternion_slerp

返回两个四元数之间的球面线性插值。

输入

from_quaternion

与零小数对应的源四元数。

to_quaternion

与完整小数对应的目标四元数。

fraction

插值参数。

clamp_negative

将负小数值钳制为 0。

clamp_above_one

如果小数值大于 1，则钳制为 1。

always_use_shorter_arc

始终使用两个旋转之间的较短弧。

use_nlerp_if_closer

如果四元数之差小于此容差，则在更接近的情况下，执行归一化线性插值。这省去了计算，并且当差值很小时更稳定。

输出

interpolated

插值四元数。
