create_usd_stage

创建新的 USD 后台文件。

输入

layer

将保存后台文件的文件路径。支持 usd（通用）、usda（ASCII 文本）和 usdc（创建二进制）格式。

file_format

将此文件保存为 USD 二进制文件还是 ASCII 文件。可以从 .usd 文件改为以 ASCII 文本形式写入（不更改文件名），以便调试 USD 输出。

sublayers

连接任何层以添加为新后台文件的子层。请注意，在 Bifrost-USD 中，第一个子层（索引 0）包含最弱的观点，这与使用 USD Python 或 C++ API 时子层的索引方式相反。

mask

连接基本体路径（字符串）数组，将后台文件填充限制为指定基本体及其子孙和祖先对象。请注意，由于后台文件遮罩未存储在 USD 层中，因此需要共享 Bifrost 后台文件才能在主机应用程序中查看效果。如果主机正在子分层或参照 Bifrost 后台文件的根层，则不会考虑遮罩。

load

控制 USD 有效负载圆弧的行为。

  • LoadAll：加载所有可载入的基本体。
  • LoadNone：记录有效负载圆弧，但不会遍历。

layer_index

目标层的索引。

输出

stage

新后台文件。
