如果您使用的是 macOS，除非另有指定，否则请使用以下特殊键，而不是其 Windows 等效键：
  • Alt = Option
  • Ctrl = Control
  • Windows 键 = Command
此外，如果您在 macOS 上使用单按钮或双按钮鼠标，请确保根据需要在界面首选项(Interface Preferences)中设置“鼠标跟踪”(Mouse Tracking)选项。
