rotate_points

按指定的旋转更新基于点的几何体的当前 point_orientation 值。这将影响实例的方向以及点形状（如果它们未与摄影机对齐）。但是，它不会影响整个几何对象的旋转。此外，此复合还会更新 point_normal、point_tangent 和 point_binormal 特性（如果存在）。

输入

points

具有要旋转的点的几何体。

weights

要应用的旋转的比例。

  • 权重为 0 时，不进行旋转。
  • 权重在 0 到 1 范围内时，按给定角度的某个比例进行旋转。
  • 权重为 1 时，旋转给定角度。
  • 权重大于 1 时，旋转角度大于给定角度。
  • 权重小于 0 时，按相反方向旋转。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

axis

旋转轴。无需归一化向量。

这是一个解释的自动端口，其接受以下类型：

  • 连接单个 float3 以将其值用于所有点。
  • 连接 array<float3> 以对每个点使用数组的相应元素。可根据需要调整数组大小以匹配点数。
  • 连接 string 以使用具有相应名称的特性给定的向量（每个点的 float3）。
  • 连接 VectorField 以使用点的当前位置之处的场的值。

请注意，也可以使用标量值而不使用向量，在这种情况下，将对所有三个轴使用单个元素。但是，这对指定轴几乎没有任何用处。

radians

旋转角度（以弧度为单位）。

local_space

控制如何组合新旋转和旧旋转。当此选项为 true 时，将在点的当前局部空间中应用随机化旋转，例如，围绕 Y 轴旋转就是围绕点当前方向的局部 Y 轴旋转。

输出

out_points

具有已旋转点的几何体。

point_orientation

在几何体上设置的 point_orientation 值。
