linear_interpolate_normalized

计算两个向量之间的线性插值并将结果归一化。

输入

first

初始向量。

second

最终向量。

fraction

插值小数。值为 0 时返回初始向量，值为 1 时返回最终向量，而其他任何值则返回连接两个向量的直线上的相对位置。

clamp_negative

如果为 true，则只要 fraction 输入小于 0，就会返回初始向量。

clamp_above_one

如果为 true，则只要 fraction 输入大于 1，就会返回最终向量。

输出

interpolated

归一化的插值向量。
