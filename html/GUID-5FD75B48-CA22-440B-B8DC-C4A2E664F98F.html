使用 componentMatch 节点创建顶点查找表

使用 componentMatch 节点自动创建提供给组件列表的查找表，这样您就不必创建自己的顶点表。componentMatch 节点使用算法来比较几何体，构建顶点之间的相关性，以及输出预配置的查找表。

如果您希望创建自己的查找表，请确保“变形选项”(Morph Options)中的“使用组件查找”(Use Component Lookup)选项处于禁用状态。

使用 componentMatch 节点创建顶点查找表

  1. 从由驱动者对象和变形对象组成的变形几何体开始。
  2. 在驱动者和变形几何体上设置组件标记。请参见为几何体指定组件标记。
    注： 在驱动者和变形几何体之间，组件必须具有相同数量的组件元素，否则无法自动生成表。如果多边形数不匹配，则 componentMatch 节点无法在形状之间切合实际地插值。
  3. 应用变形变形器，将驱动者对象的变形传递到要变形的对象。在变形选项(Morph Options)中，激活“使用组件查找”(Use Component Lookup)以生成 componentMatch 节点。
  4. 在“节点编辑器”(Node Editor)中，选择 componentMatch 节点以在“属性编辑器”(Attribute Editor)中打开“组件匹配属性”(Component Match Attributes)。
  5. 在“属性编辑器”(Attribute Editor)的“组件匹配属性”(Component Match Attributes)中的“标记表达式”(Tag Expression)字段中输入组件标记名称。（此字段支持表达式，因此您可以输入部分标记名称，例如，输入“*_left”以查找包含“left”的所有组件标记名称。）
  6. 按 Enter 键以应用组件匹配。将基于驱动者对象修改目标几何体。
