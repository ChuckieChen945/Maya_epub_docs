特定于平台的安装说明

在 Windows 上安装 Bifrost

  • Bifrost 的默认安装路径为 C:\Program Files\Autodesk\Bifrost\<MayaVersion>\<BifrostVersion>。

  • 要卸载 Bifrost，请转到 Windows“控制面板”，然后选择“程序和功能”。

  • 要以静默方式安装，请执行以下操作：

    1. 下载以 .exe 形式提供的安装程序，然后使用 7-Zip 等应用程序提取此归档。

    2. 运行以下命令之一，通过 .msi 以静默方式安装 Bifrost：
      msiexec /i Bifrost.msi /qn
      Bifrost.msi /passive（显示进度条，但无需交互）
      Bifrost.msi /quiet（完全静默选项）

在 macOS 上安装 Bifrost

Bifrost 的默认安装路径为 /Applications/Autodesk/bifrost/<MayaVersion>/<BifrostVersion>。

从适用于 Maya 2024 的 Bifrost 2.7.0.0 版开始，Bifrost 可以在 Apple Silicon 和 Intel Mac 计算机上本地运行：

  • 安装程序和安装的插件是通用二进制文件 (UB2)。
  • Bifrost 的 JIT 编译器 Amino 在 Apple Silicon 上运行时会生成本地 Arm64 代码。

在 Linux 上安装 Bifrost

Bifrost 的默认安装路径为 /usr/autodesk/bifrost/<MayaVersion>/<BifrostVersion>。

通过 Maya 单独获取的 Bifrost 安装程序采用 .run 文件形式，应按如下方式安装：

sudo sh ./Bifrost_2.2.0.0_Maya2020_Linux.run

按 Page Down 键阅读许可协议，然后根据提示单击“安装”或“提取”。

要以静默方式安装，请执行以下操作：

  1. 使用 chmod +x 运行 .run 文件。

  2. 提取文件以获取 rpm。

    ./Bifrost_2.5.0.0_Maya2022_Linux.run --tar xvf

  3. 使用 rpm --force，以便 Maya 自动识别最新安装。

    rpm -iv --force ./Bifrost2022_64-2022.2.5.0.0-2.5.0.0.x86_64.rpm
