加载工具架

实际工具架比默认情况下在用户界面中显示的工具架多得多。如果要使用在 UI 中未显示的工具架，请单击工具架图标 ，然后选择“加载工具架”(Load Shelf)。
将打开“加载工具架”(Load Shelf)窗口，其中显示 runtime/script/shelf 目录的内容。

Windows 和 Mac OS X 版的 Maya 包括了默认工具架，包含常用功能按钮。

Maya 未在 Linux 版中加入工具架，因为工具架会增加在该平台上的启动时间。但是，可以按以下说明加载这些工具架。

在 Linux 上将默认工具架添加到 Maya

  1. 打开“脚本编辑器”(Script Editor)（窗口 > 常规编辑器 > 脚本编辑器(Window > General Editors > Script Editor)）。
  2. 要加载所有工具架，请复制以下脚本并将其粘贴到脚本编辑器窗口中，然后选择“命令 > 执行”(Command > Execute)。如果不想加载所有工具架，请仅从下方列表中选择所需工具架。 
loadNewShelf "shelf_General.mel";
loadNewShelf "shelf_Curves.mel";
loadNewShelf "shelf_Surfaces.mel";
loadNewShelf "shelf_Polygons.mel";
loadNewShelf "shelf_Deformation.mel";
loadNewShelf "shelf_Animation.mel";
loadNewShelf "shelf_Dynamics.mel";
loadNewShelf "shelf_Rendering.mel";
loadNewShelf "shelf_Fluids.mel";
loadNewShelf "shelf_Fur.mel";
loadNewShelf "shelf_Hair.mel";
saveAllShelves $gShelfTopLevel;
