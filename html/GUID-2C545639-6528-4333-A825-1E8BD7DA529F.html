“静止形状”(Rest Shape)子菜单

将静止形状设置为初始形状(Set Rest to Start Shape)

将选定用于模拟的 nCloth 的静止形状设定为开始帧处的 nCloth 输入网格的状态。

将输入网格连接到静止形状(Connect Input Mesh to Rest Shape)

将选定用于模拟的 nCloth 的静止形状设定为当前帧处的 nCloth 输入网格的状态。

将选定网格连接到静止形状(Connect Selected Mesh to Rest Shape)

将选定用于模拟的 nCloth 的静止形状设定为另一个选定网格（其具有匹配拓扑）的状态。
