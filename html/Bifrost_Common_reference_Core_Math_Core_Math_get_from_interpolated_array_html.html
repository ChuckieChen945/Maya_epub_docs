get_from_interpolated_array

通过一个浮点数或浮点数组对传入数组进行插值（混合），并能够对传入数组进行切片和对生成的切片进行插值。

输入

array

要进行插值的传入数组

float_index

用于驱动插值的值 - 例如，如果您有一个颜色数组并希望跨平面的 X 位置混合这些颜色，则这将是平面的归一化 (0-1) X 位置的数组。

normalized_index

如果未选中此项，复合将在传入 array 中插值前两个条目。如果选中此项，将对整个传入数组进行插值。

on_slice

如果选中此项，可以对传入数组进行切片并在切片之间进行插值。

slice_start

要开始的传入数组的索引。

slice_end

要结束的传入数组的索引。

输出

interpolated

生成的插值数组
