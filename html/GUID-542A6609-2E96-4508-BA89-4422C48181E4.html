在 UV 集之间切换

用户可以轻松地切换获取应用于多边形模型的 UV 集。

在 UV 集之间切换

  1. 执行下列操作之一：
  • 在“UV 编辑器”(UV Editor)中，选择“UV 集 > UVsetName”(UV Sets > UVsetName)。
  • 在场景视图中，在网格上单击鼠标右键，然后选择“UV 集(UVsetName)”(UV Sets > (UVsetName))。
  • 在主菜单中，选择“UV > 设置当前 UV 集”(UV > Set Current UV Set)。在该窗口中，键入所需的 UV 集的名称。
