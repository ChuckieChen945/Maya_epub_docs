identity

Returns the multiplicative identity of the input type.

  • For square matrices, the result is the multiplicative identity for matrix_multiply, that is, the identity matrix I.
  • For vector4, the result is the multiplicative identity for multiply_quaternions, that is, the value (0, 0, 0, 1).
  • For Core::Transform::Transform, the result is a Transform struct with all of its matrix components set to identity.

Inputs

type

The desired output type. The actual value is not used.

Outputs

identity

The multiplicative identity of the given type.
