平滑网格预览概述

“平滑网格预览”(Smooth Mesh Preview)可用于快速、轻松地查看多边形网格在平滑后的外观。

在不同的平滑网格预览模式之间循环

  • 选中对象后，按 1、2 或 3

默认情况下，当“平滑网格预览”(Smooth Mesh Preview)处于禁用状态时，网格会自动显示。当“平滑网格预览”(Smooth Mesh Preview)处于启用状态时，您可以同时显示原始网格和平滑预览（“框架 + 平滑网格”(Cage + Smooth Mesh)显示模式），也可以显示平滑预览本身（“平滑网格”(Smooth Mesh)显示模式）。

此模式非常适用于在执行实际平滑网格操作之前调整网格，这是因为您对原始网格所做的任何修改均会自动更新平滑预览。但是，请记住，“平滑网格预览”(Smooth Mesh Preview)仅更改网格的显示，便于您可视化网格在平滑后的外观。不会修改或平滑处理原始网格。

例如，如果在“平滑网格预览”(Smooth Mesh Preview)处于启用状态时渲染网格，网格将以原始的未平滑状态显示在最终图像中。如果要在渲染中显示已平滑的网格，您可以将其转化为完全平滑的网格，或从平滑网格预览创建细分曲面代理。请参见将平滑网格预览转化为多边形和细分曲面代理选项(Subdiv Proxy Options)。

默认情况下，“平滑网格预览”(Smooth Mesh Preview)使用 OpenSubdiv Catmull-Clark 细分方法，该方法将对网格面应用统一的细化方案。有关详细信息，请参见 OpenSubdiv 概述。

注： 缓存平滑网格可能会导致性能下降并占用大量内存。可以使用缓存播放首选项(Cached Playback Preferences)中的“缓存平滑网格”(Cache Smooth Meshes)选项来禁用“平滑网格预览”(Smooth Mesh Preview)。
