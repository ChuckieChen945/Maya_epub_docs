HumanIK 角色控制模板

三种不同的角色控制模板：A.CharacterControlsConfig.xml、B.CustomRigControlsConfig.xml 和 C.FullRigControlsConfig.xml

Maya 附带了 3 种可用于定义骨架的 HumanIK 角色模板。您还可以使用基本文本编辑器（例如记事本或 Programmer's Notepad）修改模板以符合您的需求。

若要加载其中一个角色控制布局，请执行以下操作：

  1. 选择 >“编辑控制绑定 > 加载 UI 配置”(Edit Control Rig > Load UI Configuration)
  2. 在此处导航：
    • Windows：C:\Program Files\Autodesk\<Maya 版本>\resources\CharacterControls
    • Mac OS X： /Applications/Autodesk/<Maya 版本>/Maya.app/Contents/resources/CharacterControls
    • Linux：/usr/autodesk/Maya/resources/CharacterControls
      • 注：

      默认情况下，在安装期间，将在 CharacterControls 目录中创建角色控制模板。这些文件用于创建“控制”(Controls)和“自定义绑定”(Custom Rig)选项卡的默认布局。

有关自定义 HumanIK 界面的详细信息，另请参见自定义 HumanIK 角色界面、对自定义绑定角色定义背景进行自定义和在 HumanIK“自定义绑定”(Custom Rig)界面上重新定位效应器。

CharacterControlsConfig.xml

CharacterControlsConfig.xml 文件是 Maya 中的默认模板。它包含常用的骨架定义。

CustomRigControlsConfig.xml

CustomRigControlsConfig.xml 是一个高级模板，具有更多选项，以及用于其他手指、脚趾等的扩展窗口。

FullRigControlsConfig.xml

FullRigControlsConfig.xml 模板是一个扩展模板，将 CustomRigControlsConfig.xml 的所有附加窗口与附加的肩部骨骼以及所有手指和脚部骨骼组合在一起。

CharacterControlsConfig_T 和 CustomRigControlsConfig_T 文件

角色控制模板 A.CharacterControlsConfig 和 CharacterControlsConfig_T B.CustomRigControlsConfig 和 CustomRigControlsConfig_T

在“resources > Character Controls”文件夹中找到的 CharacterControlsConfig_T 和 CustomRigControlsConfig_T 文件是 CharacterControls 文件中模板的版本，但表示的是 T 形站姿或基本姿势，有时会成为动画师的首选。

注：

有时，T 姿势版本的角色表示在窗口中偏移或不对齐。如果发生这种情况，请将 HIK 面板调宽，然后单击“创建骨架”(Create Skeleton)。
