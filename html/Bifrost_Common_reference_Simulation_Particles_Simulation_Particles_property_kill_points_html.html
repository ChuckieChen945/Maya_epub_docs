property_kill_points

此项可用于根据指定点特性以及阈值从点对象移除点。在从一个系统向另一个系统发射粒子的情况下，此项非常有用。例如，可以在向另一个粒子系统中的源传递粒子之前先移除低于定义年龄的所有点，以便只发射年龄较大的粒子。或者，可以通过结合使用 point_velocity 和浮点阈值，移除高于或低于阈值速度的点。

输入

object

此项为要从其移除点的点几何体。

property

此项为要用于控制移除哪些点的特性的名称。它应为几何特性，例如粒子系统的 point_age、point_velocity 或 point_size。如果需要，可以忽略名称中的 point_ 前缀。

threshold

指定特性等于或大于此阈值的点将被移除。这可以是 float、float3、int、int3、scalarField 或 vectorField。对于 float3 和 vectorField，如果 x、y 和 z 中的任何一项超过阈值，则视为高于阈值。如果特性为 float3，但具有浮点阈值，则使用 float3 的长度。如果特性为 float4，则将其转化为 float3 rotation_vector 以进行比较。

reverse

如果启用此项，则指定特性低于阈值时点将被移除。

输出

out_object

输出为高于定义的阈值的点已被移除的对象。
