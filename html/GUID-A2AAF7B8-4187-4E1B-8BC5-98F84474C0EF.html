创建角色集(Create character sets)

您可以从“关键帧”(Key)菜单、Trax 编辑器或关系编辑器创建角色集。

从关键帧(Key)菜单创建角色集

  1. 选择要包含在角色集中的对象。
  2. 根据是要设定创建选项还是使用已在“创建角色集选项”(Create Character Set Options)中设定的选项，执行下列任一操作：
    • 从关键帧(Key)菜单中，选择“关键帧 > 创建角色集”(Key > Create Character Set)。
    • 从关键帧(Key)菜单中，选择“关键帧 > 创建角色集”(Key > Create Character Set) > 。

      在“创建角色集选项”(Create Character Set Options)中，设定所需的选项并单击“创建角色集”(Create Character Set)。请参见角色集。

从“Trax 编辑器”(Trax Editor)中创建角色集

    注： 不必手动在“Trax 编辑器”(Trax Editor)中创建角色集。创建动画对象的片段时，系统将在“Trax 编辑器”(Trax Editor)中自动为对象片段创建角色集。
  1. 选择要添加到新角色集中的对象。
  2. 在“Trax 编辑器”(Trax Editor)中，选择“创建 > 角色集”(Create > Character Set) > 。

    将出现“创建角色集选项”(Create Character Set Options)窗口。请参见角色集。

  3. 设定选项并单击“创建角色集”(Create Character Set)。

    角色集将创建，并且选定对象将由该集的某个成员生成。

从“关系编辑器”(Relationship Editor)中创建角色集

  1. 也可以在“角色编辑”(Character Editing)模式中使用“关系编辑器”(Relationship Editor)时创建角色（选择“窗口 > 关系编辑器 > 角色集”(Windows > Relationship Editors > Character Sets)）。选择要包含在角色集中的对象，然后在“关系编辑器”(Relationship Editor)中选择“编辑 > 创建角色集”(Edit > Create Character Set)。
