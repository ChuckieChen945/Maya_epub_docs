绘制簇变形器权重

可以使用“绘制簇权重工具”(Paint Cluster Weights Tool)绘制簇变形器权重。另请参考编辑簇变形效果主题中的“关于绘制簇权重”部分。

也可以使用“镜像变形器权重选项”(Mirror Deformer Weights Options)镜像簇变形器权重。请参见镜像变形器权重。

在簇上绘制权重

  1. 选择包含要在其上绘制权重的簇的曲面。
  2. 选择“着色 > 对所有项目进行平滑着色处理”(Shading > Smooth Shade All)或按默认热键 5 以切换到平滑着色模式。
  3. 在“装备”(Rigging)菜单集中，选择“变形”(Deform) >（“绘制权重”(Paint Weights)）“簇”(Cluster) > 打开“绘制簇权重工具”(Paint Cluster Weights Tool)设置。
    注： “绘制簇权重工具”(Paint Cluster Weights Tool)会自动检测曲面上的簇，然后选择一个进行绘制。
  4. 在“显示”(Display)区域中，启用“颜色反馈”(Color Feedback)。颜色反馈可帮助识别曲面上的权重，方法是将权重表示为灰度值（值越小越暗，值越大越亮）。
    提示：

    可以使用默认热键 Alt+C（Linux 和 Windows）或 Option+C (Mac OS X) 来在“绘制属性工具”(Paint Attributes Tool)外部启用和禁用“颜色反馈”(Color Feedback)。

  5. 选择要绘制的簇。在“绘制属性工具”(Paint Attributes Tool)的“绘制属性”(Paint Attributes)区域中，单击 clustern.weights 按钮，然后从弹出菜单中选择适当的簇权重名称。

    请注意，一次仅可以在一个簇上绘制权重。如果选择多个簇，则只能在活动簇（提供颜色反馈）上绘制权重。

    如果曲面只有一个簇，则可以单独选择曲面。

    选定簇变为白色，表示权重值为 1（默认值）。

    提示：

    如果在单个曲面上绘制，可以在不打开“工具设置”(Tool Settings)窗口的情况下选择簇，方法是右键单击曲面并从“绘制”(Paint)命令子菜单中选择适当的簇权重名称。

  6. 选择笔刷、绘制操作和值，并根据需要定义其他设置。
  7. 在簇上拖动笔刷。
    提示： 可以从一个簇快速拾取权重值，并使用热键在另一个簇或同一个簇上绘制这些权重值。
    • 选择具有要拾取权重值的簇。
    • 按住“拾取颜色模式”(Pick Color Mode)热键（默认热键：/），单击具有要拾取权重的簇的区域，然后释放热键。
    • 如果在同一个簇上绘制拾取的权重，请在簇上拖动笔刷。

    如果在另一个簇上绘制拾取的权重，请选择该簇，然后在其上拖动笔刷。
