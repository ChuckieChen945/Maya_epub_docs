软选择 UV

使用“软选择”(Soft Selection)，您可以一次性地平滑变换模型上的纹理，而无需逐个变换各个 UV。

软选择 UV

  1. 选择网格和要变换的 UV。
  2. 将组件选择模式更改为 UV。
  3. 打开 UV 编辑器。
  4. 在场景视图或 UV 编辑器中，选择要变换的一个或多个 UV。
  5. 按键盘上的“B”键。

    将启用“软选择”(Soft Selection)模式。

  6. 按住 B 键的同时使用鼠标中键左右拖动来增大或减小衰减区域。

    光标将变为一个圆形，表示衰减区域的大小。

  7. 使用一种变换工具在 UV 编辑器中变换 UV。

UV 上的软选择显示有色反馈以提供加权衰减的可视表示（请参见修改软选择衰减）。Maya 会自动缩放场景视图中的反馈，以便精确地表示正在网格曲面上变换的纹理的区域。

如果在场景视图中调整衰减区域的大小（请参见更改衰减区域的大小），Maya 会相应地自动在 UV 编辑器中缩放衰减区域，反之亦然。

注：

与处理其他组件不同，处理 UV 时，将“衰减模式”(Falloff mode)设置为“全局”(Global)并不允许您跨多个网格平滑地更改 UV。
