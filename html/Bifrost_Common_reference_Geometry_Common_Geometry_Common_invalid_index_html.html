invalid_index

Bifrost 几何体（例如 mesh 和 strands）使用无符号 32 位整数作为索引，这允许使用大约 40 亿个索引。许多几何体算法主要与查找或计算此类索引相关，在这些算法中，将一个值标记为尚未找到或尚未计算可能会很有用。这可以通过使用一组单独的布尔值来实现，但为了简便起见或提高性能，通常仅使用一个数组，但保留一个值为“无效索引”。

这称为“警示”值 - 一组整数中的一个值，它不表示常规值，而是表示值无效或没有值。如果对该特殊值使用零，则意味着零不能表示有效值。

很常用的警示值是所用类型中可能的最大值。这意味着可以编制索引的值数量仅减少一个。此节点可用作常量来设置或比较值，与数值常量相比，它更易于读取且更不易出错。

无效索引值最大为 2^32 - 1（即 4,294,967,295），这是可能的最大无符号 32 位整数。
