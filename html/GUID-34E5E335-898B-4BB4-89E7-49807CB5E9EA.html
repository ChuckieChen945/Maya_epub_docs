复制器

复制 MASH 网络并允许在复制者上执行变换。

复制器

复制者(Replicants)
确定 MASH 网络的复制次数。
变换空间(Transformation Space)
偏移位置 X (Offset Position X)、偏移位置 Y (Offset Position Y)、偏移位置 Z (Offset Position Z)
确定复制者之间的间距。
位置渐变(Position Ramp)
使用渐变自定义复制者位置。
一起缩放(Scale Together)
从轴一起缩放所有点。
缩放复制 X (Scale Replicants X)、缩放复制 Y (Scale Replicants Y)、缩放复制 Z (Scale Replicants Z)
缩放复制者之间的空间。
缩放渐变(Scale Ramp)
使用渐变自定义缩放形状。
一起旋转(Rotate Together)
从轴或驱动旋转一起旋转所有点。
旋转复制者 X (Rotate Replicants X)、旋转复制 Y (Rotate Replicants Y)、旋转复制 Z (Rotate Replicants Z)
绕共享中心旋转复制者。
旋转渐变(Rotation Ramp)
使用渐变自定义旋转形状。
旋转枢轴(Rotate Around)
设置复制者将绕其旋转的点。
偏移(Offset)
偏移复制者的编号。效果类似于先添加其他不可见的复制者，然后再添加实际复制者时发生的情况。
步骤 ID (Step ID)
随每个复制者增加 ID，而不是每个对象。

模式

可用于为编号为 n 的倍数的复制者添加额外偏移。

影响位置(Affects Position)、影响旋转(Affects Rotation)、影响缩放(Affects Scale)
指定受图案影响的通道。
图案(Pattern)
指定要将偏移应用于每 n 行。
图案偏移(Pattern Offset)、图案缩放(Pattern Scale)、图案旋转(Pattern Rotation)
设置要应用于每 n 行的变换偏移。

时间机器(Time Machine)

可用于偏移每个复制者的动画。

时间偏移(Time Offset)
设置要偏移每个复制者的动画的帧数。

曲线

可用于沿曲线排列复制者并设置其动画。

曲线(Curve)
显示将沿其均匀分布复制者的曲线。
连接曲线(Connect Curve)
将选定曲线连接到“曲线”(Curve)属性。
时间滑块(动画) (Time Slide (Animation))
确定是否要沿曲线设置复制者的动画。
曲线缩放(Curve Scale)
调整“时间滑块(动画)”(Time Slide (Animation))的比例，以适应小/大曲线。
上方向向量(Up Vector)
定义复制者的正面方向。
旋转偏移(Rotational Offsets)
将偏移添加到旋转。用于手动调整复制者的朝向。
使用曲线(Use Curve)
启用/禁用曲线的效果。

受驱动通道

可用于指定另一个 Waiter 以驱动复制者的位置、旋转和缩放。

驱动位置(Driving Position)、驱动旋转(Driving Rotation)、驱动缩放(Driving Scale)
显示目前驱动复制者的相应通道的 Waiter。

强度(Strength)

当“融合类型”(Blend Type)设置为“强度”(Strength)时，可以使用下列选项手动调整融合。

强度(Strength)
控制融合量。值越高，表示子网络影响越大。
随机强度(Random Strength)
控制添加到融合的随机度量。
阶跃强度(Step Strength)
每次禁用一个对象的节点效果。
强度贴图(Strength Map)
确定输入文件（2D 纹理、动画纹理等）来控制该节点的效果的形状。
贴图投影轴(Map Projection Axis)
确定投影“强度贴图”(Strength Map)时所沿的轴。
贴图辅助对象(Map Helper)
显示用于在场景中交互放置“强度贴图”(Strength Map)的对象。可以在该字段上单击鼠标右键来创建新的辅助对象（平面）（如果不存在）。还可以使用鼠标中键将网格拖入此字段，或者单击鼠标右键连接选定网格。如果网格已连接，还可以单击鼠标右键来断开其连接或者将其显示在大纲视图中。
注： 为获得最佳结果，请将相同的纹理指定给“强度贴图”(Strength Map)和“贴图辅助对象”(Map Helper)（在创建新的辅助对象时将自动执行此操作）。
