bake_instanced_geometry

将实例转化为非实例化几何体。它可从输入上实例化的所有几何体构建组合网格、组合股或组合点云。

输入

instances

要烘焙的实例。

prefer_preview_geometry

如果选中此项，则当实例上同时存在 render_geometry 和 preview_geometry 时，将使用实例的预览几何体。如果未选中此项，则首选 render_geometry。如果实例对象上缺少 preview_geometry 或 render_geometry，则无论此设置为何，都将使用另一个。

properties

要传递到烘焙几何体的特性。

输出

merged_meshes

组合了输入上所有实例化网格的网格。

merged_strands

由输入上所有实例化股组成的股对象

merged_points

由输入上所有实例化点组成的点云
