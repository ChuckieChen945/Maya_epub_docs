interpret_auto_port_as_transform

将各种输入转化为 double4x4 类型。使用此工具节点可以构建支持多种输入类型的可重用复合，以执行要对变换矩阵执行的任何类型的操作。例如，变换几何体对象的点。

若要使用此节点，请将 interpreted_transform_port 连接到复合中 auto 类型的输入，然后使用输出 transform 在复合内执行任何类型的矩阵操作。

支持以下多种可能的输入类型：

  • 将单个 float4x4 升级为 double4x4。

  • 按原样传递单个 double4x4。

  • 单个 Transform 返回其 world_matrix 分量。

  • array<float4x4> 解释为变换堆栈，并且按顺序缩减并转换为单个 double4x4 变换矩阵。

  • array<double4x4> 解释为变换堆栈，并且按顺序缩减为单个 double4x4 变换矩阵。

  • array<Transform> 解释为变换堆栈，并且其 world_matrix 分量按顺序缩减为单个 double4x4 变换矩阵。

  • object 返回对象的 Transform 类型的 transform 特性。如果 transform 特性不存在，则 valid 为 false，并且 transform 将输出单位矩阵。

输入

interpreted_transform_port

将此端口连接到复合的 auto 输入，以支持不同类型的输入。从这个端口创建新端口时，还会向该端口添加解释的自动类型调整。

输出

transform

将输入 interpreted_transform_port 转换为 double4x4 变换矩阵的结果。如果结果变换矩阵无效，则此项将输出单位矩阵。

valid

输出 transform 是否有效。如果无效，则 transform 将返回单位矩阵。
