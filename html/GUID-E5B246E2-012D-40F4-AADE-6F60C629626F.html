定位 2D 纹理和标签

您可以在 place2dTexture 节点的“属性编辑器”(Attribute Editor)中通过数值精度重新定位 2D 纹理，也可以使用 Maya 的“纹理放置工具”(Texture Placement Tool)以交互式方式重新定位纹理或标签。

有关定位 place2dTexture 节点的纹理的详细信息，请参见 2D 纹理位置。

以交互式方式定位 2D 纹理

  1. 在“Hypershade”中，单击连接到要重新定位的纹理的 place2dTexture 样例。

    将显示 place2DTexture 的“属性编辑器”(Attribute Editor)

    提示：
    如果无法看到 place2dTexture 样例，请执行以下操作：
    • 选择纹理样例，然后单击“Hypershade”工具栏上的“输入连接”(Input Connections)按钮。
    • 如果不存在 place2DTexture 样例，则已删除或未创建该样例（因为默认情况下禁用其自动创建）。必须添加一个节点（请参见在 Hypershade 中创建和连接节点），然后将它连接到纹理（请参见通过创建连接线在 Hypershade 中连接节点），再重新定位它。
  2. 执行下列操作之一：
    • 在“属性编辑器”(Attribute Editor)中更改属性的值。有关这些属性的详细描述，请参见 2D 放置。
    • 对于 NURBS，单击“属性编辑器”(Attribute Editor)中的“交互式放置”(Interactive Placement)按钮，以使用“纹理放置工具”(Texture Placement tool)。若要设定纹理放置工具，请参见纹理放置与标签映射。
      提示：

      可能很难只通过使用操纵器来放置纹理。可同时使用操纵器和“属性编辑器”(Attribute Editor)来获得所需结果。
