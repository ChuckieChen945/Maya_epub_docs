Suites 特定的环境变量

以下变量仅在 Maya 用作 Autodesk Entertainment Creation Suites 的一部分时适用。

SYNHUB_CONFIG_PATH
如果要在断开网络时运行 Maya，请将此变量设置为 Maya .syncfg 文件的位置。必须设置该变量才能启用“发送到”(Send to)命令。
默认情况下，安装期间在以下位置创建 .syncfg 文件：
  • Windows XP

    C:\Documents and Settings\All Users\Application Data\Autodesk\Synergy

  • Windows Vista 和 Windows 7

    C:\Program Data\Autodesk\Synergy

  • Mac OS X

    /Applications/Autodesk/Synergy

  • Linux（64 位）

    /opt/Autodesk/Synergy

MAYA_SYNSUITE_CONFIG_PATH
如果要在断开网络时运行 Maya，请将此变量设置为 Maya .synsuite 文件的位置。必须设置该变量才能启用“发送到”(Send to)命令。
默认情况下，安装期间在以下位置创建 .synsuite 文件：
  • Windows XP

    C:\Documents and Settings\All Users\Application Data\Autodesk\Synergy

  • Windows Vista 和 Windows 7

    C:\Program Data\Autodesk\Synergy

  • Mac OS X

    /Applications/Autodesk/Synergy

  • Linux（64 位）

    /opt/Autodesk/Synergy

ONECLICK_TEMP_DIR
指定活动角色在两个不同工作站之间流动时，文件所在的公用网络存储库的位置。
