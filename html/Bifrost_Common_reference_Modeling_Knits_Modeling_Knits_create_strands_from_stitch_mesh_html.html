create_strands_from_stitch_mesh

从缝合网格创建股。

输入

stitch_mesh

要使用的缝合网格。

strand_thickness_scale

股的缩放。

point_thickness_scale_name

点厚度比例特性的名称。

use_normal_property

如果为 true，则将使用法线特性。

use_face_vertex_normal

如果为 true，则将使用面顶点法线。

输出

strands

创建的股。

errors

操作失败时的错误消息。

local_strand_endpoints

局部股端点。

point_mesh_location

点网格位置。

endpoint_mesh_location

端点网格位置。

point_normal_offset

点法线偏移。

endpoint_normal_offset

端点法线偏移。

point_template_info

点模板信息。

endpoint_template_info

端点模板信息。
