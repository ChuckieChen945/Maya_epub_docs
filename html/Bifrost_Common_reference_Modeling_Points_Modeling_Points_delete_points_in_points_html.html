delete_points_in_points

从点对象（例如，粒子系统）中移除点。

输入

points

要删除点的点对象。

points_to_delete

要删除的点。此端口接受与 interpret_auto_port_as_component_tag 相同的输入：

  • array<bool> 解释为每个点的值。数组将会传递，但可调整大小以匹配 points 输入上的元素数。
  • array<long> 或 array<uint> 解释为点索引。
  • string 解释为现有标记的名称或标记表达式。有关标记表达式语法的更多详细信息，请参见有关 resolve_tag_expression 的文档。

properties

要传递到新几何体的几何特性的名称，例如 point_color。不是几何特性的特性将始终进行传递。

  • 使用空格分隔多个几何特性名称。
  • 您可以在表达式中使用通配符 *，例如使用 point_user* 来传递所有以字符串“point_user”开头的几何特性。
  • 默认值 * 将传递所有几何特性。
  • 使用空字符串可阻止传递任何几何特性，但 point_position 除外，因为它是必需的。
  • 使用 * 后跟 ! 可排除特性。例如，* !point_user* 将包括以字符串“point_user”开头的特性以外的所有特性

输出

out_points

新的点对象，不包含已删除的点。
