绘制变形器集成员身份

可以使用“绘制集成员身份工具”(Paint Set Membership Tool)，通过绘制希望添加到集、传递到集或从集中移除的点，修改可变形对象的哪些点（例如，CV 或顶点）属于多个变形器集。

修改哪些顶点属于集

  1. 选择已变形或已蒙皮的对象。
  2. 选择“着色 > 对所有项目进行平滑着色处理”(Shading > Smooth Shade All)（默认热键：5）以切换到平滑着色处理模式。
  3. 在“建模”(Modeling) (F2)、“绑定”(Rigging) (F3) 或“动画”(Animation) (F4) 菜单集中，选择“变形 > 变形器集(旧版) > 绘制成员身份”(Deform > Deformer Sets (Legacy) > Paint Membership) > 。

    将显示“绘制集成员身份工具设置”(Paint Set Membership Tool Settings)。请参见绘制集成员身份工具(Paint Set Membership Tool)。

  4. 设定这些工具设置。
  5. 选择要修改的集，如下所示：
    • 在“绘制集成员身份工具”(Paint Set MembershipTool)的“绘制操作”(Paint Operations)部分中，从“选择要修改的集”(Select Set To Modify)列表中选择一个集。选定的集名称将显示在“要修改的集”(Set To Modify)框中。
    • 使用“拾取颜色模式”(Pick Color Mode)热键（默认热键：/）来选择曲面上的集。按住该热键，单击要绘制的集（单击上色区域上的任意位置），然后释放该热键。
  6. 在要添加到、传递到或从选定集中移除的 CV 或顶点上拖动。
