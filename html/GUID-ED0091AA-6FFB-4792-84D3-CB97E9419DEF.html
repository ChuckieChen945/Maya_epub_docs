设定流体的初始状态

将流体的当前动态栅格属性值设定为初始状态

  1. 将属性值添加到流体容器中。

    请参见向流体容器添加属性。

  2. 如有必要，请使用要设定（保存）为初始状态的流体属性值在帧中播放和停止模拟。
  3. 选择流体容器。
  4. 选择“字段/解算器 > 初始状态 > 设置初始状态”(Fields/Solvers > Initial State > Set Initial State)，或者单击 。
  5. Maya 将在临时目录中创建隐藏的初始状态缓存节点。保存场景时，Maya 将创建初始状态缓存节点和磁盘缓存文件，并将该文件存储在项目的数据目录中。
    • 初始状态缓存文件已命名为 filename_fluidShape.mcfi。
    • 该节点已命名为 initialState_fluidShape。
