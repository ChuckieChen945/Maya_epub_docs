刷新“属性编辑器”(Attribute Editor)模板

如果已更新模板，则需要将其刷新，才可将“属性编辑器”(Attribute Editor)与更改同步。

刷新“属性编辑器”(Attribute Editor)模板

  1. 保存更新的 XML 文件。
  2. 将 MEL 程序 refreshCustomTemplate 输入到“脚本编辑器”(Script Editor)或“命令行”(Command Line)。
    属性编辑器将刷新。
    提示： 可以将该 MEL 命令指定为自定义工具架项目，以便可以轻松进行访问。但是，选择此工具架项将仅更新在“属性编辑器”(Attribute Editor)中显示的当前模板。

还原为默认“属性编辑器”(Attribute Editor)模板

  1. 在“属性编辑器”(Attribute Editor)中，选择以下选项之一：
    • 显示 > 设置全局视图 > 默认值(Show > Set Global View > Default)
    • 显示 > 设置当前视图 > 默认值(Show > Set Current View > Default)
