transform_joints

变换所有关节或任何关节，并将变换更改传播到其各自的所有子孙对象。

输入

skeleton

要变换其关节的骨架。

transforms

包含用于变换关节的变换值的 array<double4x4>。

joints_to_transform

包含要变换的关节的索引值的 array<long>。

如果此数组为空，则将更新所有关节，假定输入 transforms 数组大小与 joint_component 计数匹配。

enable

如果启用此项，则将处理关节变换。否则，将跳过任何计算并按原样输出输入 skeleton。

method

定义应如何解释 transforms 输入矩阵以及如何更新关节变换。

  • From Pivot：使用新的枢轴矩阵重新计算所有变换矩阵分量。

  • From Operator：使用新的运算符矩阵重新计算非枢轴变换矩阵分量。

  • From Local：计算运算符矩阵，并直接使用局部空间动画结果重新计算移动矩阵。

  • From World：计算运算符矩阵，并直接使用世界空间动画结果重新计算移动矩阵。在使用反向运动学时，通常使用此方法。

overwrite

如果启用此项，则使用输入 transforms 覆盖选定的变换分量。否则，将 transforms 乘以从 method 选择的现有变换分量。

输出

out_skeleton

更新的骨架，包含 8 个 array<double4x4> 关节变换几何特性。
