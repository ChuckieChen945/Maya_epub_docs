Arnold_ray_bitmask

与 set_Arnold_* 节点结合使用，后者设定 Arnold 特定的几何体设置。在 Arnold 中，有几个根据光线类型设置的可见性设置，包括 autobump、sidedness 和 visibility。它们被编码为位掩码。此节点允许构建这样的掩码。输出可以与 set_Arnold_*_settings 节点结合使用，以便按光线类型进行适当的设置。

这些设置被标记为“by_ray_type”，例如 set_Arnold_mesh_settings 参数 autobump_visibility_by_ray_type。光线类型包括 camera_rays、indirect_diffuse_rays、indirect_specular_rays、indirect_volume_rays、refracted_rays、shadow_rays、subsurface_scattering_rays、transmitted_diffuse_rays。有关它们在 Arnold 中的作用的详细信息，请参见 Arnold 的用户和 API 文档。

请参见：https://docs.arnoldrenderer.com/display/A5AFHUG/Visibility

在 set_Arnold_geo_settings 上，通常使用“visibility”，并且位掩码的构建内置到该节点中。
