sample_texture_file

对磁盘上的纹理文件进行采样。

输入

filepath

将读取和采样的图像文件的路径。支持某些文件名模式：

  • filename_<uvtile>.tif
    • 将在磁盘上查找以下格式的文件：filename_u0_v0.tif
  • filename_<UDIM>.tif
    • 将在磁盘上查找以下格式的文件：filename_1001.tif

file_color_space

文件的颜色空间。如果保留为默认空字符串，则节点将尝试使用 OpenColorIO 自动确定颜色空间。像素数据将从此颜色空间转化为场景相关的“工作颜色空间”（也称为“渲染颜色空间”）。使用“raw”可忽略任何颜色空间转化。

uvs

要对纹理进行采样的 UV 坐标列表。

sampling

采样策略：

  • 最近(Closest)：最近纹理元素。
  • 双线性(Bilinear)：纹理内的双线性插值。
  • 双三次(Bicubic)：纹理内的双三次插值。

输出

sampled_data

采样数据，转化为浮点 RGBA 值，并存储在 1D 数组中。

success

如果成功读取文件，则输出 true，否则输出 false。
