define_usd_geom_subset

定义用于存储一组面 ID 的 USD GeomSubset。

输入

path

GeomSubset 类型的基本体定义的路径。

element_type

此几何体子集包含的元素类型。目前仅支持“面”。

indices

此子集中包含的索引集。索引无需排序，但同一索引不应出现多次。

family_name

此子集所属子集系列的名称。这是可选项，在 DCC 应用程序之间往返传输子集时可以使用。

输出

prim_definition

新基本体定义。可以将其连接到 add_to_stage 节点的 prim_definitions 端口，或连接到其他 define_usd_prim 或类似节点的 children 端口。
