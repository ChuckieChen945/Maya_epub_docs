使用双骨骼 IK 控制柄确定关节的姿势

移动双骨骼 IK 控制柄

  1. 请参见使用 IK 控制柄为关节设定姿势。

操纵极向量

  1. 选择双骨骼 IK 控制柄。
  2. 从 QWERTY 工具箱中选择“显示操纵器工具”(Show Manipulator Tool)（热键：T）。

    将会出现 IK 控制柄的极向量、扭曲圆盘和旋转圆盘。

  3. 在当前视图中，拖动极向量以根据需要移动该向量。

    这将旋转由当前 IK 控制柄控制的关节链。

操纵扭曲圆盘

  1. 选择双骨骼 IK 控制柄。
  2. 从 QWERTY 工具箱中选择“显示操纵器工具”(Show Manipulator Tool)（热键：T）。

    将会出现 IK 控制柄的极向量、扭曲圆盘和旋转圆盘。

  3. 选择扭曲圆盘。

    扭曲圆盘变为黄色。

  4. 按住并拖动扭曲圆盘。

    这会扭曲由当前 IK 控制柄控制的关节链。
