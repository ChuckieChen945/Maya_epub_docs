offset_level_set

此节点对等值面应用统一偏移。将根据需要添加新平铺，以使等值面保持相同的分辨率。

输入

level_set

等值面体积（对象）。

property

包含要偏移的等值面的特性。

offset

要应用于输入等值面的世界空间偏移 (float)。

grid_retopology

如果启用，栅格将重新拓扑以跟随曲面。如果禁用，将保留栅格拓扑。这可能会显著提高速度并将保留体素遮罩，但由于粗化，某些区域中的细节可能较少或出现瑕疵。

mask_field

用作遮罩的场。

mask_interpretation

如何解释遮罩（特性或场）。如果 mask_interpretation 为 FogDensity，则遮罩将被钳制在 0 和 1 之间。如果 mask_interpretation 为 SignedDistanceField，则低于 0 的值（内部）将映射到遮罩值 1，高于 0 的值（外部）将映射到遮罩值 0。

mask_property

可用于在输入体积上指定特性，以用作遮罩。如果还指定了场遮罩，则使用该场遮罩。

mask_filter_iterations

如果连接了 mask_field，则将使用 mask_filter_iterations 次迭代对其进行预过滤。这可以减少因场中高频率导致的锯齿瑕疵。当前，进行平滑处理时，只能使用均值过滤器，并且最多迭代两次。如果需要更多次迭代或需要使用其他类型的过滤器，可以将场重新采样到输入体积的体素栅格上，使用 smooth_voxel_property 节点对其进行平滑处理，并将其作为 property 输入传递到 offset_level_set。

输出

out_level_set

偏移等值面（对象）。
