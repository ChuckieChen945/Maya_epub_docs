points_to_faces

返回引用输入点数组的网格的面数组。

例如：

  p0─────────p1─────────p2
  │          │          │
  │          │          │
  │    f2    │    f1    │
  │          │          │
  │          │          │
  p3─────────p4─────────p5
  │          │          │
  │          │          │
  │    f0    │    f3    │
  │          │          │
  │          │          │
  p6─────────p7─────────p8
  • 如果 points 为 [0, 1]，则 face_indices 为 [1, 2]。
  • 如果 points 为 [4]，则 face_indices 为 [0, 1, 2, 3]。

输入

mesh

要查询的具有预先计算的相邻特性的网格。

points

将返回其连接面的点的索引。

输出

face_indices

引用输入点的面的索引。

父页面: Mesh
