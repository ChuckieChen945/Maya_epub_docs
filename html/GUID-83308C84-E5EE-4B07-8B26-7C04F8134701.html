折痕集编辑器(Crease Set Editor)

通过“折痕集编辑器”(Crease Set Editor)（“窗口 > 建模编辑器 > 折痕集”(Windows > Modeling Editors > Crease Sets)），可以在场景中创建和管理折痕集。

“折痕集编辑器”(Crease Set Editor)具有以下主要区域：
  • A：“折痕集编辑器”(Crease Set Editor)菜单
  • B：“折痕集”(Crease Set)列表
  • C：操作按钮
  • 折痕集编辑器右键单击菜单
目标 操作

创建新的折痕集。

在场景中选择组件，然后执行下列操作之一：
  • 单击“新建”(New)。
  • 在编辑器中的任意位置单击鼠标右键并选择“创建集”(Create Set)。

选择折痕集中的所有成员。

选择以下方法之一：
  • 单击“选择”(Select)。
  • 在编辑器中的一个折痕集的名称上单击鼠标右键并选择“选择成员”(Select Members)。

调整选定折痕组件的硬度。

选择以下方法之一：
  • 双击“折痕集”(Crease Set)列表中的“折痕”(Crease)字段，并输入一个新值。
  • 在“折痕”(Crease)字段中按住鼠标中键并拖动以增加或减小值。
