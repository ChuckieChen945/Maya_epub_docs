在 Maya 中对对象进行变形

将变形效果添加到角色和对象，以调整其外观并增强其动画。

变形器是可以使用简单操纵和关键帧不能进行的方式变换对象或对对象设定动画的工具。在其他软件包中，术语修改器和空间扭曲用于表示 Maya 称为变形器的功能。

用于使模型成形的晶格变形器

变形器有两个主应用程序：建模曲面或将附加变形动画添加到对象。

变形器作为建模工具

可以使用变形器作为建模工具，用于塑造 NURBS 或多边形对象。可以创建变形器，for example a sculpt deformer tool, 并使用它调整模型的形状。使用变形器完成建模时，请将变形器连同其余对象历史一起删除（选择“编辑 > 按类型删除 > 历史”(Edit > Delete by Type > History)）。删除对象的历史时，变形器将会删除，但所有变形操作将烘焙到对象的形状，使其保持最终变形状态。

请注意，在建模上下文中，对象的历史可以称为其构建历史。

变形器作为动画工具

可以将变形器用作动画工具。可以创建变形器，用变形器调整目标对象，然后随时间变化为变形器的属性设定关键帧以生成动画。For example, you can create a blend shape deformer for a model of a face. Then over time, you can manipulate and key the weights of target shapes of the blend shape deformer in the Shape Editor to create facial animation.
