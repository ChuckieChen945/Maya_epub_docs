使用笔刷修改 UV

“抓取工具”(Grab Tool)、“收缩工具”(Pinch Tool)、“涂抹工具”(Smear Tool)和“对称工具”(Symmetrize Tool)。

通过在 UV 上拖动来改变它们

  1. 从“工具”(Tools)菜单中或通过“UV 工具包”(UV Toolkit)的“变换 > 工具”(Transform > Tools)区域选择适当的工具。
  2. （可选）调整“工具设置”(Tool Settings)中的选项以更改衰减半径的大小、强度和形状。
  3. （仅限“对称工具”(Symmetrize Tool)）沿网格的对称轴选择边。
  4. 在 UV 上拖动以改变它们。
    5. 获得的效果取决于所使用的工具。
    注： 您可以固定 UV，避免它们在笔刷操作期间改变。
工具 效果 示例

抓取工具

在与鼠标拖动方向相同的方向上拉动 UV。

其作用类似于“涂抹 UV 工具”(Smudge UV Tool)。

收缩工具

将 UV 拉向彼此。

涂抹工具

按与笔划方向的原始位置相切的方向移动 UV。
对称 UV 工具 跨所选轴将 UV 移动到与对应 UV 相对的位置。
