Matrix

Matrix 元素为颜色值相乘指定一个矩阵，也可以选择应用偏移。

包含

  • Description。可选，任意数量。
  • Array (Matrix)。必需，仅一个。
  • DynamicParameter。可选，至多一个。param 属性必须设置为 "DEFAULT_LOOK"。

包含于

  • ProcessList

属性

id、name、bypass、inBitDepth、outBitDepth
请参见通用运算符属性。

本节内容
  • Array（矩阵）
    Array 元素指定 LUT 或矩阵的值数组。当 Array 元素包含在 Matrix 元素中时，其内容的格式与包含在 LUT1D 或 LUT3D 元素中时的格式不同。
