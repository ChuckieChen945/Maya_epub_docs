tag_strand_ends

按股的端点标记股。使用 invert_tag 端口标记股中间的点，而不是端点。此节点的可能用途包括使用 delete_points 切掉股的末端，或将实例放置在每个股的末端。

输入

strands

要标记的股几何体。

strands_filter

这是解释的自动端口，可以接受 array<bool>、array<long>、array<uint> 或输入 geometry 上标记的字符串名称。此端口可用于过滤节点将标记的单个股。其他标记节点的输出 tag_data 可以连接到此端口，以将多个标记节点链接在一起。

标记部分

tag

要输出的标记的名称。

invert_tag

标记股中间的点，而不是端点。

权重部分

weights_curve

权重在输出之前先由此 fcurve 调整。

weights_scale

计算的权重在输出之前先与此值相乘。此端口是解释的自动端口。用户可以连接包含与 geometry_component 相同的元素数量的数组，以逐个元素缩放值。可以将来自其他标记节点的输出 weights 插入到此端口中，以组合多个标记节点的效果。

端点部分

selection_mode

可以使用不同的度量来选择股中的点：

  • Index - 使用相对于股根部的点索引选择点。
  • Length - 使用点沿股的长度选择点。
  • PointRatio - 使用点沿股的比率选择点。

例如，要仅选择每个股中的前两个点，则将 selection_mode 设置为 Index，将 start 设置为 2，将 use_end 设置为 false。要选择前 5 个长度单位内的所有点，则将 selection_mode 设置为Length，将 start 设置为 5，将 use_end 设置为 false。要选择每个股中前 10% 和后 10% 的点，则将 selection_mode 设置为 PointRatio，将 start 设置为 0.10，将 end 设置为 0.9。

use_start

从每个股的根部选择点。

start

从每个股的根部开始，标记值小于此阈值的点。

use_end

从每个股的尖端选择点。

end

从每个股的根部开始，标记值大于或等于此阈值的点。

end 值可以为负值。在这种情况下，该值是相对于每个股的尖端的值。例如，要选择每个股中的最后一个点，则将 selection_mode 设置为 Index，将 end 设置为 -1。

诊断部分

point_size

点显示的世界空间大小。

输出

strands_with_tag

以几何特性形式添加了标记的 geometry。

tag_data

计算的组件标记。此数组的大小等于输入 strands 中的点数。如果此数组中某个元素对应的点已标记，则该元素为 true。

weights

输出权重是归一化值，起始值为 1，位于每个股的端点，终止值为 0，位于任一方向上标记的最后一个点。此数组的大小等于输入 strands 中的点数。
