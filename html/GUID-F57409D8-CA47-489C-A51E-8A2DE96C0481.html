基因型编辑器

此编辑器可用于创建和修改由 MASH“世界”(World)簇模拟传播的种子类型。
1 - 创建基因型/更新模拟 分别使用任何新添加/更改的基因型值添加新基因型和更新 MASH 模拟。
2 - 基因型摘要 显示所有现有基因型。选择基因型可在右侧面板中显示其属性值。

单击鼠标右键可显示一个菜单，用于添加新基因型、更改标签颜色、访问预设、随机化基因型值、重命名或复制基因型或者删除现有基因型。

3 - 基因型属性 显示选定基因型的属性值。

基因型属性

最大年龄(Max. Age)

确定树在消亡之前的最大年龄（以模拟年份为单位）。

种子年龄(Seed Age)

确定树开始播下种子的年龄（以模拟年份为单位）。

种子计数(Seed count)

确定树每年播下多少种子。

增长速率(Growth rate)

确定树每年长高多少。

模型大小(Model size)

指定树的大小（以世界单位表示）。尝试尽可能接近地将此大小与表示此基因型的基础对象的大小匹配。

坡度阈值(Slope threshold)

确定树将在其增长的最大陡峭程度。减小该值会增加树在更陡的山丘上增长的可能性。

随机变化(Rand Variance)

基于对象应用的早期属性的随机化器。

温度(Temperature)

确定模拟的空气温度。树在更温和的气候中生长得更好。

土壤质量(Soil Quality)

确定土壤中的营养成分。树在更优质的土壤中生长得更好。

湿度(Moisture)

确定空气中的湿度。树在更潮湿的环境中生长得更好。

弹性(Resiliance)

确定树的健康程度。弹性越大，树越有可能达到其“最大年龄”(Max. Age)。

对象 ID(Object ID)
确定将表示基因型的对象 ID。
最小 ID(Min ID)、最大 ID(Max ID)

指定将随着年龄增大表示基因型的对象 ID 的范围。如果这些值不是 0，则将优先于“对象 ID”(Object ID)。

ID 颜色(ID Color)
为基因型指定颜色以方便识别。

基因型标记菜单

可以右键单击特定的基因型以打开基因型标记菜单。

菜单项 效果
添加基因型(Add Genotype) 将新的基因型添加到列表（与“创建基因型”(Create Genotype)相同）。
标签颜色 确定“基因型编辑器”(Genotype Editor)中显示在基因型名称旁边的标签颜色。
注： 与“ID 颜色”(ID Color)不同。
预设(Presets) 将预设基因型属性值加载到选定的基因型中。
随机化(Randomize) 将随机值指定给基因型属性。
自动设置模型大小(Automatically Set Model Size) 根据实例化器自动填充“模型大小”(Model Size)属性。
重命名(Rename) 将新名称指定给选定基因型。
复制(Duplicate) 创建选定基因型的副本。
删除(Delete) 移除选定的基因型。
