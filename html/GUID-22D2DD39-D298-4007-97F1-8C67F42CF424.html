设置当前角色集

设置动画时，可以指定当前角色集以仅设置该角色集的动画。当前角色集与层非常类似，它们与选择列表中的项目是分开的。这使您可以在场景中选择项目，而无需更改那个角色集作为当前角色集。可以从“时间滑块”(Time Slider)、“角色”(Character)菜单或“关系编辑器”(Relationship editor)中设置当前角色集。

查看或设置当前角色集的步骤

  • 在“时间滑块”(Time Slider)中：

    从下拉列表中选择角色集。未选择任何角色集时，将显示“无角色集”(No Character Set)。

    提示： 从该菜单中选择“角色集编辑器”(Character Set Editor)，以在“角色编辑”(Character Editing)模式下打开“关系编辑器”(Relationship Editor)。
  • 从主菜单栏中执行以下操作：

    选择“角色 > 设置当前角色集”(Character > Set Current Character Set)，然后选择所需的角色集，或选择“角色 > 选择角色集节点 > 角色”(Character > Select Character Set Node > character) n（角色的默认名称）。

    提示： 也可以通过选择“角色 > 设置当前角色集 > 角色集编辑器”(Character > Set Current Character Set > Character Set Editor)来访问“关系编辑器”(Relationship Editor)。
  • 在“关系编辑器”(Relationship Editor)中：

    确保编辑器处于“角色编辑”(Character Editing)模式，选择要置为当前的角色集，然后选择“编辑 > 将亮显角色集设置为当前”(Edit > Make Highlighted Character Sets Current)。还可以通过选择“编辑 > 亮显当前角色集”(Edit > Highlight Current Character Sets)检查哪些角色集是当前角色集。

有关使用“关系编辑器”(Relationship Editor)选择和编辑角色集的详细信息，请参见选择角色集和编辑角色集。请参见关系编辑器。
