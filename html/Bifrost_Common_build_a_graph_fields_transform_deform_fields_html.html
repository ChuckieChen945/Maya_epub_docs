变换、平流和扭曲场

特殊节点可用于变换场，以及对它们进行平流和扭曲（或变形）。可以将这些节点视为修改对场进行采样的输入位置。

变换场

变换场会移动它，以便以与通常采样位置的偏移对位置进行采样。它相当于将反向变换应用于位置，然后在修改后的位置对原始场进行采样。

  • transform_field 将 float4x4 或 float3x3 变换应用于场。
  • scale_field 按指定因子缩放场。
  • rotate field 按指定的四元数旋转场。
  • translate_field 按指定的向量平移场。

以场表示旋转

由于场不支持 float4 值，因此不能使用场将旋转表示为四元数或轴角值。但是，可以使用向量场将旋转表示为“旋转向量”。旋转向量是三维向量，其中方向表示旋转轴，向量的长度表示以弧度为单位的角度。

有几个工具节点可帮助您在旋转向量和其他表示形式之间进行转化：

  • euler_to_rotation_vector
  • quaternion_to_rotation_vector
  • rotation_vector_to_quaternion

对于其他类型的旋转，可以在将它们转化为旋转向量之前先将其转化为四元数。

平流场

使用 advect_field 沿表示空间速度的向量场变换给定场。这特别适用于在模拟期间使场流动。

扭曲场

warp_field 节点通过将其值重映射到新位置的向量场使给定场变形。它相当于先对扭曲场进行采样以查找新位置，然后使用新位置对原始场进行采样。
