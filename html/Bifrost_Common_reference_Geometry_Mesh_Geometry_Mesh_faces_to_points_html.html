faces_to_points

返回输入网格中由输入数组中的面引用的点数组。

例如：

  p0─────────p1─────────p2
  │          │          │
  │          │          │
  │    f2    │    f1    │
  │          │          │
  │          │          │
  p3─────────p4─────────p5
  │          │          │
  │          │          │
  │    f0    │    f3    │
  │          │          │
  │          │          │
  p6─────────p7─────────p8
  • 如果 faces 为 [0]，则 point_indices 为 [3, 4, 6, 7]
  • 如果 faces 为 [0, 3]，则 point_indices 为 [3, 4, 5, 6, 7, 8]

输入

mesh

要查询的网格。

faces

要查询的面。

输出

point_indices

输入 faces 引用的点。

父页面: Mesh
