Arnold_standard_volume_mat

使用与 Arnold 标准体积材质相同的特性名称创建材质定义。

输入

name

可选标识符。

interpolation

使用命名通道对体积数据进行采样时要使用的体素插值。

displacement

使用置换作为向量偏移来置换体积。

density

体积的密度。密度越低，体积越薄；密度越高，体积越厚。它用作密度通道上的倍增。

density_channel

要从体积对象读取的密度通道。

scatter

照明体积的亮度。它是被散射而不是被吸收的光的比率。

scatter_color

照明体积的颜色。

scatter_color_channel

要从体积对象读取的散射通道。它用作散射颜色的倍增，对体积的颜色设置纹理。

scatter_anisotropy、scatter_secondary_anisotropy、scatter_secondary_anisotropy_mix

散射的方向偏差或各向异性。默认值零表示各向同性散射，灯光在所有方向上均匀散射。正值在灯光方向上向前偏差散射效果，而负值则朝灯光向后偏差散射效果。

transparent

对体积密度进行额外控制，以便对通过体积看到的体积阴影和对象进行染色。

transparent_depth

对体积密度进行额外控制，以便控制透明颜色在体积中达到的深度。

transparent_channel

要从体积对象读取的透明颜色。它用作倍增来对体积中的透明颜色设置纹理。

emission_mode、emission_scaling

  • 无(none)：不发射灯光。
  • 通道(channel)：使用指定的自发光通道发射灯光。
  • 密度(density)：使用按可选自发光通道调整的密度通道发射灯光。
  • 黑体(blackbody)：根据温度发射颜色和强度，以渲染火焰和爆炸。

emission

自发光是体积发射灯光的速率。如果使用密度通道、自发光通道或黑体通道进行自发光，则它用作倍增来减少或增加自发光。

emission_color

对自发光进行染色（与自发光相乘）的颜色。

emission_channel

在“通道”(Channel)自发光模式下，从中进行自发光速率采样的自发光通道。如果不为空，则此通道将用于对体积的自发光值进行采样，而不是“自发光”(Emission)参数。

temperature

如果使用黑体通道，则它用作黑体温度的倍增。

temperature_channel

在“黑体”(blackbody)自发光模式下，要从体积对象读取的温度通道。通常，温度通道来自物理模拟。

blackbody_kelvin

来自温度体积通道的温度的倍增。

blackbody_intensity

控制黑体自发光强度。为了获得物理上正确的结果，必须使用强度 1。这可能会导致非常亮的灯光，但是，可以使用较低的值来减小强度。

blackbody_contrast

调整不同温度区域之间的相对亮度，以获得艺术效果。值越低，黑体颜色越鲜艳，但只有值 1.0 在物理上是正确的。
