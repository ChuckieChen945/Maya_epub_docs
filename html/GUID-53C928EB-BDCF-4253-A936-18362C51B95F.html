选择两个顶点之间的边路径

通过“建模”(Modeling)菜单集 (F2) 中的“选择最短边路径工具”(Select Shortest Edge Path Tool)，可以轻松选择曲面网格上两个顶点之间的边路径。它可以确定任何两个选择点之间的最直接路径并选择这两点间的多边形边。

“选择最短边路径工具”(Select Shortest Edge Path Tool)特别适合在展开 UV 壳时需要后续执行“切割 UV 边”操作的情况下，在曲面网格上选择较长且可能缠绕的边路径。

选择两个顶点之间的边路径

  1. 在场景视图中，选择包含待选边的曲面网格。
  2. 从“建模”(Modeling)菜单集 (F2) 中，选择“选择 > 最短边路径工具”(Select > Shortest Edge Path Tool)。

    选定的曲面网格将更新，以显示网格上的顶点。

  3. 若要选择网格上的边路径，请单击一个顶点以指定边路径的起点，然后再单击另一个顶点以指定路径的终点。

    将选定构成这两点之间最直接路径的边。即，路径可能遵循对角阶梯路径，但选择遵循最直接路线。

    注： 可以依次单击多个顶点，而选择将遵循经过所指示顶点的最直接路径。
  4. 如果要撤消上一选择，请按 Ctrl+Z。如果要取消现有选择并重新开始，请单击网格外的任意位置。如果要保持现有选择并开始新的选择，请按 Y 键。按 Q 键将退出工具并继续下一操作。例如，可能要使用“切割 UV 边”(Cut UV Edges)功能（“UV > 切割 UV 边”(UV > Cut UV Edges)）切割与选定边路径关联的 UV 边。
