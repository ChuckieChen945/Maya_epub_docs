文件路径编辑器(File Path Editor)

此工具列出了 Maya 场景中所含引用、纹理、音频节点和图像平面的所有目录路径。

  • 若要打开“文件路径编辑器”(File Path Editor)，请选择“窗口 > 常规编辑器 > 文件路径编辑器”(Window > General Editors > File Path Editor)。

“文件路径编辑器”(File Path Editor)用于查看嵌入到当前场景中的文件的列表。如果移动与场景关联的文件（例如，将其重新定位到计算机上的其他文件夹），那么 Maya 将无法找到该文件并显示错误。使用“文件路径编辑器”(File Path Editor)可以快速轻松地重新链接未解析的路径，例如，允许您像文件纹理编辑器一样重新指定损坏的纹理路径。

“文件路径编辑器”(File Path Editor)具有以下主要区域：
  • “文件路径编辑器”(File Path Editor)菜单
  • “文件路径编辑器”(File Path Editor)浏览器
  • “文件路径编辑器”(File Path Editor)操作按钮

使用文件路径编辑器重新链接文件

    在此示例中，衬衫模型应用了花卉纹理。

    纹理文件已移动到另一个位置，衬衫现在无纹理。

  1. 打开“文件路径编辑器”(File Path Editor)（“窗口 > 常规编辑器 > 文件路径编辑器”(Windows > General Editors > File Path Editor)）

    缺少的文件在其文件路径旁边显示一个红色 x。

  2. 选择损坏的文件路径，然后单击“重新指定文件路径”(Repath Files)。

    将显示“重新指定路径”(Repath)对话框。激活“搜索文件的子目录”(Search Subdirectoriess for Files)和“重新指定当前已解析文件的路径”(Repath Currently Resolved Files)选项。

  3. 单击“浏览”(Browse)以导航到缺少的纹理文件的位置。
  4. 找到缺少的文件后，单击“重新指定路径”(Repath)。
