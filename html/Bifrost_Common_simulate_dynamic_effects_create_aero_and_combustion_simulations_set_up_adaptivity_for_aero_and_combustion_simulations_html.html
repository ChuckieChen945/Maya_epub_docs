为 Aero 和燃烧模拟设置自适应性

采用自适应模拟时，可以减少不同区域中的细节，从而节省计算时间和内存。您可以选择根据特定标准自动减少细节，也可以使用网格几何体明确定义低细节和高细节的区域。

  1. 将 aero_adaptivity_settings 节点添加到 Aero 或燃烧图表的顶层。

  2. 将其输出连接到 aero_solver_settings 节点的 additional_settings 端口。

  3. 执行下列操作之一：

    • 激活 enable_auto_adaptivity，然后为 coarsen 选择标准。
    • 激活 enable_resolution_bounds，然后连接网格对象，为不同的分辨率定义与世界轴对齐的边界框。任何边界框之外的区域将以最粗略的分辨率进行模拟。
