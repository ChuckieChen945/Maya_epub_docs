Bifrost 2.10.0.0

Bifrost 2.10.0.0 包括新的节点库、绑定节点和更新的 SDK 等。

以下内容概述了 Bifrost 2.10.0.0 的主要新功能：
  • 借助“节点库”(Node Library)（窗口 > 节点库(Windows > Node Library)），可以通过拖放或双击将节点添加到图表中。与“选项卡”(Tab)菜单不同，“节点库”(Node Library)可以按名称空间过滤，并允许您选择多个节点。可以拖动窗格使其浮动或停靠在曲线图编辑器中的任意位置。
  • 一组新节点提供了在 Bifrost 中提前查看绑定工作流的可能性。其中许多绑定工作流都使用新的 Core::Transform::Transform 数据类型，这是一组相互关联的矩阵，可以作为变换链的一部分进行更新。要开始您自己的实验，请查看 Bifrost 浏览器中新的绑定示例图表。
  • 执行器 SDK 可以在命令行上加载和执行 Bifrost 图表，从而使 Bifrost 集成到自定义应用程序中。这使得技术用户能够运行自动化以及为美工人员构建工具和插件。
  • Bifrost 几何体 SDK 专注于各种几何体表示的可访问性和简单解释。它允许访问由图表处理或生成的 Bifrost 几何体数据，还允许用户创建和修改几何体。此 SDK 可与现有 Operator SDK 和新的执行器 SDK 结合使用。
  • 基于 Python 的插件的抢先体验版本 BifrostUSDExamples 为通用工作流和组件 USD 工作流提供了基于菜单的选项。
  • 许多其他新节点和示例图表...

有关此版本中包含的功能和错误修复的完整列表，请参见 Bifrost 发行说明。

请记住，您随时可以通过访问 Autodesk Account 门户下载最新版本的 Bifrost。
