复制片段

您可以复制一个片段，创建原始源片段的从属副本或独立副本。

对于独立复制，可以修改原始源片段的动画，而不会影响复制。对于从属复制（也称为实例），对原始源片段的动画所做的任何修改均会传播到从属片段。

创建从属片段的优点在于只需对多个角色的所有从属片段的父片段进行一次编辑，就可以对它们进行更改。

复制片段

  1. 在“Trax 编辑器”(Trax Editor)中，选择要复制的片段。
  2. 执行下列操作之一：
    • 在 Trax 编辑器中，选择“编辑 > 复制”(Edit > Duplicate) > 。
    • 在片段上单击鼠标右键，然后从显示的菜单中选择“复制片段”(Duplicate Clip) > 。

    此时将显示“复制片段选项”(Duplicate Clip Options)窗口。请参见“Trax 编辑器的编辑(Edit)菜单”主题中的复制片段选项(Duplicate Clip Options)。

  3. 启用“复制输入连接”(Duplicate Input Connections)以创建独立复制片段，或禁用“复制输入连接”(Duplicate Input Connections)以创建从属复制片段。
  4. 启用“复制时间扭曲曲线”(Duplicate Time Warp Curve)，为复制片段创建选定片段的时间扭曲曲线的独立副本；或启用“实例时间扭曲曲线”(Instance Time Warp Curve)，为复制片段创建选定片段的时间扭曲曲线的从属副本。
  5. 设定“复制片段选项”(Duplicate Clip options)的其他内容，然后单击“复制片段”(Duplicate Clip)按钮。

    如果启用“将副本置于 Trax 编辑器和 Visor 中”(Put Copy in Trax Editor and Visor)，复制片段此时将显示在原始片段的轨迹下的新轨迹和“Visor”中。

    如果启用“仅将副本置于 Visor 中”(Put Copy in Visor Only)，则复制片段将仅放置在“Visor”中。
