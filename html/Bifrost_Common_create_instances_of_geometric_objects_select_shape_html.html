在多个实例几何体之间进行选择

selector_instance 节点用于在多个形状之间进行选择，以便您可以根据多种不同的树创建森林，等等。您还可以将 selector_instance 节点呈菊花链状连接到一起，例如，根据一个标准在松树与枝叶茂密的树之间进行选择，然后根据其他标准在每种树的不同示例之间进行选择。selector_instance 的输出可以连接到 set_instance_geometry 节点或 set_instance_shape 节点。

  1. 将 selector_instance 节点添加到图表中。

  2. 将一个或多个几何对象（例如网格）连接到 instance_geometries 端口。除了几何体外，还可以连接实例形状，例如以下项的输出：

    • geo_instance，如果要对不同的实例主形状使用不同的预览几何体。
    • render_archive_instance，用于实例化磁盘上的渲染归档文件。
    • selector_instance，以便在主几何体之间进行比较复杂的切换。

  3. 将自定义 long 特性的名称指定为 instance_id_property 输入。此操作假设您已使用 set_geo_property 等上游节点指定针对点对象的 point_component 的特性。该特性应指定要实例化的几何体，其中 0 表示连接到 instance_geometries 的第一个几何体，1 表示第二个，依此类推。

  4. 如果需要，将一个简单几何体连接到 preview_geometry 端口。此几何体用作替代对象以供在视口中预览。请注意，此几何体将用于所有实例，即使这些实例已定义预览几何体也是如此，例如，如果您已将 geo_instance 节点连接到 instance_geometries 端口，而不是直接连接几何体。

  5. 将 selector_instance 连接到 set_instance_geometry 节点或 set_instance_shape 节点。
