创建 UV 集选项(Create UV Set options)

使用这些选项可设置在选择“UV > 创建空 UV 集”(UV > Create Empty UV Set)时发生的情况。

创建 UV 集名称(Create UV Set name)

在文本字段中为新 UV 集键入名称。

UV 集共享(UV Set Sharing)

您可以选择共享该 UV 集（每个实例都有 UV 集），即“逐共享实例”(Per Instance Shared)（所有选定实例共享新 UV 集）或者“逐非共享实例”(Per Instance Unshared)（所有选定实例获得一个唯一的 UV 集）。
