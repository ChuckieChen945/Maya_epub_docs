UV 集编辑器(UV Set Editor)

导航到“创建 UV > UV 集编辑器”(Create UVs > UV Set Editor)。

“UV 集编辑器”(UV Set Editor)可用于同时创建和编辑多个多边形网格的 UV 集。

“UV 集编辑器”(UV Set Editor)只列出了当前选定多边形网格的 UV 集。必须首先选择多边形网格才能编辑 UV 集。

新建(New)

为当前选定对象创建一个新的、空 UV 集。然后可以使用其中一种映射/投影方法在集中创建 UV。该功能的作用等效于“创建 UV > 创建空 UV 集”(Create UVs > Create Empty UV Set)。有关信息，请参见创建 UV 集选项(Create UV Set options)。

重命名(Rename)

可用于重命名当前选定的 UV 集。

删除(Delete)
删除当前选定的 UV 集。
注： Maya 将不允许您删除“UV 集”(UV Set)列表中的最上方条目。如果需要删除此条目，可以将其移至列表中更低的位置，然后将其删除。
复制(Copy)

在现有 UV 布局的基础上创建一个新的 UV 集，或将 UV 布局从一个集传递到另一个集。该功能的作用等效于“将 UV 复制到 UV 集”(Copy UVs to UV Set)。

传播(Propagate)

将 UV 集编辑器列表中的选定 UV 集指定给场景中的选定对象。选定的 UV 集成为那些对象的活动 UV 集。

未映射(Unmapped)

选择场景中任意选定对象上的未映射面。这有助于直观地确定任何未显示或未正确显示纹理贴图的区域。
