superquadric_shape_field

一个标量场，将形状表示为隐式曲面。形状的曲面位于零等值面。请注意，与 sphere_field 不同，这不是带符号距离场。即，渐变不一定是 1。

输入

position

形状的中心。

radius

控制形状的大小。但是，形状的实际大小取决于半径和 Gamma。为了获得更可预测的结果，将其保留为默认值 1，然后使用 scale_field 节点调整其大小。

gamma

修改形状。值介于 0 和 1 之间时，形状是具有凹边的尖八面体。值为 1 时，它是八面体。值为 2 时，它是球体。值大于 2 时，它是带圆角的长方体。较高的值会生成更尖的角，非常高的值会生成接近立方体的形状。

输出

level_set

生成的形状为隐式曲面，并且曲面位于零等值面。
