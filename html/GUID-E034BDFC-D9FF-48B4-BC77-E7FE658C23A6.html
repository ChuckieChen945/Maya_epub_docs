“空气”(Air)选项

使用这些选项可控制“空气”(Air)场的行为。

空气场的预定义设置(Air Field Predefined Settings)

风(Wind)

将“空气场”(Air Field)属性设定为与风的效果近似的默认设置。连接的对象将增加其速度，直到与 X 轴方向每帧 5 个单位的速度匹配。

尾迹(Wake)

将“空气场”(Air Field)属性设定为与由移动对象中断和牵引的空气的运动近似的默认设置。除非空气场正在移动，否则尾迹的默认设置不影响对象。

可以对空气场或拥有该空气场的对象的运动设置动画，也可以将移动对象设置为空气场的父对象（请参见示例：创建移动对象的空气尾迹）。如果需要具有静态空气场的尾迹的效果，请将速度设定为大于 0 的值。

风扇(Fan)

将“空气场”(Air Field)属性设定为与本地风扇效果近似的默认设置。连接的对象以每帧 5 个单位的速度沿着与 X 轴成 45 度的方向移动。

幅值(Magnitude)

设定空气场的强度，该选项设定沿空气移动方向的速度。“幅值”(Magnitude)以及“方向 X”(Direction X)、“方向 Y”(Direction Y)和“方向 Z”(Direction Z)属性设定风的速度。

该数字越大，力越强。可以使用负数来反转力的方向。

衰减(Attenuation)

设定场的强度随着到受影响对象的距离增加而减小的量。更改速率与距离成指数关系；“衰减”(Attenuation)是指数。如果将“衰减”(Attenuation)设定为 0，力将随着距离的变化保持不变。负数无效。

注：

当对象与场之间的距离接近“最大距离”(Max Distance)时，空气场的强度会急剧下降到零。有关“衰减”(Attenuation)随着距离的变化如何影响场强度的图形表示，请参见使用工作区操纵器设定场属性。

方向 X(Direction X)、方向 Y(Direction Y)、方向 Z(Direction Z)

指定空气吹动的方向。

速度(Speed)

控制连接的对象与空气场速度匹配的快慢。如果将“速度”(Speed)设定为 0，则对象不移动。如果将“速度”(Speed)设置为 1，则对象几乎可以即时与空气场的速度匹配。

“继承速度”(Inherit Velocity)（或继承）

当空气场移动或以移动对象作为父对象时，“继承速度”(Inherit Velocity)指定添加到“方向”(Direction)和“幅值”(Magnitude)中的移动空气场速度的大小。使用 0 到 1 之间的值。

继承旋转(Inherit Rotation)

如果启用“继承旋转”(Inherit Rotation)，且空气场正在旋转或以旋转对象作为父对象，则气流会经历同样的旋转。空气场旋转中的任何更改都会更改空气场指向的方向。这类似于旋转风扇或来自其头部正在移动的角色口中的空气。启用“继承旋转”(Inherit Rotation)会影响空气场的有效速度。

如果启用“风”(Wind)或“风“扇”(Fan)按钮，则会启用“继承旋转”(Inherit Rotation)。

仅组件(Component Only)

如果禁用“仅组件”(Component Only)，空气场会应用使受影响对象的速度与空气场的速度匹配所需的任何力。

如果启用“仅组件”(Component Only)，空气场仅在其“方向”(Direction)、“速度”(Speed)和“继承速度”(Inherit Velocity)属性的组合所指定的方向应用力。此外，不应用任何力来降低对象沿该方向的速度；仅应用力来提高速度。

只有移动速度比空气场慢的对象会受到影响。移动速度比空气场快的对象将继续以该速度移动。

在启用“尾迹”(Wake)按钮时，将启用“仅组件”(Component Only)。仅当“继承速度”(Inherit Velocity)具有非零值时，该选项才起作用。

扩散(Spread)

启用扩散(Enable Spread)

指定是否使用“扩散”(Spread)角度。如果启用“启用扩散”(Enable Spread)，空气场将只影响“扩散”(Spread)设置指定的区域内的连接对象。运动以类似圆锥的形状呈放射状向外扩散。如果禁用“启用扩散”(Enable Spread)，空气场将影响“最大距离”(Max Distance)设置内的所有连接对象。运动方向是一致的。

如果在启用“启用扩散”(Enable Spread)后将“幅值”(Magnitude)属性设定为负值，则会将粒子吸引到场中心。可以使用该选项创建粒子被“吸收”的效果。

扩散(Spread)

表示与“方向”(Direction)设置所成的角度，该角度内的对象会受到空气场的影响。当启用“启用扩散”(Enable Spread)时，空气场将从其原点以类似圆锥的形状推进。

当该值为 1 时，会将空气场前方的任何对象沿方向向量吹动 180 度。当该值为 0 时，只沿方向向量吹动恰好在空气场前面的对象。当该值在 0 和 1 之间时，将沿方向向量以类似圆锥的形状吹动对象。

提示：

如果将“扩散”(Spread)设定为 0 并且看不到粒子移动，则粒子并非恰好在空气场前面。请使用大于 0 的“扩散”(Spread)。

距离(Distance)

使用最大距离(Use Max Distance)

如果启用“使用最大距离”(Use Max Distance)，则空气场会影响“最大距离”(Max Distance)设置所定义的区域内的连接对象。空气场不会影响“最大距离”(Max Distance)之外的任何连接对象。

如果禁用“使用最大距离”(Use Max Distance)，则无论距离为何空气场都会影响所有连接的对象。

最大距离(Max Distance)

设定空气场能够施加影响的与该场之间的最大距离。还必须启用“使用最大距离”(Use Max Distance)，“最大距离”(Max Distance)才能生效。

注：

“最大距离”(Max Distance)可限制场的影响范围。但是，如果场将某个对象移动到影响范围外，该对象通常会因动量而继续移动。

衰减曲线(Falloff Curve)

在选中“使用最大距离”(Use Max Distance)时可用。

当使用场施加力时，有时受力对象在施加力后可能无法稳定下来。使用该曲线可调整力。例如，缩放场的力，以使其在“最大距离”(Max Distance)值定义的边界处逐渐衰减为零。

曲线在归一化的距离内定义；也就是 0 到 1，其中 0 对应于场的中心，1 对应于“最大距离”(Max Distance)值。该曲线中的值是要应用于力的比例因子。

注： 在创建场后，衰减曲线仅作为属性提供。它在场创建选项中不可用。

体积形状(Volume shape)

体积决定场影响粒子/刚体的区域。

体积(Volume)

选择“无”(None)、“立方体”(Cube)、“球体”(Sphere)、“圆柱体”(Cylinder)、“圆锥体”(Cone)或“圆环”(Torus)之一。

体积排除(Volume Exclusion)

启用“体积排除”(Volume Exclusion)时，体积定义空间中场对粒子或刚体没有任何影响的区域。

体积偏移 X(Volume Offset X)、体积偏移 Y(Volume Offset Y)、体积偏移 Z(Volume Offset Z)

从场的位置偏移体积。如果旋转场，也会旋转偏移方向，因为它在局部空间内操作。

注：

偏移体积仅更改体积的位置（因此，也会更改场影响的粒子）。它不会更改用于计算场力、衰减等的实际场位置。

体积扫描(Volume Sweep)

定义除立方体外的所有体积的旋转范围。该值可以是介于 0 和 360 度之间的值。

截面半径(Section Radius)

定义圆环体的实体部分的厚度（相对于圆环体的中心环的半径）。中心环的半径由场的比例确定。如果缩放场，则“截面半径”(Section Radius)将保持其相对于中心环的比例。

特殊效果(Special Effects)

逐顶点应用(Apply Per Vertex)

设定场源自对象的位置。如果启用“逐顶点应用”(Apply Per Vertex)，则选定对象的每个点（CV、粒子和顶点）会以最大强度均匀地施加场的影响。如果禁用“逐顶点应用”(Apply Per Vertex)，将只从指定点的平均位置施加场的影响。

如果使用的是“属性编辑器”(Attribute Editor)，请打开“特殊效果”(Special Effects)区域以显示“逐顶点应用”(Apply Per Vertex)属性。仅适用于对象场。
