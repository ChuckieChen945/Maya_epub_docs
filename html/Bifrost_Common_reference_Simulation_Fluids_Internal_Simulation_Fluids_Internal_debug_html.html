debug

此节点允许用户设置各种调试参数，供解算器用于输出调试信息。

输入

enable

设置为 true 可启用调试模式和切片图像渲染

visualizeTiles

设置为 true 可在渲染的切片中可视化平铺

visualizeCorners

设置为 true 可在渲染的平铺中可视化角

useLinearSampler

设置为 true 可在渲染切片时使用线性采样器

enableDebugPrint

设置为 true 可在渲染的切片中和 (tsX, tsY) 点处输出数字

flushAllocationCache

tsX

调试信息输出到标准输出时所在位置的 X 平铺坐标

tsY

调试信息输出到标准输出时所在位置的 Y 平铺坐标

frame

模拟中的输入帧

startFrame

应开始进入调试模式的帧

verboseLevel

解算器中使用的详细级别

imageScale

图像比例：输出切片图像中的每个体素都将按此量进行缩放

slice

切片坐标。切片将平行于 XY 平面

imagePath

图像将输出到的目录路径

debugFilePath

可将调试信息输出到的文件的路径

debugInt

传递到基本解算器的一个整数，可用于测试目的。

body

IO 体（对象）。
