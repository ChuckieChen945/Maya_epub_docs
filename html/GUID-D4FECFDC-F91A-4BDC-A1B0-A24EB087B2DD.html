MASH 节点概述

MASH 的主干是其节点网络。每个节点执行特定的功能，且可与其他节点组合以合成自身的行为。您可以通过 Waiter 节点的“属性编辑器”(Attribute Editor)将大多数（但并非全部）节点添加到网络。

音频(Audio) 使用音频输入文件驱动动画。
颜色(Color) 在 MASH 网络中自定义网格的 CPV 数据。
曲线(Curve) 沿曲线设置对象的动画。
延迟(Delay) 在时间上偏移对象的现有动画。
分布(Distribute) 在形式上排列对象的多个副本。
动力学(Dynamics) 将动力学作用力应用于 MASH 点。
飞行(Flight) 近似于聚集/集群行为。
ID 自定义为 MASH 点指定实例化对象的方式。
影响(Influence) 使用导向对象影响 MASH 网络的变换。
合并(Merge) 将两个 MASH 网络合并到一起。
偏移(Offset) 偏移对象变换。
方向(Orient) 将对象指向移动方向或另一个对象。
放置器(Placer) 允许通过绘制来放置 MASH 点。
Python 允许编写自定义 Python 脚本。
随机(Random) 随机数生成器。
复制器(Replicator) 复制 MASH 网络。
信号(Signal) 将 4D 噪波或三角动画添加到网络。
强度(Strength) 控制连接节点在网络上的效果程度。
弹簧(Spring) 将弹簧添加到对象的动画。
对称(Symmetry) 沿指定的轴反射整个 MASH 网络。
时间(Time) 偏移组件动画。
变换(Transform) 移动/旋转/缩放整个网络。
可见性(Visibility) 控制对象的可见性。
Waiter 充当 MASH 网络的中心，可用于添加其他节点。
世界(World) 在自然的非碰撞簇中围绕点排列对象。

禁用“仅 DAG 对象”(DAG Objects Only)时，可以通过使用 MASH 过滤器的“显示 > 对象 > MASH”(Show > Objects > MASH)在 MASH 编辑器或常规大纲视图中查看 MASH 节点。
