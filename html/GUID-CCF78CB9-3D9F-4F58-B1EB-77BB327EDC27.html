折痕集编辑器右键单击菜单

“折痕集编辑器”(Crease Set Editor)上下文相关（右键单击）菜单提供了其他方式来编辑和自定义折痕集。可通过在“折痕集编辑器”(Crease Set Editor)中的任意位置单击鼠标右键来访问此菜单。

创建集(Create Set)
创建包含选定组件（多边形边和顶点）的新折痕集，或者创建空的折痕集（如果未选定任何对象）。如果某些选定组件属于另一个折痕集，它们将被移动到新的折痕集。此选项仅在编辑器中未选定任何折痕集时才出现。
添加选定组件(Add Selected Components)
将选定组件添加到当前折痕集。如果任何选定组件属于另一个折痕集，它们将被移动到选定折痕集。
移除选定组件(Remove Selected Components)
从当前折痕集移除选定折痕组件。
删除集(Delete Set)
从“折痕集”列表中删除选定折痕集。
合并集(Merge Sets)
将“折痕集”列表中选定的多个折痕集合并成一个折痕集。此选项仅在编辑器中具有多个选定折痕集时才会出现。
使用选定组件分割集(Split Set using Selected Components)
将现有折痕集的选定成员分割成新的折痕集。
选择成员(Select Members)
在场景中选择折痕集的成员。
选择节点(Select Nodes)
显示“属性编辑器”(Attribute Editor)中选定折痕集的节点。
设置颜色(Set Color)
允许您从“颜色设置”(Color Settings)中的“用户定义”(User Defined)列表将颜色指定给每个集。
