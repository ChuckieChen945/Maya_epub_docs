创建工具动画演示

将新元素添加到 Maya 界面后，您可以创建工具动画演示为用户提供自定义的学习内容。

当您将光标悬停在 Maya 界面中某个项目之上时，工具动画演示将查找用于识别该项目的关键字名称。Maya 附带的工具动画演示将使用界面项目层次名称的最后一个元素作为“关键字名称”。在创建您自己的自定义工具动画演示时，您必须使用界面项的名称作为关键字名称。您可以通过将关键字名称分配给 ToolClips.xml 文件中的 <key> 属性，将界面项目链接到工具动画演示。（请参见工具动画演示 XML 文件的组成部分）。

创建工具动画演示

  1. 为新的界面项目指定唯一名称。
    注： 名称中不能包含尾随的数字索引值（即 myButton1、myButton2）。
  2. 将 ToolClips.xml 文件从 builtin.toolclips\Control 目录复制到 user.toolclips\Control 目录。请参见工具动画演示目录。

    您必须将 XML 文件复制到 user.toolclips 目录，因为您在 builtin.toolclips 目录中所做的任何编辑都将被软件修补程序和升级覆盖。

  3. 在您选择的文本或 XML 编辑器中，打开复制的 ToolClips.xml 文件。
    注： 如果您计划添加采用双字节编码的字符，则建议您使用支持 UTF-8 格式的编辑器。确保文件使用该指定的编码进行保存。
  4. 将新的 <toolclip> 元素添加到文件中。

    每个唯一的工具动画演示都包含在 <toolclip> 元素中。Maya 界面中显示的工具动画演示都位于 ToolClips.xml 文件中。

  5. 添加工具动画演示的内容。有关设置工具动画演示数据的格式的详细信息，请参见工具动画演示 XML 文件的组成部分。
  6. 请确保将界面项目的关键字名称分配给 <key> 属性。
  7. 保存 XML 文件。
  8. 将您在 XML 文件中指定的任何新资源添加到 user.toolclips\Resources 目录。
  9. 启动 Maya，以查看工具动画演示。
