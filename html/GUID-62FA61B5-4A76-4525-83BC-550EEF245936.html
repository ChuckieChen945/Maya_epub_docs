灯光效果(Light Effects)

“灯光效果”(Light Effects)属性控制照明（灯光）雾和光学灯光效果的外观。（有关光学效果的信息，请参见辉光、光晕和镜头光斑以及光学效果属性。）

注： 创建灯光雾时，请避免使用实例化灯光。复制灯光而不是实例化。对于灯光雾，请确保独立于灯光来复制灯光雾，因为在复制灯光时，它的灯光雾不随它一起复制。
灯光雾(Light Fog)

灯光雾属性仅可用于点光源、体积光和聚光灯。

单击“灯光雾”(Light Fog)旁边的贴图按钮时，Maya 会创建灯光雾节点，并将它连接到灯光。雾图标（本质上是灯光图标的扩展）会显示在视图中。将“灯光雾”(Light Fog)连接到灯光后，单击字段末尾的按钮可以访问灯光雾属性。

下面的示例显示使用默认设置 1.0 以及在雾扩散和雾密度设置更改为 2.0 时应用于聚光灯的“灯光雾”(Light Fog)。

注：

默认情况下，仅在边界入口点对灯光雾进行求值。当灯光雾的半径大于体积雾的大小时，照明雾的灯光量是体积光边缘处的灯光强度（默认情况下为零，但如果编辑颜色渐变，则可能为其他值）。如果对“形状”(Shape)节点（例如，sphereShapeNode）启用“体积采样数覆盖”(Volume Samples Override)，并增加采样数，则可以更好地表示体积光的照明。在这些情况下，对采样启用“深度抖动”(Depth Jitter)可避免步进。

雾扩散(Fog Spread)

仅适用于聚光灯。

确定雾亮度在聚光灯或点光源光束中如何变化。较高的“雾扩散”值产生从聚光灯的圆锥形光束投射的亮度均匀的雾。较低的“雾扩散”值产生在聚光灯光束中心较亮、边缘较模糊的雾。

聚光灯的滑块范围为 0 至 5。默认值为 1。

雾类型(Fog Type)

仅适用于点光源。

从下拉列表中选择一种“雾类型”(Fog Type)。雾类型包括：

正常(Normal)

默认的雾设置。

线性(Linear)

雾从灯光的中心缓慢减弱。

指数(Exponential)

雾从灯光的中心快速减弱。

雾半径(Fog Radius)

仅适用于点光源。

“雾半径”(Fog Radius)值确定雾在点光源光束的每个方向上的扩散程度。聚光灯或点光源的“衰退速率”(Decay Rate)控制雾亮度如何随距光源的距离而降低。

点光源的滑块范围为 0 至 10。默认值为 1。

雾密度(Fog Intensity)

雾的亮度。（灯光的“强度”(Intensity)也影响照明雾的亮度）。滑块范围为 0 至 5。默认值为 1。

灯光辉光(Light Glow)

“灯光辉光”(Light Glow)属性仅可用于点光源、体积光和聚光灯。

当单击“灯光辉光”(Light Glow)旁边的贴图按钮时，Maya 会创建光学效果节点，并将它连接到灯光（辉光、光晕或镜头光斑）。附加到灯光图标的光学效果图标显示在视图中。有关设置这些属性的详细信息，请参见光学效果属性。

强度曲线(Intensity Curve)

仅适用于聚光灯。

创建用于控制聚光灯强度衰退的强度曲线。该属性仅可用于聚光灯。另请参见创建自定义聚光灯强度或颜色衰退和编辑自定义强度和颜色衰退。

颜色曲线(Color Curves)

仅适用于聚光灯。

创建用于控制聚光灯的颜色如何随距离变化的颜色曲线。

另请参见创建自定义聚光灯强度或颜色衰退和编辑自定义强度和颜色衰退。

挡光板(Barn doors)

“挡光板”(Barn Doors)属性位于聚光灯“属性编辑器”(Attribute Editor)的“灯光效果”(Light Effects)区域，用于将圆锥形光束的圆形处理成方形。

注：

可以交互方式调整挡光板。请参见对灯光的光束应用挡光板（快门）。

左侧(Left)、右侧(Right)、顶(Top)、底部挡光板(Bottom Barn Door)

从聚光灯光束的中心到每个挡光板位置的角度（以度为单位）。有效/滑块范围为 -89.997 至 89.997。默认值为 20。

衰退区域属性(Decay Regions attributes)

“衰退区域”(Decay Regions)属性允许您将聚光灯的光束分为照明区域和非照明区域。

单击“使用衰退区域”(Use Decay Regions)可将其启用并设定聚光灯的区域 1、区域 2 和区域 3 的开始距离和结束距离。“使用衰退区域”(Use Decay Regions)默认情况下处于禁用状态。
