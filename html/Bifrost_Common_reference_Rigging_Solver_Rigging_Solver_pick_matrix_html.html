pick_matrix

创建包含从输入变换选择的变换分量的新变换矩阵。

输入

transform

要分解的 double4x4 变换。

use_shear

如果启用此项，则保留输入 transform 的斜切分量。

use_scale

如果启用此项，则保留输入 transform 的缩放分量。

use_rotation

如果启用此项，则保留输入 transform 的方向分量。

use_translation

如果启用此项，则保留输入 transform 的平移分量。

输出

out_transform

生成的 array<double4x4> 变换矩阵。
