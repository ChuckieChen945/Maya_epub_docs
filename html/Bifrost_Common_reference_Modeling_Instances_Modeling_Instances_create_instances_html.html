create_instances

创建由附加到输入几何体的点位置的形状组成的实例对象。支持多种形状，并且可以采用多种方式指定形状。

输入

points

具有要在其所在位置处创建实例化形状的点的几何体。

instance_geometries

一个或多个要实例化的几何体。您还可以连接实例形状，例如 geo_instance 节点的输出。

mask_weights

将在每个点处实例化形状的概率。

  • 小于等于 0 的值表示从不在某个点的位置处实例化某个形状。
  • 0 到 1 范围内的值表示在某个点的位置处实例化某个形状的几率。例如，0.5 表示实例化形状的概率为 50%。
  • 大于等于 1.0 的值表示始终在某个点的位置处实例化某个形状。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<bool> 解释为每点的 0 (false) 或 1 (true) 值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

选择

selection_mode

如何将形状指定给点：

  • None 表示将第一个形状指定给所有实例。
  • Sequential 表示循环形状。第一个形状指定给第一个点，第二个形状指定给第二个点，依此类推。
  • Random 表示随机指定形状。可使用 probability_curve 控制分布。

instance_id_override

（可选）连接包含形状索引的 array<long> 以逐点指定。这将直接控制指定的形状。

随机化

probability_curve

此 FCurve 可在 selection_mode 为 Random 时控制随机 ID 分布。默认形状没有偏移，但您可以编辑曲线以使用较高或较低的 ID。如果 ID 的顺序有意义（例如，如果它们按大小顺序排列），这非常有用。

seed

当 selection_mode 为 Random 时，使用的随机数生成器的种子。

预览几何体

preview_mode

指定要在视口中显示的内容，而不是最终渲染。

  • BoundingBox 显示包围实例几何体范围的边界框。
  • PreviewGeometry 显示连接到 preview_geometries 的几何体。如果未连接任何对象，则显示实例几何体。
  • InstanceGeometry 显示实例几何体，即使连接了预览几何体也是如此。

preview_geometries

（可选）连接要在视口中显示的几何体。几何体越简单，预览就越快。
