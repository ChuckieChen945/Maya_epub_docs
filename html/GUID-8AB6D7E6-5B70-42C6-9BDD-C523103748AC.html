分析器标记菜单

在“分析器”(Profiler)的绘制区域单击鼠标右键可显示以下菜单项：

隐藏选定的事件(Hide selected events) / 隐藏重复的选定事件(Hide selected events repetition)
选择第一个选项可隐藏您在绘制区域中选定的事件。选择第二个选项可隐藏您在绘制区域中选定的事件的所有实例。
仅显示选定事件(Show only selected events) / 仅显示重复的选定事件(Show only selected events repetition)
选择第一个选项可隐藏绘制区域中除选定事件以外的所有事件。选择第二个选项可隐藏绘制区域中除选定事件的实例以外的所有事件。
全部取消隐藏(Unhide all)
选择该选项可恢复为默认设置，并显示所有事件。
收拢选定事件(Collapse Selected events)/ 收拢重复的选定事件(Collapse Selected events repetition)
选择“收拢选定事件”(Collapse Selected events)可将所有子事件（在父事件的同一线程中启动且包含在父事件的时间间隔内，即在事件堆栈中处于更高位置）收拢到选定事件中。选择“收拢重复的选定事件”(Collapse Selected events repetition)将不仅收拢选定事件，而且还收拢它们在任何其他帧中的重复项。因此，如果收拢选定事件，也会在其他帧中隐藏它们。
展开选定事件(Expand Selected events) / 展开重复的选定事件(Expand Selected events repetition)
选择第一个选项可还原收拢到选定事件中的所有子事件。选择“展开重复的选定事件”(Expand Selected events repetition)将不仅展开选定事件，而且还展开它们在任何其他帧中的重复项。因此，如果展开选定事件，也会在其他帧中显示它们。
显示关键路径(Show Critical path)
关键路径是求值任务序列，这些任务确定在有无限计算资源可用的情况下，计算一帧所需的最小时间。这是最小求值时间（即，帧的求值速度不可能快于此任务链）。
激活此选项可使分析器根据当前选定帧和当前调度图检测关键路径，以及显示映射到调度节点的执行的事件。
