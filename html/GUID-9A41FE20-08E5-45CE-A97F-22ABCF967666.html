wire

参数

选择下列选项之一。

旋转(Rotation)

指定在斜切与相切之间变化的效果。值的范围是 0 到 1，0 指示最大斜切，1 指示最大相切。使用滑块选择介于 0.000 到 1.000 之间的值。默认值为 1.000。

交叉效果

指定两条影响线交叉时变形效果的幅度。值可以在 0（平滑效果）到 1（相加效果）之间变化。默认值为 0。使用滑块选择介于 0.000 到 1.000 之间的值。默认值为 0.000。

局部影响

指定在局部应用两条或更多影响线的变形效果。值的范围从 0 到 1，1 指定最大局部。使用滑块选择介于 0.000 到 1.000 之间的值。默认值为 0.000。

张力(Tension)

指定影响线的吸引强度。其值指定影响线通过吸引可变形对象的点以远离基础线的强度。因此吸引强度表示影响线与基础线之间的张力。使用滑块选择介于 -10.000 到 10.000 之间的值。默认值为 1.000。

冻结几何体

指定是否冻结线变形效果。如果冻结（启用），变形对象的受影响线影响的组件（例如 CV）会变为固定的，仅受影响线的影响，即使变换（移动、旋转或缩放）对象或基础线时也是如此。需要冻结几何体的原因是为了提高性能。请注意，您不应在选中“冻结几何体”的情况下相对于基础线移动几何体对象。默认设置为禁用。

缩放

指定每条影响线的影响比例。默认值为 1.00。

衰减距离

指定每条影响线的影响范围。默认值为 1.00。

定位器

如果已创建任何线衰减定位器，此部分会列出每条影响线上每个线衰减定位器的属性。

curveShapen -> locatorn

指定该线上的影响线形状（默认名称：curveShapen）和线衰减定位器（默认名称：locatorn）。定位器按照其创建顺序编号（从 1 开始）。

Param[n]

指定线衰减定位器在影响线曲线中的位置。n 值指示是哪个线衰减定位器，以创建顺序编号（从 0 开始）。该值依据曲线的 U 参数。

Percent[n]

指定定位器对影响线的衰减产生的局部影响。n 值表示特定线衰减定位器，以创建顺序编号（从 0 开始）。默认情况下，影响线在每端有两个隐式定位器，“百分比”(Percent)为 1.000。其他定位器的影响相对于这些定位器的“百分比”(Percent)有效。使用滑块选择介于 0.000 到 1.000 之间的值。默认值为 1.000。

Twist[n]

指定线衰减定位器周围的局部扭曲效果。n 值表示特定线衰减定位器，以创建顺序编号（从 0 开始）。在增大或减小该值时，受线衰减定位器影响的曲面区域将发生扭曲。在角色设置中，更改“扭曲”(Twist)可以为嘴唇和眼眉动作提供精细效果。默认值为 0（不指定任何扭曲效果）。此属性与“线衰减扭曲”(Wire Dropoff Twist)通道对应。

Envelope[n]

指定线衰减定位器的变形比例因子。n 值表示特定线衰减定位器，以创建顺序编号（从 0 开始）。默认值为 1。该属性与“定位器封套”(Locator Envelope)通道相对应。
