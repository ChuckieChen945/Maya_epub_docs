update_strands_point_neighbors

计算股几何体中每个点的相邻点。

例如，给定一个具有 3 个点的股：

+------+------+
p0     p1     p2

点 p1 的相邻点是 p0 和 p2。端点 p0 和 p2 仅有 p1 作为相邻点。

输入

strands

要查询的股。

输出

out_strands

具有名为 point_neighbors 的附加点几何体特性的股对象。这是一个由单位值组成的 2D 数组。每个点都有一个数组。此数组包含点的相邻点的索引。股中间的点将具有两个相邻点，即上一个点和下一个点。股开头的点将仅有一个相邻点，即下一个点。股末端的点也仅有一个相邻点，即上一个点。

point_neighbors

数据的相邻点数组。有关详细描述，请参见 out_strands 端口。
