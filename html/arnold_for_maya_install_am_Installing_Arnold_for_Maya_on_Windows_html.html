在 Windows 上安装 Arnold for Maya

默认情况下，Arnold for Maya (MtoA) 安装程序会为 MtoA 模块文件 (mtoa.mod) 设置必要的环境变量，并为批渲染创建 Arnold 渲染器描述文件。我们建议大多数用户采用此方式安装 MtoA。希望执行网络安装的专家用户请参见手动安装部分。

重要信息：

如果已安装的旧版本 MtoA 使用 Maya.env 设置环境变量，请确保先从 Maya.env 文件中移除这些环境变量，然后再安装最新的 MtoA。

  1. 从 Arnold 网站下载 MtoA。
  2. 双击下载的 .exe 文件开始安装。在 Windows 8 中，您可能会收到一条警告消息，指出“Windows 已保护你的电脑”。如果发生这种情况，请单击“更多信息”，然后单击“仍要运行”继续进行安装。
  3. 保留默认安装选项，并在安装向导中一直单击“下一步”，直到设置完成。

  1. 选择要安装的 MtoA 功能。

  1. 安装程序将自动检测安装路径。另外，您也可以选择希望的安装路径。

  1. 选择 MtoA 将出现在“开始”菜单的哪个文件夹中。

  2. 单击“完成”以关闭“设置”向导。

  3. 启动 Maya，并单击“窗口 > 设置/首选项 > 插件管理器”(Window > Settings/Preferences > Plug-in Manager)。

  1. 在“插件管理器”(Plug-in Manager)中，为 mtoa.mll 选中“已加载”(Loaded)和“自动加载”(Auto Load)复选框，然后关闭“插件管理器”(Plug-in Manager)。

这时应该可以使用 Arnold 进行渲染。

手动安装

如果愿意，您可以将 MtoA 提取到临时文件夹，然后将 MtoA 软件包复制到安装位置。在希望通过网络安装 Arnold 等情况下，这种方法可能会很有用。

要执行此操作，请在运行 MtoA 安装程序时，清除“为 Maya 配置 MtoA”(Configure MtoA for Maya)复选框。以下说明假定您使用的是 Windows 和 Maya 2023，而且您将 MtoA 复制到网络文件夹 \server\arnold\mtoa\2023。

接下来，完成以下步骤：

  1. 将 MtoA 提取到临时位置时，请清除“为 Maya 配置 MtoA”(Configure MtoA for Maya)复选框。

  2. 将 MtoA 软件包复制到安装位置（例如 \\server\arnold\mtoa\2023）。

  3. 首先，编辑位于 MtoA 安装位置的 mtoa.mod 文件，并更新第一行的路径，使它与实际安装位置一致。例如：

    + mtoa any \\server\arnold\mtoa\2023
PATH +:= bin

  4. 设置环境变量 MAYA_MODULE_PATH，使其指向 mtoa.mod 文件的位置（例如 \server\arnold\mtoa\2023）。

  5. 设置环境变量 MAYA_RENDER_DESC_PATH，使其指向 arnoldRenderer.xml 文件的位置（默认为 \server\arnold\mtoa\2023）。如果要使用 Arnold 执行批渲染，此步骤是必需的。

  6. 现在，运行 Maya。您应该看到“Arnold 渲染器”(Arnold Renderer)在“渲染 > 使用以下渲染器渲染”(Render > Render Using)菜单下列出。如果未列出该渲染器，请转到“窗口 > 设置/首选项 > 插件管理器”(Window > Settings/Preferences > Plug-in Manager)。

卸载

如果之前使用安装程序安装了先前版本的 MtoA，则安装程序会检测到此状况，并指出必须先卸载旧版本后才能安装新版本。如果安装程序在指定位置找到旧版本，则会自动运行卸载程序。或者，也可以执行以下操作：

  • 在 Windows 开始菜单中找到 MtoA 组，选择“卸载”，从而移除旧版本，或者，
  • 找到旧版本 MtoA 安装文件夹中的 Uninstall.exe 文件，然后自行执行卸载，或者
  • 使用 Windows 中的“添加/删除程序”功能。
