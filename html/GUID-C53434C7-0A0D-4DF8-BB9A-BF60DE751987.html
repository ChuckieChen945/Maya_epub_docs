雕刻工具热键

注：
如果您使用的是 macOS，除非另有指定，否则请使用以下特殊键，而不是其 Windows 等效键：
  • Alt = Option
  • Ctrl = Control
  • Windows 键 = Command
此外，如果您在 macOS 上使用单按钮或双按钮鼠标，请确保根据需要在界面首选项(Interface Preferences)中设置“鼠标跟踪”(Mouse Tracking)选项。

Ctrl

如果当前工具具有反转功能，则在雕刻时将应用反转功能。

“反转”(Invert)处于活动状态时，工具的笔刷环将变为白色。

Ctrl

Ctrl

沿平均化法线在笔刷内拉动顶点。

注： “平均化法线”(Averaged Normal)特定于“抓取”(Grab)雕刻工具。
