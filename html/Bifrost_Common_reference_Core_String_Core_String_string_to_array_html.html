string_to_array

将 comma_separated_string 转化为字符串、long 整型或 floats 数组。这使您可以在图表中快速写入字符串、浮点或（长）整型数组。

字符串用逗号分隔，子字符串两端的空格被去除。

comma_separated_string:   "2, 4, 6"
string_array:             ["2", "4", "6"]
float_array:              [2.0, 4.0, 6.0]
long_int_array:           [2, 4, 6]

子字符串两端的空格被去除。

comma_separated_string: "One   string,   two string  "
string_array:           ["One   string", "two string"]

无法转化为整型或浮点型的值在浮点或整型数组中变为零值。

输入

comma_separated_string

端口类型是：string。

处理为数组的字符串。

输出

string_array

端口类型是：array。

这些是拆分字符串，字符串两端的空格被去除。

float_array

端口类型是：array。

这些是转化为浮点型的拆分值。不可转化的值将变为 0.0。

long_int_array

端口类型是：array。

这些是转化为浮点型的拆分值。不可转化的值将变为 0。
