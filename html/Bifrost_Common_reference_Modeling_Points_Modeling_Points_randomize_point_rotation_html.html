randomize_point_rotation

随机化 point_orientation 特性。这将影响实例的方向以及点形状（如果它们未与摄影机对齐）。但是，它不会影响整个几何对象的旋转。

输入

points

具有要旋转的点的几何体。

weights

控制应用的随机化量。

  • 权重为 0 时，不应用随机化。
  • 权重在 0 到 1 范围内时，按 min_radians 和 max_radians 之间的值的一部分进行随机化。
  • 权重为 1 时，完全按 min_radians 和 max_radians 之间的值进行随机化。
  • 权重大于 1 时，点移动的距离大于给定值。
  • 权重在 0 到 1 范围以外时，按 [min_radians, max_radians) 范围以外的量进行随机化。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

随机化

min_radians、max_radians

最小和最大旋转，以 Euler 角度表示（以弧度为单位）。

seed

随机数生成器的种子。

rotate_existing_values

当此选项为 true 时，随机旋转将与任何现有 point_orientation 值组合使用。local_space 选项控制如何组合新旋转和旧方向。当 rotate_existing_values 为 false 时，随机值将覆盖任何现有方向。

local_space

仅当 rotate_existing_values 为 true 时，此选项才适用。此选项可控制如何组合新旋转和旧方向。如果此选项为 true，则将在点的当前局部空间中应用随机化旋转。

rotation_order

应用随机 Euler 旋转的顺序。

distribution

用于控制随机分布的 FCurve。默认形状不会向 min_radians 或 max_radians 偏移。

输出

out_points

具有已旋转点的几何体。

point_orientation

在几何体上设置的 point_orientation 值。
