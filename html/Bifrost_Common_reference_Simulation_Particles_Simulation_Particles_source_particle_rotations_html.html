source_particle_rotations

粒子源的一些其他特性，用于控制粒子的旋转

要使用此节点，请将 source_particles_rotations.rotation_source 输出插入 source_particles 复合的 additional_properties 输入。

输入

enable

启用此节点的粒子旋转。

orientation

要为粒子旋转设置的初始值。

spin

要赋予粒子的自旋速率。

输出

rotation_source

输出是一个包含旋转特性的对象，应连接到 source_particles 节点的 additional_properties 端口。
