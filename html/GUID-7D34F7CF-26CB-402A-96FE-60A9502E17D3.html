关于编辑簇变形效果

创建簇变形器后，可以通过以下方法编辑变形器的效果。

操纵簇控制柄（C 图标）

每个簇变形器包括一个簇变形器控制柄。在场景中，控制柄为一个 C 图标（字母“C”）。可以选择控制柄，然后移动、旋转和缩放控制柄，以创建变形效果。操纵控制柄的效果取决于簇权重，而簇权重又控制可变形对象的点（NURBS CV 点、多边形顶点或晶格点）上的簇变形器效果。另请参见编辑簇权重和绘制簇权重。

请注意，褶皱变形器使用簇变形器，而且，您也可以操纵具有簇变形器控制柄的 C 图标的褶皱变形器的效果。有关褶皱变形器的详细信息，请参见褶皱变形器。

通过在属性编辑器中更改控制柄的“原点”(Origin)值，可以控制簇控制柄的放置。选择控制柄，在属性编辑器中查找 clustern HandleShape 选项卡，并设置所需的 X、Y 和 Z 轴坐标值。

选择簇变形器控制柄

    选择“c”，然后按住 Shift 键并选择以移除其他所有对象，或在“大纲视图”(Outliner)中选择簇控制柄。

    簇控制柄在对象上用一个小的“c”表示。可以在大纲视图(Outliner)中选择它。
  1. 选择带有簇“c”图标的区域。
  2. 按住 Shift 键并选择以从选择中排除其他所有对象。

编辑簇权重

创建簇后，根据您想要的点的数量设置百分比，或控制要影响的顶点。

有些情况下，您可能会为整个对象设定一个百分比。例如，移动的水上的连杆，其中，对簇控制柄进行设定，以便在 50% 的水的变换上移动。

在其他时候，您可能会希望几何体的某些部位要多少受其他部位的影响。例如，您可以拥有具有摇曳的树的场景，其中树梢受的影响最大，而且靠近地面的树干受 0% 的影响。

通过使用“组件编辑器”(Component Editor)，可以直接修改各个簇权重的值。也可以绘制簇权重。

关于绘制簇权重

通过使用“绘制簇权重工具”(Paint Cluster Weights Tool)在簇的曲面上进行绘制，可以简单地设定簇权重。虽然可以首先变换簇，然后在曲面上绘制权重，由于“绘制簇权重工具”(Paint Cluster Weights Tool)提供颜色反馈，所以您可在变换簇之前知道簇的哪些部位具有不同的权重。权重显示为一系列的灰度值，灰度值为 1 和 0 时，权重分别显示为白色和黑色。请参见绘制簇权重工具。

也可以使用“镜像变形器权重选项”(Mirror Deformer Weights Options)来镜像簇权重请参见镜像变形器权重。

在受限区绘制权重

在选定的顶点上绘制权重时，笔划将仅适用于选定顶点对应的权重。实际上，未选定的顶点充当遮罩，其中只有选定的顶点将受您执行的任意绘制或填充的影响。

创建遮罩之前，必须首先创建簇。有关遮罩曲面的详细信息，请参见限制绘制区域。

填充簇

对簇进行填充就像是拿着一把巨大的笔刷并将其设置应用于整个簇。对簇进行填充时，簇中每个顶点的权重根据工具的值和操作的设置而改变。

若要整体应用簇，请按照绘制簇变形器权重下的步骤操作，但不执行步骤 7 中的绘制，而是单击“整体应用”(Flood)按钮或使用热键 Alt+f（Linux 和 Windows）或 Option+f (Mac OS X)。

提示：

若要使簇权重之间的过渡平滑，请选择“平滑”(Smooth)绘制操作并对簇进行填充。

删减簇变形器集

通过删减簇变形器集，可以从当前不受变形器影响的集中移除点。您可以删减变形器集，以避免对不属于变形效果的点的不必要的计算。请参考主题编辑簇变形器中的步骤“删减变形器集成员身份”。

相对于父对象变换对簇进行设定

使用“相对”(Relative)属性时，只有在簇控制柄的直接父对象变换时，才可以将簇变形设定为活动的。这样可以创建效果，其中，层次的父对象不会全影响簇变形。

例如，如果使簇控制柄成为腕关节的子对象，并启用其“相对”属性，那么即使腕部位置发生更改，也可在簇不影响腕部周围蒙皮的情况下旋转肩部。移动腕部本身时，簇会根据需要对几何体变形。

控制整个簇的变形百分比

通过使用“封套”(Envelope)通道，可以控制整个簇的变形百分比。更改“封套”(Envelope)，以便以相同的方式缩放所有的簇权重。如果设定“封套”(Envelope)值，并同时设定簇权重的各个值，则所有的值都会影响变形。

使用加权节点

可以使用簇控制柄的另一个对象，而簇控制柄的移动可以控制簇。在“加权节点”(Weighted Node)属性下，指定要在簇控制柄形状节点中使用的对象。
