删除粒子碰撞事件

可以使用“粒子碰撞事件编辑器”(Particle Collision Event Editor)删除不希望再发生的事件。

删除粒子或 nParticle 碰撞事件

  1. 对于经典粒子对象，在 FX 菜单集中，选择“nParticle > 粒子碰撞事件编辑器”(nParticles > Particle Collision Events Editor)。

    将显示“粒子碰撞事件编辑器”(Particle Collision Event Editor)。

  2. 从“粒子碰撞事件编辑器”(Particle Collision Event Editor)的“对象”(Objects)面板中，选择与要删除的事件相关联的粒子对象。

    与选定粒子对象相关联的事件会显示在“事件”(Events)面板中。

  3. 在“事件”(Events)面板中，选择要删除的事件。
  4. 单击“删除事件”(Delete Event)。
