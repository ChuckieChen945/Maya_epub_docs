XGen“风”(Wind)修改器选项

您可以使用“风”(Wind)修改器为样条线基本体应用风效果。风力由 3D 向量场控制，该场中包含主要方向力以及与主要风力垂直的二次力。设置风修改器时，请考虑以下事项：

  • 使用“方向”(Direction)属性设置向量风的主要方向。默认情况下，风力沿 X 轴移动。
  • 使用“斜切频率”(Shear Freq)和“斜切强度”(Shear Strength)属性可设置垂直于“方向”(Direction)的风向量。这样，可以在风力中生成简单的湍流。
  • 使用“阵风强度”(Gust Strength)属性可设置风力的强度。
  • 对于低频噪波，使用“低强度”(Low S)和“低偏移”(Lo Bias)属性为噪波添加主“方向”(Direction)和“斜切”(Shear)向量；对于高频噪波，使用“高强度”(Hi S)和“高偏移”(Hi Bias)为噪波添加这些向量。

风修改器(Wind Modifier)

遮罩(Mask)

可用于控制修改器如何影响表面指定区域中的基本体。例如，使用“遮罩”(Mask)加载表达式或创建遮罩修改器效果的 Ptex 贴图。

刚度(Stiffness)

刚度越高，风中的样条线弯曲越小。

方向(Direction)

风的主要向量方向。

恒定强度(Const Strength)

风在其主要向量方向的恒定强度。

阵风强度(Gust Strength)

风组件在其主要方向（由噪波定位器控制）上的最大强度。

斜切强度(Shear Strength)

风组件在其次要方向（垂直于主要方向）上的最高强度。

斜切频率(Shear Freq)

斜切频率（在横向的风组件）。

种子(Seed)

噪波生成的随机化种子。

首选噪波(Pref Noise)

当处理动画或变形网格时，启用此选项可停止噪波场中不需要的运动。

有关首选项的详细信息，请参见 XGen 面片。

低强度(Low S)

设置低频噪波。

低偏移(Low Bias)

设置低频噪波偏移。

高强度(Hi S)

设置高频噪波。

高偏移(Hi Bias)

设置高频噪波偏移。

高频率(Hi Freq)
设置高频率（“高强度”(Hi S)）的值。低频率（“低强度”(Low S)）将调整为小于此值。
刚体(Rigid Body)
启用后，基本体的 CV 处于锁定状态，因此在受风力影响时，其行为与刚性对象类似。
束(Clump)

启用后，基本体将被视为风力中的束。要达到此目的，必须创建成束贴图，例如，由“成束”(Clumping)修改器使用的贴图。请参见为 XGen 修改器创建点贴图。

可循环风(Loopable Wind)

可循环(Loopable)

使“风修改器”(Wind Modifier)效果在指定的“循环开始帧”(Loop Start Frame)和“循环结束帧”(Loop End Frame)之间循环。

循环开始帧(Loop Start Frame)

可循环噪波的开始帧。

循环结束帧(Loop End Frame)

可循环噪波的结束帧。

循环噪波跨度(Loop Noise Span)

设置风在循环中移动时的噪波场单位。
