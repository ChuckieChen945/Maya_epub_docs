“MEL 分析器”(MEL Profiler)菜单

分析

开始(Start)
启动分析过程。
停止(Stop)
停止分析过程。
分析程序(Profile Procedure)
打开“分析程序”(Profile Procedure)对话框。
分析脚本文件(Profile Script File)
打开“源 MEL 脚本文件”(Source a MEL Script File)窗口，可以选择要分析的 MEL 脚本文件。
保存分析数据(Save Profile Data)
打开“保存分析数据”(Save Profile Data)窗口，使您可以在 .csv 文件中保存生成的数据。
保存调用图形数据(Save Call Graph Data)
打开“保存调用图形数据”(Save Call Graph Data)窗口，使您可以在 .dot 文件中保存生成的数据。
重置数据(Reset Data)
从“MEL 分析器”(MEL Profiler)窗口中清除分析数据。
