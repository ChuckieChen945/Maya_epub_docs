set_layer_permission

此节点用于更改层权限。

用于以下非常具体的情况：要引用的层需要由 Bifrost 以外的其他运行时修改。

例如，如果 Bifrost 生成的后台文件引用一个“基于文件”的层，该层存储的一些 USD 材质应在 LookdevX 编辑器中编写，默认情况下这无法完成。这是因为，为了使引用的层保持“基于文件”，需要在 open_layer 节点中启用“只读”模式。如果未处于只读模式，打开的层（“基于文件”的层）将会被 Bifrost USD 自动复制到一个匿名层中（以避免产生负面影响），因此无法以持久方式在 LookdevX 中保持可编辑（因为在每次执行图表时，层标识符都会发生变化）。

要在 Bifrost 中打开层并使其在外部可编辑，节点图表应如下所示：open_layer（read_only 为“启用”）-> set_layer_permission（read_only 为“禁用”）-> add_reference_prim

输入

read_only

将层设置为不可编辑。

layer

要在其中设置权限的层。

输出

父页面: Layer
