“文件路径编辑器”(File Path Editor)菜单

选项

按路径列出(List by Path)
按文件路径列出节点。“文件路径编辑器”(File Path Editor)支持显示绝对和相对路径。
按文件类型列出(List by File Type)
按文件类型列出节点。按注册的节点类型对节点进行分组。最初已注册音频、文件、图像平面和引用等节点类型。每个注册元素的类型标签在此列表中使用。请参见在文件路径编辑器注册元素。

编辑

重新指定路径(Repath)
使用上一个重新指定路径操作中的标准重新指定选定元素的路径。选择“重新指定路径”(Repath) > ，打开“重新指定路径”窗口。
替换字符串(Replace String)
使用上一个操作中的参数对选定元素执行搜索和替换操作。选择“替换字符串”(Replace String) > ，打开“替换字符串”工具。
自动解析(Auto Resolve)
使用在“自动解析”(Auto Resolve)窗口中设置的参数重新指定选定元素的路径。选择“自动解析”(Auto Resolve) > ，打开“自动解析”窗口。

Select (选择)

允许您设置在“文件路径编辑器”(File Path Editor)中选择的内容。

全选(Select All)
选择“文件路径编辑器”(File Path Editor)浏览器中列出的所有节点。
全部不选(Select None)
取消选择您在“文件路径编辑器”(File Path Editor)浏览器中的选择。
提示： 单击“文件路径编辑器”(File Path Editor)浏览器的任意空白区域以取消选择您的选择。
选择未解析(Select Unresolved)
选择“文件路径编辑器”(File Path Editor)浏览器中列出的具有未解析文件路径的所有节点。
按类型选择(Select By Type)
允许您按注册的节点类型进行选择。最初已注册以下节点类型：
音频(Audio)
选择“文件路径编辑器”(File Path Editor)浏览器中列出的所有音频节点。
纹理(Texture)
选择“文件路径编辑器”(File Path Editor)浏览器中列出的所有纹理节点。
图像平面(Image Plane)
选择“文件路径编辑器”(File Path Editor)浏览器中列出的所有图像平面节点。
引用(Reference)
选择“文件路径编辑器”(File Path Editor)浏览器中列出的所有引用节点。
反选(Invert Selection)
选择上次选择中未包含的所有节点。

显示

“显示”(Show)菜单包含的选项可控制“文件路径编辑器”(File Path Editor)浏览器中包含的元素。在您注册新节点类型时，此菜单动态更新。请参见在文件路径编辑器注册元素。

如果在“显示”(Show)菜单中启用了某个元素，则该元素类型将显示在“文件路径编辑器”(File Path Editor)浏览器中。如果禁用了某个元素，则该元素类型将不显示在浏览器中，且不会受到重新指定路径或替换字符串操作的影响。

通过“全部选中”(Check All)和“全部不选”(Check None)选项，可以快速启用和禁用所有选项。

注： 顶层级“文件路径编辑器”(File Path Editor )浏览器将 Maya 场景中所有元素的路径考虑在内，而不管“显示”(Show)菜单中的设置如何。

文件路径编辑器上下文菜单

可通过在“文件路径编辑器”(File Path Editor)浏览器列表中的一个元素上单击鼠标右键来访问上下文菜单。

重新指定路径(Repath)
使用上一个重新指定路径操作中的标准重新指定选定元素的路径。选择“重新指定路径”(Repath) > ，打开“重新指定路径”窗口。
替换字符串(Replace String)
使用上一个操作中的参数对选定元素执行搜索和替换操作。选择“替换字符串”(Replace String) > ，打开“替换字符串”工具。
自动解析(Auto Resolve)
使用在“自动解析”(Auto Resolve)窗口中设置的参数重新指定选定元素的路径。选择“自动解析”(Auto Resolve) > ，打开“自动解析”窗口。
节点编辑器中的图形(Graph in Node Editor)
打开“节点编辑器”(Node Editor)，并绘制“文件路径编辑器”(File Path Editor)选择的节点网络。
Hypershade 中的图形(Graph in Hypershade)
打开“Hypershade”，并绘制“文件路径编辑器”(File Path Editor)选择的节点网络。如果选择多个节点，则它们将绘制在一个“Hypershade”窗口中。
显示在 Explorer 中(Reveal in Explorer)
为选定的每个已解析路径打开一个平台特定的文件浏览器。
刷新列表(Refresh List)
更新“文件路径编辑器”(File Path Editor)浏览器的内容以匹配 Maya 的当前状态。
