创建“Paint Effects”修改器

创建“Paint Effects”修改器以同时影响所有的“Paint Effects”线/实体。例如，可以编辑“Paint Effects”树的形状、修剪树叶、局部修改颜色、去除分支等。可以将同一修改器指定给多个节点；例如，它可以同时影响“Paint Effects”笔划和卡通线。您也可以将多个修改器指定给一个对象；将不同修改器的影响融合在一起。您可能希望将对象和修改器分组到一起以使它们同时移动。

从“Paint Effects”菜单中选择“创建修改器”(Create Modifier)菜单项时，将出现立方形或球形修改器，您可以将它变换为所需形状。使用 lineModifierShape 节点中的属性，可控制“Paint Effects”笔划在由修改器所占用空间内的外观。这些属性对局部编辑线宽、透明度、颜色和其他属性非常有用。

创建“Paint Effects”修改器

  1. 选择您要修改的“Paint Effects”笔划、头发或卡通线。
  2. 从“建模”(Modeling)菜单集中，选择“生成 > 创建修改器”(Generate > Create Modifier)。

    线框修改器对象将创建为位于笔划中心附近的球体或立方体。

  3. 选择修改器，然后打开属性编辑器。请注意，已创建两个节点：lineModifier 和 lineModifierShape。
  4. 在属性编辑器中，单击“lineModifierShape”选项卡并展开“线修改器属性”(Line Modifier Attributes)。

    请注意，“形状”(Shape)设置可用于将修改器对象设置为球体或立方体。有关其他属性设置的信息，请参见 lineModifierShape 节点。

  5. 使用工具箱中的工具对修改器的形状和大小进行修改，使其包围要修改的笔划的体积。
  6. （可选）对修改器设置关键帧。
