define_usd_attribute

定义 USD 属性和值（可选）。输出可以连接到 define_usd_prim 和其他节点的 attribute_definitions 端口，以在相应的基本体上设置属性。

输入

name

属性的名称。

custom

声明属性是否由用户定义。

类型

type

属性的 USD 类型。

enable_value

将属性设置为下面设定的 value。如果取消选中此选项，则定义属性，但不设置值。

value

此端口默认为 Bifrost float 值，但可以单击鼠标右键设置其他类型，或者简单地将值与所需类型连接起来。使用与上述 USD type 兼容的 Bifrost 类型，例如，Normal3f 或 Color3f 使用 float3，Token 或 Asset 使用 string，依此类推。

连接

enable_connection

将属性连接到指定基本体的属性。如果取消选中此选项，则不进行连接。

connection_mode

target_prim

要连接的基本体的路径。

target_attribute

要连接的基本体的属性。

基本体变量

enable_primvar

向渲染器声明该属性与几何基本体相关联，可随曲面或体积而变化。

interpolation

指定属性应如何跨曲面或体积插值。

时间采样

frame

当 use_frame 为 true 时要设置 value 的帧。

use_frame

是否存储具有值的帧。

输出

attribute_definition

定义的属性。将其连接到 define_usd_prim 和其他节点的 attribute_definitions 端口，以在相应的基本体上设置属性。
