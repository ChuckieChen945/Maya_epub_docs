translate_points

按指定的平移偏移基于点的几何体的 point_position 特性。

输入

points

具有要移动的点的几何体。

weights

要应用的 translation 值的比例。

  • 权重为 0 时，不进行平移。
  • 权重在 0 到 1 范围内时，按给定 translation 值的相应比例移动点。
  • 权重为 1 时，按精确的 translation 值移动点。
  • 权重大于 1 时，点移动的距离大于给定的 translation 值。
  • 权重小于 0 时，朝相反方向移动点。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

translation

要按其移动点的基本向量。此外，实际偏移还与 weights 值相关。

这是一个解释的自动端口，其接受以下类型：

  • 连接单个 float3 以将其值用于所有点。
  • 连接 array<float3> 以对每个点使用数组的相应元素。可根据需要调整数组大小以匹配点数。
  • 连接 string 以使用具有相应名称的特性给定的向量（每个点的 float3）。
  • 连接 VectorField 以使用点的当前位置之处的场的值。

请注意，也可以使用标量值而不使用向量，在这种情况下，将对所有三个轴使用单个元素。但是，这对平移几乎没有任何用处。

local_space

如果选中此选项，则将在每个点的局部坐标空间中应用平移，同时考虑每个点的当前方向和比例。

输出

out_points

具有已平移点的几何体。

point_position

在几何体上设置的 point_position 值。
