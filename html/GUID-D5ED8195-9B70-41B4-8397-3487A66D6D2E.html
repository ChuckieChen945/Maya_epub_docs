size

返回数组中元素的数量或字符串中字符的数量。

int size(array array)

int size(string string)

array 是大小符合您需要的数组的名称。

string 是字符数符合您需要的字符串。

示例 1

string $s = "Hello";
$stringlen = size($s);

size ($s) 函数将返回 5，然后语句将 5 指定给 $stringlen。

示例 2

int $myInts[] = {1,2,3,4,5,6};
$numInts = size($myInts);

size ($myInts) 函数将返回 6，然后语句将 6 指定给 $numInts。
