将脚本文件加载到脚本编辑器

在脚本编辑器(Script Editor)中，选择“文件 > 打开脚本”(File > Open Script)，可从文件加载 MEL 或 Python 脚本。

将为此文件创建一个新选项卡。Maya 会识别文件中的语言，并将 MEL 或 Python 图标添加到此选项卡中，并将其重命名为文件的名称。

将鼠标悬停在该选项卡上方时，将显示工具提示，用于指示文件的路径。

在此选项卡中修改文件时，* 将附加到选项卡名称以指示文件已更新。

选择“文件 > 保存脚本”(File > Save Script)，可直接将更改保存到文件。如果关闭“脚本编辑器”(Script Editor)或 Maya 而不保存文件，将显示一个对话框，提示您保存所做的更改。

如果在 Maya 外修改脚本文件（例如，在文本编辑器中），Maya 中将显示一个对话框，提示您重新加载该文件。

如果在 Maya 外删除脚本文件，将显示一个对话框，询问您是否要将该文件保存在“脚本编辑器”(Script Editor)中。

关闭或重新打开“脚本编辑器”(Script Editor)或 Maya 时，将还原已加载的所有文件。
