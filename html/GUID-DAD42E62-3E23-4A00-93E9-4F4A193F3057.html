建模工具包热键

注：
如果您使用的是 macOS，除非另有指定，否则请使用以下特殊键，而不是其 Windows 等效键：
  • Alt = Option
  • Ctrl = Control
  • Windows 键 = Command
此外，如果您在 macOS 上使用单按钮或双按钮鼠标，请确保根据需要在界面首选项(Interface Preferences)中设置“鼠标跟踪”(Mouse Tracking)选项。
提示： 在“建模工具包”(Modeling Toolkit)窗口中，展开“四边形绘制选项”(Quad Draw Options)、“连接选项”(Connect Options)、“多切割选项”(Multi-Cut Options)和“目标焊接选项”(Target Weld Options)中的“键盘/鼠标快捷方式”(Keyboard/Mouse Shortcuts)部分以查看每个工具的热键。
