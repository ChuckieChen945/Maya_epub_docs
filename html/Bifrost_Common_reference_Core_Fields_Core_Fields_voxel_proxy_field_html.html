voxel_proxy_field

根据体积几何体的特性，创建体素代理场。这类似于创建普通体素场，但区别在于，当使用 sample_field_with_proxies 对代理场进行采样时，可以稍后在图中提供几何体。

您可以使用 Core::Fields 和 Core::Math 名称空间中的多种节点将代理场与普通场组合在一起，生成的结果也是代理场。

输入

property

对场进行采样时应返回其值的几何体上的特性的名称。

interpolation

如何对值进行插值：kLinear 速度最快但不平滑，而 kCubicC1 最平滑，但计算时间最长。

field_type

指定场的类型。根据特性的类型，连接标量场或向量场。

输出

voxel_field

生成的代理场。
