批渲染选项(Batch Render Options)

使用这些选项可查看在选择“渲染 > 批渲染”(Render > Batch Render) > 时发生的情况。

对于 Windows

使用所有可用处理器(Use all available processors)

如果启用，渲染将使用本地计算机上的所有可用处理器。

如果禁用，渲染仅使用“要使用的处理器数”(Number of Processors to Use)指示的处理器数。默认情况下“使用所有可用处理器”(Use all available processors)处于禁用状态。

要使用的处理器数(Number of Processors to Use)

“使用所有可用处理器”(Use all available processors)处于禁用状态时用于渲染的处理器数。默认值为 1。

对于 Linux

渲染 CPU (Rendering CPU)

确定渲染是要放置在本地计算机（“本地”(Local)）还是网络中另一台计算机（“远程”(Remote)）上进行渲染。如果选择“远程”(Remote)，请将“远程计算机名称”(Remote Machine name)设置为要在其上进行渲染的计算机的名称。默认设置为“本地”(Local)。

远程计算机名称(Remote Machine Name)

设置为“远程”(Remote)时，请键入要在其上进行渲染的计算机的名称，然后按 Enter 键。如果不执行此操作，将在本地计算机上进行渲染。
