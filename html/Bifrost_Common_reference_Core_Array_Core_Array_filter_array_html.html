filter_array

使用布尔数组过滤数组，以指定要保留的元素。

输入

array

要过滤的数组。

boolean_array

布尔数组，指示要保留输入 array 的哪些元素。如果 boolean_array[i] 为 true，则 array[i] 的值将包含在输出 filtered_array 中。

通常，boolean_array 应具有与 array 相同数量的元素。如果其元素数量较少，则 array 的任何额外元素都不会包含在 filtered_array 中。如果其元素数量较多，则 filtered_array 将填充每个额外 true 元素对应类型的默认值。

输出

filtered_array

过滤后的数组。

filtered_indices

将 filtered_array 数组中的元素映射到输入 array 中元素的数组 - filtered_array[i] 的值与 array[filtered_indices[i]] 的值相同。
