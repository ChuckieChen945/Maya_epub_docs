组合场

您可以通过多种方式组合噪波和形状等场来创建复杂的效果。

基本节点

许多基本的数学节点和其他节点都与场输入兼容。生成的场在任意位置的值是合并该位置的输入场值的结果。

如果复合仅由与场一起使用的基本节点组成，则该复合也将与场一起使用。

以下节点可以与标量场和向量场一起使用：

  • absolute_value
  • add
  • clamp
  • divide
  • lerp
  • max
  • min
  • multiply
  • negate
  • power
  • subtract

此外，还可以将标量场与以下节点一起使用：

  • exponential
  • scalar_to_vector3

也可以将向量场与以下节点一起使用：

  • cross
  • dot
  • length
  • length_squared
  • vector3_to_scalar

条件场

switch_fields 节点的作用类似于场的 if 节点。它返回两个场之一，具体取决于输入 condition 标量场的值。

  • 如果 condition 为正值，则返回 true_case。
  • 如果为 0 或负值，则返回 false_case。

此外，mask_field 可用于创建倍增以遮罩复杂场中的效果。它返回一个场，该场的值在输入为正时为 1，在其他情况下均为 0。
