“重力”(Gravity)选项

使用这些选项可控制“重力”(Gravity)场的行为。

幅值(Magnitude)

设置重力场的强度。“幅值”(Magnitude)越大，对象在重力方向上的加速就越快。

衰减(Attenuation)

设定场的强度随着到受影响对象的距离增加而减小的量。更改速率与距离呈指数关系；“衰减”(Attenuation)为指数。如果将“衰减”(Attenuation)设置为 0，力与距离的关系将保持恒定。

方向 X (Direction X)、方向 Y (Direction Y)、方向 Z (Direction Z)

设定重力的方向。

距离(Distance)

使用最大距离(Use Max Distance)

如果启用“使用最大距离”(Use Max Distance)，“最大距离”(Max Distance)设置所定义的区域内的连接对象都受重力场的影响。“最大距离”(Max Distance)之外的任何连接对象则不受重力场的影响。

如果禁用“使用最大距离”(Use Max Distance)，所有连接的对象均受重力场的影响，无论它们距离重力场有多远。

最大距离(Max Distance)

设定与重力场的最大距离，在达到该距离时场将发挥效力。您还必须启用“使用最大距离”(Use Max Distance)，“最大距离”(Max Distance)才会生效。仅在属性编辑器中可用。

注： 默认情况下，重力场和大多数其他场将“最大距离”(Max Distance)值设定为 -1，并禁用“使用最大距离”(Use Max Distance)。“最大距离”(Max Distance)的 -1 值不起作用，因为“使用最大距离”(Use Max Distance)设置被禁用。如果启用“使用最大距离”(Use Max Distance)，-1 值的效果与 0 值的效果相同 - 任何距离的对象都不受场的影响。

包含 -1 值是为了与 Maya 的早期版本兼容。如果启用“使用最大距离”(Use Max Distance)并更改“最大距离”(Max Distance)的值，则只能输入一个正数。

衰减曲线(Falloff Curve)

在选中“使用最大距离”(Use Max Distance)时可用。

当使用场施加力时，有时受力对象在施加力后可能无法稳定下来。使用该曲线可调整力。例如，缩放场的力，以使其在“最大距离”(Max Distance)值定义的边界处逐渐衰减为零。

曲线在归一化的距离内定义；也就是 0 到 1，其中 0 对应于场的中心，1 对应于“最大距离”(Max Distance)值。该曲线中的值是要应用于力的比例因子。

注： 在创建场后，衰减曲线仅作为属性提供。它在场创建选项中不可用。

体积形状(Volume shape)

体积决定场影响粒子/刚体的区域。

体积(Volume)

选择“无”(None)、“立方体”(Cube)、“球体”(Sphere)、“圆柱体”(Cylinder)、“圆锥体”(Cone)或“圆环”(Torus)之一。

体积排除(Volume Exclusion)

启用“体积排除”(Volume Exclusion)时，体积定义空间中场对粒子或刚体没有任何影响的区域。

体积偏移 X(Volume Offset X)、体积偏移 Y(Volume Offset Y)、体积偏移 Z(Volume Offset Z)

从场的位置偏移体积。如果旋转场，也会旋转偏移方向，因为它在局部空间内操作。

注：

偏移体积仅更改体积的位置（因此，也会更改场影响的粒子）。它不会更改用于计算场力、衰减等的实际场位置。

体积扫描(Volume Sweep)

定义除立方体外的所有体积的旋转范围。该值可以是介于 0 和 360 度之间的值。

截面半径(Section Radius)

定义圆环体的实体部分的厚度（相对于圆环体的中心环的半径）。中心环的半径由场的比例确定。如果缩放场，则“截面半径”(Section Radius)将保持其相对于中心环的比例。

特殊效果(Special Effects)

仅在“属性编辑器”(Attribute Editor)中对对象场可用。

逐顶点应用(Apply Per Vertex)

设定场源自对象的位置。如果启用“逐顶点应用”(Apply Per Vertex)，则选定对象的每个点（CV、粒子和顶点）会以最大强度均匀地施加场的影响。如果禁用“逐顶点应用”(Apply Per Vertex)，则仅在指定点的平均位置施加场。

如果使用的是“属性编辑器”(Attribute Editor)，请打开“特殊效果”(Special Effects)区域以显示“逐顶点应用”(Apply Per Vertex)属性。
