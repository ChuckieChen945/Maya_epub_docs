特殊场运算

除了相加和相乘等基本运算外，还有一些特定于场的特殊运算。这些运算包括向量分析函数，例如渐变、散度、卷曲等。

渐变

gradient_field 节点计算标量场的渐变，它有时用“nabla”或“del”符号 ∇ 表示。渐变是一个向量场，它显示标量场在位置附近的变化情况。

如果将二维标量场想象为高度贴图，则其渐变的方向表示从任意位置开始最陡的上坡方向，而渐变的幅值为该方向的坡度。如果将三维标量想象为表示房间内的空气温度，则渐变指向温度最大增加的方向，其长度是增加的速率。

散度

divergence_field 节点计算向量场的散度，它有时用 ∇ 或 div 表示。散度测量的是向量场流入或流出某一位置周围区域的净流量。向量的起源具有正的散度，而向量收敛的汇点具有负的散度。

不可压缩流体流是无散度的，这意味着各处的散度均为 0。换句话说，区域中的密度不能增大或减小，并且没有汇点或流体源。

卷曲

curl_field 节点计算向量场的卷曲，它有时用 ∇× 或 curl 表示。卷曲测量的是向量流绕某一位置的净旋转。其方向是旋转轴，其长度是旋转量。不应将其与卷曲噪波相混淆，后者使用噪波场的卷曲来创建无散度的湍流。

FCurve 场

fcurve_field 节点类似于 evaluate_fcurve 节点。但是，您可以根据可编辑的函数曲线重映射标量场的值，而不是重映射简单的标量值。

最近点和近点场

closest_point_field 节点类似于 get_closest_point 几何体查询。它返回一个向量场，该向量场在任意位置的值都是由输入有符号距离场定义的形状的曲面（0 等值面）上的最近点。例如，可以使用它创建使粒子朝向曲面上最近点加速的力。但是，如果输入不是真正的有符号距离场，则可能无法获得所需的结果。

near_point_field 节点类似于 closest_point_field，但它适用于任何标量场。它一次或多次朝 0 等值面遵循标量场的渐变，并返回新位置。使用 tolerance 输入缩短计算时间（如果它达到高原或局部极值），使用 max_iterations 输入限制其执行的步数。
