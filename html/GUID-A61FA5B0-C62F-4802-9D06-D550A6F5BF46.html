水(Water)

模拟线性水波浪、同心水涟漪（如物体落入水中）或波浪和涟漪的组合。用作凹凸贴图或置换贴图来模拟水，或用作颜色贴图来模拟水面的光线反射或折射。

在“创建”(Create)选项卡中找到此纹理。若要作为纹理贴图应用该纹理，请参见映射 2D 或 3D 纹理。

波数(Number Of Waves)

纹理中的线性波数。有效范围为 0 到 100。滑块范围为 0 到 32。默认值为 8。

提示：
  • 对于没有线性波浪的水涟漪，请将“波数”(Number Of Waves)设定为 0。
  • 若要将线性波浪与同心涟漪组合在一起，请将“波数”(Number Of Waves)设定为较小的数值（1 到 3）。
波时间(Wave Time)

控制随着时间推移而变化的波浪外观。范围为 0 到 1。默认值为 0。

波浪效果类似于由船所产生的波浪，即波浪以特定的速度和振幅从某个点开始。这是零时间。随着时间增加，波浪达到岸边且外观发生变化（速度和振幅会减小）。若要模拟移动中的波浪，请为“波时间”(Wave Time)值设置动画。该值增加时，波浪会发生移动。Maya 通过波速度值和为“波时间”(Wave Time)值设置动画时的速率来确定波浪速度。

波速度(Wave Velocity)

波浪的速率。有效范围是 0 到无限。取值范围是 0 到 1。默认值为 1。

波振幅(Wave Amplitude)

缩放波高度。有效范围是 0 到无限。范围为 0 到 1。默认值为 0.05。

快速(Fast)

“水”(Water)纹理的一种优化形式。启用该选项时，Maya 针对每帧计算一次颜色表。禁用该选项时，Maya 将对每个采样计算颜色表。

提示：

只有在最终场景中查看水的特写时，才可能禁用该选项。

波频率(Wave Frequency)

控制主波之间的距离。该值越高，距离越短。有效范围是 0 到无限。滑块范围为 0 到 20。默认值为 4。

子波频率(Sub Wave Frequency)

控制位于主波顶部的次波之间的距离（例如，白浪）。有效范围是 0 到无限。滑块范围为 0 到 1。默认值为 0.125。

平滑度(Smoothness)

控制次波的强度。有效范围是 0 到无限。滑块范围为 0 到 5。默认值为 2。

风 UV (Wind UV)

风在 U 方向和 V 方向的强度（确定了线性波浪图案的整体方向）。有效范围从 -1 到 1。U 方向默认值为 1。V 方向默认值为 0。

同心涟漪属性(Concentric Ripple Attributes)

控制同心水涟漪的外观。“同心涟漪属性”(Concentric Ripple attributes)不影响纹理的线性波浪组件。

涟漪时间(Ripple Time)

控制随着时间推移而变化的涟漪外观。范围为 0 到 1。默认值为 0。

例如，想像一滴水落到一杯静止的水中的情形，时间为 0 时没有任何涟漪，因为该水滴还没有接触到水面。随着时间属性增加到一个大于 0 的值，该水滴落到水面上，且涟漪波浪效果会模拟杯中水在那一时刻的效果。当时间值达到 1 时，水滴在水面上的效果几乎已经完成，杯中的水再次变为静止。

若要模拟移动中的水涟漪，请为“涟漪时间”(Ripple Time)值设置动画。该值增加时，涟漪会发生移动。涟漪的实际速度由组速度值和为“涟漪时间”(Ripple Time)值设置动画时的速率确定。

涟漪频率(Ripple Frequency)

控制各涟漪之间的距离。该值越高，距离越短。有效范围是 0 到无限。滑块范围为 0 到 20。默认值为 25。

涟漪振幅(Ripple Amplitude)

缩放涟漪的高度。有效范围是 0 到无限。滑块范围为 0 到 1。默认值为 0.05。

水滴大小(Drop Size)

开启水涟漪的水滴质量。“水滴大小”(Drop Size)值为 0 不会产生任何涟漪效果，因为该水滴没有质量。随着“水滴大小”(Drop Size)值增加，水滴的质量也会增加，可以产生更理想的涟漪效果。有效范围是 0 到无限。滑块范围为 0 到 1。默认值为 0.3。

涟漪原点(Ripple Origin)

涟漪中心沿 U 方向和 V 方向的位置。范围从 0 到 1。默认值为 0.5。

组速度(Group Velocity)

主涟漪的速度。有效范围是 0 到无限。滑块范围为 0 到 10。默认值为 1。

相位速度(Phase Velocity)

子涟漪的速度。有效范围是 0 到无限。滑块范围为 0 到 10。默认值为 2.5。

扩散开始(Spread Start)

水滴在时间为 0 时对静止水面的效果程度。当“涟漪时间”(Ripple Time)值为 0 时，同心涟漪作为空间（通过“涟漪原点”(Ripple Origin)值定义）中的一个扰点开始。时间为 0 时的扰动大小由“扩散开始”(Spread Start)值确定。该值越大，时间为 0 时的扰动效果越明显。有效范围是 0 到无限。滑块范围为 0 到 1。默认值为 0.005。

扩散速率(Spread Rate)

主涟漪分解成子涟漪的速率。（在涟漪向外展开的过程中，它会分解成多个子涟漪。）有效范围是 0 到无限。滑块范围是 0 到 1。默认值为 0.3。

反射长方体(Reflection Box)

激活一个涟漪可以反射的虚拟反射边界框。默认情况下，“反射长方体”(Reflection Box)处于禁用状态。（该效果可能会降低渲染速度。）

长方体最小值(Box Min)、长方体最大值(Box Max)

在 U 方向和 V 方向定义边界框。如果“反射长方体”(Reflection Box)处于禁用状态，则这些属性均不起作用。范围为 0 到 1。“长方体最小值”(Box Min)的默认值为 0；“长方体最大值”(Box Max)的默认值为 1。
