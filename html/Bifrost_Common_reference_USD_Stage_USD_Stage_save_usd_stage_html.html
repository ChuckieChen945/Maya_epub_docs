save_usd_stage

将 Bifrost USD 后台文件中所有已修改图层保存到磁盘文件中。支持 usd（通用）、usda（ASCII 文本）和 usdc（创建二进制）格式。

输入

stage

连接要保存的后台文件。

enable

保存所有已修改图层。禁用此选项时，输入后台文件仅输出而不保存。如果希望保存 USD 文件，可以再次切换 enable 启用和禁用；如果希望继续保存所有修改，则将其保持在启用状态。

file

要保存的文件的路径。要使用输入后台文件的现有 save_file 路径，将此选项保留为空。

选项

relative_path

使用子层相对于根层文件的路径存储子层。

default_prim

后台文件的根基本体第一个子项将用作默认基本体。

enable_meters_per_unit

创建每单位后台文件元数据的米数。

meters_per_unit

每单位值的米数。

enable_up_axis

创建上方向轴后台文件元数据。

up_axis

上方向轴值。

输出

out_stage

将此输出连接到 output 节点上游。
