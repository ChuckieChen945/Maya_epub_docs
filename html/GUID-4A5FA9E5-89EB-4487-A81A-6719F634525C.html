为 nCloth 创建外力

对 nCloth 使用外力

nCloth 会对 Maya 动力学力做出反应。可以将这些外力添加到场景中以影响 nCloth 对象的行为。

请参见 Nucleus 对象和外部动态力。

将 2D 或 3D 流体作为外力作用于 nCloth

  1. 选择要通过“流体效果”(Fluid Effects)施加影响的 nCloth 对象。
  2. 在“属性编辑器”(Attribute Editor)中，选择 Nucleus 选项卡。
  3. 在“重力”(Gravity)和“风”(Wind)部分中，禁用选定 nCloth 的“重力”(Gravity)，方法是将其值设置为 0.0。
  4. 选择 nCloth，然后在按住 键的同时选择 2D 或 3D 流体。
  5. 选择“字段/解算器 > 指定给选定对象”(Fields/Solvers > Assign to Selected)。

    现在，所选的 2D 或 3D 流体将影响 nCloth。

注：

可以使用“动力学关系”(Dynamic Relationships)编辑器查看和编辑 nCloth 与流体对象之间的连接。将 2D 或 3D 流体作为外力作用于 nCloth 时，流体被认为是场，而不是碰撞对象。因此，要在“动力学关系”(Dynamic Relationship)编辑器中查看 nCloth 与流体对象之间的连接，请选择场作为选择模式。请参见动力学关系编辑器。
