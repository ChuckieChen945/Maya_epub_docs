导入或导出动画片段

提示： 您可以通过将片段从文件浏览器拖动到 Maya 场景或拖动到“Trax 编辑器”(Trax Editor)中的角色，将单个或多个片段导入到场景中。请参见剪切、复制和粘贴 Trax 片段。
注意： 在导入动画数据之前，始终应检查会影响动画数据导入的当前“播放”(Playback)首选项。请参见“时间滑块”(Time Slider)首选项。

将片段或姿势导入场景

  1. 在 Trax 菜单栏中，选择“Trax 编辑器”(Trax Editor)的“文件”(File)菜单中的“文件”(File) > 导入动画片段(Import Animation Clip)。

    将出现“导入文件”(Import file)浏览器窗口。

  2. 在文件浏览器窗口中，选择包含要导入的片段或姿势的文件。
    注： 只能导入具有有效文件扩展名（.ma、.mb 或 .mov）的片段或姿势文件。
  3. 单击“导入”(Import)。

    如果导入的是片段，则会显示在“未使用的片段”(Unused Clips)下的“Visor”中。如果导入的是姿势，则会显示在“未使用的姿势”(Unused Poses)下的“Visor”中。

  4. 执行下列操作之一：
    • 在“Visor”中的片段或姿势上单击鼠标中键并将其拖到轨迹视图区域中的角色。
    • 在“Visor”中选择片段或姿势，然后将其复制并粘贴到“Trax 编辑器”(Trax Editor)中的角色集。请参见剪切、复制和粘贴片段。

将片段或姿势导入角色

  1. 在场景视图中选择角色的任何部位（例如，从角色的骨架中选择一个骨骼）。

    您也可以从“当前角色集”(Current Character Set)下拉列表中选择一个角色，或亮显要在轨迹视图区域中显示的角色。请参见在 Trax 中亮显角色、组或子角色。

  2. 在 Trax 菜单栏中，选择“文件 将动画片段导入角色”(File > Import Animation Clip to Characters)。

    将出现“导入文件”(Import file)浏览器窗口。

  3. 在文件浏览器窗口中，选择包含所需片段或姿势的文件，然后单击“导入”(Import)。

    默认情况下，该片段会放置在“Visor”中，或直接放置在“Trax 编辑器”(Trax Editor)中的选定角色下。

将片段或姿势导入角色而不将其放置在 Trax 中

  1. 在场景视图中选择角色的任何部位（例如，从角色的骨架中选择一个骨骼）。

    您也可以从“当前角色集”(Current Character Set)下拉列表中选择一个角色，或亮显要在轨迹视图区域中显示的角色。

  2. 在 Trax 菜单栏中，选择“文件 >将动画片段导入角色”(File > Import Animation Clip to Characters) > 。
  3. 在显示的“将片段导入角色”(Import Clip To Character)选项中，启用“仅将片段放在 Visor 中”(Put Clip in Visor Only)，然后单击“导入片段”(Import Clip)。
  4. 在文件浏览器窗口中，选择包含要导入的片段或姿势的文件，然后单击“导入”(Import)。

    如果导入的是片段，则会显示在“角色片段”(Character Clips)下的“Visor”中。如果导入的是姿势，则会显示在“角色姿势”(Character Poses)下的“Visor”中。

  5. 您现在可以使用鼠标中键在“Visor”或“大纲视图”(Outliner)中拖动导入的片段或姿势，然后将其拖到 Trax 中的角色。请参见将 Visor 与 Trax 结合使用或将大纲视图与 Trax 结合使用。

从当前场景导出片段或姿势

  1. 选择要导出的片段或姿势。
  2. 在 Trax 菜单栏中，选择“文件 > 导出动画片段”(File > Export Animation Clip)。

    将出现“导出”(Export)文件”浏览器窗口。

  3. 在“导出文件”(Export file)浏览器窗口中，输入片段或姿势的文件名，选择文件类型（mayaAscii、mayaBinary、move），然后单击“导出”(Export)。
    注： 导出片段或姿势时，必须为其指定有效的文件扩展名（.ma、.mb 或 .mov）。否则，无法将片段或姿势导入 Maya 场景。

    片段或姿势将以您输入的名称和指定的文件类型导出并保存。

从“Visor”导出片段或姿势

  1. 在“Visor”中，通过单击相应的选项卡（“角色片段”(Character Clips)、“角色姿势”(Character Poses)、“未使用的片段”(Unused Clips)或“未使用的姿势”(Unused Poses)）打开要导出的片段的存储位置。
  2. 在要导出的片段或姿势上单击鼠标右键，然后从出现的菜单中选择“导出”(Export)。
  3. 在“导出文件”(Export file)浏览器窗口中，输入片段或姿势的文件名，选择文件类型（mayaAscii、mayaBinary、move），然后再次单击“导出”(Export)。

    片段将以您输入的名称和指定的文件类型导出并保存。
