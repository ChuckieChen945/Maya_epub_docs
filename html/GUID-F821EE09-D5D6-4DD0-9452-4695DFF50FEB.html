属性总表(Attribute Spread Sheet)

  • 若要打开“属性总表”(Attribute Spread Sheet)，请选择“窗口 > 常规编辑器 > 属性总表”(Window > General Editors > Attribute Spread Sheet)

属性总表(Attribute Spread Sheet)

“属性总表”(Attribute Spread Sheet)以电子表格格式列出多个对象上的多个属性，使您可以采用表格格式同时查看和编辑多个属性值，以便同时管理多个对象的属性。这对于调整关键帧值、指定材质或修改对象特性非常有用。

菜单

名称(Names)

名称 > 易读属性名称、短属性名称、长属性名称(Names > Nice Attribute Names, Short Attribute Names, Long Attribute Names)

选择是否在列的顶部显示短属性名称或长属性名称。

布局(Layouts)

布局 > 仅显示选定列/显示所有列(Layouts > Show Selected Columns Only/Show All Columns)

隐藏当前未选择的属性列。选择“显示所有列”(Show All Columns)以返回完整显示。

布局 > 记住此布局(Layouts > Remember This Layout)

创建仅具有当前可见的属性列的新选项卡。在新选项卡中选择需要的列，选择“仅显示选定列”(Show Selected Columns Only)，然后选择“记住此布局”(Remember This Layout)。

布局 > 删除当前布局(Layouts > Delete Current Layout)

删除使用“记住此布局”(Remember This Layout)创建的自定义选项卡。单击选项卡并选择“删除当前布局”(Delete Current Layout)以删除选项卡。

关键帧(Key)

关键帧 > 为选定项设置关键帧(Key > Key Selected)

在当前帧处为属性总表中当前选择的属性（单元格）设置关键帧。

层(Layer)

层 > 为选定项创建覆盖(Layer > Create Override for Selected)

为选定对象创建渲染层覆盖。

层 > 从选定项移除覆盖(Layer > Remove Override from Selected)

为选定对象移除渲染层覆盖。

属性过滤器(Attribute Filter)

使用“属性过滤器”(Attribute Filter)字段过滤出属性。过滤器支持子字符串并不区分大小写。例如，如果要隔离平移属性，请键入 trans，将会仅显示包含字符串 trans 的属性。

选项卡

不同的选项卡显示不同的属性类型。

使用选项卡的右端的箭头显示被窗口的边隐藏的更多选项卡。

使用“布局 > 仅显示选定列”(Layouts > Show Selected Columns Only)和“布局 > 记住此布局”(Layouts > Remember This Layout)来创建仅包含所需属性的新自定义选项卡。

属性总表(Spreadsheet)

  • 每个对象为一行，每个对象的属性为一列。
  • 若要选择单元格、列或行，可通过单击、按住 Shift 键并单击或者拖动方式来实现。
  • 在所有选定的单元格中键入一个值即可输入值。
