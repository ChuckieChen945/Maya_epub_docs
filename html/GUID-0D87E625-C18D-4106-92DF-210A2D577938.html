脚本绘制

脚本绘制是通过选择脚本并使用“绘制脚本工具”(Paint Scripts Tool)在曲面上绘制笔划来进行的。

进行脚本绘制

  1. 确保要用以进行绘制的脚本位于以下脚本目录中：

    (Windows) 驱动器:\Documents and Settings\username\My Documents\Maya\<版本>\scripts

    (Linux) ~username/Maya/<版本>/scripts

    (Mac OS X) /Users/username/Library/Preferences/Autodesk/Maya/<版本>/scripts
    注： 执行以下操作之一，使用 Mac OS X Lion 打开 Preferences 目录：
    • 选择“Finder > 转至 > 转至文件夹”(Finder > Go > Go to Folder)，然后键入目录路径（/Users/<用户名>/Library/Preferences）。
    • 从终端窗口导航到 Preferences 目录，然后键入 open。

    (Mac OS X) /Applications/Autodesk/Maya<版本>/Maya.app/Contents/scripts/paintEffects

    （Windows 和 Linux）mayapath/scripts/paintEffects

    注： 这些示例脚本位于以下目录中：
    • (Windows) 驱动器:\Program Files\Autodesk\Maya<版本>\scripts\others
    • (Linux) mayapath/scripts/others
    • (Mac OS X) /Applications/Autodesk/Maya<版本>/Maya.app/Contents/scripts/others

    不需要将这些示例移动到个人脚本目录中，但是，应该将任何新创建的脚本存储在个人目录中。

  2. 选择要在其上进行脚本绘制的曲面。
    注： 如果要使用 geometryPaint 或 spherePaint 样例脚本进行绘制，则必须先冻结曲面的变换，然后才能继续。有关冻结变换的信息，请参见重置和冻结变换。
  3. 选择“绘制脚本工具”(Paint Scripts Tool)，然后根据需要设置“绘制脚本工具选项”(Paint Scripts Tool Options)（“修改 > 绘制脚本工具”(Modify > Paint Scripts Tool) > ）。有关定义工具设置的详细信息，请参见绘制脚本工具设置。
  4. 转到“设置”(Setup)区域。
  5. 在“工具设置命令”(Tool Setup Cmd)框中，键入 MEL 脚本的名称（不带 .mel 扩展名），然后按 Enter 键（Windows 和 Linux）或 Return 键 (Mac OS X)。该框是区分大小写的。

    例如，对于样例脚本类型 geometryPaint、spherePaint 或 emitterPaint。

    可能会打开脚本窗口，具体取决于脚本。条目也可能会显示在“设置”(Setup)区域中的框中。这些条目是 MEL 程序，它们来自脚本。不需要对它们进行修改。有关这些程序的详细信息，请参见为绘制脚本工具写入 MEL 脚本。

  6. 更改脚本窗口中的任何设置，然后在曲面上拖动笔刷。有关样例脚本窗口中的设置的详细信息，请参见 geometryPaint 示例脚本设置和 emitterPaint 示例脚本设置。
