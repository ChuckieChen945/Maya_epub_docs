使用系统命令设定环境变量

操作系统中设置的变量优先于 Maya.env 文件中的任何设置。但是，当 Maya 验证环境设置时，可能添加默认设置。

  • 在 Linux 和 macOS 上，可以在命令行上使用 export 设置环境变量。可以将命令添加到您的 .bashrc 文件以确保环境变量始终可用。 

export MAYA_SCRIPT_PATH=/remote/scriptDisk/scriptFiles/
  • 在 Windows 上，您可以使用 set 设置环境变量。 
set MAYA_SCRIPT_PATH=/remote/scriptDisk/scriptFiles/
  • 或者，在 Windows 上，可以在“控制面板”中选择“系统”，然后单击“高级系统设置”，再单击“环境变量”。
  • 在 macOS 上，可以按上述说明在 Shell 中设置环境变量。但是，这些变量只有在您从该 Shell 运行 Maya 时才可用。如果从 Dock 运行 Maya，它不会从您的 Shell 使用任何环境变量。另一种方法是在 Maya.env 文件中设置环境变量。
