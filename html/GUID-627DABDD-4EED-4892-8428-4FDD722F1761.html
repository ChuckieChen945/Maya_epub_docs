在文件路径编辑器注册元素

“文件路径编辑器”(File Path Editor)仅识别注册的节点和属性。默认注册参考、文件、图像平面和音频节点。包含具有 usedAsFilename 标志的属性的任何其他节点均可以注册。usedAsFilename 标志可以通过 addAttr 命令进行添加。

此要求使您可以控制显示在“文件路径编辑器”(File Path Editor)中的内容，并允许支持来自插件和脚本的自定义节点类型。

在文件路径编辑器注册元素

  1. 在 Maya 中，将 MEL 命令 filePathEditor -registerType "<nodetype>.<attribute>" -typeLabel "<labelname>"; 输入到“命令行”(Command Line)或“脚本编辑器”(Script Editor)中。
    例如，若要使用标签名 Sound 注册一个音频节点，可输入命令 filePathEditor -registerType "audio.filename" -typeLabel "Sound";
    注： 若要注册场景集合(Scene Assembly)路径，请执行以下操作之一：
    • 若要包括来自集合定义节点的表示文件，可使用 filePathEditor -registerType "assemblyDefinition";
    • 若要包括来自集合引用节点的定义文件，可使用 filePathEditor -registerType "assemblyReference";

    该元素将自动保存到首选项文件夹的 filePathEditorRegistryPrefs.mel 文件中。请参见首选项。

    注册后，此标签将用于“文件路径编辑器”(File Path Editor)的以下部分：
    • 选项 > 按文件类型列出(Options > List by File Type)
    • 选择 > 按类型选择(Select > Select by Type)
    • “显示”(Show)菜单
    • 自动解析(Auto Resolve)窗口中的“解析”(Resolve)列表

在文件路径编辑器注销元素

  1. 在 Maya 中，将 MEL 命令 filePathEditor -deregisterType "<nodetype>.<attribute>"; 输入到“命令行”(Command Line)或“脚本编辑器”(Script Editor)中。

    例如，若要注销一个音频节点，可输入命令 filePathEditor -deregisterType "audio.filename";

    该元素将自动从首选项文件夹的 filePathEditorRegistryPref.mel 文件以及“文件路径编辑器”(File Path Editor)的所有部分中移除。
