Maya 所需的其他 Linux 库

默认情况下，并非所有 Linux 发行版中都安装了运行 Maya 所需的所有系统库和字体。可以使用 dnf 软件包管理器的 dnf install 命令来安装系统中缺少的任何库和字体。

这些库的名称可能会因 Linux 发行版的不同而异。

重要：
  • Maya 安装程序需要其他可能未在下面列出的库。请查看并安装 Autodesk 许可安装所需的其他 Linux 软件包。
  • Maya 2025 许可不支持 Wayland 显示。有关如何解决该问题的详细信息和说明，请参见此支持主题：不支持 Wayland 显示服务器。

所需的存储库和库

这些库的软件包名称可能会因 Linux 发行版的不同而异。在本文中，我们使用了具有“带有 GUI 的服务器”配置的 Base Rocky Linux 8 和 Rocky Linux 9。

安装某些所需的软件包时，需要 EPEL 存储库。

使用 dnf 安装并启用 EPEL 存储库：

    sudo dnf install epel-release

若要安装任何软件包，请使用 sudo dnf install <软件包名称>
