适用于 Maya 的 Unreal Live Link 插件疑难解答

下面列出了使用 Live Link 时可能遇到的常见问题及其解决方案。

Live Link 窗口已消失 (Linux)

如果找不到“实时链接编辑器”(Live Link Editor)窗口，则可能它已隐藏在 Maya 窗口的后面。如果在“实时链接编辑器”(Live Link Editor)窗口已打开时重新启动 Maya，则可能会发生这种情况。如果发生这种情况，请将 Maya 主窗口移到一边，然后单击“实时链接编辑器”(Live Link Editor)窗口，使其回到前面。

Unreal 中的摄影机焦点错误

如果在 Maya 中禁用了“景深”(Depth of Field)，则必须在 Unreal 中手动将“聚焦”(Focus)方法设置为“禁用”(Disable)。

在 Unreal 中角色看起来错误/掉落

如果角色在 Unreal 中行为异常，请检查以确保其骨架层次在 Maya 和 Unreal 之间相同。实时链接要求层次和骨骼/关节名称完全匹配。此外，Unreal 仅支持 FbxSkeleton 和 FbxMesh 节点，因此，如果原始装备使用其他节点，您可能需要进行调整以确保兼容性。

缺少“发送到 Unreal”(Send to Unreal)选项

如果您未看到“文件 > 发送到 Unreal”(File > Send To Unreal)选项，请确保已加载 GamePipeline.mll 插件。

Unreal Live Link 无法加载

如果 Unreal Live Link 无法正确加载，请确保已将 Unreal Engine 安装到同一台计算机上。此外，请确保您运行的是 Maya 2020 或更高版本以及 Unreal 5.0 或更高版本。

缺少自定义属性

如果将新的自定义属性添加到 Maya 中的链接对象，则必须将其重新导出到 Unreal 并重新导入，才能显示该属性。

“Unreal 链接资产选择”(Unreal Link Asset Selection)窗口无法刷新/超时

根据计算机的速度和文件的复杂性，您可能会发现 Unreal Live Link 的“资产选择”(Asset Selection)窗口（即，在 Unreal Live Link 窗口中单击“链接”(Link)下的插件图标后显示的窗口）可能无法及时更新。如果发生这种情况，请尝试单击“刷新”(Refresh)按钮。如果超时或再次失败，请关闭 Unreal Live Link 的“资产选择”(Asset Selection)窗口，然后重新打开它。
