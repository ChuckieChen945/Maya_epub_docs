“网格工具”(Mesh Tools)菜单

  • 位于“建模”(Modeling)菜单集中。
显示/隐藏建模工具包
显示或隐藏“建模工具包”(Modeling Toolkit)窗口。请参见建模工具包。

工具

附加到多边形(Append to Polygon)
使用该工具，您可以将多边形添加到现有网格，将多边形边用作起始点。

选择“网格工具 > 附加到多边形”(Mesh Tools > Append to Polygon) > ，可设置附加到多边形工具选项(Append to Polygon Tool Options)。

连接(Connect)
可以通过其他边连接顶点和/或边。请参见使用连接工具在多边形组件之间添加边。

选择“网格工具 > 连接”(Mesh Tools > Connect) > ，可设置连接工具选项(Connect Tool Options)。

折痕(Crease)
允许您在多边形网格上使边和顶点起折痕。可以使用折痕工具修改多边形网格，并获取在硬和平滑之间过渡的形状，而不会过度增大基础网格的分辨率。

选择“网格工具 > 折痕”(Mesh Tools > Crease) > ，可设置折痕工具选项(Crease Tool Options)。

提示： 处于边或顶点选择模式时，在按住 Shift 键的同时在对象上单击鼠标右键可打开“折痕工具”(Crease Tool)。
创建多边形(Create Polygon)
可以通过在场景视图中放置顶点来创建单独的多边形。

选择“网格工具 > 创建多边形”(Mesh Tools > Create Polygon) > ，可设置创建多边形工具选项(Create Polygon Tool Options)。

插入循环边(Insert Edge Loop)
允许您在多边形网格的整个或部分环形边上插入一个或多个循环边。循环边是由其共享顶点按顺序连接的多边形边的路径。

选择“网格工具 > 插入循环边”(Mesh Tools > Insert Edge Loop) > ，可设置插入循环边工具选项(Insert Edge Loop Tool Options)。

生成洞(Make Hole)
允许您在多边形的一个面中创建一个洞，也可以在另一个面的图形中创建一个洞。

选择“网格工具 > 生成洞”(Mesh Tools > Make Hole) > ，可设置生成洞选项(Make Hole Options)。

多切割(Multi-Cut)
允许您对循环边进行切割、切片和插入。您可以沿着切割提取或删除边，通过边流和细分插入循环边和切割，并在“平滑网格预览”(Smooth Mesh Preview)模式下进行编辑。

选择“网格工具 > 多切割”(Mesh Tools > Multi-Cut) > ，可设置多切割工具选项(Multi-Cut Tool Options)。

偏移循环边(Offset Edge Loop)
允许您在所选的任何边的任一侧上插入两个循环边。

循环边是由其共享顶点按顺序连接的多边形边的路径。循环边形成的平行边线可穿越边选择的范围。在单一边的两侧或边线上向多边形网格添加本地化详细信息时，以这种方式复制边非常有用。

选择“网格工具 > 偏移循环边”(Mesh Tools > Offset Edge Loop) > ，可设置偏移循环边工具选项(Offset Edge Loop Tool Options)。

绘制减少权重(Paint Reduce Weights)
允许您在网格上绘制区域以指定要减少多边形的位置。

“绘制减少权重”(Paint Reduce Weights)与“网格 > 减少”(Mesh > Reduce)命令一起使用时，可以支持您在尝试保留原始图形的同时，减少网格中的多边形数量。请参见减少选项(Reduce Options)。

选择“网格工具 > 绘制减少权重”(Mesh Tools > Paint Reduce Weights) > ，可设置绘制减少权重工具选项(Paint Reduce Weights Tool Options)。

绘制传递属性(Paint Transfer Attributes)

允许您基于每个顶点融合源和目标的属性值，以控制任一网格对结果变形的影响。融合由您使用 Artisan 笔刷工具在网格上绘制的属性贴图控制。

选择“网格工具 > 绘制传递属性”(Mesh Tools > Paint Transfer Attributes) > ，可设置绘制传递属性工具选项(Paint Transfer Attributes Tool Options)。

四边形绘制(Quad Draw)
可用于以自然而有机的方式建模，使用简化的单工具工作流重新拓扑化网格。使用手动重新拓扑流程时，可以在保留参考曲面形状的同时，创建整洁的网格。

选择“网格工具 > 四边形绘制”(Mesh Tools > Quad Draw) > ，可设置“四边形绘制工具选项”(Quad Draw Tool Options)。

雕刻工具(Sculpting Tools)
允许您雕刻虚拟 3D 曲面，像在粘土或其他建模材质上雕刻真正的 3D 对象那样。不是使用粘土，而是使用多边形构建虚拟 3D 曲面。有关雕刻工具的列表，请参见雕刻工具架。

选择“网格工具 > 雕刻工具 > <工具名称>”(Mesh Tools > Sculpting Tools > <tool name>) > ，可设置对应的雕刻工具设置。

滑动边(Slide Edge)

允许您重新定位多边形网格上的边或整个循环边的选择。

可以在按住 Shift 键的同时逐个选择边，或者双击某个边以选择整个循环边，然后使用鼠标中键拖动以滑动选定边。与选定边关联的顶点沿它们的共享垂直边移动。或者，可以按住 Shift 键并使用鼠标中键拖动以沿每个顶点法线移动边/循环边。

选择“网格工具 > 滑动边”(Mesh Tools > Slide Edge) > ，可设置滑动边工具选项(Slide Edge Tool Options)。

目标焊接(Target Weld)
允许您合并顶点或边以在它们之间创建共享顶点或边。只能在组件属于同一网格时进行合并。

选择“网格工具 > 目标焊接”(Mesh Tools > Target Weld) > ，可设置目标焊接工具选项(Target Weld Tool Options)。
