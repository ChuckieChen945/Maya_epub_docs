memory_cache

将已设置动画的对象或对象数组缓存到内存中，以便在填充缓存后加快播放速度。这是为了在模拟、文件读取或任何其他内存密集型操作后在图表中使用，以便加快其播放速度。可以使用 Bifrost“编辑”(Edit)菜单中的“重置反馈状态”(Reset Feedback State)清除缓存。

注意：对象按帧存储在内存中，因此内存密集型模拟可能会占用大量内存并影响系统性能。内存中的大小通常大于磁盘上的大小，因为大多数文件格式都包含压缩。

输入

objects

进入要缓存的复合的对象。

Frame Range

active_frame_range

如果启用了 active_frame_range，则复合仅缓存指定帧范围内的帧。

start_frame

活动帧范围的开始。

last_frame

活动帧范围的结束。

invert_frame_range

如果启用了此选项，则复合将切换为仅缓存指定帧范围之外的帧。

Reset Mechanism

only_in_order

如果选中此选项，则可确保缓存的帧是连续的，从而确保它对模拟有效。

输出

out_objects

缓存的对象。

first_object

objects 数组中缓存的第一个对象
