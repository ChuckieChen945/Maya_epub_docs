组合选项(Combine Options)

使用这些选项可设置在选择“网格 > 结合”(Mesh > Combine)时发生的情况。

合并 UV 集(Merge UV Sets)
选择一个选项以设置 UV 集在合并时的行为方式。
不合并(No Merge)
每个网格上的每个 UV 集都成为输出中的新 UV 集。
按名称合并(Merge by Name)
合并具有相同名称的 UV 集。
按 UV 链接合并(Merge by UV Links)
合并 UV 集以使输入网格上的 UV 链接继续工作。
合并蒙皮(Combine Skinning)

使用以前的权重绑定蒙皮。

枢轴位置(Pivot Position)

确定组合对象的枢轴点所在的位置。

中心(Center)
枢轴将位于所有组合对象的中心。
最后一个对象(Last object)
枢轴将与最后一个选定对象的枢轴位于相同点上。
世界原点(World origin)
枢轴将位于场景的 0,0,0 处。
