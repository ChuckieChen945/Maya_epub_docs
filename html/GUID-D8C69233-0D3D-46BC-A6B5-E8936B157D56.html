移动、旋转和缩放 UV

可以在 UV 编辑器中手动移动、缩放或旋转 UV 和 UV 壳。例如，可以移动单个 UV 或整个 UV 壳以改善其布局，或缩放 UV 壳以确保其充分使用 0 到 +1 纹理空间。

目标 请执行以下操作：

自由移动、旋转或缩放 UV 位置。

  1. 选择要重新定位的 UV。
  2. 在“工具箱”(Toolbox)中选择“移动”(Move)、“旋转”(Rotate)和“缩放”(Scale)工具。
  3. 使用操纵器变换 UV。
注： 在“UV 编辑器”(UV Editor)中同时缩放多个多边形对象的 UV 可能产生异常结果。一次处理一个对象的 UV。
注： 点捕捉和栅格捕捉与“UV 编辑器”(UV Editor)兼容。

调整 UV（在不进行选择的情况下变换 UV)

  1. 在“工具箱”(Toolbox)中选择“移动”(Move)、“旋转”(Rotate)和“缩放”(Scale)工具。
  2. 使用鼠标中键在 UV 上拖动。

或者，您可以使用调整 UV 工具(Tweak UV Tool)拖动 UV。

注： 使用“旋转”(Rotate)或“缩放”(Scale)工具调整单个 UV 不会产生任何效果。

移动 UV 壳，使其不会与其他壳重叠。

  1. 从“UV 编辑器”(UV Editor)菜单中，选择“工具 > 移动 UV 壳”(Tools > Move UV Shell) > 。（或单击 UV 编辑器工具栏上的“移动 UV 壳工具”(Move UV Shell Tool)。
  2. 启用“禁止重叠”(Prevent overlap)。
  3. 通过在 UV 编辑器中单击壳或拖动其周围的边界框来选择要移动的 UV 壳。
  4. 拖动操纵器来移动 UV 壳。

  3. 使用“枢轴”(Pivot)栅格指定快速枢轴点，或者单击“编辑枢轴”(Edit Pivot)手动放置枢轴操纵器。再次单击“枢轴操纵器”(Edit Pivot)可返回到普通变换模式。

将 UV 移动到纹理空间中的 (0,0)、(1,1)、(1,0) 或 (0,1)。

  1. 选择要单位化 UV 的面。
  2. 选择“修改 > 单位化”(Modify > Unitize) > 。请参见单位化 UV 选项(Unitize UVs options)。
  3. 根据需要调整设置，然后单击“应用并关闭”(Apply and Close)或“应用”(Apply)。
