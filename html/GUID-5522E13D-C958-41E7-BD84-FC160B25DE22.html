“控制”(Controls)选项卡

打开控制(Controls)选项卡

  • 打开 HumanIK（“骨架 > HumanIK”(Skeleton > HumanIK)），然后选择顶部的“控制”(Controls)选项卡。
“控制”(Controls)选项卡包含下列主要区域：
  • “控件”(Controls)选项卡工具栏
  • 角色表示
  • HumanIK 控制

下表提供使用“控制”(Controls)选项卡的任务简要概述。

目标 操作

创建控制装备。

选择下列选项之一：
  • > 创建 > 控制装备(> Create > Control Rig)
  • 源(Source)菜单中的控制装备(Control Rig)

更改装备效应器的外观。

选择 “编辑控制装备 > 装备外观”(Edit Control Rig > Rig Look)，然后选择效应器样式。

更改为“全身”(Full Body)设定关键帧和操纵模式。

单击 。

更改为“身体部位”(Body Part)设定关键帧和操纵模式。

单击 。

显示角色的骨架。

单击 。

为当前角色设定动画源。

在源(Source)菜单中，按名称选择其他 HumanIK 定义的角色。

创建一个辅助或轴效应器。

在 IK 效应器单元上单击鼠标右键并选择“创建辅助效应器”(Create Aux Effector)或“创建枢轴效应器”(Create Pivot Effector)。

移除辅助或轴效应器。

选择场景中的效应器，然后按 。

要移除一个单元的所有附加效应器，单击鼠标右键，然后选择“清除附加效应器”(Clear Extra Effectors)。

固定正在平移的效应器。

选择对应的 IK 效应器单元，然后单击 。

固定正在旋转的效应器。

选择对应的 IK 效应器单元，然后单击 。

暂时释放所有固定。

将活动源(Source)烘焙到当前角色。

从“烘焙”(Bake)菜单（ > 烘焙(Bake)）中，选择一个选项。请参见 HumanIK 窗口中的烘焙选项。

在“属性编辑器”(Attribute Editor)中编辑重定目标参数。

选择 >“编辑角色定义 > 编辑 HIK 特性”(Edit Character Definition > Edit HIK Properties)。
