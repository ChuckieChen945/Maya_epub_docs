在 Bifrost-USD 中将基本体添加到后台文件

基本体是可以添加到后台文件中的层的内容。它可以包含几何体、材质定义、覆盖等，以及其他作为子项的基本体。使用链中的一个或多个 add_to_stage 节点构建场景层级。

  1. 将 add_to_stage 节点添加到图表。

  2. 将后台文件连接到 stage 输入。

    如果 stage 端口保持未连接状态，则使用新的空后台文件。在这种情况下，最好在 stage 端口上单击鼠标右键，然后选择“创建节点 > Usd::Stage::create_usd_stage”(Create Node > Usd::Stage::create_usd_stage)，以便可以指定 layer 文件，用于在图表中保存下游文件的根层。

  3. 对于 parent_path，指定层级中父对象的完整路径（包括前导斜杠“/”），新基本体将添加到该路径下。如果保留为空，则基本体将添加到指定层的根。如果输入的路径在后台文件中尚不存在，则会在层级中创建常规 def 节点 - 您可以设置这些下游基本体的特定类型。

  4. 在 prim_definitions 路径上单击鼠标右键，从“创建节点”(Create Node)菜单中选择一个选项。

    • Usd::Prim::define_usd_prim 是一个常规节点，可用于添加任何类型的基本体。默认设置定义了一个变换，它通常用作其他基本体的父对象，但您可以更改它们以定义任何基本体。
    • “创建节点”(Create Node)菜单上的其他选项基于 define_usd_prim，并专门用于特定用途。

  5. 在刚添加的节点中，设置参数。有关相关术语和概念的基本信息，请参见 USD 词汇表。特别是：

    • 设置 path 以指定基本体的名称及其在层级中的位置。安全的做法是始终包含前导斜杠，如果 add_to_stage 节点中未指定 parent_path，且基本体定义未连接到另一个 prim_definition 的 children 端口，则前导斜杠是必填的。同样，如果输入的路径在后台文件中尚不存在，则会在层级中创建常规 def 节点，您可以设置这些下游基本体的特定类型。
    • 可用的基本类型包括范围、变换、胶囊、圆锥体、立方体、圆柱体、球体、网格、NURBS 曲线和点实例化器。可用类型的完整列表取决于适用于 Maya 的 USD 版本和其他插件已注册的内容。通过在大纲视图中的 Maya USD 代理上单击鼠标右键并展开“添加新基本体 > 所有已注册”(Add New Prim > All Registered)菜单，可以查看完整列表。

  6. 根据需要，在基本体定义节点的串联输出端口上单击鼠标右键，以添加引用、属性等。特别是，可以在 children 端口上单击鼠标右键，创建任意深度的基本体定义层级。

  7. 在 add_to_stage 节点的 prim_definitions 端口上单击鼠标右键，然后重复上述步骤，根据需要添加更多基本体定义。

  8. 将 add_to_stage 节点的 stage 输出连接到将后台文件作为输入的任意节点。这最终应连接到图表顶层的 output 节点，以显示在主场景中添加的基本体。
