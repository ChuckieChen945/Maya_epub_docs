保存首选项

启动 Maya 时，它将在默认位置中查找其 prefs 文件夹，除非已将 MAYA_APP_DIR 环境变量设置为其他路径。请参见文件路径变量。

如果首选项不存在（例如，首次运行该应用程序时），则 Maya 将使用默认设置启动。

自定义 Maya 后，新的设置将存储在 prefs 文件夹中，这样每次启动该应用程序时系统都将使用这些设置而不是默认设置。大多数首选项将以包含 MEL 命令的文本文件形式保存。
注：

  • 若要更改文件的保存位置，请参见更改 Maya 的文件保存位置。

  • 如果删除（或移动)首选项文件（或整个 prefs 文件夹），则 Maya 将再次使用默认设置。

可以自定义首选项文件夹的默认位置。详细信息请参见自定义首选项文件夹的位置。

保存 Maya 首选项

  1. 选择“文件 > 保存首选项”(File > Save Preferences)。
    注：

    如果在保存首选项时磁盘空间不足，则首选项可能损坏或无法恢复。请确保硬盘驱动器上的可用空间足以保存首选项。如果磁盘空间不足，请释放一些空间，然后再退出应用程序。

首选项文件的默认位置

Maya 将首选项文件存储在以下路径：

Windows

  • ..\Documents\maya\<版本>\prefs

Mac OS X

  • /Users/<用户名>/Library/Preferences/Autodesk/maya/<版本>/prefs
    注： 若要打开“首选项”(Preferences)目录：
    • 选择“查找器 > 转到”(Finder > Go)，按 Alt 键，Library 文件夹将显示在菜单中。
    • 从终端窗口导航到“首选项”(Preferences)目录，然后键入 open。

Linux

  • ~<用户名>/maya/<版本>/prefs
