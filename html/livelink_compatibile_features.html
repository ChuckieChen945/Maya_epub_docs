与适用于 Maya 的 Unreal Live Link 兼容的 Maya 功能

下面列出的 Maya 功能与 Unreal Live Link 插件兼容。

注意：

Live Link 与 Maya 2020 及更高版本以及 Unreal Engine 5.0 及更高版本兼容。

(a) = 动画序列兼容。

(l) = 关卡序列兼容。
功能 兼容的内容
摄影机（Maya 名称 -> Unreal 名称）
  • 变换（仅平移和旋转）-> 位置和旋转（摄影机和电影摄影机）(l)
  • F 制光圈（必须启用 DoF）-> 当前光圈（仅电影摄影机；“聚焦”(Focus)方法必须设置为“手动”(Manual)）
  • 胶片纵横比 -> 纵横比（仅摄影机）
  • 焦距 -> 当前焦距（仅电影摄影机）
  • 聚焦距离（必须启用 DoF） -> 手动聚焦距离（仅电影摄影机；“聚焦”(Focus)方法必须设置为“手动”(Manual)）
  • 视角 -> 视野（仅摄影机；在 Maya 与 Unreal 之间值不是 1:1 匹配）
  • 摄影机光圈 (mm) -> 感应器宽度/高度（仅电影摄影机）
注意：如果在 Maya 中禁用了景深，则必须在 Unreal 中手动将“聚焦”(Focus)方法设置为“禁用”(Disable)*。
注意：在控制摄影机属性之前，需要在 Unreal 中安装 *LiveLinkCamera 插件。
灯光（点光源、平行光、聚光灯）
  • 变换 (l)
  • 颜色 (l)
  • 强度 (l)
  • 圆锥体角度（仅聚光灯）
  • 半影角度（仅聚光灯）
关节
  • 变换 (a)
  • 自定义属性 (a)
融合变形
  • 主融合变形值 (a)
    • 除非使用已烘焙的曲线，否则“封套”(Envelopes)和“分组”(Group)值必须设置为 1（请参见将动画数据从 Maya 流式传输到 Unreal）
    • Maya 融合变形目标属性名称必须与 Unreal 变形目标名称匹配
道具
  • 变换 (l)
时间码 有关详细信息，请参见与时间码同步
注意：

对于变换，旋转曲线将烘焙到所有道具、摄影机和灯光。但是，仅发送已设置关键帧的帧进行平移。

注意：

您可能会注意到 Maya 中的某些摄影机属性是只读属性，因此不可设置关键帧。若要将这些属性的数据发送到 Unreal，必须改为为“焦距”(Focal Length)设置关键帧（以获取“视角”(Angle of View)），或为“摄影机光圈(英寸)”(Camera Aperture (inch))设置关键帧（以获取“摄影机光圈(mm)”(Camera Aperture (mm))和“胶片纵横比”(Film Aspect Ratio)）。
