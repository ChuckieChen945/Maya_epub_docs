“纹理”(Texturing)菜单

绘制(Painting)

3D 绘制工具(3D Paint Tool)

选择“3D 绘制工具”(3D Paint Tool)作为当前绘制笔刷。

请参见 3D 绘制工具和 3D 绘制工具设置。

PSD

创建 PSD 网络(Create PSD Network)

创建一个可用作纹理贴图的 PSD 纹理网络。将出现“创建 PSD 网络选项”(Create PSD Network Options)。

请参见将 PSD 网络用作 Maya 中的纹理

编辑 PSD 网络(Edit PSD Network)
打开“编辑 PSD 网络选项”(Edit PSD Network Options)窗口，以便您可以编辑 PSD 网络的设置（如“UV 集”(UV Set)）和材质的默认属性。请参见创建 PSD 网络选项。
更新 PSD 网络

当修改连接到 Maya PSD 节点（在 Maya 中）的 PSD 文件（在 Adobe Photoshop 中）时，可以更新（刷新）Maya 中的图像以立即显示修改。

请参见更新 PSD 网络。

Projection

创建纹理引用对象(Create Texture Reference Object)

为选定曲面创建纹理引用对象以将 3D 纹理或投影的 2D 纹理锁定到该曲面。当曲面动画显示或变形时，纹理将与曲面保持在一起，这将产生非常自然的效果。

另请参见创建纹理引用对象。

删除纹理引用对象(Delete Texture Reference Object)

删除选定曲面的纹理引用对象（如果存在）。若要选择选定曲面的一个引用对象以删除该对象，请使用“纹理 > 选择纹理引用对象”(Texturing > Select Texture Reference Object)，或在通道盒中单击引用对象的标题。

另请参见选择和删除纹理引用对象。

选择纹理引用对象(Select Texture Reference Object)

为选定曲面（如果存在）选择纹理引用对象。也可以在“通道盒”(Channel Box)中单击其标题。

另请参见选择和删除纹理引用对象。

Nurbs

NURBS 纹理放置工具(NURBS Texture Placement Tool)

提供用于选定曲面的 2D 纹理放置操纵器。创建 2D 纹理时，“纹理放置工具”(Texture Placement Tool)图标将会显示在 Maya 工具栏中。

有关详细信息，请参见纹理放置与标签映射。

选择“纹理放置工具设置”(Texture Placement Tool Settings)窗口中的“纹理 > NURBS 纹理放置工具”(Texturing > NURBS Texture Placement Tool) > ，并设置以下属性：

“修改纹理”(Modify texture) - 控制纹理的哪个“2D 纹理放置”(2D Texture Placement)属性（“曲面放置”(Surface placement)或“标签映射”(Label Mapping)）可以进行交互调整。
