equal

如果第一个输入等于第二个输入（使用该类型的本地排序），则返回 true。

如果 first == second 为 true，则 first < second 和 first > second 均为 false。

建议在比较浮点值时使用 almost_equal、equivalent_float_epsilon 或 equivalent_float_ULP，因为 equal 仅测试是否完全相等，而浮点运算的精度通常不直观。

输入

first

要比较的第一个值。

second

要比较的第二个值。

输出

output

如果在 epsilon 中 first 等于 second，则返回 true。
