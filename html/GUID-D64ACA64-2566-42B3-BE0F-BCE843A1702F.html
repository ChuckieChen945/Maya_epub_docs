Maya Python 解释器 mayapy

Maya 附带了自己的 Python 解释器 mayapy。

mayapy 对于批处理和从其他应用程序访问 Maya 库非常有用。

mayapy 位于以下目录中：

平台 位置
Windows C:\Program Files\Autodesk\Maya<版本号>\bin
Linux /usr/autodesk/Maya<版本号>/bin/
macOS /Applications/Autodesk/maya<版本号>/Maya.app/Contents/bin

mayapy 必须从命令行运行。

mayapy 还可用于通过 pip 软件包管理系统安装 Python 软件包，包括 PyMel。
