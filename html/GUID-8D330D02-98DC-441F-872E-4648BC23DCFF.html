视图热键

可以使用这些关键帧更改在视图中显示的内容。

注：
如果您使用的是 macOS，除非另有指定，否则请使用以下特殊键，而不是其 Windows 等效键：
  • Alt = Option
  • Ctrl = Control
  • Windows 键 = Command
此外，如果您在 macOS 上使用单按钮或双按钮鼠标，请确保根据需要在界面首选项(Interface Preferences)中设置“鼠标跟踪”(Mouse Tracking)选项。

按 功能

Shift + 空格键

放大显示鼠标指针下的视图，例如，“视图面板”(View Panel)或“曲线图编辑器”(Graph Editor)。这将收拢停靠在主窗口中的所有其他窗格。如果鼠标指针悬停在界面的浮动窗口或其他部分（例如，“通道盒”(Channel Box)或“工具设置”(Tool Settings)）上方，则会影响您所单击的最后一个视图。再次按住 Shift + 空格键可还原之前查看的配置。

Ctrl + 空格键

全屏显示鼠标指针下的视图。其工作方式与按住 Shift + 空格键以放大视图相同，但是还会隐藏“状态行”(Status Line)、“工具架”(Shelf)和“时间滑块”(Time Slider)等 UI 元素。再次按住 Ctrl + 空格键可还原之前查看的配置。
