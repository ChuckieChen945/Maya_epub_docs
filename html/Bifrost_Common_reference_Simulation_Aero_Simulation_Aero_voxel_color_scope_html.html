voxel_color_scope

使用此项可以通过对 Aero 体积中的点进行采样来轻松显示 Aero 颜色特性。显示颜色时不应用着色，并且可从体积进行稀疏采样以快速完成更新。

输入

volume

要从中可视化颜色的体积。

color

这是要显示的颜色特性的名称。

point_size_multiplier

此项与 voxel_fog_density 相乘，所得结果再根据最小/最大点大小剪裁。如果点都是最大点大小或最小点大小，则调整此值以获得更均匀的分布。

min_point_size

点将不小于此值。

max_point_size

点将不大于此值。

scatter_density

此项确定 points_from_grid 处于禁用状态时要生成的点数。点位置在体积内随机分散，并且随机位置在每一帧都会变化。仅当 voxel_fog_density 非零时才会生成点。

points_from_grid

如果启用此项，则点不在随机位置，而是定位在体素上。此时不使用 scatter_density，但用户可以调整 detail_level 以控制点数。如果 detail_level 为 1.0，则具有密度的每个体素上都将有一个点，如果 detail_level 小于 1.0，则将随机忽略点。

detail_level

启用 points_from_grid 时，此项确定多少比例的体素将有点。值为 1.0 时，所有体素都将有一个点，值为 0.5 时，一半的体素将随机有点。

voxel_offset

启用 points_from_grid 时，此项确定相对于体素创建点的位置。例如，如果设置为“中心”，则点将定位在体素中心。
