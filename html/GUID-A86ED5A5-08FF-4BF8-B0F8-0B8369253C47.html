指定给不同长度的粒子数组属性

您可以将一个“粒子形状”(particle shape)节点的数组属性指定给另一个具有不同粒子数的节点。指定受您在“表达式编辑器”(Expression Editor)中的“对象选择”(Object Selection)列表中选择的节点的影响。选定“粒子形状”(particle shape)节点中的粒子数设定语句执行数，因此将影响指定。

示例

假定您的场景中包含一个由两个粒子组成的名为 TwoPts 的对象。TwoPts 中的这两个粒子位于以下位置：

5 0 0 
6 0 0

假定您使用“粒子工具”(Particle Tool)创建一个名为 FivePts 的五粒子对象并将这些粒子置于工作区中的某处。进一步假定您在“表达式编辑器”(Expression Editor)中选择 FivePts 的“粒子形状”(particle shape)节点，然后在运行时表达式中执行该指定：

FivePtsShape.position = TwoPtsShape2.position;

这五个粒子将在运行时表达式首次执行时移动到这些位置：

5 0 0
6 0 0
5 0 0
6 0 0
5 0 0
