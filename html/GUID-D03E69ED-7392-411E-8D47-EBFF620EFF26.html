在平滑网格预览模式下编辑网格

可以修改网格上的组件，同时在平滑网格预览(Smooth Mesh Preview)上预览其效果。

在平滑网格预览模式下编辑网格

  1. 确保已为要编辑的多边形网格启用“平滑网格预览”(Smooth Mesh Preview)。（选定网格对象时，通过按 1、2 或 3 键可以快速在不同的平滑网格预览模式之间循环切换。）请参见预览平滑网格。
    注： 使用“框架 + 平滑网格”(Cage + Smooth Mesh)模式时，您可以在“框架”(Cage)、“平滑网格”(Smooth Mesh)或“两者”(Both)上选择组件，方法是在 polyShape 节点“属性编辑器”(Attribute Editor)的“平滑网格预览 > 编辑”(Smooth Mesh Preview > Edit)部分中选择选项。
  2. 在网格上选择并编辑组件。
    提示： 当“框架 + 平滑网格”(Cage + Smooth)的“编辑”(Edit)选项设定为“二者”(Both)时，可能需要更改“平滑网格预览”(Smooth Mesh Preview)的活动和非活动显示颜色，以便更轻松地区分网格的框架和已平滑版本。
    • 选择“窗口 > 设置/首选项”(Windows > Settings/Preferences) > 颜色设置（Color Settings），然后在“活动”(Active)和“非活动”(Inactive)选项卡中打开“平滑网格预览”(Smooth Mesh Preview)部分，以设置活动和非活动颜色设置。
注： 缓存平滑网格可能会导致性能下降并占用大量内存。可以使用缓存播放首选项(Cached Playback Preferences)中的“缓存平滑网格”(Cache Smooth Meshes)选项来禁用“平滑网格预览”(Smooth Mesh Preview)。
