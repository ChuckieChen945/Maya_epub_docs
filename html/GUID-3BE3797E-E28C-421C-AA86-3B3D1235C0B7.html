使曲线动力学化选项(Make Curves Dynamic Options)

在创建头发系统或将头发曲线约束到曲面之前，可以使用以下选项为头发的曲线建模。

将曲线附加到选定曲面(Attach Curves To Selected Surfaces)

启用时，选定的曲线将附加到最近的选定曲面。

生成的动力学曲线的毛囊将附加到开始 CV 曲线所在曲面上的最近的点。如果要改为附加选定曲线的两端，请在使曲线动力学化之前，选择曲线，然后从“建模”(Modeling)菜单集中，选择“曲线 > 反转曲线方向”(Curves > Reverse Curve Direction)。

“将曲线附加到选定曲面”(Attach Curves To Selected Surfaces)默认处于启用状态。

将曲线基础捕捉到曲面(Snap Curve Base to Surface)

启用该选项后，选定曲线的开始 CV 将在曲线附加到曲面时，捕捉到选定的曲面。这将在绘制和放置曲线时，为您提供了很高的自由度。

如果有头发曲线已经附加到选定曲面，选定曲线的开始 CV 将捕捉到曲面的当前毛囊位置。

“将曲线基础捕捉到曲面”(Snap Curve Base to Surface)仅当“将曲线附加到选定曲面”(Attach Curves To Selected Surfaces)处于启用状态时可用。“将曲线基础捕捉到曲面”(Snap Curve Base to Surface)默认处于禁用状态。

精确形状匹配(Exact Shape Match)

启用此选项时，选定（输入）曲线和动力学（输出）曲线的形状和位置将完全匹配。输出曲线继承输入曲线的“次数”(Degrees)，并且它们还继承输入曲线 CV 的数量和位置。输入曲线重建为 1 次曲线。“精确形状匹配”(Exact Shape Match)默认处于启用状态。

注：

您仅可以生成精确动力学曲线与 1 次输出曲线的碰撞。同样，如果希望精确动力学曲线的碰撞，我们建议您不要启用“精确形状匹配”(Exact Shape Match)。
