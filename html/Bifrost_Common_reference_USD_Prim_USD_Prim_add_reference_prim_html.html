add_reference_prim

在给定后台文件中添加基本体参照。

输入

stage

要在其中添加参照基本体的 USD 后台文件。

prim_path

将参照某物的基本体。

reference_layer

被参照的图层（如果为内部，则为空）。

reference_prim_path

将被参照的基本体。

layer_offset

层时间偏移。

layer_scale

图层偏移比例因子。

reference_position

参照列表中的位置。

anchor_path

定位点路径是您不希望包含在参照列表中的层标识符的前部。如果提供了空或无效的定位点路径，则不会从层标识符中移除任何内容。

输出

out_stage

修改后的 USD 后台文件。

success

指示操作是否成功的布尔值。
