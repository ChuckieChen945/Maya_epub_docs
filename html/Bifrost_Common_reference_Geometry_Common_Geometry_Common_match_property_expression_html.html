match_property_expression

检查特性名称是否与 property_expression 输入匹配。

这有助于使用通配符过滤特性名称。

一些示例表达式：要匹配所有特性，请使用：* 要匹配除 point_normal 以外的所有特性，请使用：* !point_normal 要匹配以 point_ 开头的所有特性，请使用：point_* 要匹配不是以 point_ 开头的所有特性，请使用：* !point_*

输入

property_name

特性的名称。

property_expression

用于测试 property_name 的表达式。

输出

matches

如果 property_expression 与 property_name 匹配，则设置为 true。
