选择笔划

可使用多种方法在建模和场景绘制视图中选择笔划。（不能在“Paint Effects”画布上选择笔划。）

在场景视图中选择笔划

  1. 单击笔划以选择它。拖动以框选多个笔划。

在场景视图中仅选择笔划

  1. 单击“状态行”(Status Line) 上的“按对象类型选择”(Select by object type)图标。
  2. 单击“状态行”(Status Line)上的“设置对象选择遮罩”(Set the object selection mask)向下箭头图标 ，然后从弹出菜单中选择“禁用所有对象”(All Objects Off)。
  3. 在“曲线”(Curves)图标上单击鼠标右键，然后启用“Paint Effects 笔划”(Paint Effects Strokes)。
  4. 选择所需的笔划。仅可以选择“Paint Effects 笔划”(Paint Effects strokes)。
  5. 若要返回到默认选择遮罩，请单击“状态行”(Status Line)上的“设置对象选择遮罩”(Set the object selection mask)向下箭头图标 ，然后选择“初始默认值”(Initial Default)。

    有关设置和清除选择遮罩的详细信息，请参见使用选择遮罩。

在场景绘制视图中选择笔划

  1. 按住 Ctrl 键并在笔划上单击以选择它。按住 Ctrl+Shift 并单击以选择附加的笔划。

按名称选择笔划

  1. 选择笔划。
  2. 选择“生成 > 选择包含指定字符的笔刷/笔划名称”(Generate > Select Brush/Stroke Names Containing)。
  3. 在“名称片段”(Name Fragment)文本框中，键入笔划的名称，例如 daisy。
  4. 单击“选择笔划”(Select Strokes)。会选中该名称的笔划。

选择场景中的所有笔划

  1. 在场景视图或场景绘制视图中，选择“选择 > 全部按类型 > 笔划”(Select > All by Type > Strokes)。
    注： 选择场景中所有笔划也会选择隐藏的笔划。
