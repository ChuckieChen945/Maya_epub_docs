set_particle_property_from_age

可以在 simulate_particles 复合的下游使用此项来根据归一化的粒子年龄设置特性。例如，可以根据粒子年龄设置 point_size 或 color。根据粒子年龄对为出生和消亡定义的值进行插值。

输入

particles

应从 simulate_particles 复合、缓存的粒子或具有 point_age 和 point_age_limit 特性的任何对象连接此项。

property

要设置的特性。如果需要，可创建此项。

birth_value

粒子年龄为 0 时要使用的特性值。此特性的类型应与相关特性类型一致，例如，对于 color，应使用 float3。

death_value

粒子年龄为 point_age_limit 时要使用的特性值。此特性的类型应与相关特性类型一致，例如，对于 color，应使用 float3。

dropoff

输出

out_particles

输出是具有已添加或已修改特性的粒子对象。
