transform_scope

将一组变换矩阵的轴显示为股箭头。支持多种类型：

  • array<float4x4>

  • array<double4x4>

  • array<Transform>

在将 array<Transform> 连接到此端口时，其 world_matrix 分量用于显示变换轴。

输入

transforms

要显示的变换。

axes_shape

要应用于生成的股的几何体着色器的类型。

  • Wire：将股显示为 1 像素宽的细线。

  • Ribbon：将股显示为扁平条带，宽度相对于变换的比例，并使用 axes_scale、arrowhead_size_ratio 和 stem_width_ratio 值。

  • Tube：将股显示为厚挤出管，半径相对于变换的比例，并使用 axes_scale、arrowhead_size_ratio 和 stem_width_ratio 值。

axes_scale

用于增加或减小要显示的变换轴比例的倍增。

arrowhead_size_ratio

箭头宽度的大小比率，相对于每个变换轴的比例。假定变换的比例为 {1, 1, 1}，则值为 0.1 时，箭头的宽度将为 0.1。如果变换的比例为 {10, 10, 10}，则使用相同的值时，箭头的宽度将为 1.0。

stem_width_ratio

杆宽度相对于箭头最宽部分的值。值为 0.1 表示杆宽度是箭头宽度的十分之一。

输出

diagnostic_geo

生成的股几何体。
