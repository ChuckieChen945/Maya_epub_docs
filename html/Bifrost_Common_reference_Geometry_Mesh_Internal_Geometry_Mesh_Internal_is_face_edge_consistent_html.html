is_face_edge_consistent

确定指定的 face_edge 是否一致。

如果面边是开放边，或者面边是流形边，并且面边的成对面边的方向相反（即，成对面边的第一个顶点等于原始面边的第二个顶点，且成对面边的第二个顶点等于原始面边的第一个顶点），则此面边是一致的

例如，f0 和 f1 之间的边是一致的：

 ┌───────────────┬────────────────┐
 │      s2       │       s2       │
 │             ▲ │ │              │
 │             │ │ │              │
 │s3    f0   s1│ │ │s3   f1     s1│
 │             │ │ │              │
 │             │ │ ▼              │
 │      s0       │       s0       │
 └───────────────┴────────────────┘

但是，f2 和 f3 之间的边不一致。请注意，f2 的缠绕顺序为逆时针，f3 的缠绕顺序为顺时针。

 ┌───────────────┬────────────────┐
 │      s2       │       s2       │
 │             ▲ │ ▲              │
 │             │ │ │              │
 │s3    f2   s1│ │ │s1   f3     s3│
 │             │ │ │              │
 │             │ │ │              │
 │      s0       │       s0       │
 └───────────────┴────────────────┘

输入

face_edge

要查询的面边。

face_vertex

相应网格的面顶点数组。

face_offset

相应网格的面偏移数组。

face_vertex_adjacent_edge

相应网格的面顶点相邻数组。（请参见 update_mesh_adjacency）

输出

is_consistent

如果 face_edge 是一致的，则为 true；否则为 false。
