get_mesh_geo_loc_data

它会将存储在通用 GeoLocation 结构中的位置数据解码/解释为描述网格曲面上位置的数据。

输入

location

要解码的 GeoLocation 结构。

输出

face_index

曲面位置位于具有此索引的面上。

face_vertex_indices_count

face_vertex_indices

面的面相关索引。

face_vertex_weights

与面顶点索引对应的归一化权重。权重加上索引构成了表示此曲面位置的广义重心坐标的基础。

aux_face_vertex_indices

aux_face_vertex_weights

valid

如果通用输入位置表示网格曲面上的位置，则为 True。否则为 False。如果为 False，则其他输出的值未定义。
