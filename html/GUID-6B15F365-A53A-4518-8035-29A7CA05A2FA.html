“导出蒙皮权重贴图”(Export Skin Weight Map)选项

导出值(Export Value)

选择下列选项之一。

Alpha
导出 Alpha 通道（不透明度）的值。
亮度(Luminance)
导出亮度的值。
贴图大小 X (Map Size X)、贴图大小 Y (Map Size Y)
设置图像的宽度和高度。如果蒙皮具有 1000 秒的 CV 或顶点，则大型贴图将确保目标蒙皮与原始蒙皮匹配。但是，大型贴图需要您花费更多时间保存到磁盘，还需要更多的磁盘空间。如果贴图过小，可能会导致计算 Alpha 或亮度的平均值时发生意外。
默认值 512 x 512 适用于大多数蒙皮。如果蒙皮包含许多 CV 或顶点，请使用“贴图大小”(Map Size)值 1024 x 1024。
如果蒙皮具有相对较少的 CV 或顶点，请使用值 256 x 256。
如果您使用由许多 NURBS 面片组成的蒙皮，请考虑使用 256 x 256 以避免浪费大量的磁盘空间。
保持纵横比(Keep Aspect Ratio)
导出时保持属性贴图的高宽比。
图像格式(Image Format)
指定图像的类型，例如 TIFF、JPEG 等。
蒙皮簇(Skin Cluster)

“蒙皮簇”(Skin Cluster)列表包含与当前选定几何体关联的所有蒙皮簇

“蒙皮簇”(Skin Cluster)区域可用于在应用了多个蒙皮簇的几何体上选择单个蒙皮簇。
“蒙皮簇”(Skin Cluster)列表
此处显示选定几何体上的蒙皮簇。
注： 如果未找到蒙皮簇，则此列表将保留为空。选择几何体以更新该列表。
最近添加的蒙皮簇将显示在列表的顶部。
