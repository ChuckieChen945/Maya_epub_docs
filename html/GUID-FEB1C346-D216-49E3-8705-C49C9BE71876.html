“导入交互式修饰预设”(Import Interactive Grooming Preset)选项

预设文件名(Preset File Name)
单击 图标，然后浏览到要应用的预设的位置。

传递方法(Transfer Method)

可用于指定样条线是基于源网格还是目标网格进行定位。

基于位置(Position Based)
通过复制目标网格上的曲面位置来放置样条线。如果源网格和目标网格具有不同的 UV 和拓扑、但具有类似的形状，则可以使用此方法。
基于 UV(UV Based)
使用目标曲面的 UV 放置样条线。当源网格和目标网格相同或类似且 UV 相同时，可使用此方法。

取向(Orientation)

旋转以与新曲面对齐(Rotate to align with new surface)
旋转样条线，使其与目标网格的新曲面法线对齐。当源网格和目标网格具有类似或不同的形状时，均可使用此选项。
在世界空间中使用原始方向(Use original orientation in the world space)
样条线重用源网格/或源描述中的旋转值。当源网格和目标网格具有类似的形状时，可使用此选项。
