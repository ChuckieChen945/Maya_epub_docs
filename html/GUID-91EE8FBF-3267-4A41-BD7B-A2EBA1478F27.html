导入和导出 .anim 格式的动画曲线

Maya 提供了用于将动画导入和导出到单独文件的插件。此插件可加快从一个场景向另一个场景复制动画的速度：例如，您可以为具有角色的轻量表示的场景设置动画，然后将动画传输到另一文件中该角色的最终渲染版本。

注： 为了获得最佳结果，建议使用 .atom 格式导入和导出动画。请参见 ATOM 动画文件格式和使用 ATOM 文件格式导入和导出。

导入动画曲线

在内部，此插件会导入和导出到 API 剪贴板。在导入和导出过程中，此插件会执行 copyKey 或 pasteKey MEL 命令以将动画曲线从 API 剪贴板传递到 Maya 对象。使用此插件不会更改 Maya 中的动画剪贴板上的内容。

通过导入可以设置与“关键帧”(Key)菜单中的“关键帧 > 粘贴”(Key > Paste)相同的选项（将动画应用于选定对象时使用这些选项）。

导入动画曲线

  1. 确保已加载 animImportExport 插件。
  2. 选择要设置动画的 Maya 对象。
  3. 选择文件>导入(File > Import)。
  4. 选择要导入的 .anim 文件。

导出动画曲线

导出动画曲线操作将使用 copyKey MEL 命令将关键帧复制到 API 剪贴板，然后将剪贴板内容导出到动画文件。整个操作包含在文件导出中。

注：

因为使用了 MEL 命令，因此这些命令具有的所有限制在插件中也同样存在。例如，如果动画曲线位于属性上游，但该属性与动画曲线之间存在融合节点，则导出命令将不起作用。

通过“文件 > 导出当前选择”(File > Export Selection)选项窗口，可以设置与“关键帧”(Key)菜单中的“关键帧 > 复制”(Key > Copy)相同的选项（用于从选定对象导出动画）。

导出动画文件时，以前位于 API 剪贴板上的所有信息将在文件导出后替换为新动画曲线。

导出动画曲线

  1. 加载 animImportExport 插件。
  2. 选择要从中导出动画的 Maya 对象。
  3. 选择文件 > 导出全部(File > Export All)或文件 > 导出选择”(File > Export Selection)。
  4. 将文件类型设置为 animExport。
  5. 输入或选择要导出的 .anim 文件。
