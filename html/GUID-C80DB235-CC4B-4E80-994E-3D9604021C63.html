编辑 XGen 文件路径

默认情况下，XGen 使用相对路径指向生成修饰或归档所需的所有文件。每次保存修饰时，XGen 均会将必要文件保存到 XGen 目录，然后您可以在需要时从此目录进行加载。该相对路径由当前 Maya 项目定义（有关详细信息，请参见管理 XGen 文件和目录）。

如果已设置默认 Maya 项目，XGen 将使用位置：C:\Users\<用户名>\Documents\maya\projects\default\xgen。这意味着 XGen 使用的相对路径为：${PROJECT}xgen/collections/descriptions。您还可以创建新项目，或者使用现有非默认项目并将其设置为当前项目，然后再创建 XGen 描述。这样做会将所有必要文件放在一个位置。有关如何设置项目的详细信息，请参见在 Maya 中设置项目。

自定义文件路径

有时，您可能需要为 XGen 文件引用多个位置。如果有人向您发送 XGen 项目或您的部分 XGen 项目存储在不同的位置，这将很有用。

自定义 XGen 文件路径

  1. 在 XGen 窗口中，转到“集合 > 编辑文件路径”(Collection > Edit File Path)。

    将显示“编辑文件路径”(Edit File Path)对话框。

  2. 输入所需路径名（用分号分隔）。您可以混合添加绝对路径和相对路径。
  3. 单击“确定”(OK)。

XGen 将依次搜索每个位置，直至找到所需文件。如果无法找到相应文件，将返回错误，并将此错误记录在日志文件中。
