merge_meshes

输入

geometry

要合并在一起的几何体。不是有效网格的对象将被忽略。

properties

要传递到新几何体的几何特性的名称，例如 point_color。确保包含所需的所有几何特性，包括那些以面以外的组件（例如点和面顶点）为目标的几何特性。不是几何特性的特性将始终进行传递。

  • 使用空格分隔多个几何特性名称。
  • 您可以在表达式中使用通配符 *，例如使用 point_user* 来传递所有以字符串“point_user”开头的几何特性。
  • 默认值 * 将传递所有几何特性。
  • 使用空字符串可阻止传递任何几何特性，网格需要的几何特性除外（point_position、face_offset 和 face_vertex）。
  • 使用 * 后跟 ! 可排除特性。例如，* !point_user* 将包括以字符串“point_user”开头的特性以外的所有特性

point_collections

用于根据输入几何体生成集合的前缀。如果传入空字符串，则不会在合并的几何体中生成集合。

face_collections

用于根据输入几何体生成集合的前缀。如果传入空字符串，则不会在合并的几何体中生成集合。

输出

merged_meshes

将所有输入合并在一起后的结果。
