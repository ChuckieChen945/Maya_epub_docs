sample_closest_point_accelerator

使用以 build_closest_accelerator 节点为输入构建的加速度结构，返回几何体上距每个给定位置最近的点（顶点）的索引。

可以构建一次加速器，然后在图表中的多个地方使用。但是，如果在相应节点的下游修改几何体，加速器不会自动更新。要查询修改后的几何体，请构建新的加速器。

输入

accelerator

来自 build_closest_accelerator 节点的 accelerator 输出。

position

要从其查找最近点的位置。

use_cutoff_distance

仅在指定的 cutoff_distance 中搜索。

cutoff_distance

要在其中搜索的距离。

输出

point_index

最近点的点索引。

found

用于指示输出索引是否有效的标志。如果为 false，则相应的位置无效，如果使用它对几何体进行采样，将直接返回默认值。这可能是因为中止距离内没有点，也可能是因为几何体不受支持。
