创建场并将对象连接到场

创建场时，通常也将场连接到场要影响的对象。或者，可以在创建后将对象连接到场的影响（请参见动态关系编辑器）。只要连接的对象位于场的影响范围内，其就将受场的影响。可以为场景（不是被引用场景）中对象的选定 CV、编辑点、顶点或晶格点添加场。

尽管每个场具有不同的效果，用于创建所有场的技术却是相似的。有关每个场的效果以及如何调整这些效果的信息，请参见编辑场属性。

可以通过创建体积轴曲线来沿着曲线创建体积场。请参见使用体积轴曲线。

下列情况适用于所有场：

  • 可以一次选择多个场。
  • 选择顺序并不重要。
  • 所有选定的对象将受所有选定的场的影响。

创建独立的场并将对象连接到场

  1. 选择场要影响的对象。

    可以选择几何体、粒子、nParticle、刚体或柔体。

    提示：

    默认情况下，当将其连接到场的影响时，Maya 将把“NURBS”或“多边形曲面”(polygonal surface)连接到活动的刚体。控制此功能的选项为“自动创建刚体”(Auto Create Rigid Body)（位于“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)窗口的“动力学”(Dynamics)部分中）。

  2. 从“FX”菜单集中，选择“字段”(Fields) > fieldname。

    该场将在原点处创建，并且已选定的所有对象将连接到该场。可以移动场图标，使其远离原点以更改影响的范围和方向。若要更改场设置，请参见编辑场属性。

  3. 单击播放按钮以查看影响连接对象的场。

将场添加到对象并将对象连接到场

  1. 从 FX 菜单集中，选择“字段/解算器”(Fields/Solvers) > fieldname 以创建场。如果已创建场，请选择该场。
  2. 使用 Shift 键选择一个或多个要添加场的对象。
  3. 选择“字段/解算器 > 使用选定对象作为源”(Fields/Solvers > Use Selected as Source)以将场添加到对象。

    默认情况下，大多数场添加到所有 CV 或对象的顶点。场以相同的强度从每个点发出。有关详细信息，请参见场的“逐顶点应用”(Apply Per Vertex)属性描述。

  4. 在“大纲视图”(Outliner)中，选择场要影响的对象，并使用 键（Windows 和 Linux）或 键 (Mac OS X) 选择场本身。

    该场在所属对象下缩进。

  5. 选择“字段/解算器 > “影响选定对象”(Fields/Solvers > Affect Selected Object(s))。
  6. 单击播放按钮以查看影响连接对象的场。
    提示： 如果在选择要影响的对象之前创建场，将没有对象会连接到场。若要将对象连接到场，请使用动力学关系编辑器（请参见动力学关系编辑器）。

    如果已向某个对象添加多个场，而现在要将若干对象与选定的一个或多个场连接起来（或断开连接），“动力学关系”(Dynamic Relationships)编辑器也非常有用。

    若要连接到一个对象拥有的所有场，只需选择要影响的对象，将鼠标指针移动到所属对象上方，然后单击鼠标右键并从弹出菜单中选择“连接场”(Connect Field)。

创建体积场

  1. 选择场要影响的对象。

    可以选择粒子、nParticle、刚体或柔体。

  2. 从 FX 菜单集中，选择“字段/解算器”(Fields/Solvers) fieldname > 。
  3. 在场的选项窗口中，从“体积形状”(Volume Shape)下拉列表中选择形状（而不是“无”）。

    “无”意味着场将影响所有与其连接的粒子。

  4. 根据需要设定体积场属性。请参见编辑场属性。
  5. 单击“创建”(Create)。

    体积场的隐式形状将显示在屏幕上。可以移动、旋转、缩放或斜切该场。

    注：

    不能对体积变形或使用任意体积。

可以通过创建体积轴曲线来沿着曲线创建体积场。请参见使用体积轴曲线。

将场添加到选定的点并将对象连接到场

  1. 从 FX 菜单集中，选择“字段/解算器”(Fields/Solvers) > fieldname 以创建场。如果已创建场，请选择该场。
  2. 使用 Shift 键在对象上选择所需的点。

    例如，选择对象的几个 CV。有关选择组件的详细信息，请参见“基础知识”。

  3. 选择“字段/解算器 > 将选定对象作为源”(Fields/Solvers > Use Selected as Source)。

    通过该操作会将场添加到对象的选定点。此外还会创建一个名为 fieldnameSet 的点集。可以编辑集成员身份以更改场点。请参见“基础知识”。

  4. 在“大纲视图”(Outliner)中，选择场要影响的对象，并使用 键（Windows 和 Linux）或 键 (Mac OS X) 选择场。

    该场在所属对象下缩进。

  5. 选择“字段/解算器 > “影响选定对象”(Fields/Solvers > Affect Selected Object(s))。
  6. 单击播放按钮以查看影响连接对象的场。

删除场

  1. 在“大纲视图”(Outliner)或工作区中选择场。

    在“大纲视图”(Outliner)中，对象场在所属对象下缩进。

  2. 按住 键（Windows 和 Linux）或 Delete 键 (Mac OS X)。

添加更多的场到对象

  1. 从 FX 菜单集中，选择“场”(Fields) > fieldname 以创建场。如果已创建场，请选择该场。
  2. 在“大纲视图”(Outliner)中使用 键（Windows 和 Linux）或 键 (Mac OS X) 选择要添加场的对象。
  3. 选择“字段/解算器 > 将选定对象作为源”(Fields/Solvers > Use Selected as Source)。
