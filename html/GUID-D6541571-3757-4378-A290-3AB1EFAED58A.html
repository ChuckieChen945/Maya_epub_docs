查看选定的引用

如果在场景中有大量文件引用，您可以通过使用“引用编辑器”(Reference Editor)的“视图”(View)选项和引用查看集来选择哪些文件引用显示在编辑器中。“引用查看集”(Reference View Set)是在当前场景中显示特定引用子集的书签。

使用“引用编辑器”(Reference Editor)的“视图”(View)选项，您可以执行以下操作：

  • 查看选定引用
  • 查看所有引用
  • 创建引用查看集
  • 在视图和引用查看集之间切换
提示：

从“引用编辑器”(Reference Editor)中的上下文相关菜单也可以获得“视图”(View)菜单项。若要显示该菜单，请选择并按住 ，单击一个文件引用，然后选择菜单项。

创建和编辑引用查看集

创建“引用查看集”(Reference View Set)的步骤

  1. 在“引用编辑器”(Reference Editor)中，选择要添加到引用集的引用。
  2. 选择“视图 > 查看选定引用”(View > View Selected References)。
    仅选定引用显示在“引用编辑器”(References Editor)中，而新引用视图 (referenceViewSet1) 将在“视图”(View)菜单中列出。使用该书签在当前视图和其他引用查看集之间切换。
    注： 默认情况下，引用查看集命名为“referenceViewSet”，并按顺序编号。

重命名“引用查看集”(Reference View Set)

  1. 在“引用编辑器”(Reference Editor)的“视图”(View)菜单中，选中要重命名的“引用查看集”(Reference View Set)旁边的选项框 ()。

    将显示“编辑引用查看集”(Edit Reference View Set)窗口。

  2. 为“引用查看集”(Reference View Set)键入一个新名称并单击“重命名”(Rename)。

删除“引用查看集”(Reference View Set)

  1. 在“引用编辑器”(Reference Editor)的“视图”(View)菜单中，选中要删除的“引用查看集”(Reference View Set)旁边的选项框。

    将显示“编辑引用查看集”(Edit Reference View Set)窗口。

  2. 单击“删除”(Delete)。
    提示：

    从“引用编辑器”(Reference Editor)中的上下文相关菜单也可以获得“视图”(View)菜单项。若要显示该菜单，请选择一个文件引用，然后在该项上单击鼠标右键。
