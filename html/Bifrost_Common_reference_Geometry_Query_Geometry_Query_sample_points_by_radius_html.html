sample_points_by_radius

按半径对点进行采样，将对点 property 进行加权平均。

此节点结合了 get_points_in_radius 和 sample_property 的功能，可给出所找到点的加权平均值。对于每个位置输入，它将在半径范围内搜索另一个几何体的点，并返回某些特性的加权平均值。

若要使用它，请插入要搜索的目标 geometry、positions 和要使用的特性。

特性(Property)

在此处，以与 sample_property 相同的方式指定要采样的特性，并使用 default_and_type 的值和类型调整指定类型和默认值。

搜索(Search)

max_radius、enable_max_number 和 max_number 控制考虑的点数。这些设置控制搜索，与 get_points_in_radius 相同。

权重(Weights)

对找到的点进行加权，以便距离较远的点比邻近的点具有较少的效果。

exponent 控制 falloff_fcurve。

启用 use_inverse_distance 后，距离较近的点的影响将远远超过距离较远的点，衰减类似于平方反比定律。公式为 (1 / 距离) ^ 指数。max_radius 无效果。这会产生更清晰的结果。

启用 use_inverse_distance 后，衰减从 0 到max_distance 呈线性形式，并应用指数。这意味着，接近的点往往都具有相似的权重，从而产生“更模糊”的结果。

距离衰减(Distance Falloff)

启用距离衰减 F 曲线 (use_fcurve) 后，值本身会随着距离的增加而变弱，由 falloff_fcurve 控制。随着找到的采样越来越远，值会柔和地混合到零。例如，如果在搜索半径的边上仅找到一个位置值，则返回的值可能几乎为零。这样做的结果是软衰减（由 F 曲线控制）到默认值，此时，点离得更远。

启用距离衰减曲线 (use_fcurve) 后，值是纯加权平均值。如果在搜索半径内的任意位置找到单个采样，则会精确地返回该值。这样做的结果是完整的值，但当超过半径时，这些值会急剧变为零。
