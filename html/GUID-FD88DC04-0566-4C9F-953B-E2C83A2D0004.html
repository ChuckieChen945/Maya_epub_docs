元数据元素

元数据元素提供有关可在应用程序的信息面板中显示的颜色变换的信息。

本节内容
  • Info
    Info 元素是 Copyright 和 Release 元素的容器。
  • Description
    Description 元素可用于添加有关 ProcessList 或运算符元素的函数或者用法的信息。它会显示在支持 Autodesk 颜色管理的产品的颜色变换的元数据面板中。
  • InputDescriptor
    通过 InputDescriptor 元素可以添加注释，说明某个颜色变换的预期输入颜色空间和编码。它不会影响处理，但对于支持 Autodesk 颜色管理的产品，它会显示在颜色变换的元数据面板中。
  • OutputDescriptor
    通过 OutputDescriptor 元素可以添加注释，说明某个颜色变换的输出颜色空间和编码。它不会影响处理，但对于支持 Autodesk 颜色管理的产品，它会显示在颜色变换的元数据面板中。
