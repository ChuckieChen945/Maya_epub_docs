sort_array_with_indices

对数组进行排序，并输出已排序的数组及附加索引数组。

索引有助于确定已排序元素的原始索引。可与 get_from_array 一起使用，以按同样的顺序对其他数组进行排序。

例如，给定单清度浮点数数组 [0.2, 0.1, 0.3]，此节点将返回已排序数组 [0.1, 0.2, 0.3] 和索引数组 [1, 0, 2]。

输入

array

要排序的数组。

descending_order

如果为 true，则按从大到小的顺序对元素进行排序。

输出

sorted

排序后的数组。

sorted_indices

按新顺序排序的索引。可与 get_from_array 一起使用，以对大小相同的其他数组进行排序。

父页面: Array
