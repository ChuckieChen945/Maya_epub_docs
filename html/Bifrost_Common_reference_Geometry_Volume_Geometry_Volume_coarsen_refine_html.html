coarsen_refine

此节点根据体积本身或场的特性粗化和/或细化体积。

输入

volume

输入体积。

field

用于粗化和/或细化体积的标量场。该场解释为（带符号）距离场。只要该场为负，输出体积都将完全细化（如果 mode 为 Refine 或 CoarsenRefine）或保留（如果 mode 为 Coarsen）。只要该场为正，输出体积都将逐渐细化（如果 mode 为 Refine 或 CoarsenRefine）或保留（如果 mode 为 Coarsen），范围是围绕（带符号）距离场的零交叉的带，带宽由 bandwidth 参数指定。

property

用于粗化和/或细化体积的浮点特性的名称。如果连接了非空场，则将忽略 property。property 解释为与场输入类似的（带符号）距离场。

mode

用于粗化和/或细化体积的模式。支持以下模式：

  • None：不粗化或细化体积
  • Coarsen：粗化体积
  • Refine：细化体积
  • CoarsenRefine：粗化和细化体积

bandwidth

在逐渐细化体积的情况下，围绕（带符号）距离场的零交叉的带宽。在体积平铺树的每个自适应级别测量（以体素为单位）。

输出

out_volume

输出体积。
