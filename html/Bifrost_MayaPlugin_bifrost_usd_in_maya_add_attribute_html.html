在 Bifrost-USD 中将属性添加到基本体

可以使用 define_usd_attribute 节点或专门用于特定预定义属性的类似节点将属性添加到基本体。这些节点可以连接到 define_usd_port 节点的 attribute_definitions 端口 - 这可以是最初定义基本体的 define_usd_port 节点，也可以是指向相同路径的另一个节点。

  1. 在 define_usd_port 节点的 attribute_definitions 端口上单击鼠标右键，然后从“创建节点”(Create Node)菜单中选择一个选项。

    • Usd::Attribute::define_usd_transform 用于快速缩放、旋转和平移基本体。
    • Usd::Attribute::define_usd_display_color 用于快速设置要在视口中显示的颜色。
    • Usd::Attribute::define_usd_display_color 用于设置所有属性。

  2. 在 define_usd_transform 的情况下，需要指定属性的详细信息。

    • 输入属性的 name。
    • 设置 custom 复选框以声明属性是否由用户定义。
    • 从 type 下拉框中选择 USD 数据的类型。此外，请注意，value 端口的数据类型为 auto，但默认为 float - 必须在端口上单击鼠标右键，然后将其更改为与选定 USD 数据类型兼容的 Bifrost 数据类型，例如，float3 用于 Normal3f 或 Color3f，string 用于 Token 或 Asset，以此类推。
    • 如果还要将属性设置为特定值（而不是简单地定义属性是否存在），请选中 enable_value 框，然后输入或连接 value 进行设置。
    • 若要将属性连接到另一个基本体的属性，请选中 enable_connection 框并设置 connection_mode、target_prim 和 target_attribute。
    • 向渲染器声明该属性与几何基本体相关联，可随曲面或体积而变化，请选中 enable_primvar 框并设置 interpolation_method。
    • 要将值设为特定的帧，请选中 use_frame 框并输入要使用的 frame。对于已设置动画的属性，可以连接 time 节点的 frame 输出。
