displace_by_wave

此节点使用一组输入波对象使输入几何体发生位移。这些通常是 create_spectral_waves 或 simulate_dynamic_wave 节点的输出。将在几何体点位置对不同的输入波位移进行采样，并汇总在一起得到最终位移。

输入

geometry

这是使用输入波使其发生位移的网格或点对象。

waves

这是一组波对象，例如来自 simulate_dynamic_wave 或 create_spectral_waves 节点的输出。不同的输入波位移将汇总在一起。

Displace Options

sampler

这是在几何体位置对波图像进行采样时在波图像数组上使用的插值方法。

  • Nearest：最靠近点的像素：无插值

  • Linear：在最近点之间线性插值

  • Quadratic：更平滑地插值

  • Bspline：最平滑的插值

update_normals

根据位移重新计算法线。

magnitude

位移量的比例。这也可能是标量场，例如 fractal_noise_field。

offset

添加到 Y 位移。这也可能是标量场，例如 fractal_noise_field。

输出

out_geometry

这是应用了位移的输入几何体。
