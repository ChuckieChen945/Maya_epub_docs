调整 UV 工具(Tweak UV Tool)

可用于在 UV 编辑器中重新定位 UV，无需使用操纵器。您也可以使用“移动工具”(Move Tool)，通过按住鼠标中键并拖动 UV 来完成此操作。

若要打开“调整 UV 工具”(Tweak UV Tool)，请从“UV 编辑器”(UV Editor)菜单中选择“工具 > 调整”(Tools > Tweak)。

选择“工具 > 调整”(Tools > Tweak) > 时，在“工具设置”(Tool Settings)窗口中将显示以下选项。

调整 UV 设置

工具捕捉(Tool Snapping)
指定“调整 UV 工具”(Tweak UV Tool)操纵器周围的捕捉区域大小。默认情况下，操纵器将捕捉到最近 UV 中。增加“工具捕捉”(Tool Snapping)的值，以取得更大的捕捉效果。默认值为 10。
提示： 您也可以通过双击 UV 编辑器工具栏中的“调整 UV 工具”(Tweak UV Tool) 图标来调整“工具捕捉”(Tool Snapping)。
