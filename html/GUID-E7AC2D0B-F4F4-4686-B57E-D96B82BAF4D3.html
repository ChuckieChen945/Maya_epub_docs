翻转 UV 壳

使用“翻转”(Flip)命令可以在“UV 编辑器”(UV Editor)中轻松地更改 UV 壳的方向。如果需要更正反转的纹理，翻转 UV 壳将非常有用（请参见显示 UV 缠绕顺序）。

翻转 UV 壳上的 UV

  1. 在“UV 编辑器”(UV Editor)中，选择要翻转其纹理的面。
  2. 在“UV 工具包”(UV Toolkit)中，选择“变换 > 翻转”(Transform > Flip)（或者，在菜单栏中选择“修改 > 翻转”(Modify > Flip)）。

    选定 UV 将在指定轴上翻转。

如果先启用“着色模式”(Shaded Mode)，然后将其设置为显示“前面和背面”(Front and Back Facing)壳，查看翻转壳的结果最容易。
