cull_points

基于加权概率，随机删除点对象（例如粒子系统）上的点。如果是实例对象，可以改为禁用点，即禁止在不删除某个点的情况下让某个实例化几何体显示在该点处。

输入

points

点对象或实例对象。

weights

probability 端口上的倍增。

  • 小于等于 0 的值表示点永不消隐。
  • 0 到 1 范围内的值表示修改点被消隐的几率。例如，0.5 表示某个点被消隐的概率是 50% × probability。
  • 1 表示点被消隐的几率正好为 probability。
  • 大于 1 的值表示点被消隐的几率大于 probability。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

probability

点被消隐的基本几率。

seed

随机数生成器的种子。

disable_by_selection

如果选中此项，则将禁用实例，而不是删除实例。对于禁用的点，这会将 instance_property 的值从 x 设置为 - x - 1。例如，如果选择了实例 2，这会将特性设置为 -3。这将禁止在相应点处显示实例化几何体，但允许您恢复之前实例化的几何体的索引。

instance_property

要修改以便禁用实例的特性。

输出

remaining_points

修改后的几何体，不包含已删除的点。已禁用（而非已删除）的点包含在此对象中。

deleted_points

包含已删除的点的几何体，以防您要将其用于其他情况。

remaining_indices

未删除的点的旧索引。

deleted_indices

已删除的索引。如果您有需要同步的单独数据，可以使用此项。
