resample_matrix_chain

使用各种重采样方法和选项对变换矩阵链进行重采样。

输入

transforms

要解算的变换的 array<double4x4>，解释为从头至尾的连续链。

resample_mode

要执行的重采样类型：

  • Linear：在输入链中的变换之间执行线性插值。
  • Catmull-Rom：在输入链中的变换之间执行 Catmull-Rom 插值。

spatial_resample

如果启用此项，将根据距离对链进行重采样。否则，以参数化方式进行重采样。

sample_count

定义重采样的链中的变换数。

end_point_mode

控制重采样的链端点附近的形状。

  • Straight：链将沿着直线路径穿过端点。
  • Curved：链将沿着输入链的曲线穿过端点。

tension

控制插值曲线的形状。值越大，插值的变换越接近输入链。

alpha

控制曲线的参数化。值为 0.0 时将生成统一参数化。值为 0.5 时将生成向心参数化。值为 1.0 时将生成弦参数化。

输出

out_transforms

生成的 array<double4x4>，包含重采样的变换值。
