绘图板设置

建议您使用图形绘图板，以充分利用 Maya 的雕刻功能。但是，您仍可以在 Maya 中使用标准三键鼠标进行雕刻。有关将绘图板与 Blue Pencil 工具结合使用的信息，请参见将绘图板与 Blue Pencil 结合使用。

可以在界面首选项(Interface Preferences)的“绘图板 API”(Tablet API)部分中切换绘图板 API。（请参见“窗口”(Windows) > “设置/首选项”(Settings/Preferences) > “首选项”(Preferences) > “界面首选项”(Interface Preferences)）

提示： 对绘图板进行故障排除时，请尝试使用不同的 API 设置；切换到其他 API 可能会获得更好的结果。

如果具有多点触控设备，则还可以使用以光标为中心的输入来操作 Maya。请参见多点触控输入手势。

如果您使用的是 Wacom 绘图板，则可以在以下位置找到最新的软件驱动程序：http://www.wacom.com/productsupport/select.cfm。
