将预定义的初始状态添加到流体容器

初始状态描述了流体模拟的开始帧中的栅格特性值。您可以定义要模拟开始的栅格特性值，然后设定流体的初始状态。有关设定初始状态的信息，请参见流体初始状态。

如果要使用现有栅格特性作为其他流体效果（例如，公司徽标、螺旋式速度和云）的基础，则可以保存栅格特性值，然后将其添加到场景的现有流体容器中。Maya 具有一系列的预定义初始状态，您可将其用作示例。有关创建自己的预定义初始状态的信息，请参见保存流体的当前状态。

将预定义的初始状态添加到流体容器

  1. 选择流体容器。
  2. 选择“流体 > 添加/编辑内容 > 初始状态”(Fluids > Add/Edit Contents > Initial States)。将打开“内容浏览器”(Content Browser)，并显示“流体初始状态”(Fluid Initial States)文件夹。
  3. 单击包含要添加到容器的初始状态的文件夹：如果流体是 2D 流体，请选择 2D 文件夹；或者，如果流体是 3D 流体，请选择 3D 文件夹。

    仅可将 2D 容器中创建的初始状态添加到 2D 容器中。同样地，仅可将 3D 容器中创建的初始状态添加到 3D 容器中。

  4. 使用鼠标中键将预设图标从“内容浏览器”(Content Browser)拖动到选定的流体容器中。容器中已有的任何栅格值均会替换为初始状态值。

    如果容器的分辨率与用来创建初始状态的容器的分辨率不同，则您容器的分辨率将更改为初始状态容器的分辨率，且当值均存在时，将更改回原始分辨率。
