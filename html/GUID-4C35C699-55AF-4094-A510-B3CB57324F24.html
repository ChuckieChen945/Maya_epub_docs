Bullet v3.0

Maya 预安装了两个 Bullet 版本：2.8.2 和 3.24，但使用 2.8.2 作为默认版本。

如果要使用 Bullet 3.24，您必须设置一个环境变量。有关详细信息，请参见直接修改设置文件
注： 我们建议使用 Bullet 2.8.2，因为 3.24 存在限制。请参见 Bullet 的已知限制。
.

要切换环境变量以使用 Bullet 3.24：

  1. 使用以下导向定位 Maya 首选项文件：

    Windows

    • ..\Documents\maya\<版本>\prefs

    Mac OS X

    • /Users/<用户名>/Library/Preferences/Autodesk/maya/<版本>/prefs
      • 注： 若要打开“首选项”(Preferences)目录，请执行以下操作：
      • 选择“查找器” > “转到” (Finder > Go)，按 Alt 键，Library 文件夹将显示在菜单中。
      • 从终端窗口导航到“首选项”(Preferences)目录，然后键入 open。

    Linux

    • ~<用户名>/maya/<版本>/prefs
  2. 找到以下行：
    MAYA_BULLET_B3_ENABLED=1
    首选项文件中的任何位置。
  3. 保存并关闭该文件。

    4. Maya 现在将使用 Bullet 3.24。

注： 请注意，Bullet 3.24 刚体 OpenCL 加速有一些限制：
  • 柔体不能使用 OpenCL 加速器。
  • Bullet 3.24 碰撞仅支持“长方体”(Box)和“壳线”(Hull)碰撞形状类型。
  • 请不要为任何刚体和柔体交互使用 Bullet 3.24 解算器加速。
