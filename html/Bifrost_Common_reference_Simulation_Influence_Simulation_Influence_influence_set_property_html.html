influence_set_property

这允许在模拟中设置几何特性。值的类型应与几何特性的类型匹配，因此，可以对诸如 point_size 之类的特性使用 float 或 scalarField，而如果设置 point_velocity，则该值应为 float3 或 vectorfield。如果影响物没有遮罩，则它将设置所有值，否则它将考虑任何定义的遮罩。

要使用此节点，请将其输出插入模拟节点（如 simulate_aero、simulate_particles 或 simulate_mpm）的 influences 输入。

输入

influence

可以选择将影响场 out_influence 连接到此端口。例如，可以连接 mask_influence 节点以遮罩此影响的效果。可以将任意数量的影响场呈菊花链状连接在一起，以构建复杂的场。

key

这是解算器上要设置的几何特性的名称。对于 Aero，这可能是 voxel_fog_density 或 voxel_temperature，而对于粒子解算器，可能会设置 point_size 或其他每粒子特性。

value

这是用于替换解算器每粒子或每体素特性的值。它应与所设置的类型相匹配。可以连接 scalarField 和 vectorField，采用随空间变化的方式设置 float 和 float3 特性。

输出

out_influence

可以连接到模拟节点或其他影响节点的输出影响对象。
