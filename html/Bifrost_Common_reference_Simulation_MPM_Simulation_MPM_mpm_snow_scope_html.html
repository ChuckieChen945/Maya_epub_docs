mpm_snow_scope

可视化由 simulate_mpm 输出的雪粒子的内部特性。

粒子的色调由选定的模式和颜色确定。它们的明度（即明暗程度）将由它们接近屈服的程度来确定。完全亮度的颜色表示粒子发生屈服，而较暗的粒子则发生弹性变形。

输入

snow_particles

从 simulate_mpm 输出的雪粒子。

Point Display

size_scale

控制显示的粒子的大小。

shape

设置粒子的显示形状。有关选项的更多详细信息，请参见 set_point_shape。

shadows

控制视口阴影为粒子上色的强度。

mode

选择是按 firmness 还是按 packing 为粒子上色。

Firmness

密实度是指雪的抗屈服能力（以平方米/平方秒为单位）。当外力超过其抵消形状和体积变化的能力时，雪便会屈服，如果没有进一步的阻力，便会变形或流动。密实度可以是正值，也可以是负值，其中负值表示雪必须经受进一步挤压才会抵抗压缩。

firmness_start

要可视化的 firmness 的下限。

firmness_start_color

小于或等于 firmness_start 的值要使用的颜色。

firmness_end

要可视化的 firmness 的上限。

firmness_end_color

大于或等于 firmness_end 的值要使用的颜色。

Packing

挤压模式显示粒子是压缩还是展开。

unpacking_color

要用于扩张的雪粒子的颜色。只有具有完全亮度的粒子才会主动扩张，而较暗的粒子在展开时会发生弹性变形。

packing_color

用于挤压的雪粒子的颜色。只有具有完全亮度的粒子才会主动挤压，而较暗的粒子在压缩时会发生弹性变形。

bounds_geo

连接网格或其他基于点的几何体，以将可视化限制到其边界框内的雪粒子。
