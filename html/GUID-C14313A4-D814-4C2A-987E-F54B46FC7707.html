管理自定义菜单集

创建自定义菜单集

  1. 选择状态行(Status Line)菜单下拉列表，然后选择“自定义”(Customize)。

    打开“菜单集编辑器”(Menu Set Editor)

    此时将显示“菜单集编辑器”(Menu Set Editor)。

  2. 选择“编辑 > 新建菜单集”(Edit > New Menu Set)。
  3. 为自定义菜单集输入一个唯一的名称。
  4. 选择“创建”(Create)。

    新的自定义菜单集将显示在“菜单集”(Menu sets)列、“状态行”(Status Line)下拉列表和热盒中。

向菜单集中添加菜单

  1. 在“菜单集”(Menu sets)列中，选择要添加菜单的菜单集的名称。

    如果选择的菜单集已包含菜单，则这些菜单将列在中心列“菜单集中的菜单”(Menus in menu set)中。

    “菜单集编辑器”(Menu Set Editor)的“全部菜单”(All menus)列中列出了在 Maya 中使用的全部菜单。所有这些菜单都可以添加到菜单集中。

  2. 若要将某个菜单添加到当前选定的菜单集中，请执行以下操作之一：
    • 双击“全部菜单”(All menus)列中的菜单名称。
    • 使用鼠标中键将菜单从“全部菜单”(All menus)列拖动到“菜单集中的菜单”(Menus in menu set)列。
    • 选择菜单后，单击鼠标右键，然后选择“添加到菜单集”(Add to Menu Set)。
      提示：

      若要通过一个操作将多个菜单添加到选定菜单集，请在“全部菜单”(All menus)列中选择菜单，然后使用鼠标中键将其拖动到“菜单集中的菜单”(Menus in menu Set)列。

重新排列菜单集或菜单

  1. 在“菜单集编辑器”(Menu Set Editor)中，在菜单集或菜单上单击鼠标中键，然后在其列中向上或向下拖动。

从菜单集中移除菜单

  1. 选择要从其中移除菜单的菜单集的名称。

    包含在菜单集中的菜单将列在“菜单集中的菜单”(Menus in menu set)列中。

  2. 执行以下操作之一：
    • 双击“菜单集中的菜单”(Menus in menu set)列中的菜单名称。
    • 使用鼠标中键将该菜单从“菜单集中的菜单”(Menus in menu set)列拖动到“全部菜单”(All menus)列。
    • 选择菜单后，单击鼠标右键并选择“从菜单集中移除”(Remove from Menu Set)。

移除菜单集

  1. 选择要移除的菜单集的名称。

    包含在菜单集中的菜单将列在“菜单集中的菜单”(Menus in menu set)列中。

  2. 执行以下操作之一：
    • 选择“编辑 > 移除”(Edit > Remove)。
    • 在菜单集上单击鼠标右键，然后选择“移除”(Remove)。

重命名菜单集

  1. 在“菜单集”(Menu sets)列中选择菜单集后，执行以下操作之一：
    • 单击鼠标右键，然后选择“重命名”(Rename)。
    • 选择“编辑 > 重命名”(Edit > Rename)。
  2. 输入菜单集的新名称，并单击“重命名”(Rename)。
    提示：

    也可以通过双击菜单集名称、输入新名称并按 Enter 键，重命名“菜单集”(Menu sets)列中的菜单集。

将默认菜单集还原为其原始状态

  1. 单击“状态行”(Status Line)下拉列表，然后单击“自定义”(Customize)。

    此时将显示“菜单集编辑器”(Menu Set Editor)。

  2. 若要还原默认菜单集（“建模”(Modeling)、“绑定”(Rigging)、“动画”(Animation)、“动力学”(Dynamics)和“渲染”(Rendering)），请在“菜单集”(Menu sets)列中选择菜单集，然后执行以下操作之一：
    • 单击鼠标右键，然后选择“还原 <menu_name> 为默认值”(Revert <menu_name> to Default)。
    • 选择“编辑 > 还原 <menu_name> 为默认值”(Edit > Revert <menu_name> to Default)。

还原默认菜单集

  1. 选择“状态行”(Status Line)下拉列表，然后选择“自定义”(Customize)。

    此时将显示“菜单集编辑器”(Menu Set Editor)。

  2. 选择“编辑 > 还原默认菜单集”(Edit > Restore Default Menu Sets)。

重用先前版本中的菜单集

  1. 将 menuSetPrefs.mel 文件从旧版首选项文件夹复制到当前首选项文件夹中。
    注： 重命名当前的 menuSetPrefs.mel，以确保您未覆盖当前的默认值。
  2. 重新启动时，将显示“发现新的默认菜单集”(New Default Menu Sets Found)对话框。单击“跳过”(Skip)。然后将显示一条警告，提醒您选择跳过菜单集更新将隐藏功能。
  3. 单击“确定”(OK)。
