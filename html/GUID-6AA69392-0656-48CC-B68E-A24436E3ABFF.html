禁用或启用 IK 控制柄

禁用 IK 控制柄

  1. 选择要禁用的 IK 控制柄。
  2. 执行下列操作之一：
    • ) 单击 IK 控制柄并从出现的标记菜单中选择“禁用 ikHandle”(Disable ikHandle)。
    • 选择“骨架 > 禁用选定 IK 控制柄”(Skeleton > Disable Selected IK Handles)。

    选定 IK 控制柄现在已禁用。IK 控制柄已禁用时，它不再驱动其目标关节并且不能进行编辑。

启用 IK 控制柄

  1. 选择要启用的 IK 控制柄。
  2. 执行下列操作之一：
    • ) 单击 IK 控制柄并从出现的标记菜单中选择“启用 ikHandle”(Enable ikHandle)。
    • 选择“骨架 > 启用选定 IK 控制柄”(Skeleton > Enable Selected IK Handles)。

    选定 IK 控制柄现在已启用。
