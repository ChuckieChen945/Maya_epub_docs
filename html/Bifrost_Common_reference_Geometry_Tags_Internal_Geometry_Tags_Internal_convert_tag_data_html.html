convert_tag_data

将标记数据从一种组件类型转化为另一种组件类型。

这是内部节点，专门供图表中更高级别的节点使用。

输入

geometry

包含 tag_data 的几何体。

tag_data

要转化的标记数据。

source_component

tag_data 的组件类型。

target_component

tag_data 将转化为此组件类型的标记。

conversion_style

要使用的转化方法。请参见 convert_tag。

输出

converted_tag_data

转化的标记数据。

success

如果转化有效，则设置为 true，否则设置为 false。（例如，将面标记转化为股标记将是无效转化。）
