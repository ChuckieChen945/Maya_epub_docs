获取 Python 命令帮助

有关完整的 Python 命令列表，请访问 Python 命令页面。

您还可以从 Maya 中访问“Python 命令”页面，以及访问各个 Python 命令的帮助。

目标 操作

显示命令用法和标志的简短提要。

在命令行或“脚本编辑器”(Script Editor)中，键入： 

import maya.cmds
helpInfo = cmds.help('<command name>')
print(helpInfo)

例如： 

import maya.cmds
helpInfo = cmds.help('move')
print(helpInfo)

显示命令的“Maya 帮助”(Maya Help)。

转到“帮助”(Help)菜单，然后选择“Maya 脚本参考 > Python 命令参考”(Maya Scripting Reference > Python Command Reference)。

在脚本编辑器中键入命令和对象路径时，显示命令和对象路径的自动完成结果 在“脚本编辑器”(Script Editor)中，启用“命令 > 命令完成”(Command > Command Completion)和“对象路径完成”(Object Path Completion)。然后键入命令的前几个字母并按 Ctrl+空格键，或启用“命令 > 显示工具提示帮助”(Command > Show Tooltip Help)可以自动显示自动完成选项。
