collide_field_influence

允许模拟（如粒子、mpm 或 aero）与带符号距离场 (SDF) 碰撞，如 plane_field 或 sphere_field 的输出。连接距离场，并将 out_influence 连接到模拟节点（如 simulate_aero、simulate_particles 或 simulate_mpm）的 influences 输入。

SDF 定义空间中任意点到某个对象的距离。在曲面上，距离为零，而在曲面内部，距离为负。SDF 允许与对象（如没有有限边界的无限平面）发生碰撞，而无需使用无限内存。

输入

influence

（可选）连接其他影响物节点（如 mask_influence 或 modulate_influence）的 out_influence 端口，以创建复杂效果。可以按任意顺序对任意数量的影响物呈菊花链状连接到一起。

distance_field

将表示带符号距离的 ScalarField 连接到对象，例如 plane_field 或 sphere_field 的输出。

碰撞场特性

enabled

关闭此项可禁用效果。请注意，禁用此项后，连接到 influence 输入端口的任何其他影响物场仍将处于活动状态。

distance_field_velocity

当距离场表示的对象移动时，可以选择连接相应的速度值（float3 或 VectorField）。与使用从移动场自动计算的速度相比，这可以给出更好的碰撞行为。

bounciness

粒子在碰撞时反弹的量。可以使用单个 float 值或连接 ScalarField。值为 1.0 时，粒子完全反弹，而不会损失能量。超出 0—1 范围的值在物理上是不可能的，但可以用于效果。反弹度不会影响 Aero 模拟。

friction

粒子抵抗沿曲面滑动的量。可以使用单个 float 值或连接 ScalarField。值为 1.0 表示粒子不会滑动的完全摩擦状态。摩擦力不会影响 Aero 模拟。

pushout

如果粒子或 Aero 密度值在平面的内侧发射，则将它们推送到带符号距离场的外侧。可以使用单个 float 值或连接 ScalarField。

请谨慎使用此选项。对于法向碰撞，值为 0 时效果通常更好，但在需要避免轻微重叠时，可以使用更大的值。值 1 将在一步内完全推出，并可防止粒子沉降。使用 pushout 时，MPM 和自碰撞粒子模拟可能会压缩，在这种情况下，使用 pushout_velocity 可能会更好。

pushout_velocity

缩放速度以将粒子推出距离场。对于 MPM 碰撞或堆叠的自碰撞粒子，这可以比 pushout 产生更好的结果，但这会在碰撞上增加反弹效果，因此在 pushout 不足时应谨慎使用。

offset

偏移距离场碰撞对象的“曲面”（0 等值面）。可以使用单个 float 值或连接 ScalarField。正值会将其向外推，负值会使其收缩。

输出

out_influence

将其连接到模拟节点的 influences 端口或其他影响物节点。
