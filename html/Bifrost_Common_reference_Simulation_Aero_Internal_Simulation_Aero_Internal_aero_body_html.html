aero_body

此节点创建空的 Aero 体，包含体素拓扑、容器以及烟密度和温度通道。

输入

source

一组输入源。如果为非空，将用于确定 detailSize。

detailSize

烟中最小可解析细节的大小。对于基于体素的解算器，这是体素大小。如果源输入为非空，则忽略。

ambientTemperature

烟的环境温度（摄氏度）。

create_mass_density_property

如果为 true，则向 Aero 对象添加质量密度通道。

number_of_upres_levels

平铺树中将保留 upres 通道（雾密度和温度）的最精细级别的数量

feed_upres_into_sim

如果较高分辨率的 upres 通道应反馈到模拟中，则为 True

输出

outAeroBody

输出 Aero 体（对象）。
