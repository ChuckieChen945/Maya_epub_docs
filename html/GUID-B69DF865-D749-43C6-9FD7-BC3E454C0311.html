工作区布局选项(Workspace layout options)

“工作区布局选项”(Workspace layout options)窗口可将工作区与各种界面选项相关联。若要打开该窗口，请选择“窗口 > 工作区”(Windows > Workspaces) > [工作区名称] > 。

有关如何切换或锁定工作空间的信息，请参见工作空间(Workspaces)。

将菜单集链接到此工作区(Link a Menu Set to this workspace)
切换到此工作区时激活选定菜单集。若要创建自定义菜单集，请参见自定义菜单集(Custom menu sets)。
将热键集链接到此工作区(Link a Hotkey Set to this workspace)
切换到此工作区时激活选定热键集。若要创建自定义热键集，请参见热键编辑器(Hotkey Editor)。
将 VP2.0 预设链接到此工作区(Link a VP2.0 Preset to this workspace)
切换到此工作区时加载视口选项的预设。若要创建自定义视口预设，请参见 Viewport 2.0 选项(Viewport 2.0 Options)。
自动保存布局更改
如果希望 Maya 更新窗口布局，而无需在每次重新定位窗口时都保存该窗口布局，请启用此选项。如果禁用此选项，Maya 将在重新打开后还原为保存的布局。
