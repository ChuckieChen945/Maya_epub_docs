get_geo_property_any

以 Any 值的形式获取几何特性数据。这对于在不知道数据类型的情况下操纵几何特性的算法非常有用。

输入

geometry

要从中获取几何特性的几何体。

property_name

要查询的几何特性的名称。

输出

data

几何特性的数据值，包裹在 Any 中。

default

几何特性的默认值，包裹在 Any 中。

target

几何特性的目标组件。例如，point_position 将以 point_component 为目标，face_vertex_normal 将以 face_vertex_component 为目标。

found_property

如果找到几何特性，则设置为 true，否则设置为 false。请注意，如果它返回 false，则所有其他输出将设置为其类型的默认值。

property

几何特性的相应子对象。

interpolation_mode

几何特性的插值模式。

is_indexed

如果几何特性具有关联的索引数组，则设置为 true。例如，UV 特性（例如 face_vertex_uv）通常已创建索引。

indices

几何特性的索引。如果 is_indexed 为 false，则返回的数组只是一个包含值 [0, 1, 2, ... n-1] 的序列数组，其中 n 是几何特性目标的组件数。例如，如果点几何体包含 5 个点，并且 property_name 设置为 point_position（它不是已创建索引的特性），则此数组将设置为 [0, 1, 2, 3, 4]。

range_property

几何特性的相应索引几何特性子对象。

range_property_name

相应索引几何特性的名称。当为几何特性创建索引时，其相应的索引几何特性通常具有相同的名称，后缀为 _index。例如，face_vertex_uv 的索引几何特性为 face_vertex_uv_index。
