update_face_tangent

切线有两种类型，face_tangent 是每个面的计算切线（根据拓扑进行计算），而 face_vertex_tangent 是构成面的顶点的切线（根据 UV 进行计算）。

输入

mesh

要基于其计算切线的网格。

uv_tangent_space

如果选中此项，将使用网格上的 UV 坐标 (face_vertex_tangent) 计算切线空间。如果取消选中此项，将使用网格的拓扑 (face_tangent) 计算切线空间。

输出

out_mesh

网格现在将 property（请参见下文）作为 geo_property。

property

face_tangent 或 face_vertex_tangent，具体取决于选择的设置。
