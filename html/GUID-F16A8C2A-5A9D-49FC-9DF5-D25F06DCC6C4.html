借用许可

借用许可会从许可服务器中移除许可，以便在脱机计算机上使用。

借用许可后，将在启动时自动使用借用的许可。您可以随时暂停此过程，而无需将许可返还给服务器。许可将按您最初借用许可时指定的日期自动返还给服务器。您也可以选择提前手动返还许可。

通过用户帐户菜单借用许可

您可以通过用户帐户菜单下的“管理许可”(Manage License)选项借用许可。有关从“管理许可”(Manage License)借用的详细信息，请参见查看和管理网络许可的步骤。

使用命令行工具借用许可

一般情况下，应该尽量通过用户帐户菜单借用许可。但是，如果用户帐户菜单中没有用于管理许可的选项，或者借用许可的选项不可用，则可以使用命令行工具借用许可并管理借用的许可。

注： 如果您使用的是瘦客户端许可，则需要使用命令行工具借用许可并管理借用的许可。使用瘦客户端许可时，无法通过用户帐户菜单借用许可。

在 Windows 上，LMTOOLS 实用程序用于管理借用的许可。

在 Linux 和 macOS 上，lmutil 用于管理借用的许可。

使用命令行工具借用许可

要在 Windows 上借用许可，请打开 LMTOOLS 实用程序，然后为“供应商名称”(Vendor Name)输入 adskflex。输入返还日期和返还时间，然后单击“设置借用到期”(Set Borrow Expiration)。

要在 macOS 或 Linux 上借用许可，请打开终端并将目录更改为 lmutil 的安装位置，然后执行以下命令： 

lmutil lmborrow adskflex end_date [time]

其中 end_date 是许可的返还日期（格式为 dd-mm-yyyy）。返还时间是可选的。如果使用，则必须以 24 小时格式 (hh:mm) 指定。例如：lmutil lmborrow adskflex 14-oct-2020 [14:00] 借用许可的到期时间为 2020 年 10 月 14 日下午 2:00。

使用命令行工具停止借用许可

停止借用许可时，启动时将不再自动使用借用的许可。

此操作不会返还借用的许可。

要停止在 Windows 上借用许可，请打开 LMTOOLS 实用程序并切换到“借用”(Borrowing)选项卡。单击“今天不再借用”(Don’t Borrow Anymore Today)。

要停止在 macOS 或 Linux 上借用许可，请打开终端并将目录更改为 lmutil 的安装位置，然后执行以下命令： 

lmutil lmborrow -clear

使用命令行工具提前返还借用的许可

借用的许可将按借用许可时指定的日期和时间自动返还给网络许可服务器。但是，您也可以在此日期之前手动返还许可。

注： 必须连接到网络许可服务器才能返还借用的许可。

要在 Windows 上返还借用的许可，请打开 LMTOOLS 实用程序，然后切换到“借用”(Borrowing)选项卡。单击“提前返还借用的许可”(Return Borrowed Licenses Early)。

要在 macOS 或 Linux 上提前返还借用的许可，请打开终端并将目录更改为 lmutil 的安装位置，然后执行以下命令： 

lmutil lmborrow -return
