提高变形器节点性能

可以更改依存关系图节点解算性能，以便在拖动鼠标后、仅在告知场景刷新时或仅在释放鼠标按钮按钮时使场景刷新。更改解算性能可提高场景显示速度（如果有许多复杂对象发生变形）。

更改依存关系图节点的性能

  1. 打开“性能设置”(Performance Settings)窗口。

    打开“性能设置”(Performance Settings)

      注： “性能设置”(Performance Settings)窗口只能通过 MEL 命令或“搜索”(Search)字段打开：
    1. 按 Ctrl + F，然后在“搜索”(Search)字段中键入“性能设置”(Performance Settings)。
    2. 从脚本编辑器(Script Editor)调用此 MEL 命令 PerformanceSettingsWindow；
  2. 在“性能设置”(Performance Settings)窗口中，请注意“依存关系图解算”(Dependency Graph Evaluation)部分。
  3. 启用其中一个“刷新条件”(Refresh On)选项。
  4. 单击“关闭”(Close)。

更改变形器的性能设置

  1. 打开“性能设置”(Performance Settings)窗口。

    打开“性能设置”(Performance Settings)

      注： “性能设置”(Performance Settings)窗口只能通过 MEL 命令或“搜索”(Search)字段打开：
    1. 按 Ctrl + F，然后在“搜索”(Search)字段中键入“性能设置”(Performance Settings)。
    2. 从脚本编辑器(Script Editor)调用此 MEL 命令 PerformanceSettingsWindow；
  2. 在“性能设置”(Performance Settings)窗口中，请注意“变形器”(Deformers)部分。

    可以“启用”(on)和“禁用”(off)以下项的性能或将其转化为“交互式”(Interactive)：雕刻影响对象（“雕刻”(Sculpts)）、晶格影响对象（“晶格”(Lattices)）、线条影响对象（“线条”(Wires)）、融合变形和簇。可将“簇分辨率”(Cluster Resolution)设置为“逐节点”(Per Node)、“全局”(Global)或“交互式”(Interactive)。可将“晶格分辨率”(Lattice Resolution)设置为“逐节点”(Per Node)、“全局”(Global)或“交互式”(Interactive)。

  3. 单击“关闭”(Close)。

变形对象历史中的节点比未变形对象历史中的节点多，因此可以在使用变形器完成操作后删除对象历史，以此提高变形性能。

删除对象的历史

  1. 选择对象。
  2. 选择“编辑 > 按类型删除 > 历史”(Edit > Delete by Type > History)。

在某些对象的变形链中，如果将一个（或多个）变形器的“变形顺序”(Deformation order)设置为“平行”(Parallel)，则可以使用通道盒以平行方式融合链中变形器的影响。

注：

在每个变形器的高级变形器选项中，可以将“变形顺序”(Deformation order)设置为“平行”(Parallel)。

使用通道盒(Channel Box)编辑通道

  1. 选择平行融合器节点（默认名称：parallelBlendern）。

    一种快速选择融合变形变形器节点的方法是：选择要变形的对象，然后从“通道盒”(Channel Box)（位于“输入”(INPUTS)下方）中选择其历史中的平行融合器节点。

  2. 将当前不显示为可设置关键帧属性（通道）的任何通道添加到“通道盒”(Channel Box)。请参见使通道可设置关键帧或不可设置关键帧。
  3. 在“通道盒”(Channel Box)中，单击要编辑的通道的名称。
  4. 在场景中，使用鼠标中键左右拖动。通过移动鼠标，可以交互方式更改选定通道的值。移动鼠标时，请注意按 Ctrl 键可以提供更精细的控制，按 Shift 键可提供更粗略的控制。
