validate_skeleton

通过检查骨架是否符合骨架模式的要求来验证骨架。

检查是否存在 array<uint> 类型的 joint_parent_index 几何特性以及 array<double4x4> 类型的 joint_pivot_matrix。

检查这两个几何特性数据的大小是否与 point_component 的 count 匹配。

检查 joint_parent_index 数据数组中每个值的完整性。如果父索引指向当前关节之前的位置，则该索引有效。如果关节是根或是隔离的，则其父索引值必须等于 invalid_index 节点返回的值。以下代码段包含了对每个 joint_parent_index 值的验证。

valid = joint_parent_index[i] < i || joint_parent_index[i] == 4294967295

输入

skeleton

要验证的骨架。

输出

valid

如果 skeleton 有效，则返回 true。如果无效，则返回 false。

error_description

valid 为 false（表示骨架未通过验证）时，包含错误描述的 string。
