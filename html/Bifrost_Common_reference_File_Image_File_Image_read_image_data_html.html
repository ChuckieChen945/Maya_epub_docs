read_image_data

从磁盘读取图像。

输入

filepath

将读取的图像文件的路径。

file_color_space

文件的颜色空间。如果保留为默认空字符串，则节点将尝试使用 OpenColorIO 自动确定颜色空间。像素数据将从此颜色空间转化为场景相关的“工作颜色空间”（也称为“渲染颜色空间”）。使用“raw”可忽略任何颜色空间转化。

输出

success

如果成功读取文件，则输出 true，否则输出 false。

pixels

图像的像素数据，转化为浮点 RGBA 值，并存储在 1D 数组中。数组的第一个元素对应于图像左下角的像素。数组的第二个元素是第一个像素右侧的像素，依此类推。这称为行主序。数组的最后一个元素对应于图像右上角的像素。

例如，如果图像宽度为 3 像素，高度为 2 像素，则每个像素的 pixels 数组中的索引如下所示：

┌──────┬──────┬──────┐
│      │      │      │
│  3   │  4   │  5   │
│      │      │      │
├──────┼──────┼──────┤
│      │      │      │
│  0   │  1   │  2   │
│      │      │      │
└──────┴──────┴──────┘

width

以像素为单位的图像宽度。

height

以像素为单位的图像高度。
