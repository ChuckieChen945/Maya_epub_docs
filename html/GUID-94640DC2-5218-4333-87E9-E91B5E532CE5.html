重命名组

若要重命名组节点，请选择组节点，然后单击其“组”(Group) 图标以显示组网络。在组网络视图中，选择组节点，然后在 ShaderfxShader“属性编辑器”(Attribute Editor)中，将 Name 字段中的文字替换为组的新名称。

若要重命名组节点的分类（Class name、Category name、Sub menu name），以便重命名您在其中创建组节点的标记菜单项，则必须先重新保存具有新 Class name、Category name 和/或 Sub menu name 的组，然后删除旧组。有关更多信息，请参见删除组。
