set_strands_pos_from_root_data

创建从 root_positions 开始并沿 root_normals 指定的方向延伸的股。

输入

strands

有效的股几何体。请注意，不能使用空对象。

root_positions

float3 值数组，用于指定股的起始位置。

root_normals

float3 值的匹配数组，用于指定对应于每个根位置的股的长度和方向。这些值不需要表示几何法线。当 length 为 1 时，每个股的结束位置为 root_positions + root_normals。

length

root_positions 向量的倍增。

输出

out_strands

生成的股对象。

point_position

股对象的修改点位置，由与股的结束位置交错的根位置组成。
