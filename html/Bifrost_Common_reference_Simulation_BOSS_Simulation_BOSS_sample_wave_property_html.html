sample_wave_property

此节点可用于在一组传入位置对波模拟输出进行采样。

输入

point_position

这是要对波图像进行采样的一组位置。波特性沿 Y 轴向下投影，因此仅使用位置的输入 XZ。

wave_image

这是要采样的波对象。这通常是来自 simulate_dynamic_waves 或 simulate_spectral_waves 节点的 out_wave。

sampler

这是在几何体位置对波图像进行采样时在波图像数组上使用的插值方法。

  • Nearest：最靠近点的像素：无插值

  • Linear：在最近点之间线性插值

  • Quadratic：更平滑地插值

  • Bspline：最平滑的插值

default

此项确定要采样的特性的数据类型及其默认值。

property

要采样的 wave_image 对象的几何特性的名称。典型特性为 displacement (float3)、displacement_derivatives (float3)、slope (float2) 和 velocity (float3)。“default”类型应设置为与要采样的特性匹配。（请注意，仅在模拟节点上启用了 slope 和 velocity 的输出时，这些特性才可用）

输出

sampled_data

这是数组输出数据。大小将匹配输入 point_position 数组的大小，类型由 default 特性类型确定。
