创建组合目标形状

您可以创建由一个或多个其他目标形状驱动的新目标形状。这样，您可以在执行面部动画等操作时更好地控制精细形状。例如，某个角色面部的微笑目标形状本身可能会非常逼真，但与其他目标形状组合使用时可能表现欠佳。在这种情况下，您可以添加新的目标形状，用于增强或更正组合使用时的目标形状。

组合目标形状由其他目标形状驱动。因此，当您编辑起作用的目标形状（驱动者）的形状或权重值时，组合目标形状也会更改以体现这一点。事实上，它没有自己的权重值，您无法直接更改。

如果您已拥有要使用的目标形状，则可以将该形状设置为组合目标形状。

创建组合目标形状

  1. 在形变编辑器(Shape Editor)中，选择一个或多个属于同一融合形变变形器的目标形状。这些目标形状将成为驱动者目标形状。

    您也可以选择一组或多组目标形状。

  2. 选择“创建 > 添加组合目标”(Create > Add Combination Target)，或者在其中一个选定目标形状上单击鼠标右键，然后选择“添加组合目标”(Add Combination Targey)。

    3. 将创建新的目标形状，其旁边将显示驱动者目标形状的名称。根据使用的方法(Method)，其权重（显示为黄色）将由起作用的目标形状驱动。

使用现有的目标形状作为组合目标形状

  1. 在形变编辑器(Shape Editor)中，选择一个或多个属于同一融合形变变形器的目标形状。这些目标形状将成为驱动者目标形状。

    您也可以选择一组或多组目标形状。

  2. 按 Shift 键并将要用作组合的目标形状添加到该组合（它必须是最后选择的形状）。
  3. 选择“形状 > 设置为组合目标”(Shape > Set as Combination Target)，或在其中一个选定目标形状上单击鼠标右键，然后选择“设置为组合目标”(Set as Combination Target)。

    最后选择的目标形状将成为组合目标形状，且其他选定目标形状将成为驱动者。其名称显示在组合目标形状旁边。当起作用的目标形状权重设置为 1（100% 激活）时，新目标形状的权重（显示为黄色）将由此类形状驱动。

使用现有网格作为组合目标形状

  1. 选择一个或多个网格作为驱动者目标形状。
  2. 选择要用作组合目标的网格。
  3. 在形变编辑器(Shape Editor)中，创建一个新的融合形变变形器或选择一个现有的融合形变变形器。
  4. 转到“创建 > 添加当前选择作为组合目标”(Create > Add Selection as Combination Target)，或者在选定的融合形变变形器上单击鼠标右键，然后选择“添加当前选择作为组合目标”(Add Selection as Combination Target)。

    Maya 将组合目标形状添加到该融合形变变形器，并将其他选定网格作为驱动者。其名称显示在组合目标形状旁边。当起作用的目标形状权重设置为 1（100% 激活）时，新目标形状的权重（显示为黄色）将由此类形状驱动。

选择驱动者目标形状

  1. 在形变编辑器(Shape Editor)中，选择组合目标形状。
  2. 在该形状上单击鼠标右键并选择“选择组合驱动者”(Select Combination Driver)。

添加驱动者目标形状

  1. 在形变编辑器(Shape Editor)中，选择一个或多个要添加到组合目标形状的驱动者目标形状。
  2. 按 Ctrl 键并将所需的组合目标添加到当前选择。
  3. 在当前选择上单击鼠标右键并选择“添加当前选择作为组合驱动者”(Add Selection as Combination Driver)，可将驱动者目标形状添加到组合目标的驱动者列表中。

移除驱动者目标形状

  1. 在形变编辑器(Shape Editor)中，选择一个或多个要移除的驱动者目标形状。
  2. 按 Shift 键或 Ctrl 键并将所需的组合目标形状添加到当前选择。
  3. 在当前选择上单击鼠标右键并选择“从组合驱动者中删除选择”(Remove Selection from Combination Drivers)，可从组合目标的驱动者列表中移除驱动者目标形状。
