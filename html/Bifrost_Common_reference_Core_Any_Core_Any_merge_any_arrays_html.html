merge_any_arrays

将类型为 any 的值的数组（每个值包含相同具体类型的数组）合并为包含该类型的一个数组的单个 any 值。在编写用于操纵数组的算法时，这非常有用，无需知道数组中保存的数据的类型。

输入

any_arrays

any 值的数组。每个 any 都应包含相同类型的数组。

concatenate

将所有数组连接在一起。如果为 true，则不使用 indices 输入。

indices

指定源数组中要复制到输出数组的元素的数组。此数组中的每个元素都是一对索引：每对中的第一个索引确定要选择的输入数组，第二个索引确定要选择该输入数组的哪个元素。如果索引超出范围，将为该元素返回默认值。

输出

success

指示合并操作是否成功。如果输入不是数组或不是相同类型的数组，则操作可能会失败。

any_array_values

包含合并数组的 any 值。
