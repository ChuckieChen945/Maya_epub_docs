时间编辑器中用户定义的重影(User Defined Ghost)窗口

打开用户定义的重影(User Defined Ghost)窗口

  • 在时间编辑器中，在片段上单击鼠标右键，然后选择“重影 > 设置/编辑用户定义的重影”(Ghost > Set/Edit User Defined Ghosts)
  • 从时间编辑器 时间编辑器菜单栏中，选择“片段 > 重影 > 设置/编辑用户定义的重影”(Clip > Ghost > Set/Edit User Defined Ghosts)

使用“用户定义的重影”(User Defined Ghost)窗口可设置默认“时间编辑器”(Time Editor)片段重影的替代方法。

要用于重影的对象(Objects to be used for ghosting)列表
您在视口或大纲视图(Outliner)中选择的对象将显示在此列表中。这些对象用于定义片段的新重影对象。
Add
用于将选定对象添加到“要用于重影的对象”(Objects to be used for ghosting)列表中。
带子对象添加(Add with children)
用于将选定对象的任何子对象合并到“要用于重影的对象”(Objects to be used for ghosting)列表中。例如，如果选择腕部控制器，则单击“带子对象添加”(Add with children)将包含附加的手指。
移除(Remove)
从“要用于重影的对象”(Objects to be used for ghosting)列表中删除任何亮显的对象。
包含由片段驱动的对象(Include objects driven by clip)
包含按照片段设置动画的所有对象，例如，由约束驱动的对象。
