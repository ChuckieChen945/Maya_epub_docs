特殊复制选项(Duplicate Special Options)

使用这些选项可控制在选择“编辑 > 特殊复制”(Edit > Duplicate Special)时会发生的状况。若要查看或更改它们，请单击“编辑 > 特殊复制”(Edit > Duplicate Special)右侧的选项框。另请参见复制。

几何体类型(Geometry Type)

选择希望如何复制选定对象。

复制

创建要被复制的几何体副本。

实例

创建要被复制的几何体实例。创建实例时，并不是创建选定几何体的实际副本。相反，Maya 会重新显示实例化的几何体。

下方分组

将对象分组到以下对象下：

父对象(Parent)

将选定对象分组到层次中这些对象的最低公用父对象之下。

World

将选定对象分组到世界（层次顶级）下。

新组

为副本新建组节点。

智能变换(Smart Transform)

启用“智能变换”(Smart Transform)后，当复制和变换对象的单一副本或实例时（无需更改选择），Maya 可将相同的变换应用至选定副本的全部后续副本。

平移、旋转、缩放

为 X、Y 和 Z 指定偏移值。Maya 将这些值应用至复制的几何体。可以定位、缩放或旋转对象，就如 Maya 复制对象一样。

注：

“平移”(Translate)和“旋转”(Rotate)的默认值为 0.0000。“缩放”(Scale)的默认值为 1.0000。使用默认值，Maya 会将副本放置在原始几何体的顶部。可以指定转换、旋转和缩放的偏移值（正值或负值的浮点），稍后会将这些值应用至复制的几何体。

副本数

指定要创建的副本数。范围从 1 到 1000。

提示：

对启用智能变换的对象进行复制的快捷方式：“编辑 > 复制并变换”(Edit > Duplicate with Transform)。

复制输入图形

启用此选项后，可以强制对全部引导至选定对象的上游节点进行复制。上游节点是指为选定节点提供内容的所有相连节点。

例如，如果 A、B 和 C 是连接至 D 的上游节点...

A > B > C > D

...当选择 D 并使用“复制输入图表”(Duplicate Input Graph)选项时，结果图将如下所示：

A1 > B1 > C1 > D1

（其中 A1、B1、C1 和 D1 是 A、B、C 和 D 的副本。）

复制输入连接

启用此选项后，除了复制选定节点，也会对为选定节点提供内容的相连节点进行复制。

例如，如果 A、B 和 C 相连，且为 C 提供内容...

A > B > C

...当选择 C 并使用“复制输入连接”(Duplicate Input Connections)选项时，结果图将如下所示：

A > B > C 以及

A > B > C1

（其中 C1 是 C 的副本）。

实例叶节点

对除叶节点之外的整个节点层次进行复制，而将叶节点实例化至原始层次。新菜单项改进了现有实例菜单项，在新菜单项中可以将全部非叶节点的动力学属性正确复制至新的层次。

为子节点指定唯一名称(Assign Unique Name to Child Nodes)

复制层次时会重命名子节点。
