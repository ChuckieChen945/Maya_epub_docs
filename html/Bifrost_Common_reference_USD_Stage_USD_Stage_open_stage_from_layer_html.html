open_stage_from_layer

从给定层打开后台文件。

输入

root_layer

将作为后台文件中的根的层。

mask

仅加载指定的基本体。如果为空，则加载所有基本体。

load

控制 USD 有效负载圆弧的行为。LoadAll 将加载所有可载入的基本体，而 LoadNone 则会记录（但不会遍历）有效负载圆弧（在大型场景中非常有用，可以在不拖动所有内容的情况下覆盖某些内容）。

layer_index

要设置为后台文件的 EditTarget 的子层索引。子层列表中的最后一个元素是 Pixar USD 根层中最强的子层。如果子层索引为 -1 或者它未标识现有子层，则根层将设置为打开的后台文件的 EditTarget。

输出

stage

USD 后台文件。
