菜单集编辑器(Menu Set Editor)

若要打开“菜单集编辑器”(Menu Set Editor)，请在菜单集下拉列表中选择“自定义”(Customize)。（位于状态行（工具栏）上。）

在“菜单集编辑器”(Menu Set Editor)中，可以创建、编辑、重命名、移除和还原菜单集。通过从中心列中移除菜单（菜单集中的菜单），或者从右列添加新菜单（所有菜单），也可以自定义菜单集。有关详细说明，请参见管理自定义菜单集。

“菜单集编辑器”(Menu Set Editor)菜单

“编辑”(Edit)菜单

新建菜单集(New Menu Set)

创建新的菜单集。

重命名(Rename)

重命名当前选定的菜单集。

移除(Remove)

移除当前选定的菜单集。

将 <菜单集> 还原为默认值(Revert <Menu Set> to Default)

将菜单集还原为其默认状态。如果当前在“菜单集”(Menu Sets)列中选择的菜单集是默认菜单集（“建模”(Modeling)、“绑定”(Rigging)、“动画”(Animation)、“FX”和“渲染”(Rendering)）之一，则此选项可用。

还原默认菜单集(Restore Default Menu Sets)

还原默认菜单集（“建模”(Modeling)、“绑定”(Rigging)、“动画”(Animation)、“FX”和“渲染”(Rendering)），同时保留已添加的任何自定义菜单集。
