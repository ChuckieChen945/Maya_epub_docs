“编辑影响物”(Edit Influences)菜单

本主题将介绍以下所有 4 个编辑影响物选项：
  • 蒙皮 > 编辑影响物> 添加影响物(Skin > Edit Influences > Add Influence) >
  • 蒙皮 > 编辑影响物> 移除影响物(Skin > Edit Influences > Remove Influence) >
  • 蒙皮 > 编辑影响物> 设置最大影响物...(Skin > Edit Influences > Set Max Influences...)
  • 蒙皮 > 编辑影响物> 移除未使用的影响物(Skin > Edit Influences > Remove Unused Influences) >

“添加影响物”(Add Influence)选项

几何体(Geometry)
如果您希望影响对象的形状及其变换属性（平移、旋转和缩放）影响蒙皮形状，可启用“使用几何体”(Use Geometry)。如果您仅希望影响对象的变换属性（平移、旋转和缩放）影响蒙皮形状，可禁用“使用几何体”(Use Geometry)。默认情况下“使用几何体”(Use Geometry)处于启用状态。
衰减(Dropoff)
指定随着与影响对象之间的距离增加其位置影响衰减的速率。指定介于 0.1 与 100 之间的值。默认值为 4.0。
多边形平滑度(Polygon Smoothness)
指定平滑蒙皮点与给定的多边形影响对象的匹配精确度。值越大，变形效果越圆润。设置介于 0.0 与 50.0 之间的值。默认值为 0.0。
NURBS 采样数(NURBS Samples)
指定用于对 NURBS 影响对象的图形影响解算的采样数。采样数越大，平滑蒙皮就越接近影响对象的图形。设置介于 1 与 100 之间的值。默认值为 10。
权重锁定(Weight Locking)
指定希望避免影响对象的权重被间接更改，这种情况通常是由于权重绘制和编辑期间的权重归一化引起的（请参见锁定平滑蒙皮权重）。Maya 将权重保持为“默认权重”(Default Weight)。默认设置为“禁用”。
默认权重(Default Weight)
指定“权重保持”(Weight Holding)处于启用状态时的默认保持权重。默认值为 0.000。
蒙皮簇(Skin Cluster)

“蒙皮簇”(Skin Cluster)列表包含与当前选定几何体关联的所有蒙皮簇

“蒙皮簇”(Skin Cluster)区域可用于在应用了多个蒙皮簇的几何体上选择单个蒙皮簇。
“蒙皮簇”(Skin Cluster)列表
此处显示选定几何体上的蒙皮簇。
注： 如果未找到蒙皮簇，则此列表将保留为空。选择几何体以更新该列表。
最近添加的蒙皮簇将显示在列表的顶部。

“移除影响物”(Remove Influence)选项

从蒙皮中移除当前影响对象的影响。移除影响并不会从场景中删除当前影响对象。

设置最大影响物...(Set Max Influences...)

最大影响(Max Influences)
指定可影响平滑蒙皮几何体上每个蒙皮点的关节和影响对象数量。还可以指定“衰减速率”(Dropoff/Falloff Rate)来限制关节影响物的范围。
使用滑块在 0 到 30 之间选择值。默认值为 5，这将为大多数角色生成良好的平滑蒙皮结果。

“移除未使用的影响物”(Remove Unused Influences)选项

从其平滑蒙皮断开蒙皮权重为 0 的所有关节和影响对象。请参见从平滑蒙皮曲面移除未使用的影响物。
