generate_sample_locations

在几何体对象上生成随机位置。

输入

geometry

要从其生成位置的对象。

sample_type

如何生成位置：surface 在网格曲面上或沿股的各分段生成位置，point 在网格、股或点对象的各点处生成位置。volume 选项不受支持。

amount

控制生成的位置数（可能受 amount_type 和 rate_control 影响）。负数相当于零。

amount_type

控制 amount 参数的解释：number 表示生成 amount 个位置，density 表示在每单位曲面面积或股长度生成 amount 个位置。

mask_control

控制如何应用遮罩：exact_rate 表示生成相同总数的位置，但受限于未遮罩的区域，而 mask_by_property 表示根据遮罩区域按比例减少总数。

seed

控制随机序列。此值与 additional_seeds 一起进行哈希处理。

additional_seeds

与 seed 一起进行哈希处理的一组种子值，用于确定最终随机序列。

enable_max_number

将每次执行节点时生成的位置数钳制于 max_number 值。

max_number

每次执行要生成的最大位置数。负数相当于零。

enable_mask

允许对具有几何体点上存在的特性的位置进行遮罩。特性值介于 0 和 1 之间，用于确定某个位置在某个区域中的相对概率。类型为 float 和 double 的遮罩值直接控制概率。向量视为颜色：RGB 值的解算方式为 average(R, G, B)，RGBA 值的解算方式为 A × average(R, G, B)。

mask_property

要用作遮罩的特性的名称。

invert_mask

反转遮罩值。

输出

sample_locations

生成的采样位置。
