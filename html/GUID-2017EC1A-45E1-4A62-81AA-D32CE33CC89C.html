绘制集成员身份工具(Paint Set Membership Tool)

  • 在“建模”(Modeling) (F2)、“绑定”(Rigging) (F3) 和“动画”(Animation) (F4) 菜单集中：“变形 >变形器集(旧版) > 绘制成员身份”(Deform > Deformer Sets (Legacy) > Paint Membership) >

您可以使用“绘制集成员身份工具”(Paint Set Membership Tool)，通过绘制希望添加到集、传递到集或从集中移除的点，修改可变形对象的哪些点（例如，CV 或顶点）属于多个变形器集。请参见绘制变形器集成员身份。请参见 Maya®Artisan™ 手册中的编辑变形器集成员身份和 Artisan 笔刷工具的工作方式。

在“绘制操作”(Paint Operations)部分中，有一些“绘制集成员身份工具”(Paint Set Membership Tool)所特有的属性。在下面介绍这些特有属性。有关所有其他部分中所有其他属性的描述，请参见 Artisan 手册中的 Artisan 工具设置。

“绘制集成员身份工具”(Paint Set Membership tool)设置

绘制操作(Paint Operations)

以下是“绘制操作”(Paint Operations)部分中属性的描述。

添加(Add)

如果对象包含多个变形器集，“添加”(Add)操作会将绘制的 CV 或顶点保留在当前集中，并将其添加到选定集。

如果对象是刚性蒙皮对象，那么“添加”(Add)操作与“传递”(Transfer)操作执行相同的操作：将绘制的 CV 或顶点从当前集中移除，并将其添加到选定集。

传递(Transfer)

如果对象包含多个变形器集，“传递”(Transfer)操作会将绘制的 CV 或顶点从当前集中移除，并将其添加到选定集。

如果对象是刚性蒙皮对象，那么“传递”(Transfer)操作与“添加”(Add)操作执行相同的操作：将绘制的 CV 或顶点从当前集中移除，并将其添加到选定集。

移除(Remove)

“移除”(Remove)操作会将绘制的 CV 或顶点从它们所属的集中移除，从而使这些 CV 或顶点不受任何变形器或关节的影响。

选择要修改的集(Select Set To Modify)

单击要添加到、传递到或从中移除的集的名称。这只是选择集的一种方式。另外还有两种方式。有关详细信息，请参见删减变形器集成员身份。

要修改的集(Set To Modify)

所选择的集的名称会显示在该框中。
