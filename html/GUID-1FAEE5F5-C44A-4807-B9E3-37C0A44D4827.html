XGen 编辑器 “描述”(Descriptions)菜单

创建描述(Create Description)

打开“创建 XGen 描述”(Create XGen Description )窗口，可用于创建头发、毛发和实例几何体。

请参见使用修饰笔刷创建头发和毛发、使用样条线曲线创建头发或毛发和使用 XGen 实例化几何体。

复制描述(Duplicate Description)

为选定的描述创建副本（包括所有面片绑定和导向），然后将副本添加到同一集合作为原始描述。

将描述移动到集合(Move Description to Collection)

用于将所选描述从其当前集合移动到另一个集合。

选择描述(Select Description)

选择此选项会在“大纲视图”(Outliner)中亮显当前活动描述（在 XGen 编辑器中选定的描述）。

绑定面片(Bind Patches)

可以更改将哪些面绑定到选定描述的面片。有关绑定面片选项的信息，请参见 XGen“绑定面片”(Bind Patches)子菜单。

基于几何体可见性隐藏/显示面片

用于指定在绑定几何体被隐藏时，Maya 也隐藏绑定的面片。

例如，选择此选项可隐藏绑定到隐藏网格对象的面片。

设置偏离百分比(Set Stray Percentage)

打开“设置偏离百分比”(Set Stray Percentage)窗口，其中可以设置百分比值或表达式来指定哪些基本体可以考虑偏离。

偏离基本体是一个不由修改器、贴图、遮罩或属性值控制的基本体。

删除描述(Delete Description)

删除当前描述。
