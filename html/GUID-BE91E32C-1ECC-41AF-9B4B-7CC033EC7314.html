lineModifierShape 节点

在创建 Paint Effects 线修改器（在“建模”(Modeling)菜单集中的“生成 > 创建修改器”(Generate > Create Modifier)）时，会创建此节点。

属性编辑器中提供了以下 lineModifierShape 属性。

线修改器属性(Line Modifier Attributes)

形状(Shape)

确定线修改器的初始形状：“球体”(Sphere)或“立方体”(Cube)。可以变换此形状来满足您的需求。在选定形状的边界外不会有任何效果。“衰减”(Dropoff)曲线图属性是相对于形状定义的，因此衰减的右边缘是形状的外边缘，而左边缘是形状的中心。

宽度比例(Width Scale)

乘以当前的线宽度。值为 1.0 时没有任何效果，值为零时宽度将缩放到零。

宽度偏移(Width Offset)

添加到当前的线宽度上。值为 0.0 将没有任何效果，值为 1.0 会将宽度增加 1。

不透明度比例(Opacity Scale)

乘以当前的线不透明度。值为 1.0 将没有任何效果，值为零将不透明度缩放到零。

不透明度偏移(Opacity Offset)

添加到当前的线不透明度上。值为 0.0 将没有任何效果，值为 1.0 会使线完全不透明。

曲面偏移(Surface Offset)

添加到当前的线曲面偏移上。值为 0.0 将没有任何效果，值为 1.0 会将偏移增加 1.0。负值会钳制在实现偏移 0 时。

线延伸(Line Extend)

添加到当前的线延伸上。值为 0.0 将没有任何效果，值为 1.0 会将延伸增加 1.0。负值会钳制在实现延伸 0 时。

修改颜色(Modify Color)

如果启用此选项，则颜色值会基于衰减值与线颜色融合。

颜色(Color)

如果启用“修改颜色”(Modify Color)，则该颜色会基于衰减值融合到局部线颜色中。

衰减(Dropoff)

确定修改器边界内的效果的范围。如果衰减设定为恒定值 1.0（水平线），则将在衰减形状的边界内统一修改线，并在形状边缘上进行硬过渡。如果衰减的右边缘向下渐变到零，则过渡将是逐步的，不在形状边缘上进行调整，而是朝其中心增加调整。衰减曲线图属性是相对于形状定义的，因此衰减的右边缘是形状的外边缘，而左边缘是形状的中心。

衰减噪波(Dropoff Noise)

使用 3D 噪波调整衰减值。它可以为要调整的参数提供变化。例如，使用该属性，可以采用噪波方式随机化或摆动管宽度。

噪波频率(Noise Frequency)

控制“衰减噪波”(Dropoff Noise)的详细程度。较低的值产生较平滑的噪波。3D 噪波是相对于线修改器形状的局部空间定义的。

笔刷管属性(Brush Tube Attributes)

注： 若要任何“笔刷管属性”(Brush Tube Attributes)生效，必须使用在“管”(Tubes)笔刷设置中启用了管的“Paint Effects”笔刷。
力(Force)

充当推动管远离线修改器形状中心的力。正值朝远离中心的方向推动，而负值朝中心方向推动。例如，使用该选项推动“Paint Effects”树，就好像有风一样。

平行力(Directional Force)

充当沿线修改器形状的局部 X 轴推动管的力。

位移(Displacement)

充当推动管远离线修改器形状中心的置换。正值朝远离中心的方向推动，而负值朝中心方向推动。这类似于力属性，但管上后续的管分段不受偏移影响。

平行置换(Directional Displacement)

充当沿线修改器形状的局部 X 轴推动管的置换。这类似于平行力属性，但管上后续的管分段不受偏移影响。

管缩放(Tube Scale)

可用于缩放生成的“Paint Effects”管的长度和宽度。值为 1.0 将没有任何效果，值为 0.0 会将管缩放到消失，从而导致不生成任何管。该行为类似于修改笔刷的“全局比例”(Global Scale)。

管修剪(Tube Dropout)

可用于减少生成的“Paint Effects”管的数量。值为 0.0 将没有任何效果，值为 1.0 将导致不生成任何管。

叶缩放(Leaf Scale)

乘以“Paint Effects”笔刷使用的叶的大小。值为 1.0 将没有任何效果，值为 0.5 会将叶大小减半。

叶修剪(Leaf Dropout)

添加到“Paint Effects”笔刷使用的当前叶信息缺失值上。值为 0.0 时没有任何效果，值为 1.0 时会将修剪增加 1.0。

花缩放(Flower Scale)

乘以“Paint Effects”笔刷使用的花的大小。值为 1.0 将没有任何效果，值为 0.5 会将花大小减半。

花修剪(Flower Dropout)

该值添加到“Paint Effects”笔刷使用的当前花信息缺失值上。值为 0.0 时没有任何效果，值为 1.0 时会将修剪增加 1.0。

分支修剪(Branch Dropout)

该值添加到“Paint Effects”笔刷使用的当前分支信息缺失值上。值为 0.0 时没有任何效果，值为 1.0 时会将修剪增加 1.0。

细枝修剪(Twig Dropout)

该值添加到“Paint Effects”笔刷使用的当前分支信息缺失值上。值为 0.0 时没有任何效果，值为 1.0 时会将修剪增加 1.0。

占用体积(Occupation Volume)

“占用体积”(Occupation Volume)包括一组相互依赖的属性，这些属性在影响分支大小和密度的同时还会影响线修改器的 3D 形状内分支生长的方向。

使用“占用体积”(Occupation Volume)基于分支、藤本植物和灌木丛的自然生长样式创建树叶效果。

占用吸引力(Occupy Attraction)

在线修改器体积中的点上创建吸引最近“Paint Effects”管端点的吸引力。吸引力会促使管朝体积中的点生长，直到管占用了点周围的区域。在管端点生长到体积点后，它会占用此空间，点不再吸引其他管端点。

“占用吸引力”(Occupy Attraction)值越高，管端点越倾向于朝着最近的点生长。当力增加时，随着管端点迁移到的可用空间渐渐减少，分支管的角度锐度会更大。

“吸引半径比例”(Attract Radius Scale)和“占用半径比例”(Occupy Radius Scale)属性值以及“占用分支终止”(Occupy Branch Termination)会影响“占用吸引力”(Occupy Attraction)力对管端点的影响方式。

吸引半径比例(Attract Radius Scale)

指定与“Paint Effects”管半径相乘以确定吸引管端点的非占用空间区域的值。例如，设置为 1 时，吸引管端点的区域等于管的半径。“吸引半径偏移”(Attract Radius Offset)也会影响吸引半径的计算方式。

“吸引半径比例”(Attract Radius Scale)值增加时，管端点被吸引到的点数也将随之增加。这样，增长到未占用空间的分支就会减少。当“吸引半径比例”(Attract Radius Scale)值大于“占用半径比例”(Occupy Radius Scale)时，较小分支的终止频率将减小。

吸引半径偏移(Attract Radius Offset)

指定将与“Paint Effects”管的半径相加以确定吸引管端点的未占用空间区域的值。“吸引半径比例”(Attract Radius Scale)还会影响吸引半径的大小。

占用半径比例(Occupy Radius Scale)

指定将与“Paint Effe ts” 的半径相乘以确定当管被吸引到某一点时所占用的空间区域的值。

“占用半径比例”(Occupy Radius Scale)值越低，大量分支管可以占用的空间就越多。如果增加“占用半径比例”(Occupy Radius Scale)同时将“占用半径偏移”(Occupy Radius Offset)保持在 0 值附近，则较细分支所占用的空间将与较大、较粗分支所占用的空间相同。

当“占用半径比例”(Occupy Radius Scale)值大于“吸引半径比例”(Attraction Radius Scale)时，较小分支的终止频率将增加。

占用半径偏移(Occupy Radius Offset)

指定将与管半径相加以确定当管被吸引到某一点时所占用的空间区域的值。“占用半径比例”(Occupy Radius Scale)还会影响占用半径的大小。

占用栅格分辨率(Occupy Grid Resolution)

指定线修改器的“占用体积”(Occupation Volume)所用的体素点栅格分辨率。该值将确定世界空间中沿线修改器“形状”(Shape)的最长轴分布的点数量。

“占用栅格分辨率”(Occupy Grid Resolution)增加后，吸引管端点的点总数量也会增加。

占用分支终止(Occupy Branch Termination)

如果启用此选项，将终止附加到已占用空间的分支。
