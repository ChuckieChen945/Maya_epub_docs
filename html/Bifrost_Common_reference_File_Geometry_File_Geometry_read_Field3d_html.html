read_Field3d

此节点从 Field3D 文件读取 Bifrost 体积。

输入

filename

文件名。使用 # 包含输入 frame 编号。# 的出现次数确定填充量。例如，file####.f3d 解析为 file0001.f3d、file0002.f3d 等。

directory

目录。

frame

帧编号。您可以将 time 节点的 frame 输出连接到 to_long 节点，然后在此处连接结果。

field_name

要读取的 Field3D 字段的列表（以空格分隔）。使用 *（星号）指定所有字段。

properties

要读取的特性的列表（以空格分隔）。使用 *（星号）指定所有特性。

volume_subdivision_structure

输出 Bifrost 体积的体积细分结构。

resample

如果为 true，则重新对体积采样以移除剩余平移。

输出

residual_translation

将体积世界空间坐标与存储的 Field3D 变换对齐所需的偏移。

volume

输出 Bifrost 体积。
