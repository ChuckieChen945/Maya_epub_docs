transfer_properties_by_weighted_sum

使用加权总和传递几何特性，以将几何特性从源几何体线性插值到目标几何体。如果特性类型不受支持（请参见 sample_any_array_by_weighted_sum），将传递最近的索引特性。

输入

target_geometry

几何特性将被复制到的目标几何体。

source_geometry

将从中复制几何特性的源几何体。

component

要传递几何特性的几何组件。

property_expression

将传递与此表达式匹配的所有特性。

block_schema

此模式中的所有特性将不会被传递。当所有需要传递的特性都是自定义特性时，这将非常有用。例如，在点几何体之间传递特性时，可以通过在此处连接点原型对象（通过 create_object_from_schema 创建的对象）来跳过 point_position 特性。

always_transfer_expression

与此表达式匹配的任何特性将始终被传递，即使它与 block_schema 对象中的特性匹配也是如此。

indices

此数组指定源组件和目标组件之间的映射。此数组包含的元素数应与指定 component 的 target_geometry 的组件数相同。每个元素都是源组件的索引。例如，如果源几何体包含 5 个点，而目标几何体包含 3 个点（分别对应于源几何体的 0、2 和 4），则此数组应包含 [0, 2, 4]

weights

要在加权总和中使用的 4 个权重组成的数组。权重总和不必为 1，可以为负数，也不必介于 [0,1] 之间。

输出

target_with_properties

包含已传递特性的目标几何体。

resampled_properties

重采样的几何特性对象，以允许单独使用它们。

property_names

resampled_properties 中已传递特性的名称。
