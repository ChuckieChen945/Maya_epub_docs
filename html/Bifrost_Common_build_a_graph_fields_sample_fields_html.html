对场进行采样

创建场后，必须对其进行采样。这意味着，在一个或多个位置解算其值，以便可以在计算中使用这些值。

  • 使用 sample_field 对标量场或向量场进行解算。连接 float3 或 array<float3> 以指定位置 - 通常，这是某个几何对象的 point_position 特性。

  • 使用 sample_field_with_proxies 对代理场进行解算。除了场和位置，还必须将基于点的几何体或体积几何体与代理场使用的特性连接起来。

    当处理的对象包含非常多的点，并且希望场只影响少数点时，可能需要在将 point_position 数组连接到 sample_field_with_proxies 之前对其进行过滤，以提高性能。在这种情况下，可以从 point_position 数组中移除不需要的值，并将包含其余索引的数组连接到 proxy_property_indices。
