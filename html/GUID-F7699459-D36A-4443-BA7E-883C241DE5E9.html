UV 平铺

UV 布局是制作模型纹理的重要部分。UV 映射操作会自动将 UV 适配在默认的 0 到 1 UV 纹理空间中，但使用纹理贴图可能导致难以将细节均匀分布到模型的每个区域中。您可以在默认纹理空间外使用 UV 平铺，将不同纹理应用于每个平铺，以最大程度提高纹理的分辨率。

多平铺 UV 纹理

您也可以使用在其他应用程序（包括 Mudbox、ZBrush 和 Mari）中创建的多平铺 UV 纹理。加载多平铺 UV 纹理时，UV 编辑器将自动检测平铺数，并使用原始栅格平铺布局。每个平铺均使用其相应纹理文件的名称标记。请参见通过单个纹理节点加载多平铺 UV 纹理。

提示： 若要优化 UV 布局，您可以使用“UV 工具包”(UV Toolkit)的“变换”(Transform)部分中的新控件轻松偏移壳和 UV。请参见移动、旋转和缩放 UV。
