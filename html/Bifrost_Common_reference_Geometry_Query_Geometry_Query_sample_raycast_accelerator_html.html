sample_raycast_accelerator

使用以 build_raycast_accelerator 节点为输入构建的加速度结构，返回光线投射到几何体上的最近交点的位置。

可以构建一次加速器，然后在图表中的多个地方使用。但是，如果在相应节点的下游修改几何体，加速器不会自动更新。要查询修改后的几何体，请构建新的加速器。

输入

accelerator

来自 build_raycast_accelerator 节点的 accelerator 输出。

position

光线的原点。

direction

光线的方向。

proximity_method

用于确定交点的接近度方法：semi-line 沿 direction 向量射出无限长的光线；line 沿正和负 direction 向量射出无限长的光线；segment 沿 direction 向量射出光线，但受向量的长度限制。

reverse_direction

反转光线的方向。适用于 segments 和 semi-line。

use_cutoff_distance

仅在指定的 cutoff_distance 中搜索。

cutoff_distance

要在其中搜索的距离。

min_distance

在该范围之外进行搜索的最小距离。距离该位置的距离小于此距离的交点将被忽略。这可用于在从几何体向自身进行光线投射时避免不重要的自相交点。

输出

location

包含位置的数组。可以将其插入到 sample_property 节点的位置端口。

found

如果找到有效的交点，则返回 true，否则返回 false。如果为 false，则相应的位置无效，如果使用它对几何体进行采样，将直接返回默认值。这可能是因为中止距离内没有位置，也可能是因为几何体不受支持。
