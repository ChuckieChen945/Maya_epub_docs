get_face_edge_vertices

获取指定 face_edge 的顶点索引。请注意，面边的方向是从较低的顶点索引指向较高的顶点索引。

例如：

 p0─────────────p3
 │      s2       │
 │             ▲ │
 │             │ │
 │s3    f0   s1│ │
 │             │ │
 │             │ │
 │      s0       │
 p1─────────────p2

指代侧边 s1 的面边的 first_vertex 和 second_vertex 分别等于 p2 和 p3。

输入

face_edge

要查询的面边。

face_vertex

相应网格的面顶点数组。

face_offset

相应网格的面偏移数组。

输出

first_vertex

面边的第一个顶点。

second_vertex

面边的第二个顶点。
