sample_wave_data

此项可用于对波模拟进行采样以及计算一组传入位置的特性。请注意，仅当在波解算器上启用了 output_slope、horizontal_displacement 或 output_velocity 时，才会提供某些输出。尤其是，必须同时启用 output_slope 和 horizontal_displacement 才能访问除 displacement 和 velocity 以外的内容。

输入

point_position

这是要对波图像进行采样的一组位置。波特性沿 Y 轴向下投影，因此仅使用位置的输入 XZ。

wave_image

这是要采样的波对象。这通常是来自 simulate_dynamic_wave 或 create_spectral_waves 节点的 out_wave。

sampler

这是在几何体位置对波图像进行采样时在波图像数组上使用的插值方法。

  • Nearest：最靠近点的像素：无插值

  • Linear：在最近点之间线性插值

  • Quadratic：更平滑地插值

  • Bspline：最平滑的插值

输出

输出全都返回数组大小将与输入 point_position 数组的大小匹配的数组。

displacement

这是由位移值组成的 float3 数组。波峰是从（水平）x 和 z 位移值得到的。

velocity

这是由波速度值组成的 float3 数组（以单位数/秒表示）。仅当在波解算器上启用了 output_velocity 时，此项才可用。

normal

这是表示波曲面法线的 float3 数组，但请注意，这是水平位移之前的值，因此忽略波峰影响。通常，与使用此项设置法线相比，从输出网格计算法线可以获得更好的法线。仅当在波解算器上启用了 output_slope 时，此项才可用。

slope

这是表示波的 xz 坡度的 float2 数组。仅当在波解算器上启用了 output_slope 和 horizontal_displacement 时，此项才可用。

min_eigen_value

这是表示从波位移向量派生的最小特征值的浮点数组。

max_eigen_value

这是表示从波位移向量派生的最大特征值的浮点数组。

derivatives

这是由位移导数组成的 float3 数组。xyz 向量分别表示 XX、XZ 和 ZZ 导数。仅当在解算器上同时启用了 output_slope 和 horizontal_displacement 时，才会计算导数。

jacobian

这是表示水平位移的雅可比行列式的浮点数组。它可用于确定波峰可能自相交的位置。
