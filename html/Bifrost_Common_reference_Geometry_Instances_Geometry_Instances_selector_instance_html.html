selector_instance

根据某个逐点特性或一组 ID 从多个几何体或实例形状中选择一个进行实例化。输出可以连接到 set_instance_geometry 或 set_instance_shape 节点。

也可以将多个 selector_instance 节点呈菊花链状连接到一起以实现特定目的，例如，根据一个标准在松树与枝叶茂密的树之间进行选择，然后根据另一个标准选择每种类型的特定版本。

输入

instance_shapes

要实例化的一个或多个对象或者对象数组。它们可以是普通几何对象，也可以是实例形状，例如，selector_instance、geo_instance 或 render_archive_instance 等节点的输出。

selector_property

用于指定逐点实例化哪个实例形状的点对象的 long 特性的名称，其中 0 表示 instance_shapes 数组的第一个元素，1 表示第二个元素，依此类推。

preview_geometry

要用于在视口中进行预览但不用于渲染的可选几何对象。如果在此处连接了某个对象，它将覆盖已通过 geo_instance 或类似节点为连接到 instance_shapes 的对象设置的任何预览几何体。

wrap_out_of_range

包裹超过实例形状的数组大小的实例 ID 值。

name

可选标识符。

transform

要应用于实例化几何体的变换矩阵。

输出

selector_instance

用于选择实例形状的对象。
