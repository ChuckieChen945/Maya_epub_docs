compute_transform_tree

计算解释为展平的变换树的 array<Transform>，并计算其所有变换矩阵分量。

输入

parent_indices

包含父索引值的 array<long>，定义变换父对象在输入数组中的位置。如果索引为负，则从辅助 parent_transforms 数组中选择父对象。

变换的排序相对灵活，但必须确保变换的父对象在数据数组中的位置位于变换之前。

如果变换是根或是隔离的，则其父索引值必须等于 invalid_index 节点返回的值。

pivot_matrices

在世界空间中定义变换的静止状态的枢轴矩阵。它定义缩放、旋转和平移所用的空间。

这通常是静态输入。

operator_matrices

包含变换动画的运算符矩阵，包括变换中的缩放、旋转和平移操作。

这通常是动画输入。

parent_transforms

变换树的辅助父对象变换（如果有）。

可以在主变换计算中使用辅助父对象，方法是在 parent_indices 中使用负索引值。值为 -1 时将使用第一个辅助父对象，值为 -2 时将使用第二个辅助父对象，依此类推。

输出

transforms

结果 array<Transform>，解释为展平的变换树。
