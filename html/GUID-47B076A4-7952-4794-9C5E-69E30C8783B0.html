设置命令行渲染

渲染命令的路径

渲染命令位于下列位置：

  • (Windows) C:\Program Files\Autodesk\<版本>\bin
  • (macOS) /Applications/Autodesk/<版本>/Maya.app/Contents/bin/
  • (Linux) /usr/autodesk/<版本>/bin

其中，<版本> 表示 Maya 版本；例如：2019

若要在 Linux 上使用命令进行渲染，您必须将路径作为命令的一部分包含于其中。

在 Windows 上，您可以将路径作为命令的一部分包含于其中或设定系统路径以包含路径。在 macOS 上，您可以将路径作为命令的一部分包含于其中，或使用 Maya 终端窗口。有关针对 Windows 和 macOS 的命令行渲染的详细信息，请参见下列章节。

针对 Windows 设置命令行渲染

如果要从 Windows 命令行提示符运行渲染命令，且无需每次输入命令路径，则可以将命令路径添加到系统的 Path 环境变量。

设定 PATH 环境变量

  1. 打开“控制面板”(Control Panel)，然后双击“系统”(System)图标。
  2. 在显示的“系统属性”(System Properties)窗口中，切换到“高级”(Advanced)选项卡。
  3. 单击“环境变量”(Environment Variables)。
  4. 在“环境变量”(Environment Variables)窗口中，选择“Path”变量并单击“编辑”(Edit)。
  5. 在显示的“编辑系统变量”(Edit System Variable)窗口中，通过添加 C:\Program Files\Autodesk\<版本>\bin 编辑“变量值”(Variable value)。
    注： 该字段中的变量用分号隔开。

现在您可以使用 Windows 命令提示运行渲染命令，而不必输入命令路径。

针对 macOS 设置命令行渲染

您可以使用macOS 终端应用程序 (/Applications/Utilities/Terminal) 从命令行渲染。

渲染可执行文件位于以下文件夹中： 

/Applications/Autodesk/maya202x/Maya.app/Contents/MacOS/Render

以前，渲染可执行文件位于 /Applications/Autodesk/maya202x/Maya.app/Contents/bin 中。如果您从此位置运行渲染命令，包裹器脚本将从 /Applications/Autodesk/maya202x/Maya.app/Contents/MacOS 执行并调用渲染命令。
