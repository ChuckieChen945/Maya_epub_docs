set_Arnold_geo_settings

设置 geometry 的 Arnold 特定设置，以便在 Arnold 中作为 Arnold 原生设置应用。这些设置由所有 Arnold 几何体类型共享。这些设置不影响视口，仅影响 Arnold 渲染。

这些设置与 Arnold 中的对应选项匹配。有关详细信息，请参见 Arnold 用户手册中的 polygons、curves、particles 和 volume 相关信息。

按光线类型的可见性

在 Arnold 中，可见性按光线类型进行控制。这些按光线类型标志编码为位掩码，并采用 Arnold 原生格式进行设置。有关详细信息，请参见 Arnold 用户手册。

光线类型包括 camera_rays、indirect_diffuse_rays、indirect_specular_rays、indirect_volume_rays、refracted_rays、shadow_rays、subsurface_scattering_rays、transmitted_diffuse_rays

在 Arnold 中，其他可见性设置（例如公开为 double_sided_by_ray_type 的 sidedness）也是使用按光线类型位掩码进行控制的，但某些内置的 Bifrost 节点仅公开布尔值。要使用位掩码按光线类型控制这些选项，可以使用 Arnold_ray_bitmask 节点直接设置 Arnold 参数。

不透明度

在 geometry 上设置 opaque 标志。

运动模糊

有关 transform_type 的信息，请参见 Arnold 文档。

蒙版

在 geometry 上设置 matte 标志。

体积

许多几何体都可以渲染为体积，而不仅仅是体积。这会在几何体上设置体积设置 step_size 和 volume_padding。

另请参见 set_Arnold_volume_settings。

其他

设置其他不太常用的设置，包括 id、invert_normals、receive_shadows、self_shadows。请参见 Arnold 文档。
