Mac OS X 特定变量

MAYA_MAC_SET_UMASK
该环境变量可用于设定文件共享权限。以下示例将 Maya 所创建文件的权限设定为任何用户可读/可写：
MAYA_MAC_SET_UMASK = 0
QT_MAC_WANTS_BEST_RESOLUTION_OPENGL_SURFACE
将此环境变量设置为 0 可禁用高 DPI 支持，实现此支持是为了解决 4K 显示器上的像素失真，以便您可以在高 DPI 屏幕上以本地分辨率进行渲染：
QT_MAC_WANTS_BEST_RESOLUTION_OPENGL_SURFACE = 0

将此变量设置为 0 会影响视口、曲线图编辑器、Paint Effects 和其他基于 3D 视图/GL 的构件。
