基于摄影机的映射选项(Camera-Based Mapping options)

以当前摄影机视图作为平面投影，为选定对象创建 UV 纹理坐标。也就是说，摄影机视图成为投影的平面。

使用这些选项可设置在选择“UV > 基于摄影机”(UV > Camera-Based)时发生的情况。

设置

创建新 UV 集(Create new UV set)

启用此选项可创建新 UV 集并将新创建的 UV 放置在该集中。在“UV 集名称”(UV Set Name)框中键入 UV 集的名称。
