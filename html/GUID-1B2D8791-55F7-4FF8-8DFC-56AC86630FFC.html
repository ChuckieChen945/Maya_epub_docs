支持 GPU 的变形器

为了提高播放性能，Maya 将某些变形器的处理卸载到图形处理单元 (GPU)。
注： 虽然 GPU 支持仅适用于 Viewport 2.0，但在许多情况下，通过 GPU 支持更新变形器使旧版 CPU 变形器的性能略有提升。

默认情况下，“GPU 覆盖”(GPU Override)在 Maya 中处于活动状态，但如果其处于禁用状态，可以在“动画”(Animation)首选项（在“设置”(Settings)下）中启用它。

以下变形器具有 GPU 支持：

  • blendShape
  • 簇
  • deltaMush
  • groupParts
  • jiggle
  • 晶格
  • 变形
  • 非线性变形器
  • 接近度包裹
  • 雕刻（请参见下面的注释）
  • 固化
  • skinCluster
  • softMod（请参见下面的注释）
  • 张力
  • 调整
  • 线条
注：
  • GPU 支持仅适用于隐式球体，即，当雕刻对象几何体是球体时，Maya 将雕刻变形器加入使用 GPU 运行的变形链中。如果雕刻几何体不是球体，则变形将发送到 CPU。
  • 如果要对几何体子集使用 softMod 变形器，则仅当“软修改”(Soft Modification)的“衰减模式”(Falloff Mode)设置为“体积”(Volume)时，GPU 支持才可用。

GPU 支持和选择集

使用子集时张力、Delta Mush 和固化变形器的行为已更改，以支持 GPU。

在 2021 版之前的 Maya 版本中，如果在使用几何体的子集时使用这些变形器，将按一个多边形连接级别在内部展开选择集，并且相应顶点会被指定为锚点。

在新行为中，变形器仅使用子集内的顶点。如果要恢复原始变形器行为，请使用组件标记展开选择集。
