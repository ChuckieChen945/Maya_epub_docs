保存 UV 布局的图像

在 UV 编辑器中创建 UV 最终布局后，可以使用“UV 快照”(UV Snapshot)导出 UV 壳的图像。UV 快照按照用户定义的分辨率为 UV 编辑器的 2D 视图保存位图图像。可以使用 Maya Paint Effects ™ 画布或图像编辑应用程序（如 Adobe® Photoshop®）中的图像并绘制与 UV 匹配的纹理。

“UV 快照”(UV Snapshot)可用于多个选定的对象，并可延伸快照区域以覆盖选定 UV 纹理坐标的整个范围，不论 UV 属于多边形还是细分曲面类型。通过“UV 范围”(UV Range)选项可以指定要为输出图像捕捉的 UV 编辑器视图的区域。

使用 UV 快照保存图像

  1. 在场景视图中选择网格或组件，以便其 UV 在 UV 编辑器中显示。
  2. 在“UV 编辑器”(UV Editor)中，单击“视图选项”(View options)栏中的“保存图像”(Save image)按钮 ()，或选择“图像 > UV 快照”(Image > UV Snapshot)。
  3. 在“文件名”(File name)字段中输入用于导出的名称。
  4. 使用“大小 X”(Size X)和“大小 Y”(Size Y)字段设定图像大小。
  5. 为 UV 壳选择“颜色值”(Color value)。

    除非指定，否则图像的背景色为黑色。

  6. 选择“图像格式”(Image format)。
  7. 设定“UV 范围选项”(UV Range Options)。
  8. 单击“确定”(OK)。
