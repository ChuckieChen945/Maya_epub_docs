set_point_shape

设置用于显示和渲染点的形状，例如来自粒子系统。

输入

points

点对象。

shape

要使用的形状。

注意：

目前，Numeric 仅适用于 char、uchar、short、ushort、int、uint、long、ulong、float、double、float2 和 float3 类型。

screen_aligned

使形状朝向摄影机。

numeric_property

不起作用。

default_size

形状的尺寸。如果点组件上存在 point_size 特性，此值将被覆盖。

输出

out_points

修改后的对象。
