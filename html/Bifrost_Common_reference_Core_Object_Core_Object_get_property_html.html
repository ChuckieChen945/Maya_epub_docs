get_property

从对象获取特性。请注意，特性是键值对，其中，键是名称（字符串），值几乎可以是任何内容。在很多情况下，值可以是另一个对象。

通常，您可能应该使用 get_property 以外的节点。例如，如果要获取几何对象上的 point_size 等特性的值，则应改用 get_geo_property。对于某些常用特性，可能还有特定节点，例如 get_point_position。

输入

object

几何对象。

default_and_type

连接类型与特性的值相同的 value 节点。如果特性不存在或特性的类型不同，它也是作为输出返回的值。

key

特性的名称。

输出

value

特性的值，如果失败，则为 default_and_type。
