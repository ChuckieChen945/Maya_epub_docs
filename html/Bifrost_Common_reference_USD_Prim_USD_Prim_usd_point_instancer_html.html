usd_point_instancer

此节点创建点实例化器。

输入

stage

要在其中创建点实例化器的后台文件。

prim_path

点实例化器的路径。

prototypes

包含所有实例原型的路径的数组。这些是正在实例化的基本体。

protoindices

包含所有实例原型索引的数组。

positions

包含所有实例位置的数组。此数组的大小必须与原型索引的大小相同。

orientations

包含所有实例方向的数组。此数组的大小必须与原型索引相同或为空。警告：USD 数组使用半精度浮点数而不是单精度浮点数。Bifrost 使用单精度浮点数，因此可能会有一些精度损失。

scales

包含所有实例比例的数组。此数组的大小必须与原型索引相同或为空。

velocities

包含所有实例速度的数组。此数组的大小必须与原型索引相同或为空。

accelerations

包含所有实例加速度的数组。此数组的大小必须与原型索引相同或为空。

angular_velocities

包含所有实例角速度的数组。此数组的大小必须与原型索引相同或为空。

invisible_ids

要在求值时隐藏的 ID 列表。

输出

out_stage

修改后的 USD 后台文件。

success

指示操作是否成功的布尔值。
