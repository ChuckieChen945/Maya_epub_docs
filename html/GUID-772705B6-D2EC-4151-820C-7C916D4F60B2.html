创建 Bezier 曲线

创建 Bezier 曲线

  1. 选择“创建 > 曲线工具 > Bezier 曲线工具”(Create > Curve Tools > Bezier Curve Tool)或单击“曲线/曲面”(Curves/Surfaces)工具架中的“Bezier 曲线工具”(Bezier Curve Tool)图标 ()。
  2. 执行下列操作之一：
    • 若要放置定位点，请单击场景。
    • 若要放置定位点并操纵其切线，请在场景上单击并拖动鼠标。
    • 若要将定位点添加到现有曲线，同时保持其形状，请单击曲线上的任意位置。
    • 若要删除锚点，请选择它，然后按 Delete 键。
    • 若要闭合曲线，请按住 Ctrl+Shift 键并单击曲线的第一个锚点。

  3. 如果选择另一个工具，然后选择该曲线并返回到“Bezier 曲线工具”(Bezier Curve Tool)，可以将其他定位点附加到最近创建的定位点。

  4. 完成放置锚点后，请按 Enter 键结束该曲线并退出“Bezier 曲线工具”(Bezier Curve Tool)。如果要绘制更多曲线，还可以按“g”来结束当前曲线，但仍处于该工具中。

将定位点附加到现有曲线的末端

  1. 选择要附加定位点的曲线。
  2. 选择“创建 > Bezier 曲线工具”(Create > Bezier Curve Tool)或单击“曲线”(Curves)工具架中的“Bezier 曲线工具”(Bezier Curve Tool)图标 ()。
  3. 单击曲线中的最后一个定位点。

    最后一个定位点始终为曲线的端点，它不是黄色。

  4. 单击（或单击并拖动）场景中的任何位置。

    将显示定位点，并会将该定位点连接到曲线。
