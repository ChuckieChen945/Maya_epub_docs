smooth_level_set_property

此节点使用指定的过滤器平滑名称为“property”的等值面曲面特性。假定曲面特性是一个表示为带符号距离场的等值面。在过滤期间，通过随着曲面移动调整空间分辨率来跟踪等值面曲面。

输入

property

特性的名称

mode

应用的过滤器类型

iterations

应用过滤器的迭代次数

voxel_standard_deviation

高斯过滤器的标准偏差（以体素为单位）

voxel_filter_width

过滤器的宽度（以体素为单位）

grid_retopology

如果启用，栅格将重新拓扑以跟随曲面。如果禁用，将保留栅格拓扑。这可能会显著提高速度并将保留体素遮罩，但由于粗化，某些区域中的细节可能较少或出现瑕疵。

mask_field

用作遮罩的场。

mask_interpretation

如何解释遮罩（特性或场）。如果 mask_interpretation 为 FogDensity，则遮罩将被钳制在 0 和 1 之间。如果 mask_interpretation 为 SignedDistanceField，则低于 0 的值（内部）将映射到遮罩值 1，高于 0 的值（外部）将映射到遮罩值 0。

mask_property

可用于在输入体积上指定特性，以用作遮罩。如果还指定了场遮罩，则使用该场遮罩。

volume

包含特性的体积（对象）。
