“折痕集编辑器”(Crease Set Editor)列表和操作按钮

折痕集列表

当您在场景中选择对象/组件时，包含对象/组件的折痕集将在折痕集列表中亮显。若要将命令应用于某个折痕集，必须先在折痕集列表中选择该折痕集。

提示： 单击任意列表标题可按升序或降序对折痕集进行排序。
集(Set)
折痕集名称。双击“集”(Set)字段以重命名折痕集。
折痕(Crease)
应用于集中的组件的硬度级别。若要编辑硬度级别，请双击“折痕”(Crease)字段，然后手动更改值或按住鼠标中键并在字段中拖动以增加或减小值。
成员(Members)
折痕集中折痕组件数量。
选定(Selected)
属于折痕集的选定折痕组件数量。

“折痕集编辑器”(Crease Set Editor)操作按钮

新建(New)
创建包含选定组件（多边形的边和顶点）的新折痕集。如果某些选定组件属于另一个折痕集，它们将被移动到新的折痕集。
添加(Add)
将选定组件添加到当前折痕集。如果任何选定组件属于另一个折痕集，它们将被移动到当前折痕集。
删除(Remove)
从当前折痕集移除选定组件。
选择(Select)
在场景中选择折痕集的成员。如果场景中某些选定组件不属于当前折痕集，将改为选中这些组件与当前折痕集的交集。
删除(Delete)
从当前折痕集移除选定组件。
