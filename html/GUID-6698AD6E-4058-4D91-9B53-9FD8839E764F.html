编辑现有工具动画演示

  1. 将 ToolClips.xml 文件从 builtin.toolclips\Control 目录复制到 user.toolclips\Control 目录。请参见工具动画演示目录。

    您必须将 XML 文件复制到 user.toolclips 目录，因为您在 builtin.toolclips 目录中所做的任何编辑都将被软件修补程序和升级覆盖。

  2. 在您选择的文本或 XML 编辑器中，打开复制的 ToolClips.xml 文件。
    注： 如果您计划添加采用双字节编码的字符，则建议您使用支持 UTF-8 格式的编辑器。确保文件使用该指定的编码进行保存。
  3. 编辑文件以自定义现有工具动画演示。请参见工具动画演示 XML 文件的组成部分。

    例如，您可以编辑视频、图像和指向自定义内容的链接。

    提示： 每个工具动画演示都使用 <toolclip> 元素进行定义。移除任何您不想修改的 <toolclip> 元素，以便使用原始的工具动画演示数据。
  4. 保存 XML 文件。
  5. 将您在 XML 文件中指定的任何新资源添加到 user.toolclips\Resources 目录。
  6. 启动 Maya，以查看工具动画演示。
