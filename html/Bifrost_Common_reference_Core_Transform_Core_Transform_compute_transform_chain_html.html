compute_transform_chain

计算解释为从父对象到子对象的连续链的 array<Transform>，并计算其所有变换矩阵分量。

输入枢轴和运算符矩阵中的 array<double4x4> 分量应按从父对象到子对象的顺序排序。

输入

pivot_matrices

在世界空间中定义变换的静止状态的枢轴矩阵。它定义缩放、旋转和平移所用的空间。

这通常是静态输入。

operator_matrices

包含变换动画的运算符矩阵，包括变换中的缩放、旋转和平移操作。

这通常是动画输入。

parent_transform

数组中第一个 Transform 的父对象变换。如果未连接任何对象，则此项将默认为单位变换。

输出

transforms

结果 array<Transform>，解释为从父对象到子对象的连续链。

父页面: Transform
