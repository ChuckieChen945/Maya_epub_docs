cache_fluidShape（液体缓存文件）

cache_fluidShape 节点仅可用于设定流体的初始状态，以及播放使用 Maya 2008（或更早版本）创建的流体缓存。请参见 cacheFile。

流体缓存属性(Fluid Cache Attributes)

缓存文件名(Cache File Name)

指定流体缓存文件的名称（非 nCache）。

当前流体具有“每帧一个文件”(One File per Frame) (OFpF) 或“分段”(Segmented)缓存时，此字段为只读。

开始时间(Start Time)

显示当前（非 nCache）流体缓存的开始时间。该字段是只读的。

结束时间(End Time)

显示当前（非 nCache）流体缓存的结束时间。如果流体缓存是“每帧一个文件”(One File per Frame)，则流体缓存的“开始时间”(Start Time)和“结束时间”(End Times)是相同的。该字段是只读的。

采样类型(Sampling Type)

过采样和欠采样指定 Maya 计算每帧的流体模拟的频率。这些字段为只读。

请参见 nCache 附加到缓存。

采样率(Sampling Rate)

“过采样”或“欠采样”的整数值。该字段是只读的。

缓存当前时间的内容

显示哪些流体的属性已包含在当前（非 nCache）磁盘缓存文件中。这些复选框为只读。

缓存节点(Caching Node)

磁盘缓存(Disk Cache)

显示当前（非 nCache）流体磁盘缓存文件的名称。该字段是只读的。

本地复制缓存文件(Copy Cache File Locally)

可以启用或禁用创建任意（非 nCache）流体播放缓存文件的本地副本。如果缺少磁盘空间，或者正在使用网络进行分布式渲染，此功能非常有用。

重要： 当“本地复制缓存文件”(Copy Cache File Locally)属性处于禁用状态时，您不能添加或截断流体缓存。
