vary_source_property

此复合可用于设置模拟源上的逐点值。例如，可以使用此复合来改变粒子模拟中生成的点的大小。

此复合可以随机改变源特性，也可以使用场景中输入几何体上存在的颜色集来改变源特性。使用 randomize_value 和 use_color_set 选项来选择要使用的方法。如果同时启用这两个选项，则贴图将是随机化的倍数。如果两个选项均未启用，则将使用源节点上的值。要使用颜色集，请从主场景中指定其名称。

应在输入几何体和源节点之间连接此复合。它适用于所有源上的大多数 float、float3 和 float4 特性。当用于给定的源特性时，将忽略该特性的源节点上的值。可以将任意数量的 vary_source_property 节点呈菊花链状连接在一起，以使用不同的设置改变特性。

输入

geometry

连接一个或多个几何对象以充当源。这些可以是点或网格。

常规

这些设置适用于所有方法。

enable_variation

关闭该选项可禁用变化。

property

显示在源上的要改变的源特性的名称。可以通过输入多个特性名称（用空格分隔）指定多个特性。下面是完整列表：

  • source_particles：“rate”、“speed”、“direction”、“normal_speed”、“spread”、“inherit_velocity”、“bounciness”、“age_limit”、“size”。
  • source_air：“fog_density”、“temperature”、“inherit_velocity”、“initial_speed”、“initial_speed_direction”。
  • source_fuel：“ignition_temperature”。
  • source_mpm_sand：“volume_preservation”、“viscosity”、“vibration_speed”、“friction”、“cohesion”、“initial_speed”、“inherit_velocity”。
  • source_mpm_snow：“initial_firmness”、“hardening”、“volume_preservation”、“viscosity”、“vibration_speed”、“friction”、“cohesion”、“initial_speed”、“inherit_velocity”。
  • source_mpm_fluid：“surface_tension”、“viscoelasticity”、“relaxation_time”、“viscosity”、“vibration_speed”、“initial_speed”、“inherit_velocity”。
  • source_mpm_gel：“volume_preservation”、“viscosity”、“vibration_speed”、“initial_speed”、“inherit_velocity”、“yield_stress”。
  • make_mpm_cloth：“viscosity”、“direction”、“friction”、“initial_speed”、“inherit_velocity”。
  • make_mpm_shell：“viscosity”、“direction”、“initial_speed”、“inherit_velocity”。
  • make_mpm_fiber：“viscosity”、“direction”、“initial_speed”、“inherit_velocity”。

请注意，您可以在源复合中找到与源上的参数关联的特性的名称。例如，您会发现该大小实际上是 point_size。“size”或“point_size”都会改变粒子的大小。

Random Variation

这些设置适用于特性的随机化。

randomize_value

是否随机化特性值。

layer_mode

此项可确定变化的应用方式。

  • Replace：覆盖源节点上的当前值。
  • Multiply：乘以源节点上的当前值。如果变化范围为 0-2，则发射值将在源值的 0 到 2 倍内变化。当前不支持 Aero 发射。
  • Add：与源节点上的当前值相加。如果变化范围为 -0.1 到 0.1，则发射值将在源发射值加减 0.1 范围内变化。当前不支持 Aero 发射。

min

特性的最小值。该类型应与源特性上的类型匹配。例如，size 为 float，initial_speed_direction 为 float3。

max

特性的最大值。该类型应与源特性上的类型匹配。

bias

从 -1 到 1。值为 -1 时，将会有更多的 min 值。值为 0 时，min 值和 max 值的数量相当。值为 1 时，将会有更多的 max 值。

animated

随着时间的推移改变随机化。

seed

这是用于随机变化序列的基础种子。不同的值将产生不同的随机结果。

Color Set Variation

在特性上使用颜色集时，这些设置适用。只能与 float 特性一起使用。

use_color_set

是否使用颜色集特性值。

color_set_name

输入几何体上颜色集的名称。您可以添加观察点以查看输入几何体上颜色集的名称和类型。对 Alpha、RGB 或 RGBA 通道着色以将颜色集映射到浮点。

multiplier

总变化效果的最终比例。

输出

out_geometry

一组充当源的几何对象。
