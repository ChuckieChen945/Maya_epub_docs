This is a hotfix release. It addresses the following issues:

  • Fixed crashes and errors with denoised AOVs, both in interactive renders or in batch
  • Removed warnings appearing with Material Viewer
  • 8th layer was missing from aiLayerFloat interface
  • Fixed warning in ARV about progressive render
  • Alembic parameters namespace and nameprefix were missing from Attribute Editor
