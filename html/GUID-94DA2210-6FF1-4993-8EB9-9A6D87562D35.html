Dash

Dash 是通道盒中用于对相应值执行许多常见操作的内联脚本方法。
注： 确保先加载 MASH 插件，然后再运行 Dash 脚本编辑器。要检查是否已加载插件，请打开插件管理器(Plug-in Manager)（“窗口 > 设置/首选项 > 插件管理器”(Window > Settings/Preferences > Plug-in Manager)，然后找到 Mash.mll。）

激活 Dash 脚本编辑器

  1. 在通道盒中选择一个或多个属性。
  2. 按 Alt 键并在通道盒中的任意位置单击鼠标右键。

    Maya 会显示 Dash 脚本编辑器，您可以在其中输入 Dash 命令。

默认 Dash 命令

以下是可用 Dash 命令的列表。

命令 效果
l(number) 在由“number”表示的总距离上以线性方式分布选定对象（需要选择 2 个或更多对象）。
r(number) 指定介于 0 和“number”之间的随机数。
r(low, high) 指定介于“low”和“high”之间的随机数。
e() 自动简化动画通道。
e(number) 基于一个介于 -1 和 1 之间的数字简化动画通道的加速/减速。值 -1 表示缓慢加速后急剧减速，而值 o1 恰好相反。
ts(number) 按帧的“number”偏移选定属性的任何关键帧的时间。

除此之外，还可以使用 Python Random 模块中的任何内容。

自定义 Dash 命令

可以通过编辑位于 /Program Files/Autodesk/Maya<version>/plug-ins/MASH/scripts (Windows) 或 /Applications/Autodesk/Maya<version>/plug-ins/MASH/scripts (Mac) 中的 Dash.json 文件创建自己的 Dash 命令。
