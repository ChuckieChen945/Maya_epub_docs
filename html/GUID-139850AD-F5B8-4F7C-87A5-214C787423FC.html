Getting Started with LookdevX

Workflow overview

LookdevX supports both the MaterialX and USD data models. You can choose which data model to use from either the start screen of the LookdevX Graph Editor or when creating a new graph tab.

To learn material authoring with the MaterialX data model, see Overview of the MaterialX data model for LookdevX.

To learn material authoring with the USD data model, see Overview of the USD data model for LookdevX.

If you are not familiar with USD, visit the Working with USD for Material Authoring page for more information.
