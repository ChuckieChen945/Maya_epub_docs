points_to_transforms

将 point_position、point_orientation、point_scale 和 point_size 组合到一个变换矩阵中。生成的矩阵是将应用于放置在这些点上的实例的矩阵。

输入

points

要从其获取变换的点。

输出

point_transform

相当于将 point_position、point_orientation、point_scale 和 point_size 应用于某个位置的变换矩阵。

inverse_point_transform

与 point_transform 相反。
