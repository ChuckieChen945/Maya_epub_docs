将场景导出到 Unity

可以使用“文件 > 发送到 Unity”(File > Send to Unity)选项将场景从 Maya 导出到 Unity 游戏引擎，但是与 Autodesk 产品的“发送到”(Send To)选项不同，您必须先定义项目。

注： 如果看不到“文件 > 发送到 Unity”(File > Send To Unity)选项，请确保已加载 GamePipeline.mll 插件。

将场景导出到 Unity 项目

  1. 选择“文件 > 发送到 Unity > 设置 Unity 项目”(File > Send to Unity > Set Unity Project)，然后浏览以选择有效的本地 Unity 项目（项目目录），并单击“选择”(Select)。

    （您只需要设置项目一次。不需要为后续导出操作再次设置项目。）

  2. 选择“文件 > 发送到 Unity”(File > Send to Unity)，然后选择“全部”(All)（以导出整个场景）或“选择”(Selection)（以仅导出选定对象）。
  3. 在显示的“导出当前选择”(Export Selection)窗口中，输入名称以将当前选择作为 FBX 文件保存在 Unity 项目的“资产”(Assets)文件夹（默认）中，然后单击“导出当前选择”(Export Selection)。

    4. Maya 会将选定的对象或整个场景导出到 Unity 项目。如果已在 Unity 中打开项目，则可以立即访问 FBX 文件。若要在 Unity 场景中查看该文件，请将其从项目浏览器中的“资产”(Assets)文件夹拖放到场景视图中。
