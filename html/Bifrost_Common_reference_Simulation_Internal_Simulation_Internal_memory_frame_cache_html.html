memory_frame_cache

此复合包含 memory_cache 和 memory_file_cache 节点使用的端口反馈机制。此内存缓存构建块包含的逻辑决定了是否应将传递到节点的 Bifrost 对象的当前帧保存到反馈容器内的内存中。反馈容器存储 Bifrost 对象的各个帧，以便以后作为缓存帧序列的一部分进行访问。

输入

current_value

进入要缓存的复合的对象。

only_in_order

如果选中此选项，则可确保缓存的帧是连续的，从而确保它对模拟有效。

current_frame

当前帧。

输出

out_current_value

缓存的对象。

cache_hit

如果当前帧已缓存，则返回 true。

cachable_state

确定帧是有效还是需要重置。
