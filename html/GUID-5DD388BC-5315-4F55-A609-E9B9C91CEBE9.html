导入和导出编辑内容

可以导入和导出编辑文件以帮助在“摄影机序列器”(Camera Sequencer)中设置、编辑和细化快照序列。

导入编辑内容

  1. 从“摄影机序列器”(Camera Sequencer)的菜单栏中，选择下列选项之一：
    • 若要导入文件但不设定其他导入选项，请选择“文件 > 导入编辑”(File > Import Editorial)。在打开的“导入”(Import)浏览器中进行浏览，以选择要加载的 XML FCP 文件。
    • 若要导入文件并设定其他选项，请选择“文件 > 导入编辑”(File > Import Editorial) > 。
注： 导入编辑文件时，如果 Maya 使用编辑文件中指定的路径无法找到关联的影片或声音文件，则会在“影片”(Movie)和“声音”(Sound)项目位置搜索它们。这些位置在项目窗口(Project Window)中指定。例如，默认的“影片”(Movie)项目位置设定为默认 Maya 项目的“数据”(data)目录。

可以编辑“影片”(Movie)和“声音”(Sound)项目位置使其指向新位置，例如，如果存在具有指向 Mac OS X 系统上网络驱动器的路径的编辑文件，且希望将这些文件导入到 Windows 系统。请参见将文件组织到项目中。

导出编辑内容

  1. 从“摄影机序列器”(Camera Sequencer)的菜单栏中，选择下列选项之一：
    • 若要导出文件但不设定其他选项，请选择“文件 > 导出编辑”(File > Export Editorial)。
    • 若要导出文件并设定其他选项，请选择“文件 > 导出编辑”(File > Export Editorial) > 。
