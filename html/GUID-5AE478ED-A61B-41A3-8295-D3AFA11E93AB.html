创建带有 IK 控制柄的关节链

  1. 在“装备”(Rigging)菜单集（按 F3 键）中，选择“骨架 > 创建关节”(Skeleton > Create Joints) > 。

    “关节工具”(Joint Tool)设置面板将显示。

  2. 在“关节工具”(Joint Tool)设置中，启用“创建 IK 控制柄”(Create IK Handle)。

    这将使“IK 控制柄设置”(IK Handle Settings)在同一工具面板中变得可用。

  3. 根据需要设定“IK 控制柄设置”(IK Handle Settings)。

    例如，根据在操纵此关节链时要使用的 IK 解算类型，将“当前解算器”(Current solver)选项设置为“单链”(Single-Chain)或“旋转平面”(Rotate-Plane)。另请参见 IK 解算器。

  4. 关闭“关节工具”(Joint Tool)设置面板。

现在，当您使用“关节工具”(Joint Tool)创建关节链时，会自动使用关节链创建 IK 控制柄。

提示： 也可以为现有的关节链创建 IK 控制柄。请参见创建 IK 控制柄。
