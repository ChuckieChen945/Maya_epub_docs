初始化 Maya Python 环境

可以使用 userSetup.py 脚本使 Maya 在启动时运行特定命令。

userSetup.py 在 Maya 初始化时运行。它将运行其中的任何 Python 命令。它应只包含不依赖于 Maya 功能的命令。它可用于导入库并添加到系统路径。

您需要自己创建 userSetup.py，并添加希望 Maya 在启动时运行的命令，例如导入 maya.cmds 模块。
平台 userSetup.py 位置
Linux 和 macOS $MAYA_APP_DIR/<version>/scripts
Windows %MAYA_APP_DIR%/<version>/scripts
注： 从 Maya“脚本编辑器”(Script Editor)运行 getenv("MAYA_APP_DIR") 以获取 MAYA_APP_DIR 的值。有关 MAYA_APP_DIR 的详细信息，请参见文件路径变量。
