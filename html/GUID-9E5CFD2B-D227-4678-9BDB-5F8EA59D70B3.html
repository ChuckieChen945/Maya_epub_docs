Flow Retopology for Maya Release Notes & Installation

The release notes for each version of Flow Retopology describe known limitations and workarounds, plus other important information about each release. This topic provides general installation and other information that is general to all releases.

Flow Retopology for Maya is delivered as a standalone component that you can download from your Autodesk account.

System requirements

Flow Retopology is only available to Maya subscribers and can be installed on multiple computers. A valid Autodesk account and login identity is required. Flow Retopology for Maya is supported on systems capable of running Maya 2025.

Installing and uninstalling Flow Retopology for Maya

There are two methods for installing Flow Retopology for Maya:

Autodesk Desktop App:

  1. Open the Autodesk Desktop app.
  2. Log in to your account, if necessary.
  3. Choose the Maya category on the left-hand side.
  4. Scroll to select Flow Retopology for Maya.
  5. Click Install.

Autodesk Portal:

  1. Log into your Autodesk account at manage.autodesk.com.
  2. Navigate to the installer for the Flow Retopology for Maya capability from your account:
    1. Under the Products and Services tab, select Maya.
    2. Scroll to select your version of Maya.
    3. Under the Extensions tab, select View Details.
  3. Download the installer and manually install.

Uninstalling

You can uninstall the plug-in manually.

  1. In your operating system's search windows, type Programs.
  2. Select Add or Remove Programs.
  3. Select Flow Retopology for Maya.
  4. Click Uninstall.

Load and manage plug-ins

Ensure Flow Retopology for Maya is loaded with the Plug-in Manager:

  1. Select Windows > Settings/Preferences > Plug-in Manager > flowGraphEngineShared.py and flowRetopology.py > Loaded.
  2. You can also opt to Auto Load the plug-in for each Maya session. Make sure you have both checkboxes selected to auto load.
  3. Install Flow Wedging for use alongside Flow Retopology. Flow Wedging is a plug-in that ships with Bifrost for Maya that can help with wedging the output of a Bifrost graph.
Note: Flow Retopology for Maya uses Maya USD as an intermediary.
