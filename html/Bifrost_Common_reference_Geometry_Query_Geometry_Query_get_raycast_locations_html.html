get_raycast_locations

返回光线在几何体曲面上最近交点的位置。然后，可以使用 sample_property 获取这些位置上的几何体的特性的插值。

输入

geometry

要测试是否被光线命中的曲面。仅支持网格。

positions

光线的起始位置数组。

directions

光线的对应方向数组。

proximity_method

用于计算接近度的方法：

  • semi-line 沿 direction 向量射出无限长的光线。
  • line 沿正和负 direction 向量射出无限长的光线。
  • segment 沿 direction 向量射出光线，但受向量的长度限制。

reverse_direction

反转光线投射的方向。

use_cutoff_distance

仅在指定的 cutoff_distance 中搜索。

cutoff_distance

要在其中查找光线命中的距离。

min_distance

在该范围之外进行搜索的最小距离。距离该位置的距离小于此距离的交点将被忽略。这可用于在从几何体向自身进行光线投射时避免不重要的自相交点。

输出

locations

包含位置的数组。您可以将其插入到 sample_property 节点的 locations 端口。

found

布尔值的匹配数组。如果元素为 false，则相应的位置无效，如果使用它对几何体进行采样，将直接返回默认值。这可能是因为中止距离内没有位置、光线未命中几何体或几何体不受支持。
