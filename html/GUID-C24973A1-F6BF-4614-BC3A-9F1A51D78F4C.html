将外部 Python 库与 Maya Python 结合使用

如果要在 Python 脚本中使用 Maya 外部的库，则必须在启动 Maya 之前将这些库添加到 PYTHONPATH 环境变量，或者通过从 Python 解释器中调用 sys.path.append() 将它们添加到系统路径。

启动 Maya 之前，可以在Maya.env文件或环境中设置 PYTHONPATH 环境变量。如果已在 Maya.env 和系统环境变量中设置 PYTHONPATH，则在系统环境中设置的 PYTHONPATH 的值将始终优先于在Maya.env中设置的值。

Maya.env文件位于 Windows 上的C:\Users\<Username>\Documents\maya\<version_number>\、Linux上的$HOME/maya/<version_number>/和 macOS 上的$HOME/Library/Preferences/Autodesk/maya/<version_number>/中。

有关 PYTHONPATH 的详细信息，请参见文件路径变量。

若要从 Python 解释器中将库添加到 Python 系统路径，请使用 sys.path.append()
import sys sys.path.append( '<Path_To_Libraries>' )

您还可以在启动脚本（例如 userSetup.py）中添加对 sys.path.append() 的调用。
