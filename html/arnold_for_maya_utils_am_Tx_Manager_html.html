TX 管理器(TX Manager)

“Tx 管理器”(Tx Manager)窗口可以为同一源文件生成多个具有不同颜色空间的 tx 文件。

Tx 管理器可用于在 Maya 内自动执行将 Maya 纹理文件和 Ai 图像文件转化为 .tx 文件格式的过程。它可以转化当前场景中的纹理或某个文件夹（及其子文件夹）中的所有纹理。此外，您还可以指定是否要转化特定纹理。Tx 管理器可从 Arnold 菜单和 MtoA 工具架中访问。

注意：

有关使用 .tx 的详细信息，请访问 maketx 页面。

刷新(Refresh)

搜索当前场景中的可用纹理，且在“Tx 管理器”(Tx Manager)窗口中列出纹理路径。

全选(Select all)

选择场景中所有可用的纹理文件。

选择未分片对象(Select Untiled)

选择所有尚未处理为 .tx 文件的纹理。

选择缺少对象(Select Missing)

选择所有缺少文件的纹理。

重新链接...(Relink...)

用于将所有选定纹理替换为在选定文件夹及其子文件夹中找到的图像。此功能与原生纹理管理器的“重新链接”(Relink)功能相同。

选项(Options)

使用场景纹理(Use Scene Textures)

这将显示当前 Maya 场景内存在的文件纹理。

  • hastx = 原始状态，存在 tx。
  • missing = 缺少纹理或路径不正确。
  • onlytx = 仅 .tx 文件可用。

单击文件路径名可在属性编辑器中查看相应的纹理

使用文件夹纹理(Use Folder Textures)

使用“纹理文件夹”(Textures Folder)内存在的纹理文件。

场景选项(Scene Options)

设置用于在当前 Maya 场景中查找纹理的选项。

扫描属性(Scan attributes)

要扫描文件纹理路径的节点类型和属性的列表。

添加(Add)

添加要在场景中搜索的新节点类型和属性。

删除(Remove)

从列表中移除选定的属性。

TX 选项(TX Options)

允许您向转化过程添加参数。默认参数使用标志 -u（更新图像）和 --oiio (OpenImageIO)。

Update

仅当存在输入纹理的新版本时更新 .tx 文件。

详细(Verbose)

显示有关将纹理转化为 .tx 的信息。

检查 NaN(Check NaNs)

检查源纹理中是否存在 NaN 值，如果存在，则将发出警告。

打印统计信息(Print stats)

打印有关输出 .tx 文件的统计信息。

取消预乘(Unpremult)

在转化之前对输入纹理取消预乘。

线程数(Threads)

用于转化的 CPU 线程数。

预设(Preset)

选择选项的预定义预设，Arnold 始终会向 maketx 选项添加 --oiio。

附加参数(Extra args)

添加 maketx 从 maketx 选项接受的任何自定义参数。

预览

预览当前选定的参数。

默认选项(自动 TX)(Default Options (auto tx))

选中此选项将通过 MtoA 的“自动 TX”(auto-tx)选项使用默认值。

操作

为选定对象创建 .tx(Create .tx for Selected)

开始将纹理文件转化为 .tx 文件格式。如果取消，此过程将在当前文件处理之后结束。

为选定对象移除 .tx(Remove .tx for selected)

删除与选定纹理关联的所有 .tx 纹理文件。

  • 仅当在 MtoA“渲染设置”(Render Settings)窗口中的纹理(Textures)下启用“使用现有 .tx 纹理”(Use Existing .tx Textures)选项时，才会使用经过处理的纹理 (.tx)。

Tx 管理器支持的文件格式与 maketx 支持的文件格式相同：

  • Cineon
  • DDS
  • DPX
  • FITS
  • HDR/RGBE
  • ICO
  • IFF
  • JPEG
  • OpenEXR
  • PNG
  • PNM
  • PSD（展平）
  • RLA
  • SGI
  • Softimage PIC
  • Targa
  • TIFF
  • Zfile
