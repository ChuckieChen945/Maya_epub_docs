sample_closest_in_radius_accelerator

使用以 build_closest_accelerator 节点为输入构建的加速度结构，返回几何体上距给定位置在某个距离内的点的索引。

可以构建一次加速器，然后在图表中的多个地方使用。但是，如果在相应节点的下游修改几何体，加速器不会自动更新。要查询修改后的几何体，请构建新的加速器。

输入

accelerator

来自 build_closest_accelerator 节点的 accelerator 输出。

position

要在其中查找点的球体的中心。

radius

球体的半径。

enable_max_number

将每个球体返回的点数限于 max_number。

max_number

每个位置返回的最大点数。

输出

point_index

每个输入位置边界球体内点的点索引。索引按距球体中心的距离由近到远进行排序。
