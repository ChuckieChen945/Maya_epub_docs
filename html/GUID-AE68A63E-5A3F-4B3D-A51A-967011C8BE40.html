设定头发曲线位置

创建头发后，开始曲线会自动创建。如果已在“创建头发选项”(Create Hair Options)窗口中启用“创建静止曲线”(Create Rest Curves)，则也创建线“创建静止曲线”(Create Rest Curves)。如果禁用了该选项，可以随时使用“开始位置”(Start Position)或“当前位置”(Current Position)曲线创建“静止曲线”(Rest Curves)。播放模拟时，“当前位置”(Current Position)显示头发如何对动态作用力作出反应。

创建头发曲线时，“静止”(Rest)和“当前位置”(Current Position)曲线最初分别为棕色和蓝色，有助于区别“开始位置”(Start Position)曲线。可以使用这些曲线节点的“绘制覆盖”(Draw Overrides)来完成此操作。层也取决于“绘制覆盖”(Draw Overrides) ；因此，如果在层中创建或添加层，则现在所有头发曲线的颜色是相同的，即使随后从层中删除头发。

头发建模时，但仍可以多次重设开始或静止曲线位置。

设定开始曲线位置

  1. 选择要设定的曲线。
  2. 选择“nHair > 设置开始位置”(nHair > Set Start Position)，然后选择以下项之一：
    • “来自当前”(From Current)。此操作将“开始位置”(Start Position)设定为“当前位置”(Current Position)。
    • “从静止”(From Rest)。此操作将“开始位置”(Start Position)设定为“静止位置”(Rest Position)。

设定静止曲线位置

  1. 选择要设定的曲线。
  2. 选择“nHair > 设置静止位置”(Hair > Set Rest Position)，然后选择以下项之一：
    • “来自当前”(From Current)。此操作将“静止位置”(Rest Position)设定为“当前位置”(Current Position)。
    • “从开始”(From Start)。此操作将“静止位置”(Rest Position)设定为“静止位置”(Rest Position)。
