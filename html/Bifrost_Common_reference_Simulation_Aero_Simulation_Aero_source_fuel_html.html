source_fuel

指定将特性发射到模拟栅格的燃料源的参数。这应连接到 source_air 节点上的 additional_properties 输入。多个 source_fuel 节点可以连接到 additional_properties 串联输入端口，以产生更高级的燃料和氧化剂混合物。例如，通常用于实际 VFX 的 MAPP 气体包含丙炔、丙二烯、丙烷，在某些情况下还包含丙烯和丁烷。

输入

General

enable_fuel_source

启用此燃料源。

start_frame

开始从燃料源发射的帧。

use_end_frame

停止在特定帧发射。否则，将在模拟持续时间内继续发射。

end_frame

停止从源发射的帧。

Fuel Properties

fuel_type

可从下拉菜单中选择的燃料类型。对于一个特定 source_fuel 节点，只能指定一种类型的燃料，但可以将多个 source_fuel 节点呈菊花链状连接在一起。如果您已知燃料类型，这允许您匹配真实世界片段。否则，将此选项保留为默认值，并调整其他参数以获得所需的艺术效果。

ignition_temperature

燃料点燃时的温度（以摄氏度为单位）。如果低于点燃温度，则不会发生燃烧。请注意，每种类型的燃料都有真实世界点燃温度，但为了达到艺术目的，可以在此重新定义点燃温度。

提示：

要获得较大爆炸冲击波，请使用较低的点燃温度。

oxygen_percentage

氧化剂中的氧气百分比。例如，如果 oxygen_percentage 设置为 0.4，则氧化剂将包含 40% 的氧气和 60% 的氮气。通常，如果有足够的燃料，oxygen_percentage 越高，爆炸越剧烈，火焰传播速度越快，温度也越高。

默认情况下，周围环境包含空气（主要是氮气和氧气），因此，如果有足够的氧气扩散到燃料中，即使 oxygen_percentage 为 0，燃料也能够燃烧。

burn_rate

值介于 0 和 2 之间时，定义燃料和氧化剂混合物的化学计量。值为 0 时，会出现烟雾火焰，更具体地说，即仅发射燃料。值小于 1 表示燃料过多的情况，即在燃烧过程后会有多余燃料。值从 0 增加到 1 时，模拟将从烟雾弥漫的火焰逐渐转变为更清洁但燃烧也更剧烈的火焰。值为 1 表示完美的化学计量条件，即，所有可用的氧气和燃料充分发生化学反应后产生没有多余氧气和燃料的副产物。值大于 1 表示燃料不足的情况，即在燃烧后会有多余氧气。

输出

fuel_source

输出是一个对象，其中包含 Aero 解算器的燃料源相关输入集合，应连接到 simulate_aero 节点上的 sources 端口或 source_fuel 节点上的 air_source 端口。
