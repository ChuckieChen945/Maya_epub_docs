重新集合场渲染图像

场渲染图像是专为在 NTSC 设备上进行视频播放而设计的图像序列中的图像。每个场或图像仅包含半个全帧，并且每个场都存储在单独的文件中。设计将在 NTSC 设备上播放的动画时，以这种方式渲染图像非常有用。但是，若要在计算机屏幕上查看这些图像或将其传递到某些其他计算机程序，需要重新将其集合为完整图像。发生这种情况时，最终将有一半的文件，并且每个文件包含的数据翻倍。

  1. 从主菜单中选择“文件 > 新建序列”(File > New Sequence)。
  2. 在读取“偶场序列”(Even Field Sequence)的行上单击“定义”(Define)。此时将显示“定义序列”(Define Sequence)对话框。
  3. 单击“浏览”(Browse)，然后为动画选择第一个文件。
  4. 选择文件后，“定义序列”(Define Sequence)对话框将使用正确的信息自动为动画更新场。最初读取的几个文件名将显示在“序列文件名”(Sequence File Names)下。
  5. 单击“确定”(OK)以完成序列定义。有关该对话框的更多帮助信息，请参见定义序列对话框。
  6. 在读取“输出帧序列”(Output Frame Sequence)的行上单击“定义”(Define)。
  7. 此时将显示另一个“定义序列”(Define Sequence)对话框。可以单击“浏览”(Browse)，然后选择一个目录和名称以便使用。如果将数字包括在名称中，则会将该数字用作第一个帧编号。如果直接编辑“文件名”(Filename)字段，那么“#”符号将指示帧编号定位的位置。更改选项时，可以在“序列文件名”(Sequence File Names)中看到名称是如何更改的。
  8. 单击“执行”(Go)以开始集合图像。
