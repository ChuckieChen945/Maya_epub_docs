set_indexed_geo_property

创建并初始化以组件（如几何对象上的点）为目标的索引特性。

与 set_geo_property 类似，不同之处在于可以指定其他索引数组。索引数组允许多个组件引用数据数组中的同一元素。例如，face_vertex_normals 通常使用索引数组，这样一些面顶点可以共享相同的法线，而一些面顶点可以有自己的法线。非共享法线允许软/硬边。

输入

geometry

几何对象。

property

特性的名称。

default

特性的默认值。

data

特性的数据。这些值由索引数组引用。

target

特性针对的几何组件，例如 point_component、face_component、face_vertex_component、voxel_component。

indices

特性的索引。每个组件有一个元素。这些值是数据数组的索引。

interpolation_mode

指定在修改几何体时应如何对数据进行插值。

  • Nearest：最近相邻插值。
  • Linear：通过加权和进行线性插值。
  • DefaultValue：无插值，改为使用特性的 default 值。

输出

out_geometry

修改后的几何体。
