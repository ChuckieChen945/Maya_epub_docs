access_offset_array

根据以该组件为目标的偏移数组，计算与几何对象的特定元素（如特定面或股）对应的索引。

例如，如果将网格的 face_offset 数组连接到 offset_array 输入并将 index 设置为 3，则 start 输出将返回 face_vertex 数组中多边形 3 的第一个面顶点的索引，end 输出将返回多边形 3 最后一个面顶点的索引。

同样，如果连接 strand_offset 数组并将 index 设置为 2，则输出是 point_positions 数组中股 2 的第一个点和最后一个点的索引。

输入

offset_array

以组件为目标的偏移数组，例如 face_offset 或 strand_offset。

index

特定组件的索引。

输出

start

第一个关联元素的索引。

end

最后一个关联元素的索引加一，即 start + length。

为了方便进行基于零的索引和迭代操作，这是最后一个索引加一，而不是最后一个索引本身。这意味着从头到尾接受切片或在 i < end 时进行迭代，将产生正确结果。这类似于大小为 10 的数组具有索引 0-9 的情况。

递减此值（减去 1）可获取最后一个元素的索引。

length

与组件关联的元素的数量。
