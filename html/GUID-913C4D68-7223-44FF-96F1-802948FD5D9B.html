选择文件引用的内容

您可以选择引用文件包含的所有对象，以便快速查看父场景的引用组织。

从“大纲视图”(Outliner)中选择文件引用内容的步骤

  1. 在“大纲视图”(Outliner)中，在引用上单击鼠标右键，然后从显示的菜单中选择“引用 > 选择引用成员”(Reference > Select Reference Members)。

    被引用文件成员将在场景视图和“大纲视图”(Outliner)中处于已选择状态。

使用“引用编辑器”(Reference Editor)选择文件引用内容的步骤

  1. 选择文件 > 引用编辑器(File > Reference Editor)。
  2. 在“引用编辑器”(Reference Editor)中，单击被引用文件，然后选择“文件 > 选择文件内容”(File > Select File Contents)。

    被引用对象将在场景中处于已选择状态。
