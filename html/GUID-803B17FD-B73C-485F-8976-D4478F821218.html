“自定义装备”(Custom Rig)选项卡

打开自定义装备(Custom Rig)选项卡

  • 打开 HumanIK（“骨架 > HumanIK”(Skeleton > HumanIK)），然后选择顶部的“自定义装备”(Custom Rig)选项卡。

自定义装备通常由某些 ikHandle、约束和其他杂项节点的组合驱动。“自定义装备”(Custom Rig)选项卡提供了一个可视界面，它可用于映射您的自定义装备和重定目标工具，以传递其他 HumanIK 定义的角色中的数据。

“自定义装备”(Custom Rig)选项卡包含下列主要区域：
  • “自定义装备”(Custom Rig)选项卡工具栏
  • 自定义装备角色表示
  • 映射控件

下表提供有关使用“自定义装备”(Custom Rig)选项卡的任务的简要概述。

目标 操作

映射效应器。

选择一个效应器，在“角色”(Character)表示中的相应单元上单击鼠标右键，然后选择“指定选定的效应器”(Assign Selected Effector)。

调整效应器的映射方法。

单击 或 。

更改已映射的效应器。

选择一个效应器，在“角色”(Character) 表示中的相应单元上单击鼠标右键，然后选择 “替换指定的效应器”(Replace Assigned Effector) 。

清除已映射的效应器。

在“角色”(Character)表示中的单元上单击鼠标右键，然后选择 “清除”(Clear) 。

清除当前视图（全身、颈部或脊椎视图）中的所有已映射效应器。

在视图中的任意单元上单击鼠标右键，然后选择 “在视图中清除全部”(Clear All in View) 。

清除当前角色的所有已映射的效应器。

在“角色”(Character)表示中的任意单元上单击鼠标右键，然后选择 “清除全部”(Clear All) 。

加载自定义角色布局。

选择 >“编辑 > 自定义装备 > 加载 UI 配置”(Edit > Custom Rig > Load UI Configuration)。

更新自定义角色布局

选择 >“编辑 > 自定义装备 > 更新 UI 配置”(Edit > Custom Rig > Update UI Configuration)。
