access_transform_matrices

访问 Transform 结构的各个 double4x4 变换矩阵分量。

输入

transform

要访问的变换。

输出

pivot_matrix

在世界空间中定义变换的静止状态的枢轴矩阵。它定义缩放、旋转和平移所用的空间。

pivot_inverse_matrix

枢轴矩阵的逆矩阵。

pivot_local_matrix

局部空间中的枢轴矩阵，相对于变换的父对象。

operator_matrix

定义变换动画的运算符矩阵，包括变换中的缩放、旋转和平移操作。

move_local_matrix

局部移动矩阵，定义相对于在枢轴空间中应用的运算符矩阵的位移。

如果枢轴与 Y 轴成 -45 度，则运算符的 X 平移为 2.0 时，将在 X 和 Z 方向上移动 1.41。

如果变换的运算符矩阵是单位矩阵，则此矩阵也是单位矩阵。

move_world_matrix

世界移动矩阵，定义在变换中应用的总位移，这是变换或其任何祖先中的任何操作的结果。

它是从静止位置到动画位置的增量，与在 LBS（线性融合蒙皮）中应用的变换矩阵相同。

如果变换及其所有祖先的运算符矩阵都是单位矩阵，则此矩阵也是单位矩阵。

local_matrix

在局部空间中定义变换的动画状态的局部矩阵，相对于变换的父对象。

如果变换的运算符矩阵是单位矩阵，则此矩阵等于枢轴局部矩阵。

world_matrix

在世界空间中定义变换的动画状态的世界矩阵。

如果变换及其所有祖先的运算符矩阵都是单位矩阵，则此矩阵等于枢轴矩阵。
