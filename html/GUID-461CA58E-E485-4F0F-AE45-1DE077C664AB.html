自动接缝(Auto Seams)

使用“自动接缝”(Auto Seams)命令允许 Maya 自动识别和选择选定网格或 UV 壳上的最佳边用作接缝。

设置

修复非流形几何体(Fix non-manifold geometry)
对网格执行的清理操作。如果此选项处于禁用状态，则 Maya 将改为输出一条警告并提供选项来手动运行“清理”(Cleanup)操作。
注： “自动接缝”(Auto Seams)不会在非流形网格上运行。

自动接缝设置

接缝(Seams)

确定是仅选择相应的边还是对其执行切割 UV (Cut UVs)操作。默认为“切割”(Cut)。

方法(Method)

确定是要使用默认算法，还是沿所有硬边放置接缝。有关硬化边的详细信息，请参见使用顶点法线硬化或软化多边形着色。

壳拆分容差(Shell Splitting Tolerance)

确定现有 UV 壳在剪切时将拆分为新壳的可能性。将该选项设置为 0 可防止创建全新的壳。

连接孔(Connect Holes)

尝试填充 UV 壳中的孔，这有助于减少扭曲。默认为启用。
