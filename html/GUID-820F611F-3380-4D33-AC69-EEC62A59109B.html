雕刻工具

在“雕刻”(Sculpting)工具架上和“网格工具 > 雕刻工具”(Mesh Tools > Sculpting Tools)菜单中，将显示以下工具。有关与这些工具关联的公用设置列表，请参见雕刻工具设置。

工具名称 图标 用途

雕刻(Sculpt)

建立初始形状，并按工具光标边界内所有法线的平均值确定的方向移动顶点。使用“方向”(Direction)设置修改默认设置。

在其他雕刻工具已处于活动状态时，按 Ctrl + 1 可激活“雕刻”(Sculpt)工具。

平滑(Smooth)

通过平均化顶点的位置，将相对彼此的顶点位置拉平。

在其他雕刻工具已处于活动状态时，按 Ctrl + 2 可激活“平滑”(Smooth)工具。

松弛(Relax)

平均化曲面上的顶点，而不影响其原始形状。在使用其他雕刻工具时，按 Ctrl + Shift 可暂时激活“松弛”(Relax)工具

在其他雕刻工具已处于活动状态时，按 Ctrl + 3 可激活“松弛”(Relax)工具。

抓取(Grab)

选择顶点并基于您拖动的距离和方向移动顶点。对模型的形状进行精细调整时很有用。

修改“方向”(Direction)设置以约束工具的运动。例如，“XY”约束 XY 平面中的顶点运动。

按住 Ctrl 键并拖动可沿其“平均化法线”(Averaged Normal)临时移动顶点。

在其他雕刻工具已处于活动状态时，按 Ctrl + 4 可激活“抓取”(Grab)工具。

收缩(Pinch)

向工具光标的中心拉近顶点。对于更明晰地定义现有折痕很有用。

在其他雕刻工具已处于活动状态时，按 Ctrl + 5 可激活“收缩”(Pinch)工具。

展平(Flatten)

通过向共同平面移动顶点，将受影响的顶点拉平。对于设计和细节设计很有用。

在其他雕刻工具已处于活动状态时，按 Ctrl + 6 可激活“展平”(Flatten)工具。

泡沫(Foamy)

与“雕刻”(Sculpt)工具类似，但具有更柔和的感觉。对于设计初始形状很有用，不适用于细节工作。

在其他雕刻工具已处于活动状态时，按 Ctrl + 7 可激活“泡沫”(Foamy)工具。

“喷射”(Spray)

沿笔划随机在图像上盖章。主要用于细化曲面。（默认情况下使用图章图像。）

在其他雕刻工具已处于活动状态时，按 Ctrl + 8 可激活“喷射”(Spray)工具。

重复(Repeat)

在曲面上创建图案。例如，飞机机翼上的铆钉、拉链效果、布料上的缝线。（默认情况下使用图章图像。）

在其他雕刻工具已处于活动状态时，按 Ctrl + 9 可激活“重复”(Repeat)工具。

盖印(Imprint)

将图章图像按到曲面中。在网格位置上拖动并缩放图章。请参见使用图章进行雕刻

上蜡(Wax)

在模型上构建区域，向模型曲面添加材质或从中移除材质。

擦除(Scrape)

最小化突出特征。快速计算平面（基于首先放置光标的顶点位置），然后展平平面上方的任何顶点。

填充(Fill)

通过计算平面（基于工具光标内顶点的平均值），然后将平面下的顶点拉向该平面，填充模型曲面上的型腔。

修剪(Knife)

切割曲面中的精细笔划。（默认情况下使用图章图像。）

涂抹(Smear)

按与曲面上笔划方向的原始位置相切的方向移动顶点。

凸起(Bulge)

通过沿其自己的法线移动每个受影响的顶点以创建凸起效果，置换工具下方的区域。

放大(Amplify)

与“展平”(Flatten)工具的作用相反。对于设计、细节设计和进一步强调受影响顶点相对彼此的现有差异（通过移动它们远离共同平面）很有用。

冻结(Freeze)

锁定受影响的顶点，以便您雕刻时无法修改它们。默认情况下，已冻结的面显示为蓝色。详细信息请参见冻结模型上的区域。

在其他雕刻工具已处于活动状态时，按 Ctrl + 0 可激活“冻结”(Freeze)工具。

转化为冻结(Convert to frozen)

将冻结应用于组件选择。请参见冻结模型上的区域。

打开内容浏览器(Content Browser)

打开“内容浏览器”(Content Browser)中的“雕刻基础网格”(Sculpting Base Meshes)文件夹，在其中可以找到采样对象。

创建融合变形(Create a blend shape)

为选定对象创建融合变形。

形变编辑器(Shape Editor)

打开形变编辑器以使用雕刻工具编辑变形目标。
