flatten_instance_selection

显示任意嵌套的 selector_instances 树的展平视图。

如果已使用嵌套的 selector_instances 创建点上的实例，则可以使用此复合获取当前实例到点的指定的简化展平视图。将 instance_shapes 输出插入到 set_instance_geometry 节点的 instance_geometry 输入中，并将 flattened_selection 输出插入到 instance_id_values 输入中。

输入

instances

要为其生成展平选择的实例。

输出

flattened_selection

展平视图的实例的新选择。此数组的大小等于点数，其内容为从 0 到 array_size(instance_shapes) 的整数。因此，如果 instance_shapes 包含 8 个实例，则此数组将由 0 到 7 之间的数字组成（对于禁用的点为负数）。

instance_shapes

selector_instances 的展平树。
