创建图表

在 Maya 场景中从头开始创建 Bifrost 图表：

  1. （可选）在 Maya 场景中选择一个或多个要作为新图表输入的对象。支持网格、曲线和 nParticle 对象。选定对象将添加到图表中 input 节点的“路径”(Path)部分。您也可以稍后从场景中添加更多输入 - 请参见从场景输入几何体和材质。

    • 多个选定网格将作为一组网格导入。如果打算以不同的方式（例如，作为发射器、碰撞对象等）使用网格，建议在创建图表后分别输入它们。
    • 默认情况下，对多条曲线进行采样并将其作为单个股对象导入。如果需要，可以稍后将其转化为一组股，方法是在 input 节点的输出端口上单击鼠标右键并更改其类型。
    • nParticle 对象将作为点对象导入，但首先使用自动创建的名为 convertParticleToBifrost 的第二个图表进行转化。您只需在“曲线图编辑器”(Graph Editor)中关闭此选项卡即可。
    • 混合选择将在图表中创建多个 input 节点。

  2. 从 Maya 主菜单中选择“创建 > Bifrost 图表”(Create > Bifrost Graph)。

    将在场景中创建 bifrostGraph，并且可以在 Maya 的“大纲视图”(Outliner)中进行选择。bifrostGraphShape 表示图表的输出，在图表输出内容之前，该输出为空。

    如果要将此输出转化为旧的 bifShape 输出，请参见在 bifrostGraphShape 和 bifShape 输出之间转化。

  3. 此时将打开“曲线图编辑器”(Graph Editor)，其中显示带有 input 和 output 节点的新图表。如果“曲线图编辑器”(Graph Editor)已经打开，则新图表将在新选项卡上打开。

要添加节点并构建图表，请参见构建和修改图表。
