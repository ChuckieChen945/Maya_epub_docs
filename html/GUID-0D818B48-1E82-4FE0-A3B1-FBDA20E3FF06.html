禁用文件纹理图像的初始加载

若要加快场景的加载速度，可以从磁盘基于每个节点禁用文件纹理图像的初始加载。

禁用文件纹理图像的初始加载

  1. 选择文件纹理。
  2. 在“属性编辑器”(Attribute Editor)的“文件属性”(File Attributes)部分，启用“禁用文件加载”(Disable File Load)。
    注：

    在读取文件之后禁用 disableFileTexture 不会“卸载”纹理。

    禁用文件加载(Disable File Load)

    启用该属性可以阻止加载与“文件纹理”(File Texture)节点关联的文件。
