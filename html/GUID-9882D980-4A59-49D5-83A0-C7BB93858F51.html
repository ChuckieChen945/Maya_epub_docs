“移动 UV 壳”(Move UV Shell)工具

“移动 UV 壳”(Move UV Shell)工具 () 可用于移动 UV 壳，同时防止与其他 UV 壳重叠甚至是保持特定的壳间距。

禁止重叠(Prevent overlap)
启用该选项可防止将 UV 壳放入会与另一个 UV 壳重叠的位置。
壳间距(Shell spacing)
防止将 UV 壳放入会比此值使其更接近另一个 UV 壳的任何点。
放置迭代次数(Placement iterations)
确定该工具在移动操作期间执行的检查次数。减小此值将提高性能，但会降低精度。
