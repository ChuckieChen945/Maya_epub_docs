get_component_tag

获取特定组件标记的组件标记数据。也可以使用标记表达式。

表达式语言的语法与 Maya 中的标记表达式语言匹配。

示例

表达式 含义
face_* 返回以 face_ 开头的所有标记的并集。
* 返回包含几何体的所有点的标记。
v(face_top_tag) 返回包含 face_top_tag 标记（已转化为点）的标记。
f(*) - (face_top_tag + face_bottom) 返回包含除 face_top_tag 和 face_bottom_tag 中的面以外的所有面的标记
point_up_tag ^ point_right_tag 返回包含 point_up_tag 和 point_right_tag 中同时存在的点的标记

运算符

运算符 含义
+ 对两个标记执行集并集。
- 对两个标记执行集差集。
^ 对两个标记执行集交集。
! 对两个标记执行集反集。
() 括号：控制运算的顺序。例如 tag1 + (tag2 - tag3)

通配符

通配符 含义
* 与任意数量的字符匹配
ap* 匹配以 ap 开头的任何标记，并将它们合并在一起。apples 和 apricots 符合 ap* 匹配条件，此时匹配结果将为 apples + apricots

函数

函数 含义
f() 将标记投射到面。
v() 将标记投射到点。
fv() 将标记投射到面顶点。
s() 将标记投射到股。

命名语法

标记名称中允许使用大小写字母字符、数字 0 到 9、下划线 _、句点 . 和冒号 :。

输入

geometry

要查询的几何体。

tag

几何体中要获取其数据的标记的名称。这也可以是组合多个标记的标记表达式。

输出

tag_data

采用布尔值数组形式的标记数据。此数组的大小与 target 组件中的元素数相同。例如，如果 target 是 point_component，并且输入 geometry 具有 10 个点，则此数组将包含 10 个元素，每个点对应一个元素。如果此数组中的元素设置为 true，则相应的点将包含在此标记中。如果元素设置为 false，则相应的点不包含在此标记中。

tag_indices

标记中包含的采用索引数组形式的标记数据。每个索引引用一个 target 类型的元素。例如，如果 target 是 point_component，则此数组中的索引是点索引。

target

此标记以其作为目标的几何体组件，例如 point_component。

found_tag

如果在几何体上找到标记，则设置为 true，否则设置为 false。如果输入为 tag，则在成功解析了表达式时，此输出设置为 true，否则设置为 false。
