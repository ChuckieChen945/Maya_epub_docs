points_to_grid

The points_to_grid node transfers point properties to corresponding voxel properties in the input liquid object.

Inputs

liquid

The liquid object.

point_property_expression

All point properties matching this expression will be transfered to corresponding voxel properties, which will be created if needed. The corresponding property name has the point_ prefix changed to voxel_

voxel_property_to_extrapolate

All the voxel properties matching this expression will be extrapolated to regions outside the nodes which were splatted to.

point_droplet_channel

The name of the property to use for the droplet value if any. Points with values above the droplet threshold are considered balistic and are not splat.

droplet_threshold

The value of the droplet threshold (float).
