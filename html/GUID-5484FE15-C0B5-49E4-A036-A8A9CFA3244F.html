LUT3D

LUT3D 元素指定要应用的 3D LUT。

包含

  • Description。可选，任意数量。
  • Array (LUT3D)。必需，仅一个。
  • DynamicParameter。可选，至多一个。param 属性必须设置为 "DEFAULT_LOOK"。
注： IndexMap 元素不受支持。获得相同结果的最有效方法是在运算符之前使用 LUT1D 元素。

包含于

  • ProcessList

属性

id、name、bypass、inBitDepth、outBitDepth
请参见通用运算符属性。
interpolation
支持的值包括 "trilinear" 和 "tetrahedral"。

此属性是可选的。如果未指定，则默认值为 "trilinear"。

 0.002599007  0.001144927  0.023227118

        </Array>
    </LUT3D>

本节内容
  • Array (LUT3D)
    Array 元素指定 LUT 或矩阵的值数组。当 Array 元素包含在 LUT3D 元素中时，其内容的格式与包含在 LUT1D 或 Matrix 元素中时的格式不同。
