使用 set_instance_geometry 创建实例

set_instance_geometry 是一种高级节点，用于从点对象创建几何体实例。它具有一些内置选项，用于根据存储的逐点 long 值或一组 long 输入值在多个主几何体之间切换。

  1. 将 set_instance_geometry 节点添加到图表中。

  2. 将点对象（例如粒子系统的输出）连接到 points 输入。

  3. 将一个或多个几何对象（例如网格）连接到 instance_geometries 端口。除了几何体外，还可以连接实例形状，例如以下项的输出：

    • geo_instance，如果要对不同的实例主形状使用不同的预览几何体。
    • render_archive_instance，用于实例化磁盘上的渲染归档文件。
    • selector_instance，以便在主几何体之间进行比较复杂的切换。

  4. 指定如何为每个点指定几何体。这可以通过以下两种方法之一来完成：

    • 将自定义 long 特性的名称指定为 instance_id_property 输入。此操作假设您已使用 set_geo_property 等上游节点指定针对点对象的 point_component 的特性。该特性应指定要实例化的几何体，其中 0 表示连接到 instance_geometries 的第一个几何体，1 表示第二个，依此类推。
    • 将一组 long 连接到 instance_id_values 端口，并启用 overwrite_instance_id。如果数组长于点数，多余的值将被忽略。如果数组短于点数，则将所连接的第一个几何体用于其余点。

  5. 如果需要，将一个或多个简单几何体连接到 preview_geometries 端口。这些几何体用作替代对象以供在视口中预览。请注意，某些实例形状可能已定义了预览几何体，例如，如果您已将 geo_instance 节点连接到 instance_geometries 端口，而不是直接连接几何体。此外，如果在此处连接单个几何体，则它将用于预览所有实例，包括已定义自己的预览几何体的实例 - 这有助于在处理其他事项时简化视图。

  6. 将生成的 instances 输出到场景。
