“UV 编辑器”(UV Editor)的“纹理”(Textures)菜单

本主题介绍 UV 编辑器(UV Editor)的“纹理”(Textures)菜单中的选项。若要了解 UV 映射，请参见创建 UV 主题中的链接。

打开“UV 编辑器”(UV Editor)的“纹理”(Textures)菜单

  • 在“建模”(Modeling)菜单集 (F2) 中，选择“UV > UV 编辑器”(UV > UV Editor)
注： UV 编辑器(UV Editor)和 UV 集编辑器(UV Set Editor)是两种不同的工具。“UV 编辑器”(UV Editor)用于在 2D 视图内以交互方式编辑 UV 纹理坐标，“UV 集编辑器”(UV Set Editor)用于同时为多个多边形网格创建和编辑 UV 集。

选择要在“UV 编辑器”(UV Editor)中显示哪个图像纹理。从显示的下拉列表中选择图像，或者将临时“棋盘格贴图”(Checker Map)应用于所有壳。

棋盘格贴图
在“UV 编辑器”(UV Editor)中将棋盘格贴图图案应用于所有壳。有关如何执行此操作的信息，请参见将棋盘格图案着色器应用到 UV 网格。
颜色模式(Color Mode)
确定如何对棋盘格贴图进行着色。
  • 双色(2-Color)：每个框基于“棋盘格 1”(Checker 1)和“棋盘格 2”(Checker 2)这两种替代颜色。您还可以添加其他从左下角到右上角的渐变（“渐变叠加”(Gradient Overlay)）。如果要确定在模型的哪一侧启用纹理的哪一端，这对图案纹理非常有用。
  • 多色(Multi-Color)：沿多色渐变为所有框着色。
大小(Size)
确定棋盘格贴图的分辨率。
注： 当“棋盘格贴图”(Checker Map)处于可见状态时，您也可以通过按 Ctrl + 上箭头键或 Ctrl + 下箭头键随时调整“大小”(Size)。
平铺标签(Tile Labels)
在棋盘格贴图顶部之上显示各种棋盘格贴图平铺的自动 ID。
颜色(Color)
确定“平铺标签”(Tile Labels)显示的颜色。
