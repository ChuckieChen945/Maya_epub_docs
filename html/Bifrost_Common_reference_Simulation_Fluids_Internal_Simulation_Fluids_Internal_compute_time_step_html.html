compute_time_step

此节点计算子步循环的时间步。

输入

body

假定包含体素速度场的输入体（对象）。

collision

“collision”是包含体素化碰撞对象的对象或对象列表。如果 collision 包含不同时间的对象，我们假定该列表按字母（时间、对象）顺序排序，这样，我们可以按该列表中的索引和获知的给定对象的时间戳将不同时间的同一对象关联起来。

cfl

CFL 数量，即允许在单个子步中传输某个数量的最大体素数。

minSteps

子步循环中采用的最小步数

maxSteps

子步循环中采用的最大步数

time

当前时间。

startTime

帧开始的时间。

endTime

帧结束的时间。

frame

当前帧。

startFrame

开始帧。

输出

timeStep

此时间步的大小（增量时间）。

done

如果为 true，则表示已达到 endTime。
