get_strand_geo_loc_data

它会将存储在通用 GeoLocation 结构中的位置数据解码/解释为描述股几何体上位置的数据。股上的 GeoLocation 是沿着其中一个股段的位置。

输入

geo_loc

要解码的 GeoLocation 结构。

输出

strand_index

位置位于具有此索引的股上。

strand_segment_index_count

strand_segment_indices

股的股相关的段索引。

strand_segment_weights

与股段索引对应的归一化权重。权重加上股段索引构成了线性组合，用于描述股段上的位置。

valid

如果通用输入位置表示股上的位置，则为 True。否则为 False。如果为 False，则其他输出的值未定义。
