命令行渲染器的渲染器特定标志

对于特定于每个 Maya 软件、硬件和向量渲染器的标志，在 Shell 或终端 (MacOS) 窗口中键入以下命令： 

Render -r <renderer name> -help

其中，<renderer name> 是渲染器的名称（sw、hw、vr）。

注： 请查阅第三方渲染器的文档，了解其渲染器特定的标志。

下面是命令行渲染标志的提示：

  • 请记住在选项标志与其参数之间留一个空格。
  • 任何布尔标志都会将以下值视为 True：on、yes、true 或 1；同样，会将以下值视为 False：off、no、false 或 0。例如： 
-x 512 -y 512 -cam persp -im test -of jpg -mb on -sa 180 file

“Maya 软件”(Maya Software)渲染器命令行渲染器标志

以下是特定于使用“Maya 软件”(Maya Software)渲染器实现命令行渲染的所有标志及其说明。还可以在 Shell 或终端窗口中键入 Render -r sw -help。

请参见命令行渲染器的公用标志。

抗锯齿标志

标志 定义
-iip <boolean>

忽略图像平面(Ignore Image Planes)。渲染之前关闭所有图像平面。

-eaa <int>

EAS 的抗锯齿质量 (Abuffer)。以下值之一：最高 (0)、高 (1)、中 (2)，或低 (3)。

-ss <int>

像素中每个曲面的着色采样的全局数量。

-mss <int>

像素中每个曲面的自适应着色采样的最大数量。

-mvs <int>

运动模糊可见性采样的数量。

-mvm <int>

运动模糊可见性采样的最大数量。

-pss <int>

粒子可见性采样的数量。

-vs <int>

体积着色采样的全局数量。

-ufil <boolean>

如果为 True，请使用多像素过滤；否则，请使用单像素过滤。

-pft <int>

当 useFilter 为 True 时，标识以下过滤器之一：长方体 (0)、三角形 (2)、高斯 (4)，二次方 (5)。

-pfx <float>

当 useFilter 为 True 时，定义过滤器的 X 大小。

-pfy <float>

当 useFilter 为 True 时，定义过滤器的 Y 大小。

-rct <float>

红色通道对比度阈值。

-gct <float>

绿色通道对比度阈值。

-bct <float>

蓝色通道对比度阈值。

-cct <float>

像素覆盖对比度阈值（默认值为 1.0/8.0）。

光线跟踪质量标志

标志 定义
-ert <boolean>

启用光线跟踪。

-rfl <int>

最大光线跟踪反射级别。

-rfr <int>

最大光线跟踪折射级别。

-sl <int>

最大光线跟踪阴影光线深度。

场选项标志

标志 定义
-field <boolean>

启用场渲染。启用时，图像将交替。

-pal <boolean>

如果启用场渲染，则首先渲染偶场 (PAL)。

-ntsc <boolean>

如果启用场渲染，则首先渲染奇场 (NTSC)。

运动模糊标志

标志 定义
-mb <boolean>

启用/禁用运动模糊。

-mbf <float>

运动模糊帧数。

-sa <float>

运动模糊的快门角度 (1-360)。

-mb2d <boolean>

启用/禁用运动模糊 2D。

-bll <float>

2D 运动模糊模糊长度。

-bls <float>

2D 运动模糊模糊锐度。

-smv <int>

2D 运动模糊平滑值。

-smc <boolean>

启用/禁用 2D 运动模糊平滑颜色。

-kmv <boolean>

保持 2D 运动模糊的运动向量处于启用/禁用状态。

渲染选项标志

标志 定义
-ifg <boolean>

如果为 False，则使用胶片门进行渲染。

-edm <boolean>

启用深度贴图使用。

-g <float>

Gamma 值。

-premul <boolean>

将颜色预乘 Alpha 值。

-premulthr <float>

设置预乘时，可定义用于确定是否预乘的阈值。

内存与性能标志

标志 定义
-uf <boolean>

使用镶嵌细分文件缓存。

-oi <boolean>

动态检测采用类似方式镶嵌细分的曲面。

-rut <boolean>

重用渲染几何体生成深度贴图。

-udb <boolean>

使用置换边界框比例来优化置换贴图性能。

-mm <int>

渲染器最大内存使用量（以 MB 为单位）。

其他渲染层标志

标志 定义
<RS>

过时的。

-l <boolean:name>

选择要渲染的显示和渲染层。

其他标志

标志 定义
-rep <boolean>

指定是要渲染整个序列的图像，还是仅渲染其文件在磁盘上不存在的图像。

  • 0 (false) - 重新计算所有图像
  • 1 (true) - 仅重新计算指定序列中其文件在磁盘上不存在的图像
-reg <int int int int>

设置最终图像的子区域像素边界（左侧、右侧、底部和顶部）。

-n <int>

要使用的处理器的数量。0 表示使用所有可用处理器。

-mf <boolean>

如果为 True，则将图像文件格式附加到图像名称。

-sp <boolean>

仅生成阴影深度贴图。

-amt <boolean>

在遇到缺失的纹理时，中止渲染器。

-ipr <boolean>

创建 IPR 文件。

-keepPreImage <boolean>

在后期处理之前先保留渲染。
