assign_diagnostic_material

指定简单材质，以快速将输入几何体点上的数据可视化为颜色，以便进行诊断。

输入

geometry

要在其上可视化数据的几何体。

emission_color

要可视化的数据。这是解释的自动端口，因此您可以连接以下几种数据类型之一：

  • 连接 0 到 1 范围内的单个 float，以将该值视为一个灰度。
  • 连接各分量在 0 到 1 范围内的 array<float>，以将该数据视为灰色范围。
  • 连接各分量在 0 到 1 范围内的单个 float3，以将该值视为一种颜色。
  • 连接各分量在 0 到 1 范围内的 array<float3>，以将该值视为颜色范围。
  • 连接 string 以查看具有相应名称的特性的值（每个点的 float 或 float3）。
  • 连接 ScalarField 或 VectorField 以查看每个点所在位置处的场的值。

specular

镜面反射颜色对着色的贡献的权重。值为 1 表示完全镜面反射。

emission

发射颜色对着色的贡献的权重。值为 1 表示完全发射。

输出

geometry_with_material

应用了诊断材质的输入几何体。
