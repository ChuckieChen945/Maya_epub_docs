更改项目

项目是 Maya 将默认打开的不同文件类型的文件夹集合。如果将正在处理的当前项目更改为其他项目，则需要“设置”新项目（“文件 > 设置项目”(File > Set Project)）。设置新项目后，Maya 会开始将项目数据保存到新项目位置。例如，如果更改项目并缓存当前场景，缓存现在保存到设置项目的主文件位置的 data 文件夹。请参见将文件组织到项目中。

更改项目

  1. 选择“文件 > 设置项目”(File > Set Project)。

    “设置项目”(Set project)窗口出现。

  2. 在“设置项目”(Set project)窗口中，浏览到要打开项目的位置。
  3. 单击“设置”(Set)。

    如果要置为当前位置的目录未包括 workspace.mel 项目定义文件，Maya 要求选择其他目录或为指定的目录创建项目定义文件。

  4. 执行下列操作之一：
    • 单击“选择其他位置”(Select another location)，然后浏览到包含项目定义文件的目录。
    • 单击“创建默认工作区”(Create default workspace)在指定的目录中创建 workspace.mel 项目定义文件。

      按照这种方式创建项目定义文件不会自动创建项目位置（例如主项目位置和次项目位置）。这意味着设置项目更类似于项目占位符而非完整的 Maya 项目。强烈建议通过选择“文件 > 项目窗口”(File > Project Window)在“项目窗口”(Project Window)打开项目，检查设置并接受。请参见创建新项目。
