融合和设置几何缓存的权重

如果有多个几何缓存附加到某一对象，将自动创建 cacheBlend 节点以定义和管理该对象的多个几何缓存的权重。请参见 cacheBlend。

当几何缓存在时间上重叠时，其重叠的缓存区域使用这些缓存的当前输入权重值在其 cacheBlend 节点上自动融合在一起。

此外，缓存权重在两个重叠的缓存之间进行归一化。因此，如果两个重叠的缓存的权重值相同（即 1.0 和 1.0、0.5 和 0.5 等等），则在播放时，这两个缓存对其缓存融合区域中的变形有相等贡献。

融合几何缓存并设定其权重

  1. 选择要调整其几何缓存权重的对象。
  2. 在属性编辑器中，选择 cacheBlend 节点。
  3. 在“输入权重”(Input Weights)区域中，通过拖动相应的滑块或在权重字段中输入值，来调整各个对象缓存的权重值。

    当几何缓存的权重为 1.0 时，它将与当前对象的其他缓存融合，在播放期间对该对象的变形有百分之百的贡献。

    当几何缓存的权重为 0.0 时，它不会与当前对象的其他缓存融合，在播放期间对该对象的变形无贡献。

  4. （可选）为几何缓存的权重设置动画以随时间增加或减少它们对其融合变形的贡献。

    调整每个缓存的权重值后，在其权重属性上单击鼠标右键并从显示的上下文相关菜单中选择“为选定项设置关键帧”(Key Selected)。

    这将向几何缓存中添加关键帧以创建权重曲线。然后，可以操纵这些权重曲线，平滑对象重叠（融合）缓存之间的过渡。

绘制几何缓存权重

通过在对象的曲面上直接绘制对象的几何缓存权重，您可以采用交互方式应用、分配和融合单个对象中的多个缓存的权重。例如，您可以使用“绘制缓存权重工具”(Paint Cache Weights Tool)在两足动物模型的臂部绘制一个缓存，在该模型的胸部绘制另一个缓存，然后在该模型的肩部融合这两个缓存，以创建从臂部的几何缓存到胸部的几何缓存的平滑过渡。

绘制几何缓存权重

  1. 选择要绘制权重的对象。
  2. 在“动画”(Animation)菜单集中，选择“缓存 > 几何缓存 > 绘制缓存权重工具”(Cache > Geometry Cache > Paint Cache Weights Tool) > 。
  3. 在绘制缓存权重工具选项(Paint Cache Weights Tool Options)窗口中，从“缓存”(Cache)列表中，选择要在对象上绘制其权重的几何缓存。
  4. 选择一个绘制操作，然后调整工具的笔刷的“值”(Value)和“不透明度”(Opacity)。

    有关这些工具设置的说明，请参见 Artisan 工具设置。

  5. （可选）在工具设置的“属性贴图”(Attribute Maps)区域中，为要在对象上绘制其权重的几何缓存导入一个现有的几何缓存权重贴图。请参见“属性贴图”区域。
  6. 在场景视图中，沿对象的曲面拖动“绘制缓存权重工具”(Paint Cache Weights Tool)以逐顶点更改其几何缓存权重。
    注： 绘制的几何缓存权重已进行归一化。

    这意味着如果任何几何缓存具有相同的绘制权重值（即 1.0 和 1.0、0.5 和 0.5 等等），则在播放时，这些缓存将对其对象变形有相等贡献。
