create_combustion_channels

此节点向输入对象添加燃烧所需的通道。

输入

create_combustion_rate_channel

如果为 true，解算器将创建一个名为“combustion_rate”的通道，以用于跟踪燃烧率

default_ignition_temperature

燃料通道的默认点燃温度

number_of_upres_levels

平铺树中将保留 upres 通道（雾密度和温度）的最精细级别的数量

feed_upres_into_sim

如果较高分辨率的 upres 通道应反馈到模拟中，则为 True

body

Aero 体（对象）。
