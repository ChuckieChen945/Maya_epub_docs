sample_interpreted_auto_port_as_vector

在指定位置将解释的自动端口的值作为向量进行采样，就好像数据值是几何体的特性一样。与 interpret_auto_port_as_scalar 复合一样，这是一个工具节点，用于构建支持多种输入类型的可重用复合。

支持以下多种可能的输入类型：

  • 单个 float3 输入解释为应用于所有点的常量。单个 float 会先升级为具有常量成员的 float3。
  • array<float3> 输入解释为每个点的 float3 值。数组可调整大小以匹配 geometry 输入上的组件数。
  • string 输入解释为现有几何特性的名称，并正常采样。
  • VectorField 将在几何体组件的每个位置处进行采样。ScalarField 以相同的方式进行采样，并且值会升级为具有常量成员的向量。

输入

geometry

要在其上对输入进行采样的几何体。

interpreted_vector_port

将此端口连接到复合的 auto 输入，以支持不同类型的输入。

default_value

在以下情况下要使用的默认值：

  • 未将任何内容连接到 interpreted_scalar_port。
  • 连接了 array<float>，但数组太小。
  • 连接了空标量场。
  • 提供了无效位置。

locations

要对几何体进行采样的位置。

geometry_component

如果对象不使用默认的 point_component，则更改此字符串。

position_property

如果对象不使用默认的 point_position 特性，则更改此字符串。这仅用于对标量场进行采样（如果已连接）。

输出

data

在给定位置对 interpreted_vector_port 进行采样的结果。

has_data

指示是否已将有效输入连接到 interpreted_vector_port。如果输入为 float3 或 float 值、由 float3 或 float 值组成的非空数组、非空场或作为 geometry 上有效特性的名称的字符串，则此项返回 true。
