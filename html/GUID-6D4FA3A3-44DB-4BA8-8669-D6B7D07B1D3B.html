编辑 nParticle 网格颜色集

您可以将使用 nParticle 生成的逐顶点着色数据指定为对象的当前颜色集。请参见为着色生成 nParticle 网格逐顶点数据。

设定输出网格颜色集

  1. 选择 nParticle 对象的输出网格。

    如果尚未将 nParticle 对象设定为生成逐顶点着色数据或将 nParticle 转化为多边形，请立即执行该操作。请参见为着色生成 nParticle 网格逐顶点数据。

  2. 从“建模”(Modeling)菜单集中，选择“网格显示 > 颜色集编辑器”(Mesh Display > Color Set Editor)。

    此时将显示“颜色集编辑器”(Color Set Editor)，并列出从 nParticle 对象生成的逐顶点数据。

  3. 在“颜色集编辑器”(Color Set Editor)中，选择要用作多边形对象当前颜色集的逐顶点数据。

    例如，如果希望使用来自 nParticle 对象的每粒子“白炽度”(Incandescence)渐变来为 nParticle 输出网格上色，请选择“incandescencePV(RGB)”。

    如果 nParticle 对象的逐顶点数据没有显示在“颜色集编辑器”(Color Set Editor)中，请单击“更新”(Update)。

  4. 单击“更新”(Update)。
  5. 在“属性编辑器”(Attribute Editor)中，单击“polySurfaceShape”选项卡。
  6. 在“网格组件显示”(Mesh Component Display)区域，启用“显示颜色”(Display Colors)。

    nParticle 输出网格现在应显示由选定颜色集定义的颜色。在“网格控制”(Mesh Control)区域，“当前颜色集”(Current Color Set)属性应设定选定的逐顶点数据集。
