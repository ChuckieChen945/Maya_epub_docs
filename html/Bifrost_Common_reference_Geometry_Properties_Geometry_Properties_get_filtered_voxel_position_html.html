get_filtered_voxel_position

基于 float 特性返回体积上按阈值过滤的体素位置。如果未指定特性，或者如果该特性在体积上不存在，则返回空数组。

输入

volume

传入的体积。

voxel_offset

如果将体素想象为由立方体的一个角点定义，则此选项控制在体素中将从何处对点进行采样：

  • 中心 - 体素的中心
  • X 面 - X 轴上的面中心
  • Y 面 - Y 轴上的面中心
  • Z 边 - Z 轴上边的中心
  • Z 面 - Z 轴上的面中心
  • Y 边 - Y 轴上的边中心
  • X 边 - X 轴上的边中心
  • 角点 - 体素的角点

filtering_data

要用作过滤器的 float 特性，例如 voxel_fog_density 或 voxel_signed_distance。

filter_threshold

用于返回体素位置的特性的阈值。将返回特性值大于此值的体素的位置（如未选中 less_than_threshold）。

less_than_threshold

返回特性值小于阈值的体素的位置。

输出

voxel_position

voxel_position 的值（以 float3 数组的形式）。

filtering_indices

过滤后剩余的索引。
