property_proxy_field

根据基于点的几何体的特性，创建特性代理场。

特性场与其他场的不同之处在于，它返回存储在几何体上的特性值的数组，而不是空间中的位置值。例如，通过创建特性代理场，可以创建一个场，该场将存储在对象点上的值（例如 point_weight 或 point_velocity）与点位置（如温度或风）的场值组合在一起。您可以使用 Core::Fields 和 Core::Math 名称空间中的多种节点将代理场与普通场组合在一起，生成的结果也是代理场，可以使用 sample_field_with_proxies 进行采样。

输入

property

对场进行采样时应返回其值的几何体上的特性的名称。

field_type

指定场的类型。根据特性的类型，连接标量场或向量场。

输出

property_field

生成的代理场。

success

一个标量场，如果在对场进行解算时找不到 float、float3 或 float4 类型的指定特性，则值为 0.0；如果找到这些类型的指定特性，则值为 1.0。
