捕捉关键帧选项(Snap Keys Options)

注： 本节介绍了“曲线图编辑器”(Graph Editor)和“摄影表”(Dope Sheet)的“编辑”(Edit)菜单中的“捕捉关键帧选项”(Snap Keys Options)。有关“关键帧”(Key)菜单中的“捕捉关键帧选项”(Snap Keys Options)的信息，请参见捕捉关键帧选项(Snap Keys Options)。
时间范围(Time range)

指定关键帧集的动画曲线的时间范围。

全部(All)

指定对时间范围没有任何限制。

开始/结束(Start/End)

指定从“开始时间”(Start Time)到“结束时间”(End Time)的时间范围。

时间滑块(Time Slider)

指定由“时间滑块”(Time Slider)的“播放开始”(Playback Start)和“播放结束”(Playback End)时间定义的时间范围。

开始时间(Start time)

指定时间范围的开始（“开始/结束”(Start/End)处于启用状态时可用）。

结束时间(End time)

指定时间范围的结束（“开始/结束”(Start/End)处于启用状态时可用）。

捕捉(Snap)

选择下列选项之一：

时间(Times)

仅沿时间轴捕捉关键帧。这是默认设置。

值(Values)

仅沿值轴捕捉关键帧。

二者(Both)

沿时间轴和值轴捕捉关键帧。

将时间捕捉为以下项的倍数(Snap times to a multiple of)

该值定义要捕捉到的时间单位的倍数。

将值捕捉为以下项的倍数(Snap values to a multiple of)

该值定义要捕捉到的整数属性值的倍数。
