“流体 3D 容器”(Fluids 3D Container)/“2D 容器”(2D Container)选项

使用这些选项可创建 3D 或 2D 容器。

有关“基本发射器属性”(Basic Emitter Attributes)、“距离属性”(Distance Attributes)和“体积发射器属性”(Volume Emitter Attributes)，请参见流体发射器选项。

基本流体属性

X 分辨率(X Resolution)、Y 分辨率(Y Resolution)、Z 分辨率(Z Resolution)

以体素（体积像素）为单位定义流体的分辨率。分辨率越高，细节越精细，但会增加渲染时间和降低交互式模拟性能。

3D 容器的默认分辨率为 10, 10, 10。

2D 容器默认分辨率为 40, 40。

即使不使用任何栅格定义属性值（使用所有渐变样式值），增加分辨率时也会提高渲染质量。

X 大小(X Size)、Y 大小(Y Size)、Z 大小(Z Size)

在 Maya 的工作单位集中定义流体容器的物理尺度。3D 流体容器的默认大小为 10, 10, 10。

2D 容器的默认大小为 10, 10, 0.25。

提示：
  • 即使不使用任何栅格定义属性值（使用所有渐变样式值），也应使“大小”(Size)与栅格“分辨率”(Resolution)成比例。例如，如果大小为 10.0 5.0 2.0，则有效的分辨率将为 40 20 8。如果大小与分辨率不成比例，则体素将不是正方形，从而导致在向下看时一个轴比另一个轴高的质量问题。
  • 深度（“Z 大小”(Z Size)）为 0 的 2D 流体不会渲染为体积，但可以用作高度场（请参见使用高度场）或曲面渲染（请参见曲面渲染）。
添加发射器
启用后，Maya 将创建一个流体发射器，并将其添加到容器。默认情况下，“添加发射器”(Add Emitter)已启用。
