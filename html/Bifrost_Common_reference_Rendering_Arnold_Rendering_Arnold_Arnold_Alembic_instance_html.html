Arnold_Alembic_instance

创建引用磁盘上的 Alembic 文件的实例形状。实例形状可以连接到 create_instances、set_instance_shape、set_instance_geometry 或 selector_instance。这将在使用 Arnold 进行渲染时创建 Alembic 程序节点。

filename 是 Alembic 文件的路径和文件名。

时间

启用 use_frame_property 后，点云上的 frame_property 用于在不同时间创建实例。frame_property 指定点对象特性的名称，该对象指定要实例化序列的哪一帧。应确保图表在每一帧逐点更新此特性，以播放动画。

frame 和 fps 指定在未使用 frame_property 时要使用的帧。

有许多与此创建的 Arnold 节点相关的特定设置。请参见 Arnold 节点参考中的 Arnold Alembic 程序。有关 exclude_xform、expand_hidden、flip_v、layers、make_instance、material_attribute、nameprefix、objectpath、operators、pull_user_params、radius_attribute、radius_default、radius_scale、scene_camera、use_instance_cache、velocity_ignore、velocity_scale、visibility_ignore 的信息，请参见 Arnold 节点参考。

此部分包含渲染时 Arnold 使用的设置。有关详细信息，请参见 Arnold 用户手册。

实例

preview_geometry 创建 Bifrost 上一个几何体，与其他实例形状相同。如果连接了对象，它将仅显示在视口中，而不用于渲染。

name 是 Arnold 的可选标识符。

要应用于实例化几何体的可选变换矩阵。

transform 是要应用于实例化几何体的变换矩阵。
