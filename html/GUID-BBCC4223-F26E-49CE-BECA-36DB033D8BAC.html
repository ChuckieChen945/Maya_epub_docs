Array (LUT3D)

Array 元素指定 LUT 或矩阵的值数组。当 Array 元素包含在 LUT3D 元素中时，其内容的格式与包含在 LUT1D 或 Matrix 元素中时的格式不同。

包含

  • 指定 LUT 输出值的一系列数值。相应的输入值由包含元素的 inBitDepth 和条目数（均匀分布）指示。
    对于 3D LUT，每个条目都有三个值，指定单独的 R、G 和 B 输出值。条目采用从小到大的顺序，最后一列的索引更改最快，第一列的索引更改最慢。例如，2×2×2 立方体的条目顺序为：
    条目 R G B
    1 0 0 0
    2 0 0 1
    3 0 1 0
    4 0 1 1
    5 1 0 0
    6 1 0 1
    7 1 1 0
    8 1 1 1

包含于

  • LUT3D

属性

dim
表示三维立方体维度的四个整数，后跟每个条目的组件数。

仅支持在所有三个通道中具有相同维度的 3D LUT，因此前三个值必须相同。第四个值必须为 3。

dim 是必需属性，并且其值必须匹配实际存在的条目数。

        <Array dim="48 48 48 3">
 0.000669299  0.000704187  0.000766901
 0.001848371  0.001297010  0.010368987
 0.002599007  0.001144927  0.023227118
