工具动画演示目录

默认情况下，随 Maya 一起提供的工具动画演示位于 builtin.toolclips 目录中。所有自定义工具动画演示内容都必须添加到 user.toolclips 目录下。

builtin.toolclips

您可以从以下位置访问 builtin.toolclips 目录：
  • Windows：C:\Program Files\Autodesk\Maya<版本>\resources\builtin.toolclips
  • Mac OS X：/Applications/Autodesk/maya<版本>/Maya.app/Contents/Resources/builtin.toolclips
  • Linux：/usr/autodesk/maya<版本>-x64/resources/builtin.toolclips
对于本地化的 Maya 版本，您可以从以下位置访问 builtin.toolclips 目录：
  • Windows：C:\Program Files\Autodesk\Maya<版本>\resources\l10n\<语言>\builtin.toolclips
  • Mac OS X：/Applications/Autodesk/maya<版本>/Maya.app/Contents/Resources/<语言>/builtin.toolclips
随 Maya 一起提供的工具动画演示内容位于以下子目录中：
子目录 说明

Control

包含默认的 ToolClips.xml 文件。此文件指定工具动画演示窗口中显示的内容，包括工具标题、工具摘要、键盘快捷键、影片、图像和帮助链接。

Resources

包含 ToolClips.xml 文件中指定的影片和图像文件。影片必须为 WebM 格式，而图像必须为 PNG 或 JPEG 格式。

ui

包含在工具动画演示窗口中使用的资源，包括图标。此子目录还包含用于定义窗口格式和布局的 QML 格式的文件。

注： 请勿修改 builtin.toolclips 下的 XML 文件和子目录。这可能会给工具动画演示的操作带来不利影响。

user.toolclips

您可以从以下位置访问 user.toolclips 目录：
  • Windows：\Users\<用户名>\Documents\maya\<版本>-x64\en_US\user.toolclips
  • Mac OS X：/Users/<用户名>/Library/Preferences/Autodesk/maya/<版本>-x64/user.toolclips
  • Linux：/home/mayadev/maya/<版本>-x64/user.toolclips
安装时将自动创建以下子目录：
子目录 说明

Control

将 ToolClips.xml 文件放入此目录中。

Resources

将自定义工具动画演示内容（影片、图像）放在此目录中。

注： 放入 user.toolclips 目录中的工具动画演示内容不会自动复制到新的 Maya 版本。您可以将自定义内容复制到新版本，或者使用 MAYA_TOOLCLIPS_PATH 变量指向自定义内容所在的路径。请参见 MAYA_TOOLCLIPS_PATH。

自定义目录

您可以为工具动画演示内容创建自定义目录，并将其与其他用户共享。如果要创建自定义目录，请注意以下事项：
  • 自定义目录名称必须使用 .toolclips 扩展名。
  • 自定义目录必须包含 Control 和 Resources 子目录。
  • 要在 Maya 中观看工具动画演示，请将 MAYA_TOOLCLIPS_PATH 变量设置为指向自定义目录路径。
