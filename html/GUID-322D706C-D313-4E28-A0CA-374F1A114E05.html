将发射器连接到 nParticle

有几种方法可以将发射器连接到 nParticle 对象：

  • 在设置已发射 nParticle 的颜色和其他属性后，可以创建新的发射器并将其连接到相同的 nParticle 对象。默认情况下，新的发射器将发射具有相同属性设置的 nParticle，但您可以更改任何发射器属性，包括速率、方向或发射器类型。

    例如，假定要创建火效果并扩展其大小。可以将发射器添加到原始发射对象，然后连接现有的已发射火粒子对象到新的发射器。从添加的发射器发射的粒子与现有粒子具有相同的属性。

  • 使用“动力学关系”(Dynamic Relationships)编辑器连接发射器和 nParticle 对象（请参见处理动力学对象之间的连接）。如果要将发射器连接到已与多个发射器建立连接的 nParticle 对象，则您必须使用该方法。

    通过创建多个已发射的 nParticle 对象（每个对象对应一个发射器），向曲面添加多个发射器。可以使用“动力学关系”(Dynamic Relationships)编辑器从发射器断开所有发射的粒子对象，然后重新将单个对象连接到所有发射器。每个发射器仅将发射具有其属性设置的粒子对象。

  • 创建两个或多个空 nParticle 对象，为每个对象定义唯一的属性，然后将其连接到同一个发射器。通过这种方法可以从相同的发射器发射不同的粒子效果，例如，火和烟。

    如果将多个 nParticle 对象连接到同一个发射器，那么发射器发射的所有对象均具有其各自的属性。例如，如果有一个红色粒子对象和一个绿色粒子对象，将这两个对象连接到同一个发射器，那么该发射器既发射绿色粒子又发射红色粒子。

  • 可以将已发射粒子连接到具有多个发射器的对象上的选定发射器。
  • 使用“动力学关系”(Dynamic Relationships)编辑器交换连接到发射器的 nParticle 对象。处理动力学对象之间的连接
  • 将现有的独立发射器添加到单个选定对象。发射器必须未存在于另一个对象。如果要使用 Maya API 创建发射器，使用该方法以将发射器添加到所需对象。

将 nParticle 对象连接到发射器

  1. 选择要连接到发射器的 nParticle 对象。
  2. 按住 Shift 键并单击发射器（或拥有该发射器的对象）。如果在“大纲视图”(Outliner)中选择发射器，请按住 Ctrl 键并单击（Windows 和 Linux）或按住 Command 键并单击 (Mac OS X)，以进行不连续的选择。
  3. 选择“nParticle > 使用选定发射器”(nParticles > Use Selected Emitter)。

创建空 nParticle 对象

  1. 选择“nParticle > nParticle 工具”(nParticles > nParticle Tool) > 以显示选项窗口。
  2. 在“粒子设置”(Particle Settings)窗口中，将“粒子数”(Number of Particles)设置为 0。
  3. 单击工作区，然后按 Enter 键或 Return 键。

    这将在“大纲视图”(Outliner)中创建一个空 nParticle 对象。

将发射器添加到空 nParticle 对象

  1. 选择对象，然后按住 Shift 键并选择发射器。
  2. 选择“nParticle > 从对象发射”(nParticles > Emit from Object)。

    在“大纲视图”(Outliner)中，发射器将缩进显示在对象下方。移动对象的位置会使发射器的位置发生移动。
