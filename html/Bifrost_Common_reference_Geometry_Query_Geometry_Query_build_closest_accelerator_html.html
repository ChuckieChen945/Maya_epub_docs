build_closest_accelerator

构建与 sample_closest_accelerator 和其他查询节点一起使用的加速度结构。可以构建一次加速器，然后在图表中的多个地方使用。但是，如果在相应节点的下游修改几何体，加速器不会自动更新。要查询修改后的几何体，请构建新的加速器。

输入

geo

要为其构建加速器的几何体。

geo_component

将为其加速查询的组件。如果此项保留为空，则将使用输入几何体类型的默认组件：face_component（对于网格）、strand_component（对于股）、point_component（对于点对象）。

输出

accelerator

为给定几何体计算的加速器。
