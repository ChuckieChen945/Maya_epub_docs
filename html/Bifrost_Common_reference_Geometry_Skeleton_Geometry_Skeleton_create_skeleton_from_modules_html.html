create_skeleton_from_modules

从一组无序绑定模块输出构造骨架。此构造函数特定于绑定模块框架，并且仅接受 Rigging::Module::Outputs 类型的模块输出。

骨架的构造方式是对所有输入 modules 中的所有关节路径进行排序，过滤掉所有重复项，然后将其变换数据复制到生成的骨架中。

输入

modules

一组 Rigging::Module::Module 类型的绑定模块输出。

show_rest_state

如果启用此项，则将使用 joint_pivot_matrix（表示变换的静止状态）而不是 joint_world_matrix 来显示关节。

display_joints

如果启用此项，则将骨架的关节显示为网格几何体。

joint_size

用于定义关节半径的默认大小。I

joint_color

关节的颜色。

display_transforms

如果启用此项，则将骨架关节的变换显示为股箭头。

axes_shape

要应用于生成的股的几何体着色器的类型。

  • Wire：将股显示为 1 像素宽的细线。

  • Ribbon：将股显示为条带，宽度相对于变换的比例，并使用 axes_scale、arrowhead_size_ratio 和 stem_width_ratio 值。

  • Tube：将股显示为厚管，半径相对于变换的比例，并使用 axes_scale、arrowhead_size_ratio 和 stem_width_ratio 值。

axes_scale

用于增加或减小要显示的变换轴比例的倍增。

arrowhead_size_ratio

箭头宽度的大小比率，相对于每个变换轴的比例。假定变换的比例为 [1, 1, 1]，则值为 0.1 时，箭头的宽度将为 0.1。如果变换的比例为 [10. 10. 10]，则使用相同的值时，箭头的宽度将为 1.0。

stem_width_ratio

杆宽度相对于箭头最宽部分的值。值为 0.1 表示杆宽度是箭头宽度的十分之一。

输出

skeleton

生成的骨架。
