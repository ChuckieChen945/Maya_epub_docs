simulate_aero

simulate_aero 复合包含用于 Aero 和燃烧模拟的核心解算器。

用法：

  • 将 aero_solver_settings 节点连接到 settings 输入。（可选）将其他节点（如 aero_combustion_settings、aero_adaptivity_settings、aero_uvw_settings 和 aero_refinement_settings）连接到 aero_solver_settings 节点的 additional_settings 端口。
  • 将源、碰撞对象和影响节点连接到 simulate_aero 的相应输入。

输入

sources

来自 source_air 和 source_fuel 节点的一组对象输出，用于定义向栅格中发射的位置和内容。

colliders

一组 collider 对象，对应于烟将在其周围流动和被其推动的实体对象。

influences

一组用于定义力场和艺术控制的影响场。

settings

应连接到来自 aero_solver_settings 节点的输出对象，并构成基本 Aero 和燃烧设置。

volume_material

要应用于由 Aero 模拟创建的体素体积的材质。

输出

aero_volume

烟和燃烧特性的体素体积。

solver_inputs

源、碰撞对象和其他设置的合并。这可以连接到 post_refine_aero 节点的 solver_inputs 端口，以进一步增强模拟结果。
