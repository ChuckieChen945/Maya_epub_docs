transport_points

This node transports points in a voxel-velocity field.

Inputs

liquid

The input liquid object.

colliders

colliders is an Object or a list of Objects containing voxelized colliders. If colliders holds objects at different times we assume that the list is ordered temporally per collider object such that we can relate the same object at different times by the index in the list and knowledge of the timestamp on a given object.

time

The current time.

time_step

The size of this time-step.

frame_start_time

The time at the start of the frame (useful for sub-frame interpolation).

frame_end_time

The time at the end of the frame (useful for sub-frame interpolation).

time_step_size

The maximum number of voxels a point can move in a single transport step. Lower numbers are more accurate but slower.

min_steps

The minimum number of steps taken by transport.

max_steps

The maximum number of steps taken by transport.

droplet_threshold

The droplet threshold of the liquid object.

Outputs

out_liquid

The output liquid object where points have been transported.
