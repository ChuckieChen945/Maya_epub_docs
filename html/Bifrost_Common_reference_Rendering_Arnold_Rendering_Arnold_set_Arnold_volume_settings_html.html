set_Arnold_volume_settings

设置 Arnold 特定的 volume 设置，以便在 Arnold 中作为 Arnold 原生设置应用。这些设置不影响视口，仅影响 Arnold 渲染。

这些设置与 Arnold 中的对应选项匹配。有关详细信息，请参见 Arnold 用户手册中的 volume 和 volume implicit（等值面作为曲面）相关信息。

用法

它也可以与 set_Arnold_geo_settings 结合使用，后者用于设置对 Arnold 几何体通用的设置。

Bifrost volume 类型对应于两个 Arnold 类型：volume 和 implicit。

Volume 与 Implicit

Volume 用于表示体积，例如云、雾、火或灰尘。

Implicit 用于渲染由体积定义的曲面（例如等值面）。

Arnold 体积

Bifrost 体积的默认模式是转化为 Arnold 体积。

step_size 应该足够小以捕获体积的最小特征，如果设置为零，则是自动的并使用体素大小。step_scale 设置步长比例。

step_scale=20 和 step_size=0 表示步长是体素大小的 20 倍。这可用于软化体积的渲染，而无需大量的额外采样。

volume_padding 也可以在此处手动设置。

特性控件

primary_property 指定 Arnold 将用于渲染的密度特性。默认情况下，这是自动的，无需手动指定。自动行为基于附加的 Arnold 标准体积着色器指定的密度通道。但是，如果存在不同的体积着色器，或没有应用任何着色器，则可以手动指定它。

velocity_property 是 Bifrost-Arnold 用于构建运动模糊体积表示的特性。默认情况下，使用 voxel_velocity。

user_properties 允许指定在转换到 Arnold 时要保留的自定义用户特性。默认情况下，standard volume 中使用的任何内容都将保留。此设置允许覆盖诸如自定义体积着色器等情况。

Arnold Implicit

如果启用了 as_implicit，则将体积渲染为曲面而不是体积。在此处将使用 Arnold implicit 几何体类型而不是体积。

这必须使用等值面来完成，等值面是使用 property（例如 voxel_signed_distance）和 threshold 指定的。这映射到 Arnold implicit 上的 field_channel。

solver 参数设置用于解析曲面的解算器。levelset 解算器必须使用适当的带符号距离场，速度较快，而 uniform 解算器可以利用任何数据渲染曲面，速度较慢。

threshold 定义在体积中渲染曲面的位置。内部值为负且外部值为正的带符号距离场的默认值是 0，但对于密度场，应使用自定义密度值。

samples 控制渲染时用于查找交点的采样数。
