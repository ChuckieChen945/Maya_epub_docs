创建运行时命令

使用运行时命令编辑器，可以为任何自定义脚本创建运行时命令。

运行时命令是由其他命令和/或脚本（例如 MEL 命令、Maya 命令、MEL 过程）组成的命令。可将运行时命令视为“一字包裹器”或别名，允许您使用短标签在 MEL 命令行上执行长的脚本（或脚本集）。可执行的许多（但不是所有）菜单项和图标都使用运行时命令来启动。

所创建的运行时命令将显示在热键列表器的“自定义脚本”(Custom Scripts)类别中。这样，就可以轻松地将热键指定给所创建的任何运行时命令。

有关将热键指定给命令的信息，请参见将热键指定给命令。

  1. 选择“窗口 > 设置/首选项 > 热键编辑器”(Windows > Settings/Preferences > Hotkey Editor)。
  2. 单击热键编辑器右上侧的“运行时命令编辑器”(Runtime Command Editor)选项卡。
  3. 单击“新建”(New)。
  4. 键入新命令的“名称”(Name)和“描述”(Description)。
  5. 通过选择正确的选项，选择脚本是 Python 还是 MEL。
  6. 在文本字段中，键入希望运行时命令执行的脚本命令。
  7. 单击“保存运行时命令”(Save Runtime Command)。将保存新的运行时命令，并在热键列表器的“自定义脚本”(Custom Scripts)类别中列出它。如果需要，可将热键指定给新的运行时命令。请参见将热键指定给命令。

编辑运行时命令

使用位于热键编辑器中的运行时命令编辑器，可以改变或编辑运行时命令。
注： 无法修改任何 Maya 默认运行时命令。
  1. 单击热键编辑器右上侧的“运行时命令编辑器”(Runtime Command Editor)选项卡。
  2. 选择热键列表器中“自定义脚本”(Custom Scripts)类别下的运行时命令，然后在运行时命令编辑器中单击“编辑”(Edit)。

    脚本将显示在运行时命令文本字段中。

  3. 在运行时命令编辑器中单击“编辑”(Edit)。
  4. 编辑您的脚本，然后单击“保存运行时命令”(Save Runtime Command)。
