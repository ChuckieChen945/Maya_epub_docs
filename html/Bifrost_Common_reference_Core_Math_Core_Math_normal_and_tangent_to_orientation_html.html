normal_and_tangent_to_orientation

将两个向量转化为一个方向。这类似于目标约束，其中一个用作“目标”，另一个用作“上方向”向量。

输入

normal

指向方向的正 Y 轴的向量。

tangent

指向方向的 Z 轴的向量。

prioritize_normal

控制两个输入向量不正交时发生的情况。当 prioritize_normal 为 true 时，生成的方向的 Y 轴与 normal 输入完全对齐。当此项为 false 时，Z 轴与 tangent 输入完全对齐。

输出

quaternion

以四元数表示生成的方向的四维向量。
