“绘制流体工具”(Paint Fluids Tool)选项

选择“流体 > 添加/编辑内容 > 绘制流体工具”(Fluids > Add/Edit Contents > Paint Fluids Tool) > 以指定工具设置。以下描述了“绘制属性”(Paint Attributes)、“笔划”(Stroke)和“显示”(Display)区域中的“绘制流体工具”(Paint Fluids Tool)的独特属性。有关所有其他区域中的其他标准属性的描述，请参见 Artisan 工具设置。

绘制属性(Paint Attributes)

自动设置初始状态(Auto Set Initial State)

如果启用该选项（默认），那么在退出“绘制流体工具”(Paint Fluids Tool)、更改当前时间或更改当前选择时，会自动保存流体的当前状态（动态栅格中的任何“密度”(Density)、“速度”(Velocity)、“温度”(Temperature)、“燃料”(Fuel)、“颜色”(Color)或“纹理坐标”(Texture Coordinate)值）。如果禁用该选项，并且在播放或单步执行模拟之前没有设定流体的初始状态，那么原始绘制的值将丢失。

通过选择“字段/解算器 > 初始状态 > 设置初始状态”(Fields/Solvers > Initial State > Set Initial State)，可以随时手动设置初始状态；通过选择“字段/解算器 > 初始状态 > 清除初始状态”(Fields/Solvers > Initial State > Clear Initial State)，可以清除它。有关详细信息，请参见流体初始状态。

可绘制属性(Paintable Attributes)

选择要绘制的属性。

因为要使“燃料”(Fuel)具有效果需要“密度”(Density)，所以可以同时绘制“密度”(Density)和“燃料”(Fuel)值。

同样，因为“颜色”(Color)仅显示在存在“密度”(Density)的栅格中，所以可以同时绘制“密度”(Density)和“颜色”(Color)值。

如果选择“颜色”(Color)，或同时选择“密度”(Density)和“颜色”(Color)，请注意默认栅格颜色是绿色/棕色（接近于 RGB 0.4 0.4 0.3）。如果该颜色不是可接受的栅格颜色，请将所需的颜色整体应用于该颜色栅格，并将其设定为初始状态，请参见将值整体应用到容器和流体初始状态。

默认情况下，绘制属性值时，仅会显示该值。通过将“显示”(Display)区域中的“着色显示”(Shaded Display)更改为“作为渲染”(As Render)，可以在容器中的所有值将渲染时查看它们。

选择“衰减”(Falloff)可为流体的显示绘制衰减边，以便可以阻止流体显示在体积的一部分中。请参见为流体容器定义任意衰减区域。

颜色值(Color Value)

仅当将“颜色”(Color)或者将“密度”(Density)和“颜色”(Color)选作“可绘制属性”(Paintable Attribute)时，该设置才可用。单击“颜色值”(Color Value)框可打开“颜色选择器”(Color Chooser)，然后选择要绘制的颜色。

速度方向(Velocity Direction)

“速度”(Velocity)同时具有幅值和方向。“值”(Value)设置定义所绘制的速度的幅值。使用“速度方向”(Velocity Direction)设置可选择如何定义所绘制的速度笔划的方向。在绘制“速度”(Velocity)时，会显示“速度”(Velocity)向量（在“工具设置”(Tool Settings)窗口的“显示”(Display)区域中启用“绘制速度”(Draw Velocity)）。

来自笔划(From Stroke)

速度向量值的方向来自沿当前绘制切片的笔刷的方向。

按指定(As Specified)

速度向量值的方向来自“指定的 X”(Specified X)、“指定的 Y”(Specified Y)和“指定的 Z”(Specified X)框（选择该选项时，这些框会变得可用）。

绘制操作(Paint Operation)

选择操作，以定义要对绘制的值的影响方式。

替换(Replace)

将绘制的值替换为指定的“值”(Value)和“不透明度”(Opacity)。

相加(Add)

将指定的“明度值”(Value)和“不透明度”(Opacity)与绘制的当前体素值相加。

缩放(Scale)

根据“值”(Value)和“不透明度”(Opacity)系数缩放您绘制的值。

平滑(Smooth)

将值更改为周围值的平均值。

值(Value)

设定执行任意绘制操作时要应用的值。

最小值/最大值(Min/Max Value)

设置可能的最小和最大绘制值。默认情况下，可以绘制介于 0 和 1 之间的值。通过设定“最小值/最大值”(Min/Max Values)，可以扩大或缩小值的范围。负值对于将值相减很有用。例如，如果将“最小值”(Min Value)设定为 -1，将“明度值”(Value)设定为 -0.5，然后选择“相加”(Add)作为操作，则会从绘制的值中减去 0.5。正值用作倍增。

钳制(Clamp)

选择是否要在指定的范围内钳制数值，而不考虑绘制时的“值”(Value)集。

下限(Lower)

启用该选项可将下限值钳制为指定的“钳制值”(Clamp Value)。例如，如果钳制“下限”(Lower)并将下限“钳制值”(Clamp Value)设定为 0.5，那么绘制的值永远不会小于 0.5，即使将“明度值”(Value)设定为 0.25 也是如此。

上限(Upper)

启用该选项可将上限值钳制为指定的“钳制值”(Clamp Value)。例如，如果钳制“上限”(Upper)，将上限“钳制值”(Clamp Value)设定为 0.75，然后将“明度值”(Value)设定为 1，那么绘制的值永远不会大于 0.75。

钳制值(Clamp Values)

设置要钳制的“下限”(Lower)和“上限”(Upper)值。

整体应用(Flood)

单击“整体应用”(Flood)可将笔刷设置应用于选定节点上的所有属性值。结果取决于笔刷、值、绘制操作和执行整体应用时定义的其他笔刷设置。

显示(Display)

使用该区域中的设置可定义笔刷和属性值显示的方式。除了许多标准 Artisan 笔划设置之外，“绘制流体工具”(Paint Fluids Tool)还具有下列选项。

着色显示(Shaded Display)

选择处于着色显示模式时希望属性值显示的方式。

当前属性(Current Attribute)

仅为在“可绘制属性”(Paintable Attributes)中选择的属性显示值。

如果同时绘制“密度”(Density)和“燃料”(Fuel)，那么“密度”(Density)值由着色值的不透明度表示，而“燃料”(Fuel)值由从蓝色（值为 0）到黄色（值为 1）的渐变表示。

如果同时绘制“密度”(Density)和“颜色”(Color)，那么“密度”(Density)值由着色值的不透明度表示，而“颜色”(Color)值由“颜色值”(Color Value)表示。

作为渲染(As Render)

在流体将渲染时显示流体。这与在“属性编辑器”(Attribute Editor)中将“着色显示”(Shaded Display)属性设定为“作为渲染”(As Render)相同。请参见着色显示。

绘制速度(Draw Velocity)

启用该选项可显示速度向量。这与在“属性编辑器”(Attribute Editor)中启用“绘制速度”(Draw Velocity)相同。请参见速度绘制。绘制速度时默认情况下会启用“绘制速度”(Draw Velocity)。
