创建空颜色集选项(Create Empty Color Set Options)

使用以下选项可设置在选择“网格显示 > 创建空颜色集”(Mesh Display > Create Empty Color Set)时发生的情况。

颜色范围(Color Range)
启用“钳制”(Clamped)以将颜色和 Alpha 值限制在 0-1 范围内。
组件类型(Component Type)
指定新颜色集为 RGB、RGBA 或仅为 A (Alpha)。
颜色集共享(Color Set Sharing)
共享(Shared)
对象的每个实例都具有新颜色集。
逐共享实例(Per Instance Shared)
对象的所有选定实例都共享新的颜色集。
逐非共享实例(Per Instance Unshared)
对象的所有选定实例都获得唯一的颜色集。
