安全文件加载

安全首选项中的功能通过防止在导入时未经您同意执行用户标记的特定 MEL 或 Python 命令来确保系统安全：请求您允许以继续操作，或完全拒绝执行命令，然后停止导入操作。安全 MEL 加载(Secure MEL loading)默认情况下处于启用状态，但在打开/导入源自不受信任来源（如 Internet 网站或论坛）的文件之前，我们建议您另外手动启用 Python 安全。

启用 Python 安全

  1. 转到“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)。
  2. 单击“安全 > Python”(Security > Python)部分。
  3. 将“Python”设置为“执行前检查(安全)”(Check before executing (secure))以启用 Python 安全。
  4. （可选）设置所需的“已标记命令的默认操作”(Default action for flagged commands)，以确定 Maya 在遇到已标记命令时如何反应。

加载文件后，可以重复执行相同过程，以禁用“安全 Python 加载”(Secure Python loading)。

自定义 MEL/Python 安全

尽管直接使用默认值来启用/禁用 MEL/Python 安全可提供良好的保护级别，但您还可以自定义每种类型的安全以适合您的需求。您可以在安全首选项子部分中找到这些自定义选项。
注： 仅在启用了“安全 MEL 加载”(Secure MEL loading)并且“Python”设置为“执行前检查(安全)”(Check before executing(secure))时，这些选项才有效。

启用/自定义“安全 Python 内置函数加载”(Secure Python built-in function loading)

  1. 转到“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)。
  2. 单击“安全 > Python”(Security > Python)部分。
  3. 启用“安全的内置函数”(Secure built-in functions)。
  4. （可选）选中或取消选中“已标记内置函数”(Flagged built-in functions)列表中的项，让 Maya 在其尝试执行时向您发出警报。
  5. （可选）设置所需的“已标记命令的默认操作”(Default action for flagged commands)，以确定 Maya 在遇到已标记命令时如何反应。

启用/自定义“安全 Python 模块导入”(Secure Python module importing)

  1. 转到“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)。
  2. 单击“安全 > Python”(Security > Python)部分。
  3. 单击“已标记内置函数”(Flagged built-in functions)列表中相应的框，以标记/取消标记它们（已标记函数旁边将有一个复选标记）。
  4. （可选）单击“受信任的模块”(Trusted modules)旁边的“添加”(Add)按钮，然后添加模块名称以将其设为受信任，或者单击列表中的现有模块，然后单击“移除”(Remove)按钮以将其设为不受信任。
  5. （可选）设置所需的“已标记命令的默认操作”(Default action for flagged commands)，以确定 Maya 在遇到不受信任的模块时如何反应。

启用/自定义“安全 MEL 加载”(Secure MEL loading)

  1. 转到“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)。
  2. 单击“安全 > MEL”(Security > MEL)部分。
  3. 启用“安全 MEL 加载”(Secure MEL loading)。
  4. （可选）单击“已标记命令”(Flagged commands)旁边的“添加”(Add)按钮，然后添加命令名称以将其标记为受信任，或者单击列表中的现有命令，然后单击“移除”(Remove)按钮以将其设为不受信任。
  5. （可选）设置所需的“已标记命令的默认操作”(Default action for flagged commands)，以确定 Maya 在遇到已标记命令时如何反应。
