注释和过程

以下主题介绍一些在 macOS X 中使用 Maya 的特定过程和注释。这些主题介绍在 macOS X 中使用 Maya 时，功能和一般过程的不同之处。

适用于 macOS X 但不适用于 Windows 或 Linux 的功能

以下功能适用于 macOS X 版本的 Maya，但不适用于 Windows 或 Linux 版本。

  • QuickTime 集成
  • 拖动时在时间线上播放线程声音
  • AppleScript 执行 Maya MEL 命令，并返回结果。（另请参见 MEL 和表达式指南中的从 AppleScript 调用 MEL 或相反情况。）
  • 全屏模式（另请参见进入全屏模式。）

不适用于 macOS X 版的第三方程序

  • 运动捕捉驱动程序

在平台之间传递文件

macOS X 文件使用的行结束符不同于 UNIX 或 Windows 文件。macOS X 程序通过兼容其他行结束符来解决这一问题，但 Windows 和 UNIX 程序（包括从 macOS X 终端窗口运行的类似 Unix 的程序）仅使用其各自专有的行结束符形式。

Maya Bin 文件夹中包括两个可以从终端窗口运行的 UNIX 工具，可以帮助用户轻松将文件从一个平台传递至另一个平台：tounix 和 tomac。
