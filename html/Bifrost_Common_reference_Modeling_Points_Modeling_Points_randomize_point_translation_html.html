randomize_point_translation

随机化 point_position 特性。

输入

points

具有要移动的点的几何体。

weights

控制应用的随机化量。

  • 权重为 0 时，不应用随机化。
  • 权重在 0 到 1 范围内时，按 min 和 max 之间的值的一部分进行随机化。
  • 权重为 1 时，完全按 min 和 max 之间的值进行随机化。
  • 权重大于 1 时，点移动的距离大于给定值。
  • 权重在 0 到 1 范围以外时，按 [min, max) 范围以外的量进行随机化。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

min、max

要移动的最小和最大 XYZ 距离。

seed

随机数生成器的种子。

translate_existing_values

当此选项为 true 时，随机平移将与现有 point_position 值相加。当此选项为 false 时，随机值将覆盖现有位置。

local_space

当此选项为 true 时，随机平移将应用于点的局部空间。这意味着它将被定向并缩放到该点或实例。

distribution

用于控制随机分布的 FCurve。默认形状不会向 min 或 max 偏移。

输出

out_points

具有已平移点的几何体。

point_position

在几何体上设置的 point_position 值。
