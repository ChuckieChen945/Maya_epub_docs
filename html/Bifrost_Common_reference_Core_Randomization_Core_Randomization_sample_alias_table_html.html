sample_alias_table

对别名表进行采样。别名表是用于有效计算加权选择的数据结构。此复合用于 randomize_selection_by_probabilities，并且要与 build_alias_table 一起使用。有关别名方法和这些节点的更多文档，请参见该节点。

输入

alias_table

别名表采用 array<float2> 格式，由 build_alias_table 构造。

index

随机数序列中的索引，如 random_value 所示。每个种子都会生成一个不同的随机数序列，索引是在相应序列中的位置。此随机数用于对别名表进行采样。

seed

随机数生成器的种子，如 random_value 所示。
