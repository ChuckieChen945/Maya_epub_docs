get_closest_locations

返回输入几何体曲面上最接近每个输入位置的位置数组。然后，可以使用 sample_property 获取这些位置上的几何体的特性的插值。

输入

geometry

要在其上查找最近位置的几何体。仅支持网格和股对象；其他几何体类型不会产生有用的结果。

positions

要从中查找最近位置的位置数组。

use_cutoff_distance

仅在指定的 cutoff_distance 中搜索。如果最近位置太远，则 found 数组的对应元素为 false。

cutoff_distance

要在其中搜索的距离。

输出

locations

包含位置的数组。您可以将其插入到 sample_property 节点的 locations 端口。

found

布尔值的匹配数组。如果元素为 false，则相应的位置无效，如果使用它对几何体进行采样，将直接返回默认值。这可能是因为中止距离内没有位置，也可能是因为几何体不受支持。
