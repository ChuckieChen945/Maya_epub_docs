“摄影表”(Dope Sheet)大纲视图上下文菜单

在“摄影表”(Dope Sheet)大纲视图中单击鼠标右键时，将显示“摄影表”(Dope Sheet)大纲视图上下文菜单。此菜单中提供的选项与在“摄影表”(Dope Sheet)视图区域中单击鼠标右键时显示的选项不同。有关详细信息，请参见“摄影表”(Dope Sheet)视图区域主题中的“摄影表”(Dope Sheet)视图区域上下文菜单部分。

有关使用摄影表编辑器(Dope Sheet Editor)的信息，请参见使用“摄影表”(Dope Sheet)编辑关键帧。

仅显示 DAG 对象(Show DAG Objects Only)

仅显示可变换节点。有关 DAG 节点的描述，请参见 DAG 层次。

形状(Shapes)
显示与选定对象关联的所有形状。
收拢所有属性/展开所有属性(Collapse All Attributes/Expand All Attributes)
用于显示或隐藏所有属性，以便于导航。根据场景的大小，可能会有延迟。
显示选定项(Reveal Selected)

在视图中查找选定对象的节点。还可以在大纲视图中按 F 键。大纲视图展开层次以显示当前选定的节点。

在大纲视图中隐藏(Hide in Outliner)

选择“在大纲视图中隐藏”(Hide in Outliner)可在大纲视图中隐藏选定节点。

由于隐藏节点在大纲视图中看不到，因此必须先选择此菜单中的忽略“在大纲视图中隐藏”(Ignore 'Hidden In Outliner')暂时显示它，或者从视图面板或其他编辑器中选择它，然后才能将其取消隐藏。再次显示之后，选择该节点，然后选择“取消隐藏”(Unhide)更改其隐藏状态。

引用(Reference)
用于为一个或多个选定对象创建文件引用。文件引用是场景中对象的占位符，使您无需导入文件或对象即可工作。
单击“将当前选择导出为引用”(Export Selection as Reference)以打开将当前选择导出为引用选项(Export Selection as Reference options)
单击“引用 > 创建引用...”(Reference > Create Reference...)以打开文件浏览器，您可以在其中找到要用于替换对象的代理对象。
单击“引用 > 创建引用...”(Reference > Create Reference...) > 以打开文件引用选项(File Reference Options)对话框。
场景集合(Scene Assembly)
用于根据选定对象创建场景集合。通过场景集合，您可以将大型场景分解为较小的组件来进行管理，从而专注于场景的特定部分。有关详细信息，请参见使用场景集合构建场景。
在“摄影表”(Dope Sheet)大纲视图中选择一个对象，而不是属性，然后选择“场景集合 > 创建集合引用...”(Scene Assembly > Create Assembly Reference...)或“场景集合 > 创建集合定义”(Scene Assembly > Create Assembly Definition)。
选择“创建集合引用...”(Create Assembly Reference...)会打开“设置定义文件”(Set Definition File)对话框，您可以在其中选择要引用的集合定义文件。请参见使用集合引用来集合场景。
通道集(Channel Sets)
打开“摄影表”(Dope Sheet)的“通道集”(Channel Sets)子菜单：
选项 操作
创建/删除通道集(Create/Delete Channel Set) 将通道集添加到摄影表/从摄影表中移除通道集。
将当前选择添加到集/从集中移除当前选择(Add/Remove Selection to Set) 将卷展栏中显示的选定对象添加到摄影表通道集/从中移除选定对象。
选择集成员(Select Set Members) 将常规选择设置为通道集所包含属性的父节点（变换）。
重置通道集颜色(Reset Channel Set Colors) 将通道颜色重置为最初指定的颜色。（若要更改通道集颜色，请参见在摄影表中创建通道集中的更改通道集颜色。）
导入(Import)... 打开“导入通道集”(Import Channel Sets)窗口，从中可以找到已保存的通道集（.JSON 文件）并将其加载到场景中。请参见在摄影表中创建通道集中的导入通道集。
导出全部...(Export All...) 将加载到“摄影表”(Dope Sheet)大纲视图中的所有通道集保存到所选位置。请参见在摄影表中创建通道集中的导出通道集。
导出当前选择...(Export Selection...) 将选定的通道集保存到所选位置。请参见在摄影表中创建通道集中的导出通道集。
注： 如果某个选项在“通道集”(Channel Sets)子菜单中已禁用，原因是该选项不可用。例如，如果未创建通道集，则无法删除通道集。
资产(Asset)
“选择资产内容”(Select Asset Contents)可用于选择此资产节点内包含的所有元素。它们将在大纲视图中亮显。

有关详细信息，请参见资产。

显示(Display)

“摄影表”(Dope Sheet)大纲视图上下文菜单中的“显示”(Display)菜单可用于选择要在“摄影表”(Dope Sheet)大纲视图中显示的内容。该菜单包含与 Maya“大纲视图”(Outliner)主题相同的选项。

显示(Show)

“摄影表”(Dope Sheet)大纲视图上下文菜单中的“显示”(Show)菜单可用于选择要在“摄影表”(Dope Sheet)大纲视图中显示的内容。该菜单包含与 Maya“大纲视图”(Outliner)主题相同的选项。

渲染设置(Render Setup)
如果存在渲染层，则可以使用“大纲视图”(Outliner)查找对象所属的渲染层集合。
  • 使用“在所有渲染层中查找”(Find in All Render Layers)和“在选定渲染层中查找”(Find In Selected Render Layers)可查找此对象所属的渲染层集合。
  • 使用“在渲染层中查找”(Find in Render Layer)菜单可显示场景中的所有渲染层。
