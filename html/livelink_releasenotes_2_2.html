适用于 Maya 的 Unreal Live Link 2.2.0 发行说明

发布日期：2023 年 3 月 1 日

适用于 Autodesk® Maya® 的 Unreal® Live Link 2.2.0 版支持 Unreal Engine 5.1。

适用于 Maya 的 Unreal Live Link 的最新可用版本为 2.4.1。访问官方 Autodesk 下载页面 以获取最新版本。

兼容性

此版本的插件与以下各项兼容：

  • Maya 2020、2022 和 2023
  • Unreal Engine 5.1

2.2.0 版新特性：最新的 Live Link 功能

适用于 Autodesk Maya 的 Unreal Live Link 已重新编译，可支持 Unreal Engine 5.1。它不支持 Unreal Engine 5.0 或更低版本。
