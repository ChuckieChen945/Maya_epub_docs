动画层编辑器(Animation Layer Editor)

若要访问“动画层编辑器”(Animation Layer Editor)，请执行以下操作：

  • 切换到“通道盒”(Channel Box)下面的“动画”(Anim)选项卡。
  • 若要将“动画层编辑器”(Animation Layer Editor)作为单独的窗口打开，请切换到“通道盒”(Channel Box)底部的“动画”(Anim)选项卡，然后从“动画层编辑器”(Animation Layer Editor)菜单栏中选择“显示 > 浮动窗口”(Show > Floating Window)。
“动画层编辑器”(Animation Layer Editor)可用于操纵动画层和更改动画层融合在一起以创建结果动画的方式。
注： 您也可以在时间编辑器片段上创建层，以便修改现有片段动画。请参见将片段层添加到时间编辑器。时间编辑器片段层的行为与 Maya 动画层类似，但这些层专用于时间编辑器片段，因为您无法同时使用动画层和片段层驱动时间编辑器动画。

“动画层编辑器”(Animation Layer Editor)包含菜单栏、工具栏和动画层窗格。

  • “动画层编辑器”(Animation Layer Editor)菜单栏
  • 动画层工具栏
  • 动画层窗格

更改动画层模式。

在层上单击鼠标右键，选择“层模式”(Layer Mode)，然后选择“相加”(Additive)、“覆盖”(Override)或“通过”(Passthrough)。有关信息，请参见动画层模式。

将动画从场景中的选定对象提取到某一层。

在“基础动画”(BaseAnimation)栏上单击鼠标右键，并选择“为选定对象提取未分层的动画”(Extract Non-Layered Animation for Selected Objects)。

重新排列堆栈中的层。

选择该层，并使用鼠标中键将其拖到堆栈中的新位置，也可使用上移 和下移 按钮。

父层。

选择层，并使用鼠标中键将其拖动到希望作为其父层的层上。

打开“关系编辑器”(Relationship Editor)添加或移除层的属性。

从菜单栏中选择“层 > 成员身份”(Layers > Membership)。

在“大纲视图”(Outliner)中显示资产。

在“动画”(Animation)首选项（在“设置”(Settings)下）的“曲线图编辑器”(Graph Editor)部分中，启用“显示资产”(Display Assets)。

删除层。

选择要删除的层，然后从菜单栏中选择“层 > 删除”(Layers > Delete)。
