emit_liquid

此节点细化体积并将液体点发射到液体对象中。

输入

liquid

液体对象。

sources

液体源。

colliders

碰撞对象。

frame

当前帧。

time

当前时间。

time_step

当前时间步。

velocity_time_step

用于速度的时间步。某些时间积分算法需要 velocity_time_step 与 time_step 不同。

frame_start_time

当前帧的开始时间。

frame_end_time

当前帧的结束时间。

surface_bandwidth

液体表面的宽度（以体素为单位）。在此带宽内应用 surface_point_density，在其他位置应用 interior_point_density。

interior_point_density

液体整个内部体积中的点密度。每个体素的点计数是此值的立方。例如，如果指定 2，则每个体素的粒子数为 8 (2^3 = 8)。请勿使用小于 1.0 的值。

surface_point_density

液体表面上的点密度。您可以使用此值进行密度较大的点分布，同时在液体表面上显示更多细节，而对细节没有要求的内部无需提供额外点。对于 interior_point_density，每个体素的点计数是此值的立方。请勿使用小于 1.0 的值。

输出

out_liquid

输出液体对象。
