XGen“预览设置”(Preview Settings)

开始百分比(Start Percent)

控制为预览生成基本体前计算和丢弃的基本体位置的百分比。如果以特定顺序（非随机）创建基本体，请使用此设置。

百分比(Percent)

控制要在预览中显示的生成的基本体的百分比。调低后，将加快刷新速度，同时提供最终渲染的良好表示。范围从 100 到“开始百分比”(Start Percent)。

限制(Limit)

设置在预览中生成的基本体的最大数量。

使用此选项来减少大型多边形对象上生成的基本体数量，以避免 Maya 的性能问题。

仅视图中的基本体(Only primitives in the view)

在摄影机视图（视锥）中预览基本体。这不会影响最终渲染。

视图边距(View Margin)

控制基本体位于视图外但仍将包括在预览中的距离（以度为单位）。可将此选项与“仅视图中的基本体”(Only primitives in the view)一起使用。

绘制(Draw)
面片名称(Patch Names)
显示视图中的面片名称。
重要： 选择几何体或描述，然后再激活“面片名称”(Patch Names)，否则它们将不显示。
面 ID(Face IDs)
在视图中显示面 ID。您可以使用该选项来确定网格的 XGen 面 ID 是否不同于 Maya 面 ID。在使用非仅四边形网格时，可能就会出现这种情况。XGen 会将非四边形面细分为四边形，并指定这些子面的面 ID。
重要： 选择几何体或描述，然后再激活“面 ID”(Face IDs)，否则它们将不显示。
基本体 ID(Primitive IDs)

显示预览中的每个基本体的 ID 编号。基本体按面编号。

顶点(Vertices)

显示基本体上的顶点。

显示的顶点并不总是计算的基本体的顶点。启用“多边形”(Poly)将更准确地显示顶点。

多边形(Poly)

将基本体显示为未经平滑的简单多边形。

已消隐(Culled)

显示预览中的已消隐基本体。

单位立方体(Unit Cube)

显示围绕归档文件内容的单位立方体。这仅适用于“归档”(Archive)基本体。

基本体 ID 位于(Primitive IDs At)

控制沿基本体显示 ID 的位置。0 表示基本体的底端，1 表示尖端。

此选项仅在启用“基本体 ID”(Primitive IDs)时启用。

基本体颜色(Primitive Color)
设置用于显示基本体的颜色。当设置为 0.0 时，基本体将采用其默认着色器颜色进行显示。
导向颜色(Guide Color)
设置用于显示导向的颜色。
样条线段(Spline Segments)

通过增加对样条线中每个 CV 绘制的线性分段数来平滑预览的样条线。较高的值会降低预览生成速度。您可以使用“样条线段”(Spline Segment)作为生成预览时的样条线采样值。
