blend_matrix

将变换矩阵与指定目标混合。

输入

transform

要解算的 double4x4 变换矩阵。

targets

包含要与输入 transform 混合的目标结构的 array<T>。

enable

如果启用此项，将处理此节点。否则，将跳过任何计算并按原样输出输入 transform。

weight

用于控制此节点对 transform 的影响的权重。

值为 0.0 时，将按原样传递 transform；值为 1.0 时，将完全应用节点的影响；值介于 0.0 和 1.0 之间时，将在输入和修改的 transform 之间混合。

输出

out_transform

解算的 transform。
