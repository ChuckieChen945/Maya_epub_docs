fog_density_to_level_set

此节点将输入雾密度体积转化为等值面体积。计算出的 voxel_signed_distance 将添加到现有 fog_density 体积。如果输入体积已包含等值面，则将替换旧的 voxel_signed_distance 值。

输入

fog_density

具有 voxel_fog_density 特性的体积。

density_threshold

定义曲面（等值面）所在位置的密度值。值越高，曲面将收缩到密度越大的区域。

smooth

启用后，如果密度连续分级，则等值面将更加平滑。密度阈值也有助于实现平滑效果。

输出

volume

已添加计算出的 voxel_signed_distance 的输入体积。
