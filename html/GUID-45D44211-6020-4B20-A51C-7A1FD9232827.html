创建 UV 集

UV 集最快的创建方法就是映射 UV。映射 UV 时，应打开选项窗口，然后启用“创建新 UV 集”(Create New UV Set)并在“UV 集名称”(UV Set Name)框中键入名称。例如，如果创建两个平面映射（一个从顶部映射，一个从侧边映射），那么每次映射都需键入新的 UV 集名称。

或者可以在开始时使用空 UV 集，然后创建 UV。

创建空 UV 集

  1. 选择对象，然后打开 UV 编辑器(UV Editor)。
  2. 在“UV 编辑器”(UV Editor)菜单中，选择“UV 集 > 创建空 UV 集”(UV Sets > Create Empty UV Set) > 。将显示创建 UV 集选项(Create UV Set options)窗口。
  3. 在“创建 UV 集名称”(Create UV Set Name)框中，键入空集的名称并单击“创建”(Create)。
  4. 选择新的 UV 集（请参见在 UV 集之间切换）。
  5. 使用其中一种映射操作创建 UV，或通过复制创建 UV。
