使用 OpenSubdiv 预览平滑网格

默认情况下，平滑网格预览使用“细分方法”(Subdivision Method)，该方法在“建模首选项”(Modeling preferences)中设置。此设置为全局首选项，会影响场景中所有新的和现有的对象。

提示： 在属性编辑器中，选择 polyShape 选项卡，打开“平滑网格”(Smooth Mesh)面板，然后禁用“使用全局细分方法”(Use global subdivision method)选项。这样做将覆盖全局首选项，因此可以基于每个对象设置“细分方法”(Subdivision Method)。

使用 OpenSubdiv 预览平滑网格

  1. 选择您要平滑的网格。
  2. 执行下列操作之一：
    • 按 1 键可禁用“平滑网格预览”(Smooth Mesh Preview)。
    • 按 2 键可启用“框架 + 平滑网格”(Cage + Smooth Mesh)显示模式。

      Maya 会同时显示原始网格（作为线框框架）和平滑预览，这样您可以选择任一版本网格上的组件。

    • 按 3 键可启用“平滑网格”(Smooth Mesh)显示模式。

      Maya 仅显示平滑预览。当您直接在平滑预览上选择和编辑组件时，更改将反映回原始网格上。

  3. 在“属性编辑器”(Attribute Editor)中，选择“polyShape”选项卡 >“平滑网格 > OpenSubdiv 控件”(Smooth Mesh > OpenSubdiv Controls)。

    调整显示的 OpenSubdiv 控件(OpenSubdiv Controls)。

  4. （可选）将平滑网格预览转化为多边形。
