erase_component_properties

删除以几何对象的组件为目标的几何特性。

输入

geometry

要修改的几何体。

component

要从中删除几何特性的目标组件。例如，使用 point_component 从几何体的点删除几何特性。

properties

按名称指定要删除的几何特性，例如 point_color。

  • 使用空格分隔多个几何特性名称。
  • 您可以在表达式中使用通配符 *，例如使用 point_user* 来删除所有以“point_user”开头的特性。
  • 默认值 * 将删除指定组件的所有几何特性，这可能会根据其模式使几何体无效。
  • 使用 * 后跟 ! 可排除特性。例如，* !point_user* 将包括以字符串“point_user”开头的特性以外的所有特性

输出

out_geometry

删除了指定特性的几何体。
