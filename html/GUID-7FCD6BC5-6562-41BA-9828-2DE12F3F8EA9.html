“初始状态”(Initial State)子菜单

为选定对象设置
为选定的动力学对象设置初始状态。
为所有动力学对象设置
为场景中的所有动力学对象设置初始状态。
清除初始状态

从选定的动力学对象（Nucleus 和流体对象）中清除现有的初始状态，并将对象重置为其原始状态。对于 nCloth，它将布料设置为其输入网格形状。对于发射的 nParticle，它将对象设置回存到预发射状态。

Maya 会删除 initialState_fluidShape 节点，但不会删除初始状态缓存文件（如果它正由另一个场景使用）。当转到模拟开始时，容器的内容将为空。

注： 您可能需要倒至模拟开头，以查看“清除初始状态”(Clear Initial State)效果。

Nucleus

从网格设置
将要进行模拟的选定 nCloth 的初始状态（或开始位置），设置为其他选定网格（具有匹配拓扑）的状态。
松弛初始状态
模拟选定 Nucleus 对象指定数量的帧，然后将其当前状态保存为初始状态。对于 nCloth，如果想要在其模型上设置 nCloth 布料，以便使其看起来受重力影响，而不必播放整个场景的动画，“松弛初始状态”(Relax Initial State)将十分有用。

选择“字段/解算器 > 初始状态 > 松弛初始状态”(Fields/Solvers > Initial State > Relax Initial State) > 以设置松弛初始状态选项。

解决穿透

选择“字段/解算器 > 初始状态 > 解决穿透”(Fields/Solvers > Initial State > Resolve Interpenetrations)> 以设置解决穿透选项。

流体

设置初始状态

使用“设置初始状态选项”(Set Initial State Options)窗口，可以设定流体初始状态的内容。将以初始状态设定在“设置初始状态选项”(Set Initial State Options)窗口中选中的内容。默认情况下，启用所有初始状态内容选项。

从动态或静态栅格内容创建流体的初始状态。设定初始状态时，如果内容方法具有动态或静态栅格属性，但未在“设置初始状态选项”(Set Initial State Options)窗口中选中，则会出现一则警告，提示您应包括这些栅格属性或从缓存中略去它们。

将状态另存为示例
可用于将流体的当前状态另存为 MA、MB 或 MOV 文件。
