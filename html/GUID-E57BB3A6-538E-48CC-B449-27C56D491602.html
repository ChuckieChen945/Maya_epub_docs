从现有曲线创建交互式修饰头发、导向和线

您可以使用当前 Maya 场景中的曲线或 Alembic 缓存文件来生成头发、导向和线。这些曲线可以源自 Maya 之外的内容创建应用程序，它们可以是静态的也可以是设置动画的。

继续操作之前，请确保您已完成下列操作：

  • 创建了交互式修饰描述。请参见使用交互式修饰工具创建头发和毛发。

    您可以使用默认设置创建头发。添加的曲线将覆盖其中许多设置。

  • 向描述中添加了“曲线到样条线”(Curve to Spline)或“导向”(Guide)修改器。请参见使用交互式修饰修改器。

使用“曲线到样条线”(Curve to Spline)修改器从现有曲线创建头发

从场景中的曲线创建头发

  1. 选择场景中的曲线。
  2. 在“XGen 交互式修饰编辑器”(XGen Interactive Groom Editor)中，按 Ctrl 键并单击 curveToSpline 修改器。
  3. 在修改器的“属性编辑器”(Attribute Editor)中，为“输入源”(Input From)启用“曲线”(Curves)并禁用“缓存”(Cache)。
  4. 单击“添加选定曲线”(Add Selected Curves)。

    5. Maya 将从选定曲线创建交互式修饰头发。 在“属性编辑器”(Attribute Editor)中，“输入”(Input)列出了“曲线到样条线”(Curve to Spline)节点上的输入窗口的名称，而“曲线名称”(Curve Name)列出了附加到输入的曲线。

    通过在“输入”(Input)或“曲线名称”(Curve Name)下选择一条曲线，然后单击“添加选定曲线”(Add Selected Curve)，即可移除该曲线。

从 Alembic 缓存文件中的曲线创建头发

  1. 在“XGen 交互式修饰编辑器”(XGen Interactive Groom Editor)中，选择 curveToSpline 修改器。
  2. 在修改器的“属性编辑器”(Attribute Editor)中，为“输入源”(Input From)禁用“曲线”(Curves)并启用“缓存”(Cache)。
  3. 在“输入缓存”(Input Cache)部分中，单击浏览图标并找到包含曲线的 Alembic 缓存文件。

    Maya 将从 Alembic 缓存文件中的曲线创建交互式修饰样条线。

从现有曲线创建导向

在此工作流中，您可以使用“线性线”(Linear Wire)修改器，而不是“导向”(Guide)修改器，但是某些设置可能会有所不同。

从场景中的曲线创建导向

  1. 选择场景中的曲线。
  2. 在“XGen 交互式修饰编辑器”(XGen Interactive Groom Editor)中，按 Ctrl 键并单击“导向”(Guide)修改器。
  3. 在“属性编辑器”(Attribute Editor)的“输入导向”(Input Guide)部分中，单击“使用选定曲线作为导向”(Use Selected Curves as Guide)。
  4. 在“使用选定曲线作为导向的选项”(Use Selected Curves as Guide Options)窗口中，选择以下选项之一：
    • “删除曲线”(Delete Curves)可从场景中永久删除源曲线。
    • “保留动态链接”(Preserve Dynamic Link)可保持源曲线和目标导向之间的链接。这样，曲线编辑和动画将自动应用于导向。将一组曲线用作多个描述中导向的源时，此选项很有用。

      使用 nHair 动态输出曲线驱动导向形状和动画时，启用“保留动态链接”(Preserve Dynamic Link)。

    Maya 将创建 transform_fromCurve 子描述节点和 xgmCurveToSpline 节点，用于根据曲线的形状和动画生成导向。

从 Alembic 缓存文件中的曲线创建导向

  1. 在“交互式修饰编辑器”(Interactive Groom Editor)中，选择“导向”(Guide)修改器。
  2. 在“属性编辑器”(Attribute Editor)中，找到“输入导向”(Input Guide)属性组，并单击“创建”(Create)。
  3. 在“交互式修饰编辑器”(Interactive Groom Editor)中，展开“导向”(Guide)修改器，并选择 InGuide 节点。
  4. 选择“添加修改器”(Add Modifier)，然后选择“曲线到样条线”(Curve to Spline)。
  5. 执行上述“从 Alembic 文件中的曲线创建头发”中的步骤，以便从 Alembic 缓存文件中的曲线生成导向。

继续修改修饰

通过执行以下操作，您可以继续修改修饰：

  • 使用交互式修饰工具雕刻头发。为此，请将“雕刻”(Sculpt)修改器放在堆栈中的“曲线到样条线”(Curve to Spline)或“导向”(Guide)修改器上方，选择该修改器，然后开始使用交互式修饰工具进行雕刻。

    请参见使用雕刻层。

  • 添加多个修改器以定形和控制头发。请参见 XGen 交互式修饰修改器。
  • 使用该描述中的头发定形其他描述中的头发。请参见使用导向或线性线修改器控制多个描述。
  • 通过添加“线性线”(Linear Wire)或“导向”(Guide)修改器模拟头发。请参见使用线性线修改器模拟头发。
