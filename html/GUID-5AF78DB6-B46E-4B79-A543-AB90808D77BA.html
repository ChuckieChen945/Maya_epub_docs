三角化数学节点

Acos
输出给定输入的反余弦。
反余弦是其余弦为给定输入的角度。
输出范围为 0 到 PI。
Asin
输出给定输入的反正弦。
反正弦函数是正弦函数的反函数。
输出范围为 -PI/2 到 PI/2。
Atan
输出给定输入的反正切。
反正切是其正切为给定输入的角度。
输出以弧度为单位，范围为 -PI/2 到 PI/2。
使用 atan2 节点可获取 -PI 到 PI 范围内的角度。
Atan2
输出给定输入的反正切。
反正切是其正切为给定输入的角度。它计算角度 a 的明确值：

从笛卡尔坐标 (x, y) 转化为极坐标 (r, a)，其中 x = r * cos(a)，y = r * sin(a)，r^2 = x^2 + y^2

输出以弧度为单位，范围为 -PI 到 PI。

Cos
以弧度为单位输出给定角度的余弦。
角度的余弦为邻边长度与斜边长度之比。
Pi
将数字 PI 常数输出为角度（180 度）。
Sin
以弧度为单位输出给定角度的正弦。
角度的正弦为对边长度与斜边长度之比。
Tan
以弧度为单位输出给定角度的正切。
角度的正切为对边长度与斜边长度之比。
