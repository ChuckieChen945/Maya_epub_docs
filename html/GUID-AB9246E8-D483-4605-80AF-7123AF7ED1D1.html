FCheck 选项列表

重要： 自 Maya 2020 起，FCheck 在 macOS 上不再受支持。可以在应用程序首选项（“窗口 > 设置/首选项 > 首选项”(Windows > Settings/Preferences > Preferences)）中选择其他默认图像查看器。

有关如何使用“FCheck”选项的详细信息，请参见 FCheck 选项。

Windows Linux 描述

-a

-a

显示所有块，提供每个分片（与 -c 选项一起使用）的坐标和权重。

注意：块和分片是 Maya 在其渲染图像过程中使用的图像空间。

 

-B

禁用图像周围的边界。

 

-C <x> <y> <宽度> <高度>

裁剪图像。<x> 和 <y> 表示左下方起始平面点。使用 <宽度> 和 <高度> 设置图像的大小。

-c

-c

检查显示在序列中的 .iff 标记的完成度。如果序列不完整或已损坏，“FCheck”将打印错误。无需打开序列即可完成该操作。

 

-D

将计算机设置为使用立体模式。但是，如果立体模式没有激活，则需要退出“FCheck”，以便还原视频模式。

-d

-d

使用全屏窗口以演示模式运行。

-F

-F

以完全分辨率显示图像可以防止自动重缩放大图像。

-f

-f

查找动画中的第一个可用图像。

-G <gamma>

-G <gamma>

在 RLA 图像中设定 gamma 值。使用 <gamma> 指定值。

-g

-g

初始 gamma 校正，可随时更改。

-h

-h

列出“FCheck”功能命令（帮助）。

-K

-K

反转 Kodak Cineon 颜色校正（记录到线性）。

-k <mem>

-k <mem>

将图像存储或保存在内存中，使用在 <mem> 中定义为可用内存百分比的内存量。值 1 表示分配了 100% 的内存。

-l <level>

-l <level>

复杂 IFF（图像文件格式）样式文件的分析级别。

-m <mag>

-m <mag>

放大图像。<mag> 是一个指定放大因子的整数。

-n <start> <interval>

<end> <filename>

-n <start> <interval>

<end> <filename>

打开动画，指明文件的开始、结束和步长（整数）。如果给定的名称以时间结尾，“FCheck”将自动查找相应动画的开始、结束和步长。

有关详细信息，请参见使用 -n 选项打开动画。

-p

-p

使用零填充帧编号，某些图像格式需要这样。

-r <rate>

-r <rate>

动画的显示速率。默认值为 ntsc，但是您也可以指定 pal、film 或帧每秒整型值（例如，15）。

 

-R <x> <y>

通过减小图像的大小来更改其分辨率。<x> 和 <y> 分别表示新的宽度和高度值。

-s <shrink>

-s <shrink>

减小图像。<shrink> 是一个指定减小因子的整数。

-S

-S

显示没有按任何特定顺序排列的图像序列，创建图像、动画和声音的循环。

-T <timer>

-T <timer>

设定每个帧的显示秒数。该选项与 ‑S 选项组合使用。

-v

-v

详细模式允许最大信息输出。

-w

-w

等待直到图像完成渲染。图像仅在渲染完成后显示。

-z

-z

防止加载图像的 Z 深度信息。没有足够内存时，使用该选项非常有用。

-Z

-Z

在动画模式下加载图像的 Z 深度信息。

在 Windows 上，可以通过从“FCheck”窗口中选择“帮助 > 键盘”(Help > Keyboard)来访问键盘快捷方式。
