创建或删除折痕集

折痕集是要向其应用同一折痕值的多边形边或顶点集。

创建新的折痕集

  1. 选择要包含在折痕集中的组件（多边形边或顶点）。
  2. 选择“窗口 > 建模编辑器 > 折痕集”(Windows > Modeling Editors > Crease Sets)。

    此时将显示“折痕集编辑器”(Crease Set Editor)。

  3. 执行下列操作之一：
    • 单击“折痕集编辑器”(Crease Set Editor)操作按钮中的“新建”(New)。
    • 在编辑器中的任意位置单击鼠标右键，然后从右键单击菜单中选择“创建集”(Create Set)。
      新的折痕集将出现在折痕集列表中。
      提示： 要更改默认的名称，请选择“设置 > 默认折痕集名称”(Settings > Default Crease Set Name)，并输入新的名称，然后再创建折痕集。
提示： 指定颜色以区分场景中的折痕集。若要执行此操作，请使用“折痕集编辑器”(Crease Set Editor)右键单击菜单中的“设置颜色”(Set Color)或启用“属性编辑器”(Attribute Editor)中的“设置选项 > 覆盖默认组件颜色”(Set Options > Override Default Component Color)，然后为折痕集组件选择“组件颜色”(Component Color)。

删除折痕集

  1. 在“折痕集编辑器”(Crease Set Editor)中，从“折痕集”列表中选择折痕集。
  2. 执行下列操作之一：
    • 单击“删除”(Delete)。
    • 在要删除的折痕集上单击鼠标右键，然后从上下文相关菜单中选择“删除集”(Delete Set)。
