准备包含日语或简体中文文本的 Maya 文件

如果在使用日语/简体中文的操作系统上运行 Maya，并且需要与其他平台上的用户共享 Maya ASCII 文本文件（.ma 和 .mel），则需要将 Maya 文件转换为适合目标平台的正确文本编码格式。可使用常用文本编辑器或工具（如 iconv）的“另存为”功能。文本编码依赖于平台，如不适用于平台则无法保证 Maya 文本文件可以正确打开。

如果在 Maya 日语或简体中文语言用户界面中使用 .mb（Maya 二进制）格式保存 Maya 文件，则这些文件无需转换即可在各种受支持的平台之间共享。

如果需要保存和使用包含日语或简体中文的 Maya 文本文件（.ma 和 .mel），则必须根据平台使用下列文本编码保存文本文件：

操作系统 语言 支持的文本编码 (.ma .mel)

Windows

日语

Shift_JIS (CP932)

Windows

简体中文

GBK (CP936)

macOS

日语/简体中文

UTF-8
