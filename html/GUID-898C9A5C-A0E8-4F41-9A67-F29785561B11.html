如何执行在运行时创建的语句？

使用 eval 命令。使用 eval 命令可以执行在运行时构建的字符串。

例如：

switch($timeOfDay) {
 case "morning":
 $shape = "circle";
 break;
 case "afternoon":
 $shape = "sphere";
 break;
 case "evening":
 $shape = "cone";
 break;
 default:
 $shape = "cylinder";
}
eval $shape -r 5; // create specified shape with radius 5.

或者可以使用具有以下函数语法的 eval：

eval ($shape+" -r 5");

也可以使用 evalEcho 和 evalDeferred 命令。
