update_fv_indices_by_angle

此节点将基于相邻面之间的夹角更新网格的法线索引。当夹角超过给定阈值时，相邻面的法线将分离。当夹角低于阈值时，各个面将共享一条法线。

输入

fv_indices

现有法线索引数组。

mesh

要为其生成法线索引的网格。

angle_between_faces

角度阈值（以度为单位）。

face_indices

要为其生成新索引的面。

输出

updated_fv_indices

新的面顶点索引。

num_data_elements

face_vertex_normal_index 数组引用的法线数。这对于调整相应法线数组的大小非常有用。
