aim_matrix

确定变换矩阵的方向，使其指定主轴和次轴指向指定的主要目标和次要目标。

输入

transform

要解算的 double4x4 变换矩阵。

enable

如果启用此项，将处理此节点。否则，将跳过任何计算并按原样输出输入 transforms。

weight

用于控制此节点对 transform 的影响的权重。

值为 0.0 时，将按原样传递 transform；值为 1.0 时，将完全应用节点的影响；值介于 0.0 和 1.0 之间时，将在输入和修改的 transform 之间混合。

primary_mode

处理输入变换的 primary_input_axis 的对准方向的方法。

  • None：按原样使用轴，不受目标影响。
  • Aim：使用目标的位置确定轴的方向。
  • Align：仅使用目标的方向确定轴的方向。必须设置 primary_target_vector 才能定义要对齐的目标轴。

primary_input_axis

要对准主要目标的输入 transform 的轴。

primary_target_vector

用于使输入 transform 的 primary_input_axis 朝向主要目标的主要目标向量（在 primary_target_matrix 的空间中）。对此向量的解释取决于选定的 primary_mode：

  • Aim 对 primary_target_matrix 应用位置偏移。它们共同定义 primary_input_axis 应指向的最终位置。
  • Align 指定 primary_input_axis 应与其对齐的 primary_target_matrix 的局部向量。

primary_target_matrix

表示主要目标位置的变换矩阵。仅使用矩阵的平移部分推断目标向量。

secondary_mode

处理输入变换的 secondary_input_axis 的对准方向的方法。

  • None：按原样使用轴，不受目标影响。
  • Aim：使用目标的位置确定轴的方向。
  • Align：使用目标的方向确定轴的方向。必须设置 secondary_target_vector 才能定义要对齐的目标轴。

secondary_input_axis

要对准次要目标的输入 transform 的轴。

secondary_target_vector

用于使输入 transform 的 secondary_input_axis 朝向次要目标的次要目标向量（在 secondary_target_matrix 的空间中）。对此向量的解释取决于选定的 primary_mode：

  • Aim 对 secondary_target_matrix 应用位置偏移。它们共同定义 secondary_input_axis 应指向的最终位置。
  • Align 指定 secondary_input_axis 应与其对齐的 secondary_target_matrix 的局部向量。

secondary_target_matrix

表示次要目标位置的变换矩阵。仅使用矩阵的平移部分推断目标向量。

输出

out_transform

解算的 transform。
