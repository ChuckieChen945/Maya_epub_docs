disconnect_mesh_faces

创建一个网格，其中面不再共享顶点并且每个面都是断开连接的。这意味着，新网格将针对旧网格的每个面顶点具有一个点。

输入

mesh

输入网格。

weights

控制应用的收缩量。此值将乘以 shrink 输入。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<bool> 解释为每点的 0 (false) 或 1 (true) 值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

shrink

朝每个面的中心向内收缩网格面。

properties

要传递到输出网格的特性。这是要传递的基于表达式的特性列表，对于特定特性，请使用以空格分隔的列表：
point_normal point_color。

若要传递每个传入特性，请使用 *。

若要排除特定特性，请使用 !：
* !point_normal。

默认为 *，表示将传递所有受支持的传入特性。

输出

disconnected_mesh

生成的断开连接的网格。
