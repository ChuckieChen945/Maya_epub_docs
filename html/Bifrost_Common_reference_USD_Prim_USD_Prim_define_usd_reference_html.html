define_usd_reference

定义 USD 参照。可以将输出连接到 define_usd_prim 和类似节点的 reference_definitions 端口。

输入

arc_type

合成圆弧的类型。

prim_path

被参照的基本体的路径。

relative_prim_path

基本体路径是否为相对路径。

layer

连接层（可选）。

layer_offset

要应用于动画的时间偏移。

layer_scale

要应用于动画的时间比例。

position

要添加参照的列表位置。

anchor_path

在默认的 USD 资产解析器 ArDefaultResolver 中，参照可以相对于“锚点”路径，这样，参照就可以相对于较大的结构。如果 file 仅是此路径的结尾部分，则 anchor_path 是此路径的开头部分。

输出

reference_definitions

参照定义。可以将此项连接到 define_usd_prim 和类似节点的 reference_definition 端口。

父页面: Prim
