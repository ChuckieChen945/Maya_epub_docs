“骨架”(Skeleton)菜单

位于“绑定”(Rigging)菜单集中。

关节(Joints)

注： 可以通过选择关节并在属性(Attribute)或节点编辑器(Node Editor)中查看，找到其他关节选项，例如“绘制样式”(Draw Style)、“关节方向”(Joint Orient)和“自由度”(Degrees of Freedom)。
创建关节(Create Joints)

打开关节工具(Joint Tool)。

插入关节(Insert Joints)
打开插入关节工具(Insert Joint Tool)。
镜像关节(Mirror Joints)
跨指定平面复制当前父对象的关节链。请参见镜像关节链和镜像关节链。
选择“骨架 > 镜像关节”(Skeleton > Mirror Joint) > 以打开镜像关节选项(Mirror Joint Options)，可以在此指定镜像关节所跨的平面以及生成的重复关节的方向。
确定关节方向(Orient Joint)
若要打开并更改确定关节方向选项(Orient Joint Options)选项，请单击“骨架 > 确定关节方向”(Skeleton > Orient Joint) > 。
移除关节(Remove Joints)
删除当前关节，并将其余关节和骨骼联接为一个关节链。请参见移除关节。
连接关节

连接选定关节。请参见连接关节。

选择“骨架 > 连接关节”(Skeleton > Connect Joint) > 以打开“连接关节”(Connect Joint)选项，然后选择以下选项之一：

连接关节
连接选定关节，并将第一个关节的关节链移动到第二个关节的位置。连接关节后，使用“移除关节”(Remove Joints)以删除额外关节。
将关节设为父子关系(Parent joint)
通过在选定关节之间插入新的骨骼将其连接。另请参见连接关节链和骨架。
断开关节(Disconnect Joint)

在当前关节断开骨架，将骨架分为两个关节链。请参见断开关节。

重定骨架根
将当前关节指定为其层级的父或根关节。请参见重定骨架根。
请参见：
  • 关节和骨骼
  • 关节链
  • 骨架层级
  • 重定骨架根
关节标签设置(Joint Labelling)
打开“关节标签设置”(Joint Labelling)菜单。

IK

创建 IK 控制柄(Create IK Handle)
打开 IK 控制柄工具(IK Handle Tool)
创建 IK 样条线控制柄(Create IK Spline Handle)

打开 IK 样条线控制柄工具(IK Spline Handle Tool)。

设置首选角度(Set Preferred Angle)
设置当前骨架的首选角度。另请参见了解关节的属性。
单击“骨架 > 设置首选角度”(Skeleton > Set Preferred Angle) > 以查看“设置首选角度选项”(Set Preferred Angle Options)，并指定是要为当前关节还是为从当前关节到其链的末关节的所有关节设置首选角度。
选定关节(Selected Joint)
将选定关节的当前角度设置为其首选角度。
递归(Recursive)
将所有关节（从选定关节到链末尾的关节）的当前角度设置为其首选角度。
采用首选角度(Assume Preferred Angle)
返回当前骨架的关节到初始设置其首选角度时的角度。还可以在关节链或骨架上单击鼠标右键，从标记菜单访问该功能。
单击“骨架 > 采用首选角度”(Skeleton > Assume Preferred Angle) > 以打开“设置首选角度”(Set Preferred Angle)选项，指定是要将当前关节的角度重置为其首选角度，还是将从当前关节到其链的末关节的所有关节都重置为其首选角度。
选定关节(Selected Joint)
重置选定关节的当前角度为其首选角度。
递归(Recursive)
重置从当前关节到其链末关节的所有关节为其首选角度。

另请参见：

  • 了解关节的属性
  • 设置首选角度
启用 IK 控制柄捕捉(Enable IK Handle Snap)
使 IK 控制柄捕捉回其末关节的位置。默认情况下，“启用 IK 控制柄捕捉”(Enable IK Handle Snap)处于启用状态。
另请参见：捕捉启用(Snap Enable)
启用 IK/FK 控制(Enable IK/FK Control)
启用“启用 IK/FK 控制”(Enable IK/FK Control)后，就可以旋转处于 IK 链中的关节，无论 IK 控制柄是否具有关键帧。此外，“启用 IK/FK 控制”(Enable IK/FK Control)会禁用“启用 IK 控制柄捕捉”(Enable IK Handle Snap)。请参见“骨架”(Skeleton) > 启用 IK 控制柄捕捉(Enable IK Handle Snap)。
禁用“启用 IK/FK 控制”(Enable IK/FK Control)后，将无法旋转处于 IK 链中的关节，除非禁用其 IK 控制柄。
启用选定 IK 控制柄(Enable Selected IK Handles)
启用选定 IK 控制柄的可编辑性，并将其“IK 融合”(Ik Blend)值（在其“IK 解算器属性”(IK Solver Attributes)中）设置为 1.000。当“IK 融合”(Ik Blend)为 1.000 时，IK 动画可完全控制骨架。
禁用选定 IK 控制柄(Disable Selected IK Handles)
禁用选定 IK 控制柄的可编辑性，并将其“IK 融合”(Ik Blend)值（在其“IK 解算器属性”(IK Solver Attributes)中）设置为 0.000。当“IK 融合”(Ik Blend)为 0.000，FK 动画可完全控制骨架。
另请参见：启用选定 IK 控制柄(Enable Selected IK Handles)。
快速绑定(Quick Rig)
打开“快速绑定”(Quick Rig)工具。请参见为网格创建自动角色绑定。
HumanIK
打开 HumanIK 窗口。
