复制 Bullet 刚体和约束

有两种方法可以复制刚体。一种方法是使用复制、剪切和粘贴命令组合，另一种方法是使用“特殊复制”(Duplicate Special)。

如果您使用复制、剪切、粘贴，原始和复制的刚体共享现有解算器。

如果您使用“特殊复制”(Duplicate Special)，则与刚体一起复制解算器。
注： “特殊复制”(Duplicate Special)与“复制输入图表”(Duplicate Input Graph)一起使用效率非常低。此过程会复制所有附加到刚体的对象，这意味着您会获得每一个刚体的复制解算器和被动碰撞对象，而设置不再连接到一个解算器。

要使用“特殊复制”(Duplicate Special)方法复制刚体，请执行以下操作：

  1. 在场景中选择刚体及其关联的约束。
  2. 选择“编辑(Edit) > 特殊复制”(Edit > Duplicate Special)。将出现“特殊复制选项”(Duplicate Special Options)窗口。
  3. 在“特殊复制选项”(Duplicate Special Options)窗口中，激活“复制输入图表”(Duplicate Input Graph)。（请参见以上有关“复制输入图表”(Duplicate Input Graph)方法的限制的注释。）
  4. 单击“特殊复制”(Duplicate Special)。
