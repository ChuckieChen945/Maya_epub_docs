interpolate_transform_matrix

在两个分解的 4x4 变换矩阵的斜切、缩放、四元数和平移分量之间插值。

输入

first

第一个 4x4 变换矩阵。

second

第二个 4x4 变换矩阵。

fraction

在 first 与 second 变换矩阵之间插值的比率。

  • 值为 0.0 时，将输出一个等于 first 变换矩阵的矩阵。
  • 值为 1.0 时，将输出一个等于 second 变换矩阵的矩阵。
  • 值介于 0.0 和 1.0 之间时，将在 first 和 second 变换矩阵的每个分解分量之间插值。
  • 值小于 0.0 或大于 1.0 时，将基于 first 和 second 变换矩阵的每个分解分量进行外推。

shear_fraction

特定于分解变换的斜切分量的插值比率。此值将乘以全局 fraction。

scale_fraction

特定于分解变换的缩放分量的插值比率。此值将乘以全局 fraction。

quaternion_fraction

特定于分解变换的四元数分量的插值比率。此值将乘以全局 fraction。

translate_fraction

特定于分解变换的平移分量的插值比率。此值将乘以全局 fraction。

clamp_negative

钳制小于 0.0 的比例值。

clamp_above_one

钳制大于 1.0 的比例值。

always_use_shorter_arc

始终使用两个旋转之间的较短弧。

use_nlerp_if_closer

如果四元数之差小于此容差，则在更接近的情况下，执行归一化线性插值。这省去了计算，并且当差值很小时更稳定。

输出

interpolated

从每个插值分量重新组成的结果 4x4 变换矩阵。
