创建 UV 壳

可以将选定组件分离或分组到新 UV 壳中，用于将这些组件排布在一起。

创建 UV 壳

  1. 在 UV 编辑器中，选择要包含在新 UV 壳中的组件。
    注： 可以选择任何组件类型（顶点、边、面或 UV），并且选择中可包含多组组件。例如，您可以在 UV 壳的某部分中框选一组 UV，然后按住 Shift 键，在 UV 壳的其他部分中选择另一组 UV。
  2. 在 UV 编辑器中执行以下操作之一：
    • 选择“修改 > 创建 UV 壳”(Modify > Create UV Shell)。
    • 按住 Shift 键并单击鼠标右键，然后选择“创建 UV 壳”(Create UV Shell)。
    • 从“UV 工具包”(UV Toolkit)中，选择“切割和缝合 > 创建 UV 壳”(Cut & Sew > Create UV Shell)。

    Maya 沿选择的周长进行切割以创建新壳，并且 UV 编辑器会自动进入壳组件模式。
