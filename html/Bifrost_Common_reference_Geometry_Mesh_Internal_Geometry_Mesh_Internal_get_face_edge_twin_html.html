get_face_edge_twin

获取对应于与指定 face_edge 相邻的面的面边。

例如：

 ┌───────────────┬────────────────┐
 │      s2       │       s2       │
 │             ▲ │ │              │
 │             │ │ │              │
 │s3    f0   s1│ │ │s3   f1     s1│
 │             │ │ │              │
 │             │ │ ▼              │
 │      s0       │       s0       │
 └───────────────┴────────────────┘

对指代面 f0、侧边 s1 的面边调用 get_face_edge_twin 将返回指代面 f1、侧边 s3 的面边。

如果边是开放边，则返回的面边无效。在上面的示例中，指代面 f1、侧边 s1 的面边没有有效的成对面边。

请注意，非流形边的面边呈菊花链状连接在一起。在这种情况下，对返回的边调用 get_face_edge_twin 并不会返回此边，而是返回菊花链中的下一个边。对链中的最后一个面边调用 get_face_edge_twin 将返回链中的第一个面边。

输入

face_edge

要查询的面边。

face_offset

相应网格的面偏移数组。

face_vertex_adjacent_edge

相应网格的面相邻数组。（请参见 update_mesh_adjacency）

输出

twin

face_edge 的成对面边。
