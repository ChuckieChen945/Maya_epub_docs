岩石(Rock)

使用两种不同的颗粒材质类型的随机 3D 分布模拟岩石。

在“创建”(Create)选项卡中找到此纹理。若要作为纹理贴图应用该纹理，请参见映射 2D 或 3D 纹理。

提示：

将该纹理指定给材质的凹凸贴图，以获得较粗糙的模拟纹理。

颜色 1 (Color1)、颜色 2 (Color2)

纹理中两种颗粒的颜色。若要选择不同的颜色，请单击颜色栏以打开“颜色选择器”(Color Chooser)。

颗粒大小(Grain Size)

指定颗粒大小并缩放整个纹理。有效范围是 0 到无限。滑块范围是 0（无颗粒）到 0.1（大颗粒）。默认值为 0.01。

扩散(Diffusion)

控制融合到“颜色 2”(Color2)中的“颜色 1”(Color1)量。范围为 0 到无穷大。滑块范围是 0（无融合）到 1（平滑融合）。默认值为 1。

混合比(Mix Ratio)

确定主色。有效范围是 0 到无限。滑块范围是 0（“颜色 1”(Color1)完全占主导）到 1（“颜色 2”(Color2)完全占主导）。默认值为 0.5。
