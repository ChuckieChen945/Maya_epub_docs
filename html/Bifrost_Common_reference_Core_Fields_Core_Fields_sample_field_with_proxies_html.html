sample_field_with_proxies

返回代理场值的数组。可以在一些位置对体素代理场进行解算，并且可以在一些索引处对特性代理场进行解算。

输入

field

要解算的体素代理场或特性代理场。

positions

对场进行解算的位置数组。即使对于简单的特性代理场，这也是必需的（但不使用）。

proxy_geometry

一个或多个用于构建场的几何体。对于体素代理场，它们应为体积。对于特性代理场，它们可以是带有点的任何几何体。将使用其特性名称与代理场使用的特性相对应的第一个几何体。

proxy_property_indices

要返回的分量索引值的数组（在特性代理场的情况下）。如果未连接任何内容，则默认情况下将返回所有索引的值。如果已连接数组，其大小应与点位置的输入数组匹配，并且仅返回相应位置和索引组合的场值。

输出

result

场值的数组。
