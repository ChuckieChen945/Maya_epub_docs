clear

清空阵列的内容，释放所有阵列保留的内存。清除阵列之后，其大小为 0。如果不再需要使用阵列，请使用清除功能以释放内存。

int clear(array array)

array 是要清除的阵列名称。

如果清除函数成功，函数将返回 1，如果失败，将返回 0。在表达式中通常不使用返回值。

示例

int $myInts[] = {1,2,3,4,5,6};
print("size of $myInts is: "+size($myInts)+"\n");
clear($myInts);
print("size of $myInts is: "+size($myInts)+"\n");

上方第三个语句清除阵列 $myInts。

第二个和第四个语句在“脚本编辑器”(Script Editor)中显示以下文字：

size of $myInts is: 6
size of $myInts is: 0
