复制、重命名或删除 UV 集

选择要重命名、复制或删除的 UV 集。

执行下列操作之一：

  • 选择“UV > 将 UV 复制到 UV 集”(UV > Copy UVs to UV Set)。

    该操作可将当前 UV 集复制到新的 UV 集。

  • 选择“UV > 重命名当前 UV 集”(UV > Rename Current UV Set)。

    在显示的窗口中键入新的 UV 集名称。

  • 选择“UV > 删除当前 UV 集”(UV > Delete Current UV Set)。
