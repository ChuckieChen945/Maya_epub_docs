创建复合

  1. （可选）在图表中或其他可编辑复合中选择一个或多个节点。
  2. 按 Ctrl+G。或者，可以选择“编辑 > 生成复合”(Edit > Make Compound)。

选定节点将包含在新复合中，并且任何现有连接都将成为端口。

  • 双击复合的名称可对其重命名。
  • 右键单击端口可对其重命名、移动端口或者设置其数据类型和其他属性。
  • 双击复合可进入复合并编辑其内部子图。
注意：

从“选项卡”(Tab)菜单中选择 compound 将添加一个空复合。当前选定的节点不包含在其中。

炸开复合

炸开复合与创建复合相反。炸开复合会将复合的内容“展开”一层深度。所有连接都保持不变。

  1. 右键单击复合（而不是其中一个端口）。
  2. 选择“炸开”(Explode)。
注意：

如果复合是引用的，则会自动导入该复合，然后再炸开。
