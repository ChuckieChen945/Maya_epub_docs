关于 Maya 帮助

Maya 帮助（您所查看的内容）提供有关 Maya 中所有功能（从初级到高级技术）的可搜索文档。此外，它还包含脚本信息的技术文档以及 Maya API 详细信息的开发人员帮助。

  • 如果您是从 Maya 中通过按 F1 键并选择“帮助 > Maya 帮助”(Help > Maya Help)或在应用程序主页中心单击 到达此处，则只需使用左侧的目录或右上角的搜索栏来查找所需内容。

  • 如果您是从 Internet 到达此处，例如通过 Google 等搜索引擎，您可能看到的是此页面的 Autodesk Knowledge Network (AKN) 版本。单击上面（Autodesk 标识正下方）的 链接访问完整的 Maya 用户手册，然后使用目录和搜索栏查找所需的内容。您可以在任一 AKN 版本的帮助页面中进行此操作。
    注： 确保帮助文章中的 Maya 版本与您当前使用的 Maya 版本相匹配也是一种非常好的做法。可以在 URL 中看到文章版本，也可以在 AKN 中通过单击“涵盖的产品和版本”(Products and versions covered)按钮了解版本信息。

特定帮助任务疑难解答

目标 操作

打开并搜索联机帮助。

选择“帮助 > Maya 帮助”(Help > Maya Help)或按 F1 键，然后在搜索栏中输入关键字。

查找界面中的工具或菜单项。

选择“帮助 > 查找菜单”(Help > Find Menu)。

系统将返回菜单项名称以及该菜单项所属的菜单和菜单集。

界面概述是 Maya 界面的图像，其中包含所有主界面元素的名称和基本描述。通过它可链接至所描述的内容，以查找更多信息。

查找特定窗口或工具的帮助。

查找工具设置中的“帮助”(Help)菜单（而不是主“帮助”(Help)菜单）。

获取光标下有关操作的描述/说明。

查看帮助行（左下角），或启用帮助首选项中的“显示工具动画演示”(Display ToolClips)。

查找热键。

所有 Maya 热键列出了在 Maya 中可以使用的各个键盘快捷键。

查找技术文档。

您可以在脚本主题（包括 MEL、Python 和 Dash）中访问此内容。

请参见与您的 Maya 版本相匹配的帮助版本。

有时，在使用搜索引擎查找 Maya 帮助时，结果可能无法反映您的软件版本。您可以在 Web 浏览器顶部的帮助页面 URL 中检查/更改版本号。

与客户支持联系。

在线查找“联系我们”(Contact Us)页面：https://www.autodesk.com/company/contact-us 上。
