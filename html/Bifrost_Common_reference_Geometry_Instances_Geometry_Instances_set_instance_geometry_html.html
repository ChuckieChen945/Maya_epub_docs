set_instance_geometry

在点对象的位置实例化几何对象，例如粒子系统。

输入

points

点对象。

instance_geometries

要实例化的一个或多个对象或者对象数组。它们可以是普通几何对象，也可以是实例形状，例如，selector_instance、geo_instance 或 render_archive_instance 等节点的输出。

preview_geometries

要用于在视口中进行预览但不用于渲染的一个或多个可选几何对象。对于连接到 instance_geometries 但自己没有 geo_instance 或类似节点定义的预览几何体的对象，将显示此处连接的任何对象。

提示：

如果连接单个对象，它将覆盖以前定义的任何预览几何体，并且对所有实例显示。

instance_id_property

用于指定逐点实例化哪个实例形状的点对象的 long 特性的名称，其中 0 表示 instance_geometries 数组的第一个元素，1 表示第二个元素，依此类推。

overwrite_instance_id

如果为 true，则将覆盖 instance_id_property，并改用连接的 instance_id_values。

instance_id_values

一组用于指定要逐点实例化哪个实例形状的 long 值。

wrap_out_of_range

包裹超过实例形状的数组大小的实例 ID 值。

输出

instances

实例对象。
