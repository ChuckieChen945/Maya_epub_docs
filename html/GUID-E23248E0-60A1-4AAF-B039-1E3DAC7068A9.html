“流体渐变”(Fluids Gradients)选项

密度/速度/温度/燃料(Density/Velocity/Temperature/Fuel)

启用要按渐变表示的属性。必须至少选择一个属性。如果禁用某属性，将应用默认方法（对于“密度”(Density)和“速度”(Velocity)，会应用“动态栅格”(Dynamic Grid)，对于“温度”(Temperature)和“燃料”(Fuel)，会禁用所有属性（零））。

密度/速度/温度/燃料 渐变(Density/Velocity/Temperature/Fuel)

选择要置于容器中的值的预定义集：

恒定(Constant)

在整个流体中将值设定为 1。

X 渐变(X Gradient)

设定值沿 X 轴从 1 到 0 的渐变。

Y 渐变(Y Gradient)

设定值沿 Y 轴从 1 到 0 的渐变。

Z 渐变(Z Gradient)

设定值沿 Z 轴从 1 到 0 的渐变。

-X 渐变(-X Gradient)

设定值沿 X 轴从 0 到 1 的渐变。

-Y 渐变(-Y Gradient)

设定值沿 Y 轴从 0 到 1 的渐变。

-Z 渐变(-Z Gradient)

设定值沿 Z 轴从 0 到 1 的渐变。

中心渐变(Center Gradient)

在中心沿边缘方向设定 1 到 0 之间的渐变值。
