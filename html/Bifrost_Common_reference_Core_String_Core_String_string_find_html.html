string_find

在输入字符串的搜索范围内查找字符串。

在从 start 开始到 end 之前结束的给定范围（即 [start, end) 范围）内搜索子字符串。

如果切换反转标志，将从 stop 开始到 start 之前结束的给定范围（即 [stop, start) 范围）内反向搜索输入字符串。

函数返回在指定范围内是否找到子字符串，以及找到子字符串处的索引（从输入字符串起点开始）。

输入

string

输入字符串。

find

要在输入字符串中搜索的子字符串。

reverse

指明字符串搜索方向的标志。

start

搜索范围的起点。

end

搜索范围的终点。

输出

found

指明在输入字符串中是否找到子字符串的标志。

index

找到的子字符串在输入字符串中的位置，如果在给定的搜索范围内未找到子字符串，则返回 -1。
