set_property

在对象上设置键值对。该键是一个标识字符串，对应的值可以是任何内容：单个值、float3 等结构、数组或其他对象。

此节点是设置特性的最常用节点。在许多情况下，使用其他节点更容易或更合适。

  • 使用 set_geo_property_data 为现有基于每个组件的特性设置值，例如粒子系统中的 point_velocity。此节点通过截断或使用默认值填充，确保值数组与几何体上的组件数匹配。
  • 使用 set_geo_property 创建和初始化基于每个组件的特性。
  • 对于某些常用特性，还可以使用特定复合，例如 set_point_position。

输入

object

Bifrost 对象。这可以是几何对象（如网格），也可以是各种特性的容器。将此端口保留为空，以使用新的空对象。

key

特性的名称。如果对象上已存在此特性，将使用新值覆盖此特性。如果没有，将创建新特性。

value

要为特性设置的值。

输出

out_object

具有更新特性的对象。
