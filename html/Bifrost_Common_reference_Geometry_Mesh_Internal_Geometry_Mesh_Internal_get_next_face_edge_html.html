get_next_face_edge

获取相应面中的下一个面边。下一个面边是指按逆时针顺序沿面遍历面边时指定面边后面的面边。next_face_edge 的第一个顶点是 face_edge 的第二个顶点。

例如：

 ┌───────────────┐
 │      s2       │
 │             ▲ │
 │             │ │
 │ s3   f    s1│ │
 │             │ │
 │      s0     │ │
 │ ──────────►   │
 └───────────────┘

面 f、侧边 s0 的下一个面边是指面 f、侧边 s1。

输入

face_edge

要查询的面边。

face_offset

相应网格的面偏移数组。

输出

next_face_edge

面中 face_edge 后面的下一个面边。
