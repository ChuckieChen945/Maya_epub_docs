names_to_chained_paths

将一组名称转换为一组路径（按从父对象到子对象的顺序链接），其中每个路径还包含其父对象的路径。例如，对于 [a, b, c] 这样一组名称，将返回下面一组路径：[/a, /a/b, /a/b/c]。

输入

names

包含要转换为路径的名称的 array<string>。

parent_path

生成的链中第一个实体的父对象路径（如果有）。

prefix

要在每个输入 names 的开头添加的前缀（如果有）。

suffix

要在每个输入 names 的末尾添加的后缀（如果有）。

输出

paths

生成的包含路径的 array<string>。
