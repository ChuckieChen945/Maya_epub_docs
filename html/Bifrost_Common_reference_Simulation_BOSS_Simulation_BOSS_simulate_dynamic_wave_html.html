simulate_dynamic_wave

此节点计算二维栅格上的动态波模拟，并将其输出到二维波对象中，该对象可用于使几何体发生位移或用作其他模拟（例如泡沫）的输入。二维模拟的范围由设置中的边界特性定义。如果启用了 periodic，则在这些边界之外重复操作。

输入

sources

这是可用于船尾迹和碰撞的一组源。应从 source_dynamic_wave 节点连接它。

settings

此项应该连接到 dynamic_wave_settings 节点，该节点为模拟提供所有控制。

输出

out_wave

这是二维图像格式的波数据。可以将其连接到 wave_displace 等节点，使几何体发生位移。可以使用 sample_wave_property 从此波采样其特性（例如 displacement、displacement_derivatives、slope 和 velocity），以获得三维点数组。
