get_geo_component_indices

返回一个包含几何对象中组件索引的数组。例如，如果输入几何体是具有 4 个面的网格，并且我们需要 face_component 索引，则输出为 [0, 1, 2, 3]

输入

geometry

几何对象。

component

组件的名称，例如 point_component、face_component、face_vertex_component、voxel_component。

index_type

返回的索引的类型。默认情况下，输出为 long 数组，但是您可以连接其他类型以进行更改。这支持任意长度的带符号和无符号整数类型。

输出

indices

索引的数组。
