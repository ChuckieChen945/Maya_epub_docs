This version includes the following fixes:

  • Fixed memory leak when animated standins were displayed in the viewport
  • Exporting selected shading groups wasn't including the custom AOVs
  • Fixed regression in "Bake selected geometry" with displacement
  • Matte controls weren't working with Standard Hair
  • Fixed regression in "Render To Texture" with standins
  • Fixed UI error when filter nodes were selected
  • Volume grids list wasn't filled for animated VDB
