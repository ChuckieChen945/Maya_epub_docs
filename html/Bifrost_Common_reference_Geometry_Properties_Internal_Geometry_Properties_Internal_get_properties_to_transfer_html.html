get_properties_to_transfer

获取要传递的特性数组及对应的特性名称。

输入

source_geometry

传递的源几何体。

component

要传递的特性的几何组件。

user_property_expression

将返回与此表达式匹配的特性。

block_schema

要过滤的特性的对象模式。

always_transfer_expression

将始终返回与此表达式匹配的特性。请注意，这优先于 block_schema 中指定的特性。

输出

properties

与给定表达式匹配的特性。

names

与给定表达式匹配的特性的名称。

父页面: Internal
