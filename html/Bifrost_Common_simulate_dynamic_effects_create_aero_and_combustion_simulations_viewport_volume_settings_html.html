使用 NanoVTT 在视口中预览体积

set_viewport_volume_settings 节点可用于控制视口中体积的 NanoVTT 渲染质量和速度。NanoVTT 渲染视口中与 Arnold 渲染紧密匹配的体积，并支持彩色体积和基于物理的热力学发射缩放。此外，视口体积使用 Alpha 通道进行渲染，这对于可以合成的播放预览很有用。

要使用 set_viewport_volume_settings，请将要渲染的体积连接到 volume 输入，然后将 out_volume 输出连接到图表输出或 terminal 节点。当 mode 设置为 NanoVTT 时，可以调整质量、交互质量和其他设置。

特别是，如果图形卡上出现超时检测和恢复 (TDR) 错误，应该降低质量设置。

当 mode 设置为 Legacy Volume 时，将使用旧着色器在视口中渲染体积。该着色器不使用 set_viewport_volume_settings 节点中的质量和其他设置。
