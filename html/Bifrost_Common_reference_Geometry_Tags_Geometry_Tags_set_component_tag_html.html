set_component_tag

在几何对象上创建命名组件标记并指定其成员。

输入

geometry

要查询的几何体。

tag

要设置的标记的名称。

tag_data

要指定给标记的组件。这是一个可接受不同类型的值的自动端口：

  • array<bool>。此数组的大小应与 target 组件中的元素数相同。例如，如果 target 是 point_component，并且输入 geometry 具有 10 个点，则此数组应包含 10 个元素，每个点对应一个元素。如果此数组中的元素设置为 true，则相应的点将包含在此标记中。如果元素设置为 false，则相应的点不包含在此标记中。数组将调整大小以匹配 target 组件中的元素数。

  • array<long>。数组中的每个元素是要包含在标记中的组件的索引。例如，如果 target 为 point_component，则此数组中的索引是点索引。

  • string。字符串引用几何体中的另一个标记。

target

标记的目标组件。

invert

在几何体中设置数据之前，反转输入标记数据。

输出

out_geometry

在几何体中设置了组件标记的输入。
