创建 2D 纹理并选择映射方法

创建 2D 纹理时，可以从以下三种映射方法中选择一种：正常创建（创建纹理）、创建为投影和创建为蒙板。

使用 Tab 键在“节点编辑器”(Node Editor)中创建节点后，输入所需的 2D 纹理类型的名称时，将显示纹理的三种变化（投影、蒙板、纹理），从而可以在三种映射方法中进行选择。有关如何使用 Tab 键创建节点的详细信息，请参见在节点编辑器中创建节点。

使用拖放操作创建文件节点

您可以使用鼠标中键将图像文件从“内容浏览器”(Content Browser)拖动到“节点编辑器”(Node Editor)，或将图像文件从 Windows 资源管理器拖动到“节点编辑器”(Node Editor)，以创建文件节点并自动将其指定给图像文件。
