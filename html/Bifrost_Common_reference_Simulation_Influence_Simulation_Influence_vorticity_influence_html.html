vorticity_influence

此项对模拟的速度应用漩涡限制。有助于促进在模拟中生成较小比例漩涡，否则这些漩涡可能会被抑制。此项使模拟看起来更湍急且粘性更低。当前，此节点仅可与 simulate_aero 一起使用。

要使用此节点，请将其输出插入 simulate_aero 的 influences 输入。

输入

influence

可以选择将影响场 out_influence 连接到此端口。例如，可以使用 modulate_influence 节点实现仅在 voxel_temperature 较高时漩涡才会较高。可以将任意数量的影响场呈菊花链状连接在一起，以构建复杂的场。

enable

关闭此项可禁用效果。如果禁用，将使其成为穿过节点。请注意，禁用此项后，连接到影响输入端口的任何其他影响场仍将处于活动状态。

vorticity

要应用于模拟的附加加速度的强度。通常，此项的最佳值取决于模拟的 detail_size。如果细节大小较小，则应使用较小的漩涡值以避免模拟变得过于杂乱。

scale

此项为针对 Aero 模拟中的较大体素以自适应方式将漩涡值增大的程度。通常，这些较大的体素位于密度很低或没有密度的区域中。默认比例 0 表示直接对所有体素应用相同的推力，无论其大小如何。值为 1.0 时，推力将完全与平铺树中的相关体素大小成比例。最终会得到较大比例的漩涡。此特性的范围介于 0.0 到 1.0 之间。

输出

out_influence

可以连接到模拟节点或其他影响节点的输出影响对象。
