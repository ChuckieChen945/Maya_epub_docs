将 UV 从一个 UV 集复制到另一个 UV 集

将 UV 从一个现有 UV 集复制到另一个 UV 集

  1. 在 UV 编辑器中，选择要复制的 UV。
  2. 在“UV 编辑器”(UV Editor)中，打开“UV 集 > 将 UV 复制到 UV 集”(UV Sets > Copy UVs to UV Set)子菜单，然后选择一个 UV 集。

注释

  • 复制包含历史的多边形 UV 集时，会将源 UV 集与目标链接到一起。通过使用构建历史对源 UV 集所做的任何更改都会影响复制的 UV 集。若要避免该问题，请在复制 UV 集后删除构建历史。
