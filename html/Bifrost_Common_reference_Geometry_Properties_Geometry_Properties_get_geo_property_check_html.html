get_geo_property_check

返回几何对象的组件上某个特性的值数组，例如点的 point_size 值。这不同于 get_geo_property，因为它还会返回特性是否存在于对象上。

输入

geometry

几何对象。

property

特性的名称。

type

特性的类型。如果未找到特性，则这也是返回的数组的默认值。

输出

data

值数组。

default

特性的默认值。

target

特性针对的几何组件，例如 point_component、face_component、face_vertex_component、voxel_component。

interpolation_mode

修改特性的关联几何体时要对该特性执行的所需插值。

found_property

是否找到特性。
