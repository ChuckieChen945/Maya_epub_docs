山脉(Mountain)

在“创建”(Create)选项卡中找到此纹理。

使用 2D 分形图案模拟岩石地形。若要应用该纹理作为纹理贴图，请参见映射 2D 或 3D 纹理。

使用“山脉”(Mountain)纹理（在平面上）同时用作颜色贴图和凹凸/置换贴图，以模拟积雪覆盖的山脉。若要执行此操作，请按如下方式应用“山脉”(Mountain)纹理：

  • 纹理将根据凹凸/置换贴图计算颜色贴图，例如，雪的位置以曲面的置换为依据。
  • 颜色属性（“雪颜色”(Snow Color)和“岩石颜色”(Rock Color)）仅与颜色贴图相关，对凹凸/置换贴图没有任何影响 — 所有非颜色属性仅与凹凸/置换贴图相关，对颜色贴图没有任何影响。
  • 凹凸/置换贴图的“边界”(Boundary)、“雪海拔高度”(Snow Altitude)、“雪衰减”(Snow Dropoff)、“雪渐变”(Snow Slope)和“最大深度”(Depth Max)属性值将覆盖颜色贴图的这些属性。例如，凹凸/置换贴图的“边界”(Boundary)值控制颜色贴图的雪/岩石粗糙度。
    雪颜色(Snow Color)、岩石颜色(Rock Color)

    雪和岩石的颜色。单击颜色文件以从“颜色选择器”(Color Chooser)中选择不同的颜色。

    振幅(Amplitude)

    应用于纹理中所有值的比例因子。有效范围是 0 到无限。取值范围是 0 到 1。默认值为 1。

    雪粗糙度(Snow Roughness)

    雪的粗糙度。取值范围是 0（非常平滑的雪）到 1（非常粗糙的雪）。默认值为 0.4。

    岩石粗糙度(Rock Roughness)

    岩石的粗糙度。取值范围是 0（非常平滑的岩石）到 1（非常粗糙的岩石）。默认值为 0.707。

    边界(Boundary)

    岩石/雪边界的粗糙度。取值范围是 0（非常平滑的岩石/雪边界）到 1（极其粗糙的岩石/雪边界）。默认值为 1。

    雪海拔高度(Snow Altitude)

    岩石和雪之间的过渡级别（海拔高度）。有效范围是 0 到无限。取值范围是 0 到 1。默认值为 0.5。

    雪衰减(Snow Dropoff)

    雪不再粘住山脉的突然性。有效范围是 0 到无限。默认值为 2。

    雪渐变(Snow Slope)

    雪不会粘住山脉的最大角度（用小数值来表示）。例如，如果渐变超过“雪渐变”(Snow Slope)值，则它会成为光秃秃的岩石。有效范围是 0 到无限。取值范围是 0 到 3。默认值为 0.8。

    最大深度(Depth Max)

    用于计算分形图案的最大迭代数，其控制分形图案的粗细度。取值范围是 0 到 40。默认值为 20。
