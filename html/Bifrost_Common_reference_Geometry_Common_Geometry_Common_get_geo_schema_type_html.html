get_geo_schema_type

确定给定对象的几何体模式类型。

输入

object

要查询的对象。

输出

volume

如果对象是体积，则设置为 true，否则设置为 false

instances

如果对象是实例几何体，则设置为 true，否则设置为 false

mesh

如果对象是网格，则设置为 true，否则设置为 false

strands

如果对象是股几何体，则设置为 true，否则设置为 false

points

如果对象是点云，则设置为 true，否则设置为 false

not_a_geometry

如果对象不是上述几何体类型之一，则设置为 true。

geometry_type

此枚举设置为输入对象匹配的几何体的类型。

父页面: Common
