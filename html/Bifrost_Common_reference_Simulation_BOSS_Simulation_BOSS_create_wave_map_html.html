create_wave_map

此节点用于定义二维像素贴图，以将源输入用于动态波模拟。此贴图位于 XZ 方向并沿 Y 轴向下投影，而且包含 ZX 方向上的边界。它可用于尾迹生成、波遮挡和水深贴图等内容。

输入

value

这是要用于 wave_map 的值。它可以是恒定浮点、数组 或场。如果初始化为数组，则数组大小应为 resolutionX * resolutionZ。将使用要创建贴图的 UV 调用一个场。如果解算 position_field，则 xy 将在 0.0 到 1.0 范围内，这是贴图在 XZ 方向上的最小/最大边界映射到的范围（归一化的 XZ 位置映射到 XY 场，Z 为 零）。如果未启用边界，则这是 dynamic_wave 模拟的最小/最大 XZ。

resolution_xz

贴图的分辨率。这是相对于由边界设置确定的贴图边界的值。如果未设置边界，则使用连接的模拟的边界。如果分辨率的纵横比与边界的纵横比不匹配，则像素可能不是方形的，尽管并不要求像素是方形的。

enable_bounds

如果设置了此项，则贴图将具有自己的边界，与模拟的边界分开，否则贴图图像将拉伸以适合其连接的任何 dynamic_wave 模拟的边界。在某些情况下，单独的边界可能更有效。例如，可以创建一个具有平滑衰减的小尾迹，它可能具有较低分辨率（例如 10x10），并使用 Size_Center 边界方法。然后，可以使用 size 为模拟设置此发射器的大小，并使用 center 使其偏移。这样，就可以使用小贴图并在模拟区域中移动它，而不必为了计算一点发射而使用大量像素。

Bounds

边界确定贴图在 X 和 Z 方向上的尺寸。如果 enable_bounds 处于禁用状态，则这将是贴图连接到的波模拟的最小/最大 XZ。

bounds_method

此项确定如何为贴图定义边界。

  • Min_Max：边界由特性 min_bound 和 max_bounds 确定

  • Size_Center：边界由特性 size 和 center 确定

  • From_Geometry：边界由连接到 bounds_geometry 的几何体确定

min_bound

贴图在 X 和 Z 方向上的最小范围。（当 bounds_method 为 Min_Max 时）

max_bound

贴图在 X 和 Z 方向上的最大范围。（当 bounds_method 为 Min_Max 时）

size

贴图在 X 和 Z 方向上的总体大小（当 bounds_method 为 Size_Center 时）

center

贴图边界在 X 和 Z 方向上的中心位置（当 bounds_method 为 Size_Center 时）

bounds_geometry

当 bounds_method 设置为 From_Geometry 时，边界由连接到 bounds_geometry 的对象的 xz 边界确定。连接几何体（基于点的对象）。这可以是任何点对象或网格。

输出

out_wave_map

这是输出波贴图对象，它具有名为“map”的像素数组特性以及 min_bound 和 max_bound 特性。
