update_joint_chain_index

更新关节链索引，即关节所属的链的索引。

joint_chain_index 取决于 chain_joint_indices 几何特性，并且使用其计算得出。如果输入骨架上不存在此特性，则此节点将在更新 joint_chain_index 几何特性之前先更新它。

输入

skeleton

要更新的骨架。

输出

out_skeleton

更新的骨架，包含 joint_chain_index 几何特性。

joint_chain_index

从更新的 joint_chain_index 几何特性生成的 array<uint> 数据。

父页面: Skeleton
