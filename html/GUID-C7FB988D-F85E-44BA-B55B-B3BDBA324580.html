使用 parentmatrix 节点将对象更改为其父对象

可以使用 parentMatrix 节点将对象更改为其“父对象”，方法是将原始输入矩阵与多个加权父对象矩阵输入组合在一起。请参见变换数学节点中的“parentMatrix”。

多个加权父对象矩阵输入合并后会产生一个输出矩阵，该矩阵可用作对象的父对象矩阵，通常与对象的 offsetParentMatrix 属性相连。可以设置权重动画，例如，在任何时候只有一个权重具有完整效果，或者该权重可以是输入的归一化组合。

例如，假设将对象 A 放置在两个其他对象（P1 和 P2）之间。我们可能希望对象 A 随 P1 一起移动，也可能希望它随 P2 一起移动，或者可能希望它同时随两者一起移动。由于 A 到 P1 的相对位置与 A 到 P2 的相对位置不同，我们不能使用 P1 和 P2 的世界矩阵。因此，parentMatrix 节点将捕获每个目标的 offsetMatrix 以表示这些相对位置。

可以使用“管理目标”(Manage Targets)菜单中的选项来初始化和捕捉目标的 offsetMatrix：

  • 初始化(Initialize)：初始化目标是指计算 offsetMatrix，以便当目标成为唯一的影响因素时（目标的权重为 1.0，而所有其他对象的权重为 0.0），对象与 inputMatrix 保持在同一位置。
  • 捕捉(Snap)：捕捉目标是指计算 offsetMatrix，以便当目标成为唯一的影响因素时（目标的权重为 1.0，而所有其他对象的权重为 0.0），对象不会移离当前位置。
