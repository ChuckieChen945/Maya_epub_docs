displace_points

沿给定向量置换几何体的点。

输入

points

具有要置换的点的对象。

weights

用作置换的基本幅值（比例 × 长度）的倍增的权重。

  • 权重为 0 时，不进行置换。
  • 权重在 0 到 1 范围内时，按基本幅值的相应比例移动点。
  • 权重为 1 时，按基本幅值精确移动点。
  • 权重大于 1 时，点移动的距离大于基本幅值。
  • 权重小于 0 时，朝相反方向移动点。

如果未连接任何对象，则所有点的默认权重为 1。这是一个解释的自动端口，其接受以下类型：

  • 单个 float 解释为应用于所有点的常量值。
  • array<float> 解释为每个点的值。可根据需要调整数组大小以匹配点数。
  • array<bool> 被解释为每点的 0 (false) 或 1 (true) 值。
  • string 解释为现有几何特性的名称，并使用其值。
  • ScalarField 将在几何体组件的每个位置处进行采样。
  • array<long> 指定要为其指定值 1 的索引。对于其他索引，则指定值 0。

置换

scale

置换向量的幅值（长度）的倍增。

displacement_vector

要沿其置换点的向量。如果未设置任何向量或未将任何向量连接到此端口，则默认情况下将使用 point_normal 特性。置换的幅值取决于此向量的长度，以及 weights 和 scale。

这是一个解释的自动端口，其接受以下类型：

  • 连接单个 float3 以将其值用于所有点。
  • 连接 array<float3> 以对每个点使用数组的相应元素。可根据需要调整数组大小以匹配点数。
  • 连接 string 以使用具有相应名称的特性给定的向量（每个点的 float3）。
  • 连接 VectorField 以使用点的当前位置之处的场的值。

请注意，也可以使用标量值而不使用向量，在这种情况下，将对所有三个轴使用单个元素。但是，这对置换几乎没有任何用处。

输出

out_points

点置换的几何体。
