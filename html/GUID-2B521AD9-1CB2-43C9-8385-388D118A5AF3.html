绘制 NURBS 曲线

您可以通过放置控制顶点 (CV)、放置编辑点或徒手绘制的方式来绘制 NURBS 曲线。如果使用 CV 或编辑点绘制，则曲线会自动平滑；如果徒手绘制，则曲线会完全遵循工具提示的路径。

开始绘制之前，您可能需要通过选择选项框来查看或设置工具选项。例如，选择“创建 > 曲线工具 > CV 曲线工具”(Create > Curve Tools > CV Curve Tool) > ，可查看“CV 曲线工具”(CV Curve Tool)选项。

通过放置 CV 绘制 NURBS 曲线

  1. 选择“创建 > 曲线工具 > CV 曲线工具”(Create > Curve Tools > CV Curve Tool)。
  2. 单击以放置这些 CV。
    • 第一个 CV 看起来像空盒子，后续 CV 是点。放置时，将在 CV 之间绘制壳线。
    • 对于放置的第三个 CV 之后的每个 CV，将绘制曲线的形状。
    • 若要移除放置的最后一个 CV，请按 Delete 键。
    • 若要切换到编辑 CV，请按 Home 或 Insert 键。操纵器将显示在最后一个
    • CV 上。使用操纵器移动该 CV，按 Delete 键删除一个分段，或单击另一个 CV 对其进行编辑。再次按 Home 或 Insert 键可返回到添加 CV。
  3. 按 Enter 键以完成曲线。

通过放置编辑点绘制 NURBS 曲线

    如果希望曲线通过特定点，请使用编辑点绘制。此绘制方法将根据指定编辑点的位置计算 CV 的位置。

  1. 选择“创建 > 曲线工具 > EP 曲线工具”(Create > Curve Tools > EP Curve Tool)。
  2. 单击以放置编辑点。
    • 对于您放置的第一个编辑点之后的每个编辑点，Maya 绘制曲线的形状。
    • 若要切换到编辑编辑点，请按 Home 或 Insert 键。操纵器将显示在最后一个点上。使用操纵器移动该点，按 Delete 键删除一个分段，或单击另一个点对其进行编辑。再次按 Home 或 Insert 键可返回到添加点。
  3. 按 Enter 键以完成曲线。

徒手绘制 NURBS 曲线

  1. 选择“创建 > 曲线工具 > 铅笔曲线工具”(Create > Curve Tools > Pencil Curve Tool)。
  2. 拖动以绘制曲线草图。松开鼠标按钮时，曲线已绘制。
  3. “铅笔曲线工具”(Pencil Curve tool)会创建具有大量数据点的曲线。使用“曲线 > 重建”(Curves > Rebuild)可简化曲线。
    • 绘制曲线时，如果点距上一个点至少五个屏幕像素，则点将放在视图中。
    • 如果在正交视图（前视图、顶视图或侧视图）中绘制草图，曲线将放在原点处的视图平面上。如果在透视视图中绘制草图，曲线将放在栅格平面上。
