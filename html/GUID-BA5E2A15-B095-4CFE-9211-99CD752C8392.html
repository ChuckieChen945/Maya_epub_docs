管理文件路径

使用“文件路径编辑器”(File Path Editor)来解决路径错误，并为场景重新指定文件路径，保持文件组织有序，以便于访问。

构成场景的元素不包含在单个文件中。例如，纹理和声音是单独的文件，这些文件需要链接到场景中。“文件路径编辑器”(File Path Editor)包含场景中包括的引用、纹理、音频节点和图像平面的目录路径的列表。在场景中使用的所有文件应位于同一目录中，或在项目中组织，以便应用程序可以找到它们。文件移至其他目录中时未进行解析，而应用程序将使用不再有效的原始文件路径来查找它们。“文件路径编辑器”(File Path Editor)会搜索未解析的文件，然后重新指定文件路径并将其保留在原始位置，或者重新指定文件路径并将其复制到其他文件所在的目录。

注： 要重新指定引用节点的路径，则目标文件必须存在。与“文件路径编辑器”(File Path Editor)中的其他节点类型不同，无法重新指定不存在文件的引用节点的路径。

使用“重新指定路径”(Repath)窗口来重新指定文件路径

  1. 在“文件路径编辑器”(File Path Editor)（“窗口 > 常规编辑器 > 文件路径编辑器”(Windows > General Editors > File Path Editor)）中，选择要更改的文件路径。
    提示： 可以按住 Ctrl 或 Shift 键单击来选择多个文件路径。
  2. 单击“重新指定文件路径”(Repath Files)。
  3. 在显示的“重新指定路径”(Repath)窗口中，手动在“新路径”(New Path)字段中输入新的文件路径，或单击“浏览”(Browse)导航到新位置。
  4. （可选）如果要在执行重新指定路径操作之前预览该操作的结果，请单击“预览”(Preview)。

    此时将显示“操作预览”(Actions Preview)窗口，并显示有关即将发生的重新指定路径操作的详细信息。有关详细信息，请参见“操作预览”(Actions Preview)窗口。

  5. 要执行重新指定路径操作，请执行下列操作之一：
    • 在“操作预览”(Actions Preview)窗口中单击“继续”(Continue)。
    • 在“重新指定路径”(Repath)窗口中单击“重新指定文件路径”(Repath Files)。

使用“自动解析”(Auto Resolve)窗口来重新指定文件路径

  1. 在“文件路径编辑器”(File Path Editor)（“窗口 > 常规编辑器 > 文件路径编辑器”(Windows > General Editors > File Path Editor)）中，选择要更改的文件路径。
    提示： 可以按住 Ctrl 或 Shift 键单击来选择多个文件路径。
  2. 单击“自动解析”(Auto Resolve)。

    在显示的“自动解析”(Auto Resolve)窗口中，设置选项来有选择地重新指定文件和节点路径，并自动将文件复制到指定的位置。有关详细信息，请参见“自动解析”(Auto Resolve)窗口。

  3. （可选）如果要在执行自动解析操作之前预览该操作的结果，请单击“预览”(Preview)。
  4. 单击“自动解析”(Auto Resolve)执行自动解析操作。
