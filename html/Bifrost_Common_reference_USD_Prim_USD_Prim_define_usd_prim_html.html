define_usd_prim

创建 Bifrost-USD 基本体定义。将输出连接到 add_to_stage 节点下游时，这会在后台文件中创建一个 USD 基本体。

输入

path

新基本体的路径，例如：/car 或 /street/car。此处输入的路径将添加到任何已连接子对象指定的路径前面。同样，如果此节点的输出连接到基本体定义下游的 children 端口，则此路径将被附加到该处指定路径的后面。

variant_set_definitions

为新基本体连接一个或多个变量集定义（可选）。可以在端口上单击鼠标右键，然后选择“连接节点”(Connect Node)以快速添加和连接 define_usd_variant_set 节点。

reference_definitions

为新基本体连接一个或多个参照定义（可选）。可以在端口上单击鼠标右键，然后选择“连接节点”(Connect Node)以快速添加和连接 define_usd_reference 节点。

attribute_definitions

为新基本体连接一个或多个属性定义（可选）。可以在端口上单击鼠标右键，然后选择“连接节点”(Connect Node)以快速添加和连接 define_usd_attribute 节点或常用预设（如 define_usd_transform 或 define_usd_display_color）。

children

为新基本体连接一个或多个子对象（可选）。可以在端口上单击鼠标右键，然后选择“连接节点”(Connect Node)以快速添加和连接任一常用节点。

relationship_definitions

为新基本体连接一个或多个关系定义（可选）。可以在端口上单击鼠标右键，然后选择“连接节点”(Connect Node)以快速添加和连接 define_usd_relationship 节点。

常规

type

基本体的类型，例如：Scope、Xform、Capsule、Cone、Cube、Cylinder、Sphere、Mesh、NurbsCurves、PointInstancer 等。完整的类型列表取决于渲染器和其他插件的注册内容。

specifier 是 Over 或 Class 时，type 将被忽略。specifier 是 Def 时，如果保留 type 为空，则会创建一个基本体定义，其类型可稍后指定。

specifier

基本体的说明符：

  • Def：定义基本体。
  • Over：可在基本体或 PrimSpec 上合成。
  • Class：创建可供其他基本体参照的抽象基本体。

kind

  • None：未创建 kind 元数据。
  • Assembly：特殊类型的组（例如：已发布资产或对已发布资产的引用）。
  • Component：不能包含其他模型的叶模型。
  • Group：用于对其他模型进行分组。
  • SubComponent：组件的特殊标识部分。

purpose

控制可见性：

  • None：未创建 purpose 元数据。事实上，这意味着基本体通常是可见的。
  • Render：仅在以最终质量渲染时可见。
  • Proxy：仅在执行轻量代理渲染时可见。
  • Guide：仅在明确显示导向工具时可见。

active

  • None：未创建 active 元数据。默认情况下，基本体被视为是活动的。
  • False：取消激活定义的基本体及其子对象。这可用于删减不必要的场景描述。
  • True：允许正常合成和处理定义的基本体。

instanceable

声明定义的基本体是否可用于实例。只有将 instanceable 显式设置为 True 的基本体才能实例化。

  • None：未创建 instanceable 元数据。
  • False：基本体不是实例化的候选对象。
  • True：基本体是实例化的候选对象。在后台文件上合成时，如果基本体还包含一个或多个直接合成圆弧，则其会成为隐式主基本体的实例。

applied_schema_names

USD 应用的 API 模式的逗号分隔名称。

变量选择

这些参数是可选的。

variant_set_name

输入基本体定义变量的变量集的名称（可选）。如果保留为空，则基本体不是任何变量集的成员。

variant_name

此基本体定义的变量的名称。

material

连接材质定义（可选），例如：define_usd_preview_surface 节点的输出。

输出

prim_definition

新基本体定义。可以将其连接到 add_to_stage 节点的 prim_definitions 端口，或连接到其他 define_usd_prim 或类似节点的 children 端口。
