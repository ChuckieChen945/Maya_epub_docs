duplicate_points

复制指定点。所有指定的逐点特性也将被复制。此项适用于所有基于点的几何体类型（点、股和网格）。

输入

geometry

要从中复制点的基于点的几何体。

point_indices

要复制的点的索引。

properties

要传递到新几何体的几何特性的名称，例如 point_color。确保包含所需的所有几何特性，包括那些以点以外的组件（例如网格中的面和面顶点）为目标的几何特性。不是几何特性的特性将始终进行传递。

  • 使用空格分隔多个几何特性名称。
  • 您可以在表达式中使用通配符 *，例如使用 point_user* 来传递所有以字符串“point_user”开头的几何特性。
  • 单独使用值 * 将传递所有几何特性。
  • 使用空字符串可阻止传递任何几何特性，几何体模式需要的几何特性除外（用于点对象的 point_position、用于网格的 face_offset 和 face_vertex 等）。
  • 使用 * 后跟 ! 可排除特性。例如，* !point_user* 将包括以字符串“point_user”开头的特性以外的所有特性

输出

out_geometry

包含复制的点的新 geometry。

original_point_count

输入 geometry 中的点数。

new_point_count

输出 out_geometry 中的点数。
