update_joint_path

使用默认的关节名称更新关节路径。

使用 joint 作为前缀，且后跟自然索引编号，由此创建唯一的关节名称。然后，使用关节层级树数据构造路径，使每个路径都包含其名称，且前面带有其祖先的所有关节名称。

包含两个隔离关节链的骨架将生成如下关节路径：

/joint0
/joint0/joint1
/joint0/joint1/joint2
/joint0/joint1/joint2/joint3
/joint4
/joint4/joint5
/joint4/joint5/joint6
/joint4/joint5/joint6/joint7

输入

skeleton

要更新的骨架。

输出

out_skeleton

更新的骨架，包含 joint_path 几何特性。

joint_path

从更新的 joint_path 几何特性生成的 array<string> 数据。
