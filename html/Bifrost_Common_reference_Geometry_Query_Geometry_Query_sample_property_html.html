sample_property

返回几何体上的位置的特性值。值可以从最近的几何组件进行插值。

输入

geometry

要采样的几何体。

locations

包含几何体上的位置的数组，例如，来自 get_closest_locations 的输出。对于位置的 2D 数组，请使用 sample_property_2D。

default

连接与特性相同类型的 value 节点。如果失败，默认将返回此值，例如，如果特性不存在或位置无效。

property

特性的名称。

method

如何在组件之间返回值：

  • interpolated 返回最近的组件上特性的平均值，按距离计算权重。
  • from_closest 返回最近的组件上特性的值。

输出

sampled_data

包含位置的特性值的数组。

success

布尔值的匹配数组。如果元素为 false，则表示特性不存在或对应的位置无效。在此类情况下，sampled_data 数组的对应元素为 default 值。
