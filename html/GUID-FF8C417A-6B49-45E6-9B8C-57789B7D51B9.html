设置受控关键点

可以在“动画”(Animation)菜单集的“关键帧 > 设置受控关键点关键帧”(Key > Set Breakdown Key)上设置受控关键点。受控关键点可用的选项与通过选择“关键帧 > 设置关键帧”(Key > Set Key) > 获得的选项相同。请参见“设置受控关键点”(Set Breakdown)选项。

有关受控关键点的详细信息和示例，请参见受控关键点。

也可以从“通道盒”(Channel Box)为属性设置受控关键点，如下所述。

为选定属性设置受控关键点

  1. 选择对象。
  2. 单击要设置受控关键点的“时间滑块”(Time Slider)帧编号。
  3. 选择要为其设置受控关键点的属性。
  4. 输入属性的值。
  5. 选择“通道 > 为选定项设置受控关键点”(Channels > Breakdown Selected)。

    或

    在属性名称或文本框上单击鼠标右键并选择“为选定项设置受控关键点”(Breakdown Selected)。

为所有属性设置受控关键点

  1. 选择对象。
  2. 单击要设置受控关键点的“时间滑块”(Time Slider)帧编号。
  3. 输入属性的值。
  4. 在“通道盒”(Channel Box)中，选择“通道 > 为所有项设置受控关键点”(Channels > Breakdown All)。

    或

    在属性名称或文本框上单击鼠标右键并选择“为所有项设置受控关键点”(Breakdown All)。
