创建褶皱变形器

若要创建褶皱变形器，您可以使用“褶皱工具”(Wrinkle Tool)。创建的褶皱变形器的特性取决于“褶皱工具”(Wrinkle Tool)的工具设置。

重要： 创建变形器之后，请避免更改可变形对象的点（例如，CV、顶点或晶格点）的数量。更改点数可能会导致意外的变形效果。请尽可能确保您对可变形对象的拓扑满意之后，再开始使用变形器。您可能需要保存该对象的副本，以防将来需要执行进一步的建模。

创建褶皱变形器

  1. 选择一个或多个可变形对象。
  2. 选择“变形 > 褶皱”(Deform > Wrinkle) > 。

    此时将出现“工具设置”(Tool Settings)窗口。

  3. 设置“褶皱工具”(Wrinkle Tool)选项。

    此时会亮显曲面的一个 UV 区域，使您可以塑造线簇的形状以变形曲面。

  4. 用鼠标中键拖动以塑造 UV 区域的形状。使用每一侧中间的圆圈进行缩放、使用角点进行旋转并使用 UV 区域中间的点进行移动。
  5. 在 UV 区域适合可变形对象的区域时，按 Enter 键（Linux 和 Windows）或 Return 键 (Mac OS X)。

    “C”图标是褶皱变形器的簇变形器控制柄。

创建褶皱变形效果

  1. 移动、旋转或缩放簇变形器控制柄（“C”图标）。
  2. 移动、旋转或缩放影响线。

    有关创建和编辑变形效果的详细信息，请参见编辑褶皱变形效果。
