transform_matrix_chain

通过其他变换来变换选定范围的变换矩阵。

输入

transforms

要解算的变换的 array<double4x4>，解释为从头至尾的连续链。

transform

用于变换 transforms 的 double4x4 变换矩阵。

enable

如果启用此项，将处理此节点。否则，将跳过任何计算并按原样输出输入 transforms。

start_index

定义变换链处理操作的起始数组位置。

end_index

定义变换链处理操作的结束数组位置。支持负值。例如，-1 表示处理到链中的最后一个变换，-2 表示处理到倒数第二个变换，依此类推。

输出

out_transforms

解算的 transforms。
