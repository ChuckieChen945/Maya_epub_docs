transform_vector

使用各种变换方法变换向量。

输入

vector

要变换的向量。

transform

变换向量时要使用的 4x4 变换矩阵。

vector_mode

变换 vector 的方法。

-As Position：将输入 vector 解释为位置并对其进行变换，完全受变换影响。

-As Direction：将输入 vector 解释为方向向量，并对其进行变换，使向量受 transform 的斜切、缩放和方向的影响。

-As Normal：将输入 vector 解释为法线向量，并通过 transform 的归一化 3x3 旋转矩阵分量的逆转置（伴随）对其进行变换，使变换的向量与曲面保持垂直。

输出

transformed_vector

变换的 vector。
