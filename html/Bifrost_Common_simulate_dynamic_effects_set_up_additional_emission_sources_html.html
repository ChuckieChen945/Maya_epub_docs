设置其他发射源

您可以将其他几何对象连接到位于模拟图表顶层的源节点的 geo 端口。连接到相同源节点的所有发射器几何体都使用相同设置。

要使用不同的设置（如不同的速率、大小或速度）进行发射，请执行以下操作：

  1. 将另一个源节点添加到图表中。

  2. 将发射器几何体连接到其 geometry 端口。

  3. 将源的输出连接到模拟节点的 sources 输入。

可以将任意数量的源节点连接到 sources 端口。具体地说，对于 Aero 模拟，还可以通过将 simulate_aero 节点的 source_volumes 连接到 output 节点来输出体素化源。
