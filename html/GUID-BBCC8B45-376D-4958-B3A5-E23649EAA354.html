加快曲面渲染速度

请执行下列任一操作：

  • 在对象的“属性编辑器”(Attribute Editor)上，使用单面而不是双面曲面（默认设置）。最大速度增益用于 Maya 硬件渲染器。
  • 细分大型曲面需要大量内存，因此尽可能使用多个小曲面来代替一个大型曲面。处理较小的曲面时，渲染器更有效。
  • 对于 Maya 软件渲染和 Maya 硬件渲染，请使用凹凸贴图，而不是置换贴图。
  • 对于 Maya 软件渲染，请使凹凸贴图更平坦。若要执行该操作，请减小“Alpha 增益”(Alpha Gain)属性的值，这样可平滑凹凸贴图并减少自适应着色的采样数。仅当“边缘抗锯齿”(Edge Anti-aliasing)设定为“最高质量”(Highest Quality)时，该技术才适用。“Alpha 增益”(Alpha Gain)较低时，纹理凹凸外观较平坦。
  • 对于 Maya 软件渲染，使用置换贴图时请启用“使用置换边界框”(Use Displacement Bounding Box)。
  • 对于 Maya 软件渲染，尽可能使用分层纹理，而不是分层着色器。（有关详细信息，请参见“分层着色器”(Layered Shaders)和 2D 和 3D 纹理。）
  • 对于 Maya 软件渲染，如果对场景进行光线跟踪，请将“反射限制”(Reflection Limit)和“折射限制”(Refraction Limit)设置为产生可接受结果的最小值。
  • 若要避免在使用 Maya 软件渲染时重新细分同一曲面，请在“内存和性能选项 > 细分”(Memory and Performance Options > Tessellation)下的“渲染设置”(Render Settings)：“Maya 软件”(Maya Software)选项卡中选择“使用文件缓存”(Use File Cache)。选择“使用文件缓存”(Use file cache)时，几何数据将存储在系统临时目录的文件中。
    操作系统 临时目录的位置
    Windows 由 TEMP 环境变量定义

    使用 echo %TEMP% 查找精确位置

    macOS 由 TMPDIR 环境变量定义

    使用 echo $TMPDIR 查找精确位置

    Linux 由 TMPDIR 环境变量定义

    使用 echo $TMPDIR 查找精确位置

    可以通过更改 TMPDIR 或 TEMP 环境变量指向的位置来更改缓存文件的存储位置。请勿在系统设置中修改 TMPDIR 或 TEMP。而是在终端或命令窗口中设置这些环境变量，然后从窗口而不是桌面图标启动 Maya。如果经常执行此操作，则可能需要创建一个启动脚本，用于在启动 Maya 之前设置这些变量。
