face_group_to_points

查找被一组指定的面引用的点。这些点分类为内部点或边界点。内部点是完全被输入 faces 组中的面包围的点，并且不在输入网格的边界上。边界点是位于输入 faces 组中面的边界上的点：这意味着它没有被选定面包围，或者位于输入网格的边界上。

例如，对于此网格以及输入 faces f0、f1、f2 和 f3：

  p0──────────p1───────────p2──────────p9
  │     s0     │     s0     │    s0     │
  │            │            │           │
  │s1   f0   s3│s1   f1   s3│s1  f4   s3│
  │            │            │           │
  │     s2     │     s2     │    s2     │
  p3──────────p4───────────p5──────────p10
  │     s0     │     s0     │    s0     │
  │            │            │           │
  │s1   f2   s3│s1   f3   s3│s1  f5   s3│
  │            │            │           │
  │     s2     │     s2     │    s2     │
  p6──────────p7───────────p8──────────p11
  • 边界点是 p0、p1、p2、p3、p5、p6、p7 和 p8。
  • 内部点是 p4。
  • 边界面边是 (f0, s0)、(f0, s1)、(f2, s1)、(f2, s2)、(f3, s2)、(f3, s3)、(f1, s3) 和 (f1, s0)。

输入

mesh

要查询的网格。

faces

要查询的 mesh 的面。

输出

interior_points

网格中被指定 faces 包围且不在输入网格边界上的点。每个引用点添加到此数组一次，即使点被面组中的多个面引用也是如此。

border_touching_non_selected_face

指定 faces 的边界上与至少一个未被选中的面相邻的点。

border_points

指定 faces 的边界上的点。每个引用点添加到此数组一次，即使点被面组中的多个面引用也是如此。

border_face_vertices

面顶点的索引，其中每个面顶点位于面组内的面与不在面组内的面之间的边界上。

border_face_edges

指定的 faces 的面边的子集，这些面边位于面组内的面与不在面组内的面之间的边界上。
