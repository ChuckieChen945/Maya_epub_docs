sqrt3_subdivide_mesh

此节点使用 SIGGRAPH 2000 中 Leif Kobbelt 的 sqrt(3) 细分算法对网格进行细分。

如果每个三角形沿 X、Y 或 Z 轴的任何边的范围超过 detail_size 的两倍，则将其细分。这样可以确保三角形的大小与从具有相同细节大小的体积转化的网格三角形大致相当。

存储在输入网格上的点特性将插值到细分网格。面特性和面顶点特性（例如 UV 坐标）当前不受支持，将被放弃。

请注意，具有边界的网格（例如壳）和非流形网格当前也不受支持，不会进行细分。

输入

mesh

输入网格。

volume

可选体积，包含 VariedFromProperty 模式下要用于自适应细分的细节大小特性。

detail_size_scale

与输入体积的细节大小相比，值小于 1 时会生成更多更小的三角形。值大于 1 时会生成更少更大的三角形。请注意，将在所有 adaptivity 模式下应用此比例。

adaptivity

在 Off 模式下，每个三角形都根据节点上指定的全局 detail_size 进行细分。在 VariedFromProperty 模式下，在输入 volume 中根据 adaptivity_property 对细节大小进行采样。这意味着可以在空间中改变三角形的大小。请注意，在每个三角形的质心处对细节大小进行采样。在 Automatic 模式下，如果找到 adaptivity_property，则节点的工作方式与 VariedFromProperty 模式类似；否则，其工作方式与 Off 模式类似。

detail_size

在 Off 模式下使用的绝对细节大小。值将按 detail_size_scale 缩放。

adaptivity_property

包含在 VariedFromProperty 模式下使用的绝对细节大小的 float 体素特性的名称。值将按 detail_size_scale 缩放。请注意，为了防止由于输入体积中包含非常小的细节大小而导致挂起，体素值将以 detail_size 作为区间下限。

输出

triangle_mesh

细分网格。
