create_arrow_strands

创建发自股的箭头，以用于诊断目的。箭头股是四顶点股，其中厚度 (point_size) 用于创建箭头形状。它们显示为带状。

输入

start_positions

表示箭头起点的位置数组。

end_positions

表示（指向目标的）箭头终点的位置数组。

colors

表示股颜色的可选 float3 数组。如果未提供，则基于规格化光线方向生成颜色。

arrowhead_size

箭头的大小。此大小与可选的 arrowhead_sizes 相乘计算出每个箭头的大小。

arrowhead_sizes

表示箭头大小的可选浮点数。杆厚度是此大小的若干分之一，此项值与 arrowhead_size 相乘。

stem_width_ratio

杆宽度相对于箭头最宽部分的值。值 0.1 表示杆宽度是箭头宽度的十分之一。
